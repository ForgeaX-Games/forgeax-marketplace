## 图例说明

| 标识 | 含义 | 说明 |
|:---:|------|------|
| ⭐⭐⭐ | **必须** | 该类型游戏的核心系统，没有则无法成立 |

## 1. RPG 角色扮演

> 经典回合制/半即时RPG，如勇者斗恶龙、最终幻想

### 必须系统 ⭐⭐⭐

| 系统 | 理由 |
|------|------|
| 实体系统 | 基础架构 |
| 输入系统 | 玩家操作 |
| 场景系统 | 地图切换 |
| 事件系统 | 系统通信 |
| 存档系统 | 进度保存（核心需求） |
| 属性系统 | 角色数值基础 |
| 战斗系统 | 核心玩法 |
| 背包系统 | 道具管理 |
| 装备系统 | 角色强化 |
| 等级系统 | 成长体验 |
| 任务系统 | 游戏引导和目标 |
| 对话系统 | 剧情和NPC交互 |

### 推荐系统 ⭐⭐

| 系统 | 理由 |
|------|------|
| 技能系统 | 丰富战斗 |
| Buff系统 | 状态效果 |
| AI系统 | 敌人行为 |
| 经济系统 | 金币流通 |
| 商店系统 | 购买装备道具 |
| 技能树系统 | 角色培养 |
| 剧情系统 | 章节管理 |
| 地图系统 | 区域导航 |
| UI交互系统 | 复杂菜单交互 |

### 可选系统 ⭐

| 系统 | 理由 |
|------|------|
| 制作系统 | 额外玩法 |
| 掉落系统 | 战斗奖励 |
| 成就系统 | 额外目标 |
| 收集系统 | 图鉴收集 |
| 教程系统 | 新手引导 |

### 不需要 ➖

塔防系统、波次系统、载具系统、匹配系统、Roguelike系统
