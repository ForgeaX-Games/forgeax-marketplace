# 一、系统概述

## 什么是声望系统

声望系统管理玩家与游戏中各个阵营/势力的关系，影响NPC态度、商店解锁、任务接取等。

## 核心职责

| 职责 | 说明 |
|------|------|
| 声望管理 | 管理各阵营声望 |
| 等级计算 | 计算声望等级 |
| 效果应用 | 应用声望效果 |
| 奖励解锁 | 解锁声望奖励 |

## 适用游戏类型

| 游戏类型 | 声望重要性 | 特点 |
|---------|----------|------|
| **MMORPG** | 高 | 多阵营系统 |
| **开放世界** | 高 | 势力影响 |
| **RPG** | 中 | NPC好感 |
| **SLG** | 中 | 外交系统 |

# 三、声望设计

## 1. 阵营配置

```
FactionConfig {
  id: "alliance"
  name: "联盟"
  icon: "faction_alliance.png"
  
  // 声望等级
  levels: [
    { name: "仇恨", min: -36000, max: -6000 },
    { name: "敌对", min: -6000, max: -3000 },
    { name: "中立", min: -3000, max: 3000 },
    { name: "友善", min: 3000, max: 9000 },
    { name: "尊敬", min: 9000, max: 21000 },
    { name: "崇拜", min: 21000, max: 42000 }
  ]
  
  // 敌对阵营
  enemies: ["horde"]
  
  // 等级奖励
  rewards: {
    friendly: [{ type: "discount", value: 10 }],
    honored: [{ type: "recipe", id: "alliance_recipe" }],
    exalted: [{ type: "mount", id: "alliance_horse" }]
  }
}
```

# 六、功能速查表

## 声望系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| getReputation | 获取声望值 | 阵营ID |
| addReputation | 增加声望 | 阵营ID, 数值 |
| getLevel | 获取声望等级 | 阵营ID |
| isHostile | 是否敌对 | 阵营ID |
| getDiscount | 获取折扣 | 阵营ID |
| getUnlocks | 获取解锁内容 | 阵营ID |

## 声望事件

| 事件 | 数据 | 说明 |
|------|------|------|
| reputation:changed | {faction, old, new} | 声望变化 |
| reputation:level_up | {faction, level} | 声望升级 |
| reputation:unlock | {faction, content} | 内容解锁 |
