# 一、系统概述

## 什么是Buff/Debuff系统

Buff/Debuff系统管理游戏中的临时状态效果，包括增益效果(Buff)和减益效果(Debuff)。这些效果可以修改属性、造成持续伤害/治疗、施加控制等。

## 核心职责

| 职责 | 说明 |
|------|------|
| 效果应用 | 将Buff/Debuff应用到目标 |
| 效果管理 | 管理持续时间和层数 |
| 效果更新 | 每帧/每跳更新效果 |
| 效果移除 | 过期或被驱散时移除 |
| UI同步 | 同步状态图标显示 |

## 适用游戏类型

| 游戏类型 | Buff复杂度 | 特点 |
|---------|----------|------|
| **MOBA** | 高 | 大量控制效果 |
| **MMORPG** | 高 | 复杂叠加规则 |
| **RPG** | 中 | 战斗增益/减益 |
| **卡牌** | 中 | 回合制状态 |
| **Roguelike** | 中高 | 大量临时效果 |
| **塔防** | 低中 | 减速、易伤 |

# 二、核心功能模块

## 1. Buff应用

**功能说明**：将状态效果应用到目标单位。

| 功能 | 说明 |
|------|------|
| 应用Buff | 给目标添加Buff |
| 来源追踪 | 记录Buff的施加者 |
| 初始化 | 设置初始层数和持续时间 |
| 生效通知 | 触发Buff生效事件 |

**示例**：

> **中毒效果**：敌人攻击玩家，附加"中毒"Debuff，持续10秒，每秒造成50点伤害。
>
> **攻击加成**：战士使用战吼，给自己附加"狂暴"Buff，持续15秒，攻击力+30%。

---

## 2. 持续时间管理

**功能说明**：管理Buff的持续时间和过期。

| 功能 | 说明 |
|------|------|
| 倒计时 | 每帧更新剩余时间 |
| 过期检测 | 时间到后自动移除 |
| 永久Buff | 不会自动过期 |
| 暂停/恢复 | 某些情况下暂停倒计时 |

**示例**：

> **定时Buff**：力量药水效果持续60秒，UI显示倒计时。
>
> **永久被动**：装备提供的属性加成是永久Buff，卸下装备时移除。
>
> **死亡暂停**：玩家死亡时所有Buff暂停计时，复活后继续。

---

## 3. 周期性效果

**功能说明**：按固定间隔触发效果的Buff。

| 功能 | 说明 |
|------|------|
| 跳数 | 效果触发的次数 |
| 间隔 | 每跳之间的时间 |
| DOT | 持续伤害(Damage Over Time) |
| HOT | 持续治疗(Heal Over Time) |

**示例**：

> **中毒DOT**：持续10秒，每2秒造成100伤害，共5跳，总伤害500。
>
> **回春HOT**：持续12秒，每3秒恢复200生命，共4跳，总恢复800。
>
> **加速跳跃**：某些技能可以加速DOT/HOT，如双倍速度跳完。

---

## 4. Buff移除

**功能说明**：移除目标身上的Buff/Debuff。

| 移除方式 | 说明 |
|---------|------|
| 自然过期 | 持续时间结束 |
| 手动移除 | 技能或物品移除 |
| 驱散 | 驱散所有负面效果 |
| 偷取 | 偷取敌人的Buff |
| 死亡移除 | 死亡时移除所有效果 |
| 替换移除 | 被新Buff替换 |

**示例**：

> **净化技能**：移除目标身上所有负面效果。
>
> **Buff偷取**：偷取敌人2个增益效果到自己身上。
>
> **死亡清除**：玩家死亡时所有Buff/Debuff被移除（特殊标记的除外）。

---

## 5. Buff图标与UI

**功能说明**：在界面上显示当前状态效果。

| 功能 | 说明 |
|------|------|
| 图标显示 | 显示Buff/Debuff图标 |
| 剩余时间 | 显示倒计时 |
| 层数显示 | 显示叠加层数 |
| 分类显示 | Buff和Debuff分开显示 |
| 悬停提示 | 鼠标悬停显示详情 |

**示例**：

> **状态栏**：角色头像下方显示所有Buff图标，增益为绿色边框，减益为红色边框。
>
> **倒计时**：图标上用扇形或数字显示剩余时间。
>
> **层数角标**：图标右下角显示层数数字。

# 三、Buff类型设计

## 1. 效果分类

### 属性修改
| 效果 | 说明 | 示例 |
|------|------|------|
| 属性加成 | 增加某项属性 | 攻击力+50 |
| 属性减少 | 降低某项属性 | 防御力-30% |
| 属性覆盖 | 设置属性为特定值 | 攻击速度=0 |

### 持续效果
| 效果 | 说明 | 示例 |
|------|------|------|
| DOT | 持续造成伤害 | 中毒、燃烧 |
| HOT | 持续恢复生命 | 回春、再生 |
| 持续回蓝 | 持续恢复魔法 | 魔力潮汐 |

### 触发效果
| 效果 | 说明 | 示例 |
|------|------|------|
| 攻击触发 | 攻击时触发 | 攻击回血 |
| 受击触发 | 受击时触发 | 受击反弹 |
| 击杀触发 | 击杀时触发 | 击杀刷新 |
| 条件触发 | 满足条件触发 | 低血量暴击 |

### 控制效果
| 效果 | 说明 | 示例 |
|------|------|------|
| 眩晕 | 无法行动 | 锤击眩晕 |
| 沉默 | 无法施法 | 沉默术 |
| 定身 | 无法移动 | 冰冻 |
| 缴械 | 无法普攻 | 缴械 |

### 特殊效果
| 效果 | 说明 | 示例 |
|------|------|------|
| 无敌 | 不受伤害 | 金钟罩 |
| 隐身 | 不可被选中 | 潜行 |
| 免疫 | 免疫特定效果 | 魔法免疫 |
| 复活 | 死亡后复活 | 守护天使 |

## 2. Buff配置结构

```
BuffConfig {
  id: "burn"
  name: "燃烧"
  icon: "buff_burn.png"
  description: "每秒受到{damage}点火焰伤害"
  
  // 基础属性
  type: "debuff"           // buff/debuff
  category: "dot"          // 分类
  element: "fire"          // 元素
  
  // 持续时间
  duration: 5              // 秒，0为永久
  
  // 叠加规则
  stackable: true
  maxStacks: 5
  stackBehavior: "duration_refresh"  // 叠加时行为
  
  // 效果
  effects: [
    { type: "dot", damage: 50, interval: 1 }
  ]
  
  // 特殊标记
  dispellable: true        // 可驱散
  deathPersist: false      // 死亡保留
  hidden: false            // UI隐藏
}
```

# 四、叠加与刷新规则

## 1. 叠加类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 不可叠加 | 只能存在一个 | 护盾 |
| 层数叠加 | 增加层数，效果增强 | 中毒层数 |
| 独立叠加 | 每个实例独立 | 多人DOT |
| 来源叠加 | 不同来源可叠加 | 不同玩家的Buff |

## 2. 刷新行为

| 行为 | 说明 |
|------|------|
| 刷新时间 | 重置持续时间为最大 |
| 延长时间 | 增加持续时间 |
| 增加层数 | 层数+1，时间可能刷新 |
| 替换 | 新效果替换旧效果 |
| 取较强 | 保留效果更强的 |
| 无效果 | 已有时不能再次施加 |

**示例**：

> **刷新时间**：中毒效果剩余3秒，再次中毒后刷新为10秒满时间。
>
> **层数叠加**：每次攻击叠加1层印记，最多5层，每层增加10%伤害。5层后再攻击只刷新时间不增加层数。
>
> **独立叠加**：A和B都对目标施加DOT，两个DOT独立存在和计时。

---

## 3. 互斥与覆盖

| 规则 | 说明 | 示例 |
|------|------|------|
| 同类互斥 | 同类效果只能存在一个 | 加速和减速 |
| 元素互斥 | 特定元素互相抵消 | 冰火互斥 |
| 等级覆盖 | 高等级覆盖低等级 | 大回春覆盖小回春 |
| 来源互斥 | 同一来源只能存在一个 | 同一技能的Buff |

**示例**：

> **冰火互斥**：目标身上有燃烧效果，施加冰冻时移除燃烧，施加冰冻。
>
> **效果覆盖**：目标有+10%攻击Buff，施加+20%攻击Buff时，覆盖为+20%。

# 五、控制效果系统

## 1. 控制类型

| 类型 | 效果 | 可做 | 不可做 |
|------|------|------|--------|
| 眩晕 | 完全失去控制 | 无 | 移动、攻击、施法 |
| 沉默 | 无法使用技能 | 移动、攻击 | 施法 |
| 定身 | 无法移动 | 攻击、施法 | 移动 |
| 缴械 | 无法普攻 | 移动、施法 | 普攻 |
| 恐惧 | 随机移动 | 无（自动移动） | 主动操作 |
| 嘲讽 | 强制攻击施法者 | 无（自动攻击） | 选择目标 |
| 击飞 | 被击飞到空中 | 无 | 任何操作 |
| 击退 | 被推开一段距离 | 短暂无法操作 | 被推开期间 |

---

## 2. 控制抵抗

| 机制 | 说明 |
|------|------|
| 韧性 | 减少控制时间 |
| 控制免疫 | 完全免疫控制 |
| 控制递减 | 短时间内控制效果递减 |
| 霸体 | 不会被打断/击飞 |

**示例**：

> **韧性减免**：眩晕2秒，目标有30%韧性，实际眩晕1.4秒。
>
> **控制递减**：第一次眩晕2秒，10秒内再次眩晕只有1秒，第三次只有0.5秒。
>
> **Boss免疫**：Boss免疫所有控制效果，但可以被减速。

---

## 3. 解控技能

| 类型 | 说明 |
|------|------|
| 净化 | 移除当前所有控制 |
| 免疫 | 短时间免疫控制 |
| 解除并免疫 | 移除控制并短暂免疫 |
| 特定解控 | 只能解除特定类型 |

**示例**：

> **净化技能**：解除自身所有控制效果，3秒内免疫所有控制。
>
> **水银饰带**：受到控制时自动触发，解除控制效果（90秒冷却）。

# 六、各类型游戏应用

## 1. MOBA

### Buff特点
- 大量控制效果
- 复杂的叠加规则
- 装备被动Buff
- 召唤师技能

### 典型Buff
```
燃烧（引燃）
- 持续5秒
- 总伤害：70-410（基于等级）
- 真实伤害
- 降低治疗效果40%

护盾（屏障）
- 持续2秒
- 吸收伤害：115-455（基于等级）
- 冷却180秒
```

---

## 2. MMORPG

### Buff特点
- 职业专属Buff
- 团队光环
- 长时间增益
- 副本机制Debuff

### 典型系统
```
战士：
- 狂暴：攻击+30%，持续15秒
- 破甲：降低目标护甲20%，持续10秒

牧师：
- 真言术盾：吸收5000伤害，持续15秒
- 恢复：每3秒恢复500生命，持续15秒

Boss机制：
- 必杀标记：3秒后受到致命伤害
- 减益累积：每层降低治疗效果10%
```

---

## 3. 卡牌游戏

### Buff特点
- 回合制持续
- 层数机制
- 简单直观

### 典型效果
```
中毒：回合结束受到X点伤害，X=层数
虚弱：造成伤害-25%
易伤：受到伤害+50%
力量：攻击+X，X=层数
敏捷：防御+X，X=层数
```

---

## 4. Roguelike

### Buff特点
- 大量临时增益
- 词条即Buff
- 可能有负面效果

### 典型系统
```
祝福：
- 狂战士之血：攻击+50%，防御-25%
- 吸血鬼契约：攻击吸血30%，无法自然回血

诅咒：
- 虚弱诅咒：攻击-30%
- 迟缓诅咒：移速-50%
```
