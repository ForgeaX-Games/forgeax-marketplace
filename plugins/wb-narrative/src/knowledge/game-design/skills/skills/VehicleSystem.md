# 一、系统概述

## 什么是载具系统

载具系统管理游戏中的可驾驶交通工具，包括车辆、飞行器、船只、坐骑等。

## 核心职责

| 职责 | 说明 |
|------|------|
| 载具管理 | 管理载具获取和存储 |
| 上下载具 | 进入和离开载具 |
| 驾驶控制 | 载具操控逻辑 |
| 载具战斗 | 载具武器系统 |
| 载具养护 | 燃料、损坏等 |

## 适用游戏类型

| 游戏类型 | 载具重要性 | 特点 |
|---------|----------|------|
| **竞速** | 核心 | 驾驶为主 |
| **开放世界** | 高 | 多种载具 |
| **FPS** | 中 | 战斗载具 |
| **飞行** | 核心 | 飞行模拟 |
| **RPG** | 低 | 坐骑系统 |

## 2. 载具属性

```
VehicleConfig {
  id: "sports_car"
  name: "跑车"
  type: "ground"
  
  // 性能属性
  maxSpeed: 200          // 最高时速
  acceleration: 10       // 加速度
  handling: 8            // 操控性
  braking: 7             // 制动力
  
  // 基础属性
  health: 1000           // 耐久度
  armor: 50              // 护甲
  seats: 2               // 座位数
  
  // 消耗
  fuelCapacity: 100      // 油箱容量
  fuelConsumption: 1     // 每公里油耗
}
```

# 七、功能速查表

## 载具系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| enterVehicle | 进入载具 | 载具, 座位? |
| exitVehicle | 离开载具 | - |
| spawnVehicle | 召唤载具 | 载具ID |

| accelerate | 加速 | 力度 |
| brake | 刹车 | 力度 |
| steer | 转向 | 角度 |
| refuel | 加油 | 数量 |
| repair | 修理 | - |
