# 一、系统概述

## 什么是背包系统

背包系统管理玩家拥有的物品，包括物品的存储、使用、整理和转移等功能。

## 核心职责

| 职责 | 说明 |
|------|------|
| 物品存储 | 保存玩家物品 |
| 物品操作 | 使用、丢弃、移动 |
| 容量管理 | 管理背包容量 |
| 物品查询 | 按条件查找物品 |
| UI交互 | 与界面系统交互 |

## 适用游戏类型

| 游戏类型 | 背包复杂度 | 特点 |
|---------|----------|------|
| **RPG** | 高 | 多分类、装备系统 |
| **生存** | 高 | 负重、制作材料 |
| **ARPG** | 中高 | 快速拾取、自动整理 |
| **冒险** | 中 | 任务物品、收集 |
| **休闲** | 低 | 简单道具 |

# 二、核心功能模块

## 1. 物品获取

**功能说明**：将物品添加到背包。

| 功能 | 说明 |
|------|------|
| 拾取物品 | 从地面拾取 |
| 任务奖励 | 任务获得物品 |
| 购买物品 | 商店购买 |
| 制作获得 | 制作系统产出 |
| 邮件领取 | 邮件系统领取 |

**示例**：

> **拾取物品**：玩家靠近掉落物，按F键拾取，物品进入背包。若背包已满，提示"背包已满"。
>
> **堆叠物品**：拾取99个金币，背包中已有50个金币，合并为149个（假设堆叠上限999）。

---

## 2. 物品使用

**功能说明**：使用背包中的物品。

| 功能 | 说明 |
|------|------|
| 直接使用 | 双击或右键使用 |
| 快捷使用 | 快捷栏快速使用 |
| 使用条件 | 检查使用条件 |
| 使用效果 | 执行物品效果 |
| 消耗处理 | 使用后减少数量 |

**示例**：

> **使用药水**：右键点击生命药水，恢复500HP，药水数量-1。
>
> **使用限制**：传送卷轴只能在安全区使用，战斗中使用提示"无法在战斗中使用"。

---

## 3. 物品丢弃

**功能说明**：从背包中移除物品。

| 功能 | 说明 |
|------|------|
| 单个丢弃 | 丢弃一个物品 |
| 批量丢弃 | 丢弃指定数量 |
| 全部丢弃 | 丢弃一格所有 |
| 丢弃确认 | 稀有物品二次确认 |
| 丢弃保护 | 某些物品不可丢弃 |

**示例**：

> **丢弃物品**：将物品拖到背包外区域，弹出确认框"确定丢弃？"，确认后物品掉落在地面。
>
> **任务物品**：任务物品不可丢弃，尝试丢弃时提示"任务物品无法丢弃"。

---

## 4. 物品整理

**功能说明**：自动整理背包物品。

| 功能 | 说明 |
|------|------|
| 自动排序 | 按规则排序物品 |
| 自动堆叠 | 合并相同物品 |
| 按类型分类 | 装备、消耗品分开 |
| 按稀有度排序 | 稀有物品靠前 |
| 自定义排序 | 玩家自定义规则 |

**示例**：

> **一键整理**：点击整理按钮，物品按类型→稀有度→名称排序，碎片合并，空位移到末尾。

---

## 5. 物品移动

**功能说明**：在背包内或背包间移动物品。

| 功能 | 说明 |
|------|------|
| 拖拽移动 | 拖动物品到目标位置 |
| 交换位置 | 与目标位置物品交换 |
| 分堆 | 将堆叠物品分成两份 |
| 转移 | 移动到其他容器 |

**示例**：

> **交换位置**：将A格物品拖到B格，A和B物品交换位置。
>
> **分堆操作**：Shift+拖动99个药水到空格，弹出输入框选择分出多少个。

# 三、物品系统设计

## 1. 物品分类

| 分类 | 说明 | 示例 |
|------|------|------|
| 装备 | 可穿戴物品 | 武器、防具、饰品 |
| 消耗品 | 使用后消失 | 药水、卷轴、食物 |
| 材料 | 制作原料 | 矿石、布料、木材 |
| 任务物品 | 任务相关 | 钥匙、信件 |
| 货币 | 交易媒介 | 金币、点券 |
| 其他 | 杂项 | 收藏品、装饰 |

---

## 2. 物品属性

### 基础属性
```
ItemConfig {
  id: "potion_hp_large"
  name: "大生命药水"
  icon: "item_potion_hp_large.png"
  description: "恢复500点生命值"
  
  type: "consumable"
  subType: "potion"
  rarity: "uncommon"    // common/uncommon/rare/epic/legendary
  
  maxStack: 99          // 最大堆叠
  sellPrice: 50         // 出售价格
  buyPrice: 100         // 购买价格
  
  usable: true          // 是否可使用
  tradable: true        // 是否可交易
  droppable: true       // 是否可丢弃
  destroyable: true     // 是否可销毁
}
```

### 使用效果
```
ItemEffects {
  onUse: [
    { type: "heal", value: 500 },
    { type: "buff", buffId: "regen", duration: 10 }
  ]
  
  useCooldown: 30           // 使用冷却
  useConditions: [
    { type: "not_in_combat" },
    { type: "health_not_full" }
  ]
}
```

# 四、背包管理

## 1. 容量系统

| 功能 | 说明 |
|------|------|
| 初始容量 | 默认背包格子数 |
| 容量扩展 | 购买/解锁更多格子 |
| 分页/标签 | 多页或分类标签 |
| 临时容量 | 活动期间额外容量 |

**示例**：

> **背包扩展**：初始50格，使用背包扩展卡+10格，最大200格。
>
> **分页背包**：装备页、消耗品页、材料页、任务页分开显示。

---

## 2. 负重系统

**功能说明**：物品有重量，总重量不超过负重上限。

| 功能 | 说明 |
|------|------|
| 物品重量 | 每个物品有重量值 |
| 负重上限 | 角色可承受的总重量 |
| 超重惩罚 | 超重时移速降低等 |
| 负重加成 | 属性/装备增加负重 |

**示例**：

> **负重计算**：负重上限100，当前携带物品总重80，剩余可负重20。
>
> **超重惩罚**：超重后移动速度-50%，无法奔跑和跳跃。

---

## 3. 快捷栏

| 功能 | 说明 |
|------|------|
| 绑定物品 | 将物品放入快捷栏 |
| 快速使用 | 按数字键使用 |
| 自动补充 | 使用后自动从背包补充 |
| 冷却显示 | 显示物品冷却 |

**示例**：

> **药水快捷栏**：生命药水放在1号位，按1键使用，使用后自动从背包补充到快捷栏。

---

## 4. 物品锁定

| 功能 | 说明 |
|------|------|
| 锁定物品 | 防止误操作 |
| 锁定后限制 | 无法丢弃、出售、分解 |
| 批量锁定 | 按稀有度批量锁定 |

**示例**：

> **锁定装备**：右键装备选择"锁定"，出现锁图标，该装备无法出售或分解。

# 六、各类型游戏应用

## 1. RPG

### 背包特点
- 大容量
- 装备为主
- 详细物品信息

### 功能配置
```
容量：100格，可扩展
分类：装备/消耗品/材料/任务/其他
排序：稀有度 > 等级 > 名称
特殊：装备对比、套装提示
```

---

## 2. 生存游戏

### 背包特点
- 负重系统
- 制作材料多
- 快捷栏重要

### 功能配置
```
容量：受负重限制
分类：工具/武器/材料/食物/建筑
特殊：物品腐烂、温度影响
```

---

## 3. ARPG

### 背包特点
- 快速拾取
- 装备爆率高
- 自动分解

### 功能配置
```
容量：有限，需要频繁清理
分类：装备为主
特殊：一键分解、装备评分
```

# 七、功能速查表

## 背包系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| addItem | 添加物品 | 物品ID, 数量 |
| removeItem | 移除物品 | 物品ID, 数量 |
| useItem | 使用物品 | 槽位索引 |
| moveItem | 移动物品 | 源槽位, 目标槽位 |
| splitStack | 分割堆叠 | 槽位, 数量 |
| getItem | 获取物品 | 槽位索引 |
| hasItem | 检查是否有物品 | 物品ID, 数量? |
| getItemCount | 获取物品数量 | 物品ID |
| sortInventory | 整理背包 | 排序规则? |
| expandCapacity | 扩展容量 | 数量 |

## 物品类型

| 类型 | 说明 |
|------|------|
| EQUIPMENT | 装备 |
| CONSUMABLE | 消耗品 |
| MATERIAL | 材料 |
| QUEST | 任务物品 |
| CURRENCY | 货币 |
| MISC | 杂项 |

| inventory:item_removed | {item, slot, count} | 物品移除 |
| inventory:item_used | {item, slot} | 物品使用 |
| inventory:item_moved | {item, fromSlot, toSlot} | 物品移动 |
| inventory:full | {} | 背包已满 |
