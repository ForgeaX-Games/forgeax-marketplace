# 一、系统概述

## 什么是粒子系统

粒子系统用于创建视觉特效，如火焰、烟雾、爆炸、魔法效果等，通过大量小粒子的组合实现复杂视觉效果。

## 核心职责

| 职责 | 说明 |
|------|------|
| 粒子发射 | 生成粒子 |
| 粒子更新 | 更新粒子状态 |
| 粒子渲染 | 绘制粒子 |
| 生命周期 | 管理粒子生命 |
| 对象池 | 复用粒子 |

## 适用游戏类型

大部分游戏类型都需要粒子系统来增强视觉表现。

| 火焰 | 上升粒子+颜色渐变 |

# 六、功能速查表

## 粒子系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| emit | 发射粒子 | 预设ID, 位置 |
| stop | 停止发射 | 发射器ID |
| burst | 爆发发射 | 预设ID, 数量, 位置 |
| setEmissionRate | 设置发射率 | 发射器ID, 速率 |
| attach | 附着到对象 | 发射器ID, 对象 |
| detach | 取消附着 | 发射器ID |

## 粒子事件

| 事件 | 数据 | 说明 |
|------|------|------|
| particle:emitter_started | {emitterId} | 发射开始 |
| particle:emitter_stopped | {emitterId} | 发射停止 |
| particle:effect_complete | {effectId} | 效果完成 |
