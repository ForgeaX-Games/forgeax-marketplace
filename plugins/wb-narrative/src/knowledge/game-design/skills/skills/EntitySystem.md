# 一、系统概述

## 什么是实体系统

实体系统是游戏的核心架构系统，管理游戏世界中所有对象（实体）的创建、销毁、组件挂载和生命周期。

## 核心职责

| 职责 | 说明 |
|------|------|
| 实体管理 | 创建、销毁、查询实体 |
| 组件管理 | 添加、移除、获取组件 |
| 生命周期 | 管理实体生命周期 |
| 对象池 | 复用实体减少GC |
| 层级管理 | 父子实体关系 |

## 适用游戏类型

所有游戏类型都需要实体系统作为基础架构。

# 六、功能速查表

## 实体系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| createEntity | 创建实体 | 预制体ID? |
| destroyEntity | 销毁实体 | 实体ID |
| getEntity | 获取实体 | 实体ID |
| findByTag | 按标签查找 | 标签 |
| findByType | 按类型查找 | 类型 |
| addComponent | 添加组件 | 实体, 组件 |
| removeComponent | 移除组件 | 实体, 组件类型 |
| getComponent | 获取组件 | 实体, 组件类型 |
| setParent | 设置父实体 | 子实体, 父实体 |
| getChildren | 获取子实体 | 实体 |

## 实体事件

| 事件 | 数据 | 说明 |
|------|------|------|
| entity:created | {entityId} | 实体创建 |
| entity:destroyed | {entityId} | 实体销毁 |
| entity:component_added | {entityId, component} | 组件添加 |
| entity:component_removed | {entityId, component} | 组件移除 |
