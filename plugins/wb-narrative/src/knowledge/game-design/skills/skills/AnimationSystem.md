# 一、系统概述

## 什么是动画系统

动画系统管理游戏中的动画播放，包括帧动画、骨骼动画、补间动画等，为游戏提供动态视觉表现。

## 核心职责

| 职责 | 说明 |
|------|------|
| 动画播放 | 播放动画 |
| 状态管理 | 动画状态机 |
| 混合控制 | 动画混合 |
| 事件触发 | 动画事件 |

## 适用游戏类型

所有游戏类型都需要动画系统。

# 四、动画控制

| 功能 | 说明 |
|------|------|

| 状态切换 | 条件切换 |

# 五、各类型游戏应用

## 1. 像素游戏

### 动画特点
- 帧动画为主
- 低帧率风格
- 简单状态机

---

## 2. 动作游戏

### 动画特点
- 骨骼动画
- 复杂状态机
- 动画取消

---

## 3. 3D游戏

### 动画特点
- 骨骼动画
- 动画混合
- 物理融合

# 六、功能速查表

## 动画系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| play | 播放动画 | 动画名, 选项? |
| stop | 停止动画 | - |
| pause | 暂停动画 | - |
| resume | 继续动画 | - |
| setSpeed | 设置速度 | 速度倍率 |
| setFrame | 设置帧 | 帧数 |
| crossFade | 交叉淡化 | 动画名, 时间 |
| setLayer | 设置层动画 | 层, 动画名 |

## 动画事件

| 事件 | 数据 | 说明 |
|------|------|------|
| animation:start | {name} | 动画开始 |
| animation:complete | {name} | 动画完成 |
| animation:loop | {name, count} | 动画循环 |
| animation:frame | {name, frame} | 帧事件 |
| animation:event | {name, event} | 自定义事件 |
