# 一、系统概述

## 什么是对话系统

对话系统管理游戏中NPC与玩家之间的对话交互，支持剧情叙述、任务接取、分支选择等功能。

## 核心职责

| 职责 | 说明 |
|------|------|
| 对话展示 | 显示对话内容 |
| 分支管理 | 管理对话选项 |
| 条件检查 | 检查对话条件 |
| 效果执行 | 执行对话触发的效果 |
| 历史记录 | 记录对话历史 |

## 适用游戏类型

| 游戏类型 | 对话复杂度 | 特点 |
|---------|----------|------|
| **AVG** | 极高 | 核心玩法 |
| **RPG** | 高 | 剧情推进 |
| **开放世界** | 高 | NPC互动 |
| **MMORPG** | 中 | 任务对话 |
| **动作** | 低 | 简单对话 |

# 二、核心功能模块

## 1. 对话触发

**功能说明**：启动与NPC的对话。

| 触发方式 | 说明 |
|---------|------|
| 点击NPC | 玩家主动触发 |
| 接近触发 | 进入范围自动触发 |
| 事件触发 | 游戏事件触发 |
| 任务触发 | 任务需要对话 |

**示例**：

> **主动对话**：点击村长，开始对话"勇士，你来了！"

| 继续提示 | 提示点击继续 |

# 三、对话设计

## 1. 对话结构

```
DialogueConfig {
  id: "elder_greeting"
  npcId: "elder"
  
  // 对话节点
  nodes: [
    {
      id: "start",
      speaker: "elder",
      text: "欢迎来到我们的村庄，年轻人。",
      next: "ask_help"
    },
    {
      id: "ask_help",
      speaker: "elder",
      text: "你愿意帮助我们吗？",
      choices: [
        { text: "当然愿意", next: "accept" },
        { text: "让我考虑一下", next: "decline" }
      ]
    },
    {
      id: "accept",
      speaker: "elder",
      text: "太好了！请去森林清除那些野狼。",
      effects: [{ type: "accept_quest", questId: "kill_wolves" }],
      next: null
    },
    {
      id: "decline",
      speaker: "elder",
      text: "好的，需要帮助时再来找我。",
      next: null
    }
  ]
}
```

---

## 2. 节点类型

| 类型 | 说明 |
|------|------|
| 普通节点 | 显示文本，自动继续 |
| 选择节点 | 显示选项供选择 |
| 条件节点 | 根据条件分支 |
| 效果节点 | 执行特定效果 |
| 结束节点 | 对话结束 |

---

## 3. 动态文本

**功能说明**：在对话中插入动态内容。

| 变量 | 说明 | 示例 |
|------|------|------|
| 玩家名 | 插入玩家名称 | "你好，{player_name}" |
| 数值 | 插入数值 | "你有{gold}金币" |
| 物品名 | 插入物品名 | "我需要{item_name}" |
| 条件文本 | 根据条件显示 | {if_vip: "尊贵的会员"} |

# 四、分支与选择

## 1. 选项设计

| 选项类型 | 说明 |
|---------|------|
| 普通选项 | 始终可选 |
| 条件选项 | 满足条件才显示 |
| 灰色选项 | 显示但不可选 |
| 隐藏选项 | 不满足条件不显示 |

**示例**：

> **条件选项**：
> - "我可以帮忙"（始终可选）
> - "用钱解决"（需要1000金币）[灰色：金币不足]
> - "我是VIP"（仅VIP可见）

---

## 2. 选择后果

| 后果类型 | 说明 |
|---------|------|
| 对话分支 | 进入不同对话 |
| 声望变化 | 影响NPC/阵营好感 |
| 物品获取 | 获得或失去物品 |
| 任务影响 | 接取或改变任务 |
| 剧情影响 | 影响后续剧情 |

---

## 3. 条件检查

| 条件 | 说明 |
|------|------|
| 等级条件 | 玩家等级要求 |
| 物品条件 | 拥有特定物品 |
| 声望条件 | 声望达到要求 |
| 任务条件 | 完成特定任务 |
| 属性条件 | 属性达到要求 |

# 六、各类型游戏应用

## 1. AVG/视觉小说

### 对话特点
- 核心玩法
- 大量分支
- 多结局

### 配置
```
对话量：10万+字
分支点：50+
结局数：5-10
立绘：大量
```

---

## 2. RPG

### 对话特点
- 任务驱动
- NPC互动
- 剧情叙述

### 配置
```
NPC对话：每个NPC独立
任务对话：任务相关
剧情对话：主线支线
```

# 七、功能速查表

## 对话系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| startDialogue | 开始对话 | 对话ID |
| showNode | 显示节点 | 节点ID |
| selectChoice | 选择选项 | 选项索引 |
| skipDialogue | 跳过对话 | - |
| endDialogue | 结束对话 | - |
| getHistory | 获取历史 | - |

## 对话事件

| 事件 | 数据 | 说明 |
|------|------|------|
| dialogue:started | {dialogueId, npcId} | 对话开始 |
| dialogue:node | {nodeId, text} | 显示节点 |
| dialogue:choice | {nodeId, choiceIndex} | 选择选项 |
| dialogue:ended | {dialogueId} | 对话结束 |
