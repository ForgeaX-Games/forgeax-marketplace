# 一、系统概述

## 什么是收集系统

收集系统管理游戏中的可收集内容，如怪物图鉴、物品收集、CG回顾等，激励玩家探索和收集。

## 核心职责

| 职责 | 说明 |
|------|------|
| 收集管理 | 管理收集进度 |
| 图鉴展示 | 展示收集内容 |
| 解锁记录 | 记录解锁状态 |
| 奖励发放 | 发放收集奖励 |

## 适用游戏类型

| 游戏类型 | 收集重要性 | 特点 |
|---------|----------|------|
| **宝可梦like** | 核心 | 收集即玩法 |
| **RPG** | 高 | 怪物/物品图鉴 |
| **卡牌** | 高 | 卡牌收集 |
| **开放世界** | 中 | 探索收集 |
| **AVG** | 中 | CG回顾 |

# 三、收集设计

## 1. 收集类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 怪物图鉴 | 遭遇的怪物 | 宝可梦图鉴 |
| 物品图鉴 | 获得的物品 | 装备收集 |
| 角色收集 | 解锁的角色 | 抽卡角色 |
| CG收集 | 解锁的CG | AVG回顾 |
| 音乐收集 | 解锁的音乐 | 音乐鉴赏 |
| 成就收集 | 达成的成就 | 成就系统 |

---

## 2. 图鉴信息

```
CollectionEntry {
  id: "monster_001"
  name: "火焰龙"
  category: "dragon"
  
  // 展示
  icon: "monster_001_icon.png"
  model: "monster_001.fbx"
  description: "生活在火山的龙类..."
  
  // 解锁
  unlocked: false
  unlockCondition: { type: "encounter" }
  
  // 详细信息
  info: {
    habitat: "火山",
    rarity: "稀有",
    dropItems: ["龙鳞", "龙牙"]
  }
}
```

# 四、奖励机制

## 1. 收集奖励

| 奖励类型 | 说明 |
|---------|------|
| 单个奖励 | 解锁单个获得 |
| 里程碑奖励 | 收集到一定数量 |
| 全收集奖励 | 收集全部获得 |
| 分类奖励 | 完成某分类 |

---

## 2. 奖励设计

```
收集10个怪物 → 称号"初级冒险家"
收集50个怪物 → 稀有道具
收集100个怪物 → 传说坐骑
收集全部 → 终极称号 + 特效
```

# 六、功能速查表

## 收集系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| unlock | 解锁收集项 | 收集ID |
| isUnlocked | 检查是否解锁 | 收集ID |
| getProgress | 获取进度 | 分类? |
| getCollection | 获取收集列表 | 分类? |
| getInfo | 获取详细信息 | 收集ID |

## 收集事件

| 事件 | 数据 | 说明 |
|------|------|------|
| collection:unlocked | {id, category} | 解锁收集 |
| collection:milestone | {count, reward} | 里程碑达成 |
| collection:complete | {category} | 分类完成 |
