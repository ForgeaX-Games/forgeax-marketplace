# 一、系统概述

## 什么是装备系统

装备系统管理角色可穿戴的装备，包括装备的穿戴、属性计算、强化升级等功能。

## 核心职责

| 职责 | 说明 |
|------|------|
| 装备管理 | 穿戴、卸下装备 |
| 属性计算 | 计算装备提供的属性 |
| 强化升级 | 装备强化、精炼等 |
| 词条系统 | 装备随机词条 |
| 套装效果 | 套装激活与效果 |

## 适用游戏类型

| 游戏类型 | 装备复杂度 | 特点 |
|---------|----------|------|
| **Diablo-like** | 极高 | 随机词条、暗金装备 |
| **MMORPG** | 高 | 强化、附魔、洗练 |
| **RPG** | 中 | 固定属性装备 |
| **ARPG** | 中高 | 刷装备玩法 |
| **Roguelike** | 中 | 局内装备 |

# 二、核心功能模块

## 1. 装备槽位

**功能说明**：定义角色可穿戴装备的位置。

| 槽位 | 说明 | 装备类型 |
|------|------|---------|
| 武器 | 主手武器 | 剑、斧、杖 |
| 副手 | 盾牌或副武器 | 盾、书、双持武器 |
| 头部 | 头盔 | 头盔、帽子 |
| 胸部 | 胸甲 | 重甲、轻甲、布甲 |
| 手部 | 手套 | 护手 |
| 腿部 | 腿甲 | 腿甲、裤子 |
| 脚部 | 鞋子 | 靴子 |
| 项链 | 颈部饰品 | 项链 |
| 戒指1/2 | 手指饰品 | 戒指 |
| 腰带 | 腰部 | 腰带 |
| 披风 | 背部 | 披风 |

**示例**：
> 角色有10个装备槽位，每个槽位只能装备对应类型的物品。

---

## 2. 装备穿戴

**功能说明**：将装备穿戴到对应槽位。

| 功能 | 说明 |
|------|------|
| 穿戴检查 | 检查等级、职业要求 |
| 穿戴操作 | 装备移动到槽位 |
| 属性生效 | 装备属性加到角色 |
| 外观更新 | 更新角色外观 |

**示例**：

> **穿戴流程**：双击装备，检查是否满足等级30要求，满足则装备到武器槽，角色攻击力+100。
>
> **替换装备**：已装备剑，双击新剑，新剑装备，旧剑回到背包。

---

## 3. 装备卸下

**功能说明**：将装备从槽位移除。

| 功能 | 说明 |
|------|------|
| 卸下操作 | 装备回到背包 |
| 属性失效 | 移除装备属性加成 |
| 背包检查 | 背包满时无法卸下 |
| 诅咒装备 | 某些装备无法卸下 |

---

## 4. 装备限制

| 限制类型 | 说明 |
|---------|------|
| 等级限制 | 需要达到指定等级 |
| 职业限制 | 只有特定职业可装备 |
| 属性限制 | 需要满足属性要求 |
| 声望限制 | 需要特定声望等级 |
| 唯一限制 | 只能装备一件该类型 |

**示例**：
> 装备要求：等级50，战士专属，力量100以上。不满足任一条件无法装备。

# 三、装备属性设计

## 1. 基础属性

| 属性 | 说明 | 装备类型 |
|------|------|---------|
| 攻击力 | 增加物理攻击 | 武器 |
| 防御力 | 增加物理防御 | 防具 |
| 生命值 | 增加最大HP | 所有 |
| 魔法值 | 增加最大MP | 法师装备 |

---

## 2. 词条系统

**功能说明**：装备可有随机附加属性。

### 词条类型
| 类型 | 说明 |
|------|------|
| 前缀词条 | 装备名称前的词条 |
| 后缀词条 | 装备名称后的词条 |
| 固定词条 | 该装备必有的词条 |
| 随机词条 | 随机生成的词条 |

### 词条数量
| 稀有度 | 词条数 |
|--------|--------|
| 白色 | 0 |
| 绿色 | 1 |
| 蓝色 | 2 |
| 紫色 | 3 |
| 橙色 | 4 |
| 暗金 | 5+ |

**示例**：
> **暴风之剑 of 吸血**
> - 前缀"暴风"：攻击附加雷电伤害
> - 后缀"吸血"：攻击回复生命5%

---

## 3. 词条池设计

```
词条池 {
  // 攻击类词条
  attackPercent: { min: 5, max: 20, weight: 100 },
  critRate: { min: 3, max: 10, weight: 80 },
  critDamage: { min: 10, max: 50, weight: 60 },
  
  // 防御类词条
  defensePercent: { min: 5, max: 20, weight: 100 },
  healthPercent: { min: 5, max: 15, weight: 90 },
  
  // 特殊词条
  lifesteal: { min: 3, max: 8, weight: 30 },
  cooldownReduce: { min: 5, max: 15, weight: 40 }
}
```

# 四、装备强化系统

## 1. 基础强化

**功能说明**：消耗材料提升装备等级。

| 功能 | 说明 |
|------|------|
| 强化等级 | 装备强化的等级 |
| 属性提升 | 每级提升基础属性 |
| 材料消耗 | 消耗强化石/金币 |
| 成功率 | 高等级可能失败 |

**示例**：
> +0 → +1：100%成功，消耗强化石×1
> +9 → +10：50%成功，失败不掉级，消耗强化石×10

---

## 2. 强化失败处理

| 处理方式 | 说明 |
|---------|------|
| 仅失败 | 材料消耗，等级不变 |
| 等级下降 | 降低1-2级 |
| 装备损坏 | 需要修复后继续 |
| 装备消失 | 装备永久损坏 |
| 保护系统 | 使用保护道具免失败 |

**示例**：
> +13 → +14：30%成功，失败掉2级，使用保护符可防止掉级。

---

## 3. 附魔系统

**功能说明**：为装备添加额外效果。

| 功能 | 说明 |
|------|------|
| 附魔选择 | 选择附魔类型 |
| 附魔材料 | 消耗特定材料 |
| 附魔替换 | 新附魔替换旧附魔 |
| 附魔等级 | 附魔有多个等级 |

**示例**：
> 武器附魔"烈焰"：攻击附加50点火焰伤害，5%几率点燃目标。

---

## 4. 宝石镶嵌

**功能说明**：在装备孔位镶嵌宝石。

| 功能 | 说明 |
|------|------|
| 孔位数量 | 装备的宝石孔数 |
| 宝石镶嵌 | 将宝石放入孔位 |
| 宝石取出 | 取出已镶嵌宝石 |
| 宝石合成 | 低级宝石合成高级 |

**示例**：
> 武器有3个孔位，镶嵌3颗红宝石，每颗+10攻击力，共+30攻击力。

---

## 5. 装备洗练

**功能说明**：重新随机装备词条。

| 功能 | 说明 |
|------|------|
| 全部洗练 | 重随所有词条 |
| 单条洗练 | 只重随一条词条 |
| 锁定词条 | 锁定某条不变 |
| 洗练范围 | 词条数值范围 |

**示例**：
> 装备有4条词条，锁定暴击率词条，消耗洗练石重随其他3条。

# 五、套装与特效

## 1. 套装系统

**功能说明**：穿戴同套装多件获得额外效果。

| 件数 | 效果 |
|------|------|
| 2件 | 生命值+500 |
| 4件 | 攻击力+15% |
| 6件 | 技能伤害+30% |

**示例**：
> 穿戴"烈焰套装"的头盔和胸甲（2件），激活套装效果：火焰伤害+20%。

---

## 2. 装备特效

| 特效类型 | 说明 |
|---------|------|
| 触发特效 | 攻击/受击触发 |
| 主动特效 | 主动使用技能 |
| 被动特效 | 永久生效 |
| 套装特效 | 套装激活 |

**示例**：
> **狂战士之斧**：攻击有10%几率触发"狂暴"，攻击速度+30%持续5秒。

---

## 3. 装备传承

**功能说明**：将旧装备的强化转移到新装备。

| 功能 | 说明 |
|------|------|
| 强化传承 | 转移强化等级 |
| 经验传承 | 转移强化经验 |
| 材料回收 | 回收部分材料 |

---

# 六、各类型游戏应用

## 1. Diablo-like

### 装备特点
- 大量随机词条
- 传奇/暗金装备
- 套装系统

### 装备配置
```
词条数量：0-6条
词条类型：100+种
套装：10+套
特殊：赛季专属装备
```

---

## 2. MMORPG

### 装备特点
- 强化系统复杂
- 副本专属装备
- 交易系统

### 装备配置
```
强化等级：+0 ~ +20
附魔：5种选择
宝石：3孔
特殊：装备绑定、交易限制
```

---

## 3. RPG

### 装备特点
- 固定属性为主
- 等级解锁装备
- 商店/掉落获取

### 装备配置
```
属性：固定数值
获取：商店、Boss、任务
特殊：职业专属
```
