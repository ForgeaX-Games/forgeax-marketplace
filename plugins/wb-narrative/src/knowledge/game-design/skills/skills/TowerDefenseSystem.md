# 一、系统概述

## 什么是塔防系统

塔防系统管理防御塔的放置、升级和敌人路径，是塔防游戏的核心机制。

## 核心职责

| 职责 | 说明 |
|------|------|
| 塔管理 | 放置、升级、出售 |
| 目标选择 | 塔的攻击目标 |
| 路径管理 | 敌人行进路线 |
| 资源管理 | 建塔消耗 |

## 适用游戏类型

| 游戏类型 | 塔防特点 |
|---------|---------|
| **经典塔防** | 固定路径 |
| **迷宫塔防** | 自建路径 |
| **TD-RPG** | 角色即塔 |
| **RTS混合** | 策略混合 |

---

## 2. 塔属性

```
TowerConfig {
  id: "arrow_tower"
  name: "箭塔"
  
  // 基础属性
  damage: 10
  attackSpeed: 1.0
  range: 5
  
  // 费用
  buildCost: 100
  upgradeCost: [150, 250, 400]
  sellRatio: 0.7
  
  // 攻击类型
  attackType: "single"
  damageType: "physical"
}
```

# 六、功能速查表

## 塔防系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| placeTower | 放置塔 | 塔类型, 位置 |
| upgradeTower | 升级塔 | 塔ID |
| sellTower | 出售塔 | 塔ID |
| setTargetMode | 设置目标模式 | 塔ID, 模式 |
| canPlace | 检查可否放置 | 位置 |
| getTowerAt | 获取位置的塔 | 位置 |

## 塔防事件

| 事件 | 数据 | 说明 |
|------|------|------|
| tower:placed | {towerId, position} | 塔放置 |
| tower:upgraded | {towerId, level} | 塔升级 |
| tower:sold | {towerId} | 塔出售 |
| tower:attack | {towerId, target} | 塔攻击 |
| tower:kill | {towerId, enemy} | 塔击杀 |
