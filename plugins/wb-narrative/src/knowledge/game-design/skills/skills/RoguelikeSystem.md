# 一、系统概述

## 什么是Roguelike系统

Roguelike系统提供程序化内容生成、永久死亡、局内局外成长等核心机制。

## 核心职责

| 职责 | 说明 |
|------|------|
| 地图生成 | 程序化生成地图 |
| 词条系统 | 随机增益/减益 |
| 局内成长 | 单局内升级 |
| 局外成长 | 永久解锁 |
| 随机事件 | 商店/事件等 |

## 适用游戏类型

| 游戏类型 | Rogue程度 | 特点 |
|---------|----------|------|
| **传统Roguelike** | 核心 | 永久死亡 |
| **Roguelite** | 核心 | 局外成长 |
| **卡牌Rogue** | 核心 | 构筑式 |
| **动作Rogue** | 高 | 快节奏 |
