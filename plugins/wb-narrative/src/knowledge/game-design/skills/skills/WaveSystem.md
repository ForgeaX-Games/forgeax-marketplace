# 一、系统概述

## 什么是波次系统

波次系统管理敌人的波次出现，控制敌人种类、数量和时机，常用于塔防、生存类游戏。

## 核心职责

| 职责 | 说明 |
|------|------|
| 波次配置 | 配置每波敌人 |
| 波次触发 | 控制波次开始 |
| 敌人生成 | 按配置生成敌人 |
| 进度追踪 | 追踪波次进度 |

## 适用游戏类型

| 游戏类型 | 波次特点 |
|---------|---------|
| **塔防** | 固定波次 |
| **生存** | 无尽波次 |
| **无双** | 大量敌人 |
| **吸血鬼幸存者** | 时间波次 |

# 三、波次设计

## 1. 波次配置

```
WaveConfig {
  waveNumber: 5
  
  // 敌人配置
  spawns: [
    { enemyId: "goblin", count: 10, interval: 0.5 },
    { enemyId: "orc", count: 5, interval: 1.0, delay: 5 },
    { enemyId: "boss", count: 1, delay: 15 }
  ]
  
  // 波次属性
  duration: 30          // 波次持续时间
  restTime: 10          // 休息时间
  
  // 奖励
  rewards: {
    gold: 100,
    exp: 50
  }
}
```

# 六、功能速查表

## 波次系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| startWave | 开始波次 | - |
| pauseWave | 暂停波次 | - |
| skipRest | 跳过休息 | - |
| getCurrentWave | 获取当前波 | - |
| getTotalWaves | 获取总波次 | - |
| getEnemiesRemaining | 剩余敌人数 | - |

## 波次事件

| 事件 | 数据 | 说明 |
|------|------|------|
| wave:started | {waveNumber} | 波次开始 |
| wave:completed | {waveNumber} | 波次完成 |
| wave:rest_started | {duration} | 休息开始 |
| wave:enemy_spawned | {enemyId} | 敌人生成 |
| wave:all_cleared | {} | 全部波次完成 |
