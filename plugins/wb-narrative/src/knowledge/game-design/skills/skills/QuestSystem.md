# 一、系统概述

## 什么是任务系统

任务系统管理游戏中的目标和任务，引导玩家进行游戏，推进剧情发展，提供奖励激励。

## 核心职责

| 职责 | 说明 |
|------|------|
| 任务管理 | 管理任务的接取和完成 |
| 目标追踪 | 追踪任务目标进度 |
| 奖励发放 | 完成后发放奖励 |
| 剧情推进 | 推进游戏剧情 |
| 引导系统 | 引导玩家熟悉游戏 |

## 适用游戏类型

| 游戏类型 | 任务重要性 | 特点 |
|---------|----------|------|
| **RPG** | 核心 | 主线支线系统 |
| **MMORPG** | 核心 | 日常/周常 |
| **开放世界** | 高 | 大量支线 |
| **AVG** | 高 | 剧情任务 |
| **手游** | 高 | 引导+活跃 |

# 二、核心功能模块

## 1. 任务接取

**功能说明**：玩家获取新任务。

| 接取方式 | 说明 |
|---------|------|
| NPC接取 | 与NPC对话接取 |
| 自动接取 | 满足条件自动接取 |
| 物品触发 | 使用物品接取 |
| 区域触发 | 进入区域接取 |
| 事件触发 | 特定事件后接取 |

**示例**：

> **NPC接取**：与村长对话，接取任务"清剿野狼"。
>
> **自动接取**：达到10级后自动接取"觉醒之路"主线任务。

---

## 2. 任务追踪

**功能说明**：追踪任务目标进度。

| 功能 | 说明 |
|------|------|
| 目标显示 | 显示当前目标 |
| 进度更新 | 实时更新进度 |
| 导航指引 | 指向目标位置 |
| 追踪设置 | 选择追踪的任务 |

**示例**：

> **任务追踪**：追踪"清剿野狼"任务，界面显示"击杀野狼 3/10"，地图标记野狼位置。

---

## 3. 任务完成

**功能说明**：完成任务目标后的处理。

| 功能 | 说明 |
|------|------|
| 完成检测 | 检测所有目标完成 |
| 交付任务 | 返回NPC交付 |
| 自动完成 | 目标完成自动交付 |
| 奖励发放 | 发放任务奖励 |

**示例**：

> **任务交付**：击杀10只野狼后，返回村长处交付，获得经验500、金币100。
>
> **自动完成**：击杀Boss后任务自动完成，弹窗显示奖励。

---

## 4. 任务放弃

**功能说明**：放弃当前任务。

| 功能 | 说明 |
|------|------|
| 放弃确认 | 确认放弃任务 |
| 进度重置 | 放弃后进度清空 |
| 重新接取 | 可以重新接取 |
| 不可放弃 | 某些任务不可放弃 |

# 三、任务类型设计

## 1. 按重要性分类

| 类型 | 说明 | 特点 |
|------|------|------|
| 主线任务 | 核心剧情 | 必做、高奖励 |
| 支线任务 | 额外内容 | 可选、中奖励 |
| 日常任务 | 每日刷新 | 日常活跃 |
| 周常任务 | 每周刷新 | 周活跃 |
| 隐藏任务 | 触发条件隐藏 | 探索奖励 |

---

## 2. 按机制分类

| 类型 | 说明 |
|------|------|
| 一次性任务 | 完成后不可重复 |
| 重复任务 | 可重复完成 |
| 链式任务 | 完成后触发下一个 |
| 分支任务 | 选择影响后续 |
| 限时任务 | 有时间限制 |
| 随机任务 | 随机生成 |

---

## 3. 任务配置

```
QuestConfig {
  id: "main_001"
  name: "觉醒之路"
  description: "前往长老处学习技能"
  
  // 类型
  type: "main"
  repeatable: false
  
  // 接取条件
  prerequisites: {
    level: 10,
    quests: ["tutorial_complete"]
  }
  
  // 目标
  objectives: [
    { type: "talk", npcId: "elder", count: 1 },
    { type: "learn_skill", skillId: "fireball", count: 1 }
  ]
  
  // 奖励
  rewards: {
    exp: 500,
    gold: 100,
    items: [{ id: "skill_book", count: 1 }]
  }
  
  // 后续
  nextQuest: "main_002"
  
  // 限制
  timeLimit: 0,        // 0=无限制
  abandonable: true
}
```

# 四、任务目标

## 1. 目标类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 击杀 | 击杀指定目标 | 击杀10只野狼 |
| 收集 | 收集指定物品 | 收集5个狼皮 |
| 对话 | 与NPC对话 | 与长老对话 |
| 到达 | 到达指定位置 | 前往迷失森林 |
| 护送 | 护送NPC到目的地 | 护送商人到城镇 |
| 使用 | 使用指定物品/技能 | 使用信号弹 |
| 交付 | 交付物品给NPC | 把信交给村长 |
| 探索 | 探索指定区域 | 探索3个洞穴 |
| 交互 | 与物体交互 | 调查神秘石碑 |

---

## 2. 目标配置

```
QuestObjective {
  id: "obj_1"
  type: "kill"
  target: "wolf"
  count: 10
  
  // 可选参数
  area: "forest",          // 限定区域
  weapon: "sword",         // 限定武器
  timeLimit: 300,          // 时间限制
  hidden: false            // 是否隐藏进度
}
```

# 五、高级功能

## 1. 任务分支

**功能说明**：玩家选择影响任务走向。

| 功能 | 说明 |
|------|------|
| 选择点 | 关键决策点 |
| 分支路径 | 不同选择不同结果 |
| 影响记录 | 记录玩家选择 |
| 后果展示 | 显示选择的影响 |

**示例**：

> **分支选择**："帮助村民"或"帮助盗贼"，选择不同获得不同奖励和声望。

# 六、各类型游戏应用

## 1. RPG

### 任务特点
- 主线驱动
- 剧情丰富
- 分支选择

### 配置
```
主线任务：50+
支线任务：100+
隐藏任务：20+
分支系统：多结局
```

---

## 2. MMORPG

### 任务特点
- 日常周常
- 重复任务
- 组队任务

### 配置
```
主线任务：大量
日常任务：10个/天
周常任务：10个/周
世界任务：随机刷新
```

---

## 3. 手游

### 任务特点
- 新手引导
- 活跃任务
- 简单目标

### 配置
```
新手任务：引导功能
每日任务：活跃度
成长任务：长期目标
活动任务：限时活动
```

# 七、功能速查表

## 任务系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| getQuests | 获取任务列表 | 类型? |
| acceptQuest | 接取任务 | 任务ID |
| abandonQuest | 放弃任务 | 任务ID |
| completeQuest | 完成任务 | 任务ID |
| getProgress | 获取进度 | 任务ID |
| updateProgress | 更新进度 | 任务ID, 目标ID, 进度 |
| trackQuest | 追踪任务 | 任务ID |
| getActiveQuests | 获取进行中任务 | - |
| getCompletedQuests | 获取已完成任务 | - |

## 任务类型

| 类型 | 说明 |
|------|------|
| MAIN | 主线任务 |
| SIDE | 支线任务 |
| DAILY | 日常任务 |
| WEEKLY | 周常任务 |
| EVENT | 活动任务 |
| HIDDEN | 隐藏任务 |

## 任务状态

| 状态 | 说明 |
|------|------|
| NOT_STARTED | 未开始 |
| IN_PROGRESS | 进行中 |
| COMPLETED | 已完成(未交付) |
| TURNED_IN | 已交付 |
| FAILED | 失败 |
| ABANDONED | 已放弃 |

## 任务事件

| 事件 | 数据 | 说明 |
|------|------|------|
| quest:accepted | {questId} | 任务接取 |
| quest:progress | {questId, objectiveId, current, total} | 进度更新 |
| quest:completed | {questId} | 任务完成 |
| quest:turned_in | {questId, rewards} | 任务交付 |
| quest:failed | {questId, reason} | 任务失败 |
| quest:abandoned | {questId} | 任务放弃 |
