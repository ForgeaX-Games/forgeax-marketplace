# 一、系统概述

## 什么是回合战斗系统

回合战斗系统是按回合进行的战斗模式，玩家和敌人轮流行动，每回合可选择攻击、技能、道具、防御等行动。

## 核心职责

| 职责 | 说明 |
|------|------|
| 回合管理 | 管理回合流转 |
| 行动顺序 | 确定行动顺序 |
| 行动选择 | 处理行动选择 |
| 效果执行 | 执行行动效果 |
| 胜负判定 | 判定战斗结果 |

## 适用游戏类型

| 游戏类型 | 回合特点 | 典型作品 |
|---------|---------|---------|
| **JRPG** | 经典回合制 | 最终幻想 |
| **SRPG** | 战棋回合 | 火焰纹章 |
| **卡牌** | 出牌回合 | 炉石传说 |
| **Roguelike** | 简化回合 | 杀戮尖塔 |
| **策略** | 回合策略 | 文明 |

# 三、行动系统

## 1. 行动顺序

### 速度优先制
```
行动顺序 = 按速度属性排序
速度高的先行动
```

### 回合制
```
我方全体 → 敌方全体
或
敌方全体 → 我方全体
```

### ATB系统
```
行动条充满 → 可行动
速度影响充能速度
```

---

## 2. 行动值系统

| 功能 | 说明 |
|------|------|
| 行动值 | 行动消耗的点数 |

# 四、战斗流程

## 1. 标准战斗流程

```
战斗开始
    │
    ▼
初始化战场
    │
    ▼
┌─▶ 回合开始
│       │
│       ▼
│   计算行动顺序
│       │
│       ▼
│   ┌─▶ 当前单位选择行动
│   │       │
│   │       ▼
│   │   执行行动
│   │       │
│   │       ▼
│   │   检查胜负条件 ──满足──▶ 战斗结束
│   │       │不满足
│   │       ▼
│   └───下一行动单位
│       │
│       ▼
│   回合结束
│       │
└───────┘
```

# 七、功能速查表

## 回合战斗方法

| 方法 | 说明 | 参数 |
|------|------|------|
| startBattle | 开始战斗 | 敌人配置 |
| startTurn | 开始回合 | - |
| endTurn | 结束回合 | - |
| selectAction | 选择行动 | 行动类型 |
| selectTarget | 选择目标 | 目标 |
| executeAction | 执行行动 | - |
| checkVictory | 检查胜利 | - |
| escape | 尝试逃跑 | - |

## 战斗事件

| 事件 | 数据 | 说明 |
|------|------|------|
| battle:started | {enemies} | 战斗开始 |
| battle:turn_start | {turnNumber} | 回合开始 |
| battle:turn_end | {turnNumber} | 回合结束 |
| battle:action | {actor, action, target} | 执行行动 |
| battle:damage | {target, damage} | 造成伤害 |
| battle:victory | {} | 战斗胜利 |
| battle:defeat | {} | 战斗失败 |
| battle:escape | {success} | 逃跑尝试 |
