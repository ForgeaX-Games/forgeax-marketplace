# 一、系统概述

## 什么是UI交互逻辑系统

UI交互逻辑系统管理用户界面的交互行为，包括面板管理、焦点控制、输入响应、状态管理和反馈机制，是连接玩家操作与游戏逻辑的桥梁。

## 核心职责

| 职责 | 说明 |
|------|------|
| 面板管理 | 面板打开、关闭、层级控制 |
| 焦点控制 | 焦点导航、焦点记忆 |
| 输入响应 | 点击、拖拽、悬停等事件处理 |
| 状态管理 | 组件状态、数据绑定 |
| 反馈机制 | 音效、动画、震动反馈 |

## 适用游戏类型

| 游戏类型 | UI复杂度 | 特点 |
|---------|----------|------|
| **RPG/MMORPG** | 极高 | 多层面板、复杂交互 |
| **手游** | 高 | 触摸优化、手势交互 |
| **策略** | 高 | 信息密集、嵌套面板 |
| **主机游戏** | 中 | 手柄导航、焦点系统 |
| **休闲** | 低 | 简洁直观 |

# 七、功能速查表

## UI交互系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| openPanel | 打开面板 | 面板ID, 参数? |
| closePanel | 关闭面板 | 面板ID? |
| closeAll | 关闭所有面板 | - |
| setFocus | 设置焦点 | 元素ID |
| getFocus | 获取当前焦点 | - |
| navigateTo | 导航到元素 | 方向 |
| enableInput | 启用输入 | - |
| disableInput | 禁用输入 | - |
| showToast | 显示提示 | 消息, 时长? |
| showDialog | 显示对话框 | 配置 |
| startDrag | 开始拖拽 | 数据 |
| endDrag | 结束拖拽 | - |

## UI交互事件

| 事件 | 数据 | 说明 |
|------|------|------|
| ui:panel_opened | {panelId} | 面板打开 |
| ui:panel_closed | {panelId} | 面板关闭 |
| ui:focus_changed | {from, to} | 焦点变化 |
| ui:click | {elementId, position} | 点击事件 |
| ui:double_click | {elementId} | 双击事件 |
| ui:long_press | {elementId} | 长按事件 |
| ui:drag_start | {elementId, data} | 拖拽开始 |
| ui:drag_end | {elementId, target} | 拖拽结束 |
| ui:drop | {targetId, data} | 放置事件 |
| ui:hover_enter | {elementId} | 悬停进入 |
| ui:hover_leave | {elementId} | 悬停离开 |
| ui:scroll | {elementId, position} | 滚动事件 |
| ui:input_mode_changed | {mode} | 输入模式切换 |

| 回调 | 参数 | 说明 |
|------|------|------|
| onClick | event | 点击回调 |
| onDoubleClick | event | 双击回调 |
| onLongPress | event | 长按回调 |
| onHover | isHover | 悬停回调 |
| onFocus | isFocus | 焦点回调 |
| onDragStart | data | 拖拽开始回调 |
| onDragEnd | result | 拖拽结束回调 |
