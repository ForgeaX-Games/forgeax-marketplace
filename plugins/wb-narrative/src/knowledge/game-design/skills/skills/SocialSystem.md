# 一、系统概述

## 什么是社交系统

社交系统管理玩家之间的互动功能，包括好友关系、公会组织、即时聊天、排行竞争等。

## 核心职责

| 职责 | 说明 |
|------|------|
| 关系管理 | 管理玩家间关系 |
| 组织管理 | 管理公会组织 |
| 通讯管理 | 管理聊天消息 |
| 竞争管理 | 管理排行竞争 |
| 匹配管理 | 管理玩家匹配 |

## 适用游戏类型

| 游戏类型 | 社交重要性 | 特点 |
|---------|----------|------|
| **MMORPG** | 核心 | 全功能社交 |
| **MOBA** | 高 | 组队匹配 |
| **手游** | 高 | 好友互动 |
| **休闲** | 中 | 社交竞争 |
| **单机** | 低 | 排行榜 |

## 3. 匹配流程

```
加入匹配队列
    │
    ▼
搜索匹配玩家
    │
    ▼
找到匹配 ──否──▶ 扩大范围继续搜索
    │是
    ▼
确认匹配
    │
    ▼
进入游戏
```

# 七、功能速查表

## 社交系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| addFriend | 添加好友 | 玩家ID |
| removeFriend | 删除好友 | 好友ID |
| blockPlayer | 拉黑玩家 | 玩家ID |
| createGuild | 创建公会 | 公会名 |
| joinGuild | 加入公会 | 公会ID |
| leaveGuild | 离开公会 | - |
| sendMessage | 发送消息 | 频道, 内容 |
| getRanking | 获取排行 | 排行类型 |
| startMatch | 开始匹配 | 模式 |
| cancelMatch | 取消匹配 | - |

## 社交事件

| 事件 | 数据 | 说明 |
|------|------|------|
| social:friend_added | {friendId} | 添加好友 |
| social:friend_removed | {friendId} | 删除好友 |
| social:guild_joined | {guildId} | 加入公会 |
| social:message_received | {from, content} | 收到消息 |
| social:match_found | {matchId, players} | 匹配成功 |
| social:rank_updated | {rank, type} | 排名更新 |
