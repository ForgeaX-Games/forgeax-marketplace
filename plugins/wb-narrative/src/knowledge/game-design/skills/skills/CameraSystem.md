# 一、系统概述

## 什么是相机系统

相机系统控制游戏的视角呈现，包括跟随、缩放、震动等效果，直接影响玩家的视觉体验。

## 核心职责

| 职责 | 说明 |
|------|------|
| 视角控制 | 控制相机位置和方向 |
| 目标跟随 | 跟随游戏对象 |
| 效果处理 | 震动、缩放等 |
| 边界限制 | 限制相机范围 |

## 适用游戏类型

所有游戏类型都需要相机系统。

# 六、功能速查表

## 相机系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| follow | 跟随目标 | 目标 |
| setPosition | 设置位置 | x, y, z? |
| moveTo | 移动到位置 | 位置, 时间 |
| setZoom | 设置缩放 | 缩放值 |
| zoomTo | 缩放到 | 缩放值, 时间 |
| shake | 震动 | 强度, 时间 |
| setBounds | 设置边界 | 边界 |
| setOffset | 设置偏移 | x, y |

## 相机事件

| 事件 | 数据 | 说明 |
|------|------|------|
| camera:move | {position} | 相机移动 |
| camera:zoom | {zoom} | 相机缩放 |
| camera:shake | {intensity} | 相机震动 |
| camera:target_changed | {target} | 目标改变 |
