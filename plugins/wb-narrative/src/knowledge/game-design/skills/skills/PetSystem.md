# 一、系统概述

## 什么是宠物系统

宠物系统管理可跟随玩家的宠物或随从，包括获取、养成、战斗和骑乘等功能。

## 核心职责

| 职责 | 说明 |
|------|------|
| 宠物管理 | 获取、存储宠物 |
| 宠物召唤 | 召唤/收回宠物 |
| 宠物养成 | 升级、进化 |
| 宠物战斗 | 战斗辅助 |
| 宠物交互 | 喂养、抚摸 |

## 适用游戏类型

| 游戏类型 | 宠物重要性 | 特点 |
|---------|----------|------|
| **宝可梦like** | 核心 | 收集战斗 |
| **养成** | 核心 | 养成为主 |
| **MMORPG** | 中 | 战斗辅助 |
| **RPG** | 中 | 伙伴系统 |

# 六、功能速查表

## 宠物系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| getPets | 获取宠物列表 | - |
| summon | 召唤宠物 | 宠物ID |
| dismiss | 收回宠物 | - |
| feed | 喂养宠物 | 宠物ID, 食物ID |
| evolve | 进化宠物 | 宠物ID |
| learnSkill | 学习技能 | 宠物ID, 技能ID |
| mount | 骑乘 | 宠物ID |
| dismount | 下坐骑 | - |

## 宠物事件

| 事件 | 数据 | 说明 |
|------|------|------|
| pet:obtained | {petId} | 获得宠物 |
| pet:summoned | {petId} | 召唤宠物 |
| pet:level_up | {petId, level} | 宠物升级 |
| pet:evolved | {petId, newForm} | 宠物进化 |
| pet:skill_learned | {petId, skillId} | 学习技能 |
