# 一、系统概述

## 什么是技能系统

技能系统管理游戏中角色可使用的主动或被动能力，包括技能的配置、释放条件检查、冷却管理、效果执行等完整流程。

## 核心职责

| 职责 | 说明 |
|------|------|
| 技能配置 | 定义技能属性和效果 |
| 条件检查 | 检查释放条件是否满足 |
| 资源消耗 | 扣除MP/怒气等资源 |
| 冷却管理 | 管理技能冷却时间 |
| 效果执行 | 执行技能效果 |
| 目标选择 | 确定技能影响目标 |

## 适用游戏类型

| 游戏类型 | 技能复杂度 | 特点 |
|---------|----------|------|
| **MOBA** | 高 | 4技能+被动，精确控制 |
| **MMORPG** | 高 | 大量技能，技能栏 |
| **ARPG** | 中高 | 技能连招 |
| **RPG** | 中 | 回合制技能 |
| **ACT** | 中 | 动作技能 |
| **卡牌** | 低中 | 卡牌即技能 |

# 二、核心功能模块

## 1. 技能注册与配置

**功能说明**：定义和注册游戏中的所有技能。

| 功能 | 说明 |
|------|------|
| 技能定义 | 配置技能基本属性 |
| 技能注册 | 将技能加入系统 |
| 技能查询 | 根据ID获取技能 |
| 技能分类 | 按类型/元素分类 |

**示例**：

> **火球术配置**：ID=fireball，名称=火球术，消耗MP=50，冷却=5秒，伤害=攻击力×150%，目标类型=单体敌人，射程=10米。

---

## 2. 技能学习与装备

**功能说明**：角色学习和装备技能。

| 功能 | 说明 |
|------|------|
| 学习技能 | 角色获得新技能 |
| 技能等级 | 技能可升级 |
| 技能装备 | 将技能放入技能栏 |
| 技能替换 | 更换技能栏技能 |
| 技能遗忘 | 移除已学技能 |

**示例**：

> **学习新技能**：角色达到10级，自动学习"冰冻术"。
>
> **技能升级**：使用技能点将火球术从1级升到2级，伤害从150%提升到180%。
>
> **技能装备**：将火球术拖放到技能栏第1格，绑定快捷键Q。

---

## 3. 冷却管理

**功能说明**：管理技能的冷却时间。

| 功能 | 说明 |
|------|------|
| 冷却时间 | 技能使用后的等待时间 |
| 冷却缩减 | 属性减少冷却时间 |
| 冷却重置 | 某些效果重置冷却 |
| 共享冷却 | 同类技能共享CD |
| 全局冷却 | 所有技能的短暂CD |

**计算公式**：
```
实际冷却 = 基础冷却 × (1 - 冷却缩减率)
冷却缩减上限通常为 40-45%
```

**示例**：

> **普通冷却**：火球术冷却10秒，使用后需等待10秒才能再次使用。
>
> **冷却缩减**：有30%冷却缩减，火球术实际冷却 = 10 × 0.7 = 7秒。
>
> **共享冷却**：治疗类技能共享5秒冷却，使用任一治疗技能后其他治疗技能也进入冷却。

---

## 4. 资源消耗

**功能说明**：使用技能需要消耗的资源。

| 资源类型 | 说明 | 示例 |
|---------|------|------|
| 魔法值 (MP) | 最常见资源 | 火球术消耗50MP |
| 怒气 | 战斗积累 | 旋风斩消耗30怒气 |
| 能量 | 快速恢复 | 背刺消耗40能量 |
| 生命值 | 消耗自身血量 | 血祭消耗10%当前HP |
| 道具 | 消耗特定道具 | 召唤术消耗召唤石×1 |
| 充能 | 有限次数 | 闪现3次充能 |

**示例**：

> **MP消耗**：火球术消耗50MP，当前MP=40，无法释放，提示"魔法值不足"。
>
> **充能系统**：闪现有3层充能，使用后消耗1层，每30秒恢复1层。

---

## 5. 目标选择

**功能说明**：确定技能影响哪些目标。

| 目标类型 | 说明 | 示例 |
|---------|------|------|
| 自身 | 只对自己生效 | 治疗术、护盾 |
| 单体敌人 | 选择一个敌人 | 火球术 |
| 单体友军 | 选择一个队友 | 复活术 |
| 地点 | 选择一个位置 | 陨石术 |
| 方向 | 选择一个方向 | 冲刺、激光 |
| 范围 | 指定范围内所有 | 暴风雪 |
| 锥形 | 扇形范围 | 火焰吐息 |
| 全体 | 所有敌人或友军 | 群体治疗 |

**示例**：

> **单体技能**：点击敌人释放火球术，火球飞向目标造成伤害。
>
> **范围技能**：选择地点释放暴风雪，8米范围内所有敌人受到伤害。
>
> **锥形技能**：向鼠标方向释放火焰吐息，前方45度扇形范围内敌人受到伤害。

# 三、技能配置设计

## 1. 基础配置

```
SkillConfig {
  id: "fireball"
  name: "火球术"
  icon: "skill_fireball.png"
  description: "发射一颗火球，造成{damage}点火焰伤害"
  
  // 基础属性
  type: "active"           // active/passive
  element: "fire"          // 元素类型
  category: "magic"        // 分类
  
  // 消耗和冷却
  cost: { mp: 50 }
  cooldown: 5              // 秒
  
  // 释放
  castTime: 0.5            // 前摇时间
  channelTime: 0           // 引导时间
  
  // 目标
  targetType: "enemy"
  range: 10                // 射程
  aoeRadius: 0             // 0为单体
  
  // 等级
  maxLevel: 5
  levelRequirement: 1
}
```

## 2. 技能效果配置

```
SkillEffects {
  // 伤害效果
  damage: {
    type: "magical"
    formula: "attack * 1.5 + level * 20"
    element: "fire"
  }
  
  // Buff效果
  buffs: [
    { id: "burn", duration: 5, chance: 0.3 }
  ]
  
  // 位移效果
  displacement: {
    type: "knockback"
    distance: 3
  }
  
  // 召唤效果
  summon: {
    unitId: "fire_elemental"
    duration: 30
    maxCount: 1
  }
}
```

## 3. 技能升级配置

```
SkillLevelData [
  { level: 1, damage: 150, cooldown: 10, cost: 50 },
  { level: 2, damage: 180, cooldown: 9, cost: 55 },
  { level: 3, damage: 210, cooldown: 8, cost: 60 },
  { level: 4, damage: 250, cooldown: 7, cost: 65 },
  { level: 5, damage: 300, cooldown: 6, cost: 70 }
]
```

# 四、技能释放流程

## 1. 完整释放流程

```
玩家按下技能键
      │
      ▼
  技能是否存在 ──否──▶ 返回
      │是
      ▼
  是否在冷却中 ──是──▶ 提示"技能冷却中"
      │否
      ▼
  资源是否足够 ──否──▶ 提示"资源不足"
      │是
      ▼
  是否可施法 ──否──▶ 提示"无法施法"（沉默等）
      │是
      ▼
  需要选择目标？ ──是──▶ 进入目标选择模式
      │否              │
      ▼              ▼
  开始前摇 ◀────── 选择目标确认
      │
      ▼
  前摇是否被打断 ──是──▶ 技能取消
      │否
      ▼
  扣除资源
      │
      ▼
  执行技能效果
      │
      ▼
  播放特效和音效
      │
      ▼
  进入后摇
      │
      ▼
  开始冷却
```

## 2. 前摇与后摇

| 阶段 | 说明 | 可打断 |
|------|------|--------|
| 前摇 | 施法前的准备动作 | 通常可打断 |
| 施法 | 技能效果生效 | 通常不可打断 |
| 后摇 | 施法后的恢复 | 可被取消 |

**示例**：

> **火球术**：0.5秒前摇（举杖蓄力）→ 瞬间发射 → 0.3秒后摇（收势）。
>
> **前摇打断**：法师蓄力火球时被敌人攻击，施法被打断，技能取消但冷却和消耗不触发。
>
> **后摇取消**：高手可以通过翻滚/移动取消后摇，更快进行下一个动作。

---

## 3. 引导技能

**功能说明**：需要持续施法的技能。

| 功能 | 说明 |
|------|------|
| 引导时间 | 持续施法时间 |
| 持续效果 | 引导期间持续生效 |
| 引导打断 | 被攻击或移动打断 |
| 可移动引导 | 某些技能可边移动边引导 |

**示例**：

> **激光**：引导3秒，期间持续发射激光造成伤害。玩家需要保持按住技能键。
>
> **治疗之泉**：引导5秒，范围内友军持续恢复生命。引导期间无法移动。

# 五、高级技能机制

## 1. 技能连招

**功能说明**：技能之间可以衔接形成连招。

| 功能 | 说明 |
|------|------|
| 连招规则 | 技能A后可接技能B |
| 连招窗口 | 衔接的时间窗口 |
| 连招加成 | 连招中伤害加成 |
| 连招分支 | 不同按键不同后续 |

**示例**：

> **战士连招**：普攻×3 → 上挑 → 空中斩×3 → 下砸
> 在普攻第3下命中后0.5秒内按技能键可接上挑，连招中伤害+20%。

---

## 2. 蓄力技能

**功能说明**：按住技能键蓄力，蓄力时间影响效果。

| 功能 | 说明 |
|------|------|
| 蓄力时间 | 可蓄力的最大时间 |
| 蓄力阶段 | 不同蓄力时间对应不同效果 |
| 蓄力指示 | 显示当前蓄力进度 |
| 蓄力取消 | 按其他键取消蓄力 |

**示例**：

> **蓄力射击**：
> - 不蓄力：100%伤害
> - 蓄力1秒：150%伤害
> - 蓄力2秒：200%伤害 + 穿透效果
> - 最大蓄力3秒：300%伤害 + 穿透 + 爆炸

---

## 3. 充能技能

**功能说明**：技能有多次充能，消耗充能而非传统冷却。

| 功能 | 说明 |
|------|------|
| 最大充能 | 可储存的使用次数 |
| 充能恢复 | 恢复一次充能的时间 |
| 充能条件 | 特定条件恢复充能 |

**示例**：

> **闪现**：最大3次充能，每30秒恢复1次。可以连续使用3次后等待恢复。
>
> **弹药系统**：技能有6发弹药，使用后需要换弹（2秒）才能继续使用。

---

## 4. 被动技能

**功能说明**：不需要主动释放，自动生效的技能。

| 类型 | 说明 | 示例 |
|------|------|------|
| 永久被动 | 始终生效 | 攻击力+10% |
| 触发被动 | 满足条件触发 | 暴击时回复生命 |
| 光环被动 | 影响周围单位 | 周围友军攻击+10% |
| 叠加被动 | 需要叠加层数 | 攻击叠加印记 |

**示例**：

> **杀戮本能**：击杀敌人后5秒内移动速度+30%，可无限刷新。
>
> **吸血被动**：普通攻击回复造成伤害10%的生命值。
>
> **破甲印记**：攻击敌人叠加印记，每层-5%护甲，最多5层。

---

## 5. 技能预览

**功能说明**：显示技能的作用范围预览。

| 类型 | 说明 |
|------|------|
| 射程指示 | 显示技能射程圆圈 |
| 范围指示 | 显示AOE范围 |
| 方向指示 | 显示技能方向和轨迹 |
| 落点预览 | 显示技能落点 |
| 可命中高亮 | 高亮可命中的目标 |

**示例**：

> **范围技能**：按下技能键后显示红色圆圈，移动鼠标调整位置，点击确认释放。
>
> **方向技能**：显示一条从角色延伸的箭头，指向鼠标方向，显示技能将经过的范围。

# 六、各类型游戏应用

## 1. MOBA

### 技能设计
- QWER 四个主动技能 + 被动
- 技能随等级解锁和升级
- 技能冷却和资源管理
- 智能施法/手动施法选项

### 示例技能
```
Q - 火球术
  冷却: 8/7.5/7/6.5/6 秒
  消耗: 60/65/70/75/80 魔法值
  伤害: 80/120/160/200/240 (+0.6AP)
  射程: 850
```

---

## 2. MMORPG

### 技能设计
- 大量技能可学习
- 技能栏自定义摆放
- 技能连招宏
- 技能天赋树

### 特殊机制
- 技能组合效果
- 职业专属技能
- 技能突破/觉醒

---

## 3. 动作游戏

### 技能设计
- 技能=动作招式
- 轻重攻击基础
- 派生和连招
- 无敌帧技能

### 示例
```
□ - 轻攻击 (可连续4次)
△ - 重攻击 (蓄力)
□□□△ - 上挑连招
空中△ - 下砸
○ - 闪避 (有无敌帧)
```

---

## 4. 回合制RPG

### 技能设计
- 每回合选择技能
- MP消耗控制
- 速度决定顺序
- 全队技能配合

### 示例
```
攻击 - 无消耗普通攻击
火球术 - 30MP 单体火属性伤害
暴风雪 - 50MP 全体冰属性伤害
治疗术 - 20MP 恢复单体生命
```

# 七、功能速查表

| 方法 | 说明 | 参数 |
|------|------|------|
| registerSkill | 注册技能 | 技能配置 |
| learnSkill | 学习技能 | 角色, 技能ID |
| equipSkill | 装备技能 | 角色, 技能ID, 槽位 |
| useSkill | 使用技能 | 角色, 技能ID, 目标? |
| canUseSkill | 检查能否使用 | 角色, 技能ID |
| getCooldown | 获取剩余冷却 | 角色, 技能ID |

| upgradeSkill | 升级技能 | 角色, 技能ID |

## 技能类型

| 类型 | 说明 |
|------|------|
| ACTIVE | 主动技能 |
| PASSIVE | 被动技能 |
| TOGGLE | 开关技能 |
| CHANNEL | 引导技能 |
| CHARGE | 蓄力技能 |

## 目标类型

| 类型 | 说明 |
|------|------|
| SELF | 自身 |
| SINGLE_ENEMY | 单体敌人 |
| SINGLE_ALLY | 单体友军 |

| DIRECTION | 方向 |
| AOE_ENEMY | 范围敌人 |
| AOE_ALLY | 范围友军 |
