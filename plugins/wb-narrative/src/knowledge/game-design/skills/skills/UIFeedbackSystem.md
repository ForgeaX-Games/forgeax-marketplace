# 一、系统概述

## 什么是UI与反馈系统

UI系统管理游戏界面的显示和交互，反馈系统通过视觉、音效、震动等方式给玩家提供操作反馈。

## 核心职责

| 职责 | 说明 |
|------|------|
| 界面管理 | 管理UI界面的显示隐藏 |
| 数据绑定 | 将数据显示在界面上 |
| 交互处理 | 处理玩家的UI操作 |
| 反馈表现 | 提供操作反馈 |
| 引导系统 | 新手引导UI |

## 适用游戏类型

几乎所有游戏类型都需要UI与反馈系统。

# 七、功能速查表

## UI系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| openPanel | 打开界面 | 面板ID |
| closePanel | 关闭界面 | 面板ID |
| showToast | 显示Toast | 消息 |
| showDialog | 显示对话框 | 配置 |
| showFloating | 显示飘字 | 内容, 位置 |
| updateHUD | 更新HUD | 数据 |
| playUIAnimation | 播放动画 | 元素, 动画 |
| shake | 屏幕震动 | 强度, 时长 |

## UI事件

| 事件 | 数据 | 说明 |
|------|------|------|
| ui:panel_opened | {panelId} | 界面打开 |
| ui:panel_closed | {panelId} | 界面关闭 |
| ui:button_clicked | {buttonId} | 按钮点击 |
| ui:item_dragged | {from, to} | 物品拖拽 |
