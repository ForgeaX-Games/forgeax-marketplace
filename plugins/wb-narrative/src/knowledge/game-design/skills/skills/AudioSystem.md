# 一、系统概述

## 什么是音频系统

音频系统管理游戏中的所有声音，包括背景音乐、音效、环境音、语音等。

## 核心职责

| 职责 | 说明 |
|------|------|
| 音乐播放 | 播放背景音乐 |
| 音效播放 | 播放游戏音效 |
| 音量控制 | 管理各类音量 |
| 空间音频 | 3D音效定位 |
| 资源管理 | 音频资源加载 |

## 适用游戏类型

所有游戏类型都需要音频系统。

# 七、功能速查表

## 音频系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| playBGM | 播放背景音乐 | 音乐ID |
| stopBGM | 停止背景音乐 | - |
| playSFX | 播放音效 | 音效ID |
| playVoice | 播放语音 | 语音ID |
| setVolume | 设置音量 | 类型, 音量 |
| mute | 静音 | 类型 |
| unmute | 取消静音 | 类型 |
| fadeIn | 淡入 | 时长 |
| fadeOut | 淡出 | 时长 |
| crossFade | 交叉淡化 | 新音乐, 时长 |

## 音频事件

| 事件 | 数据 | 说明 |
|------|------|------|
| audio:bgm_started | {trackId} | BGM开始 |
| audio:bgm_ended | {trackId} | BGM结束 |
| audio:sfx_played | {sfxId} | 音效播放 |
| audio:volume_changed | {type, volume} | 音量变化 |
