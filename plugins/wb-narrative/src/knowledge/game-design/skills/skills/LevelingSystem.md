# 一、系统概述

## 什么是等级系统

等级系统管理角色或账号的成长等级，通过积累经验值提升等级，解锁内容和提升能力。

## 核心职责

| 职责 | 说明 |
|------|------|
| 经验管理 | 管理经验值获取和消耗 |
| 等级计算 | 根据经验计算等级 |
| 升级处理 | 处理升级时的奖励和变化 |
| 内容解锁 | 根据等级解锁游戏内容 |

## 适用游戏类型

| 游戏类型 | 等级重要性 | 特点 |
|---------|----------|------|
| **RPG** | 核心 | 等级决定实力 |
| **MMORPG** | 核心 | 高等级上限 |
| **手游** | 高 | 多重等级系统 |
| **Roguelike** | 中 | 局内等级 |
| **休闲** | 低 | 简单进度 |

# 三、经验系统设计

## 1. 经验曲线

### 线性增长
```
升级经验 = 基础经验 + 等级 × 增量

1级→2级：100 + 1×50 = 150
2级→3级：100 + 2×50 = 200
10级→11级：100 + 10×50 = 600
```

### 指数增长
```
升级经验 = 基础经验 × 系数^等级

1级→2级：100 × 1.2^1 = 120
2级→3级：100 × 1.2^2 = 144
10级→11级：100 × 1.2^10 = 619
```

### 阶梯增长
```
等级1-10：每级需要100经验
等级11-20：每级需要500经验
等级21-30：每级需要2000经验
```

---

## 2. 经验配置

```
LevelConfig {
  maxLevel: 100
  
  // 经验曲线
  expCurve: {
    type: "polynomial"      // linear/exponential/polynomial
    base: 100
    factor: 1.5
    power: 2
  }
  
  // 或直接配置每级
  expTable: [
    0,      // 1级
    100,    // 2级需要100经验
    300,    // 3级需要300经验
    600,    // 4级需要600经验
    // ...
  ]
}
```

## 3. 经验修正

| 修正类型 | 说明 |
|---------|------|
| 等级差修正 | 打低级怪经验减少 |
| 组队加成 | 组队获得额外经验 |
| 双倍经验 | 活动或道具加成 |
| VIP加成 | VIP专属加成 |
| 休息加成 | 离线后累积加成 |

**示例**：

> **等级压制**：玩家50级打20级怪，经验减少90%。
>
> **组队加成**：5人组队，每人获得原本经验的120%。

# 五、高级机制

## 1. 多重等级

| 等级类型 | 说明 |
|---------|------|
| 角色等级 | 角色的战斗等级 |
| 账号等级 | 账号的总体等级 |
| 声望等级 | 阵营的声望等级 |
| 技能等级 | 技能熟练度 |
| 通行证等级 | 赛季通行证 |

---

## 2. 转生/觉醒

**功能说明**：满级后重置获得永久加成。

| 功能 | 说明 |
|------|------|
| 转生条件 | 达到满级 |
| 重置内容 | 等级重置为1 |
| 永久加成 | 获得永久属性加成 |
| 转生次数 | 记录转生次数 |

**示例**：

> **转生系统**：100级满级后可转生，等级重置为1级，获得+10%全属性永久加成。第二次转生获得+20%。

---

## 3. 突破系统

**功能说明**：突破等级上限。

| 功能 | 说明 |
|------|------|
| 等级上限 | 初始等级上限 |
| 突破条件 | 消耗材料突破 |
| 提升上限 | 提高等级上限 |
| 突破奖励 | 突破时获得奖励 |

**示例**：

> **突破系统**：初始上限50级，消耗突破石突破到60级，再突破到70级。

---

## 4. 等级压制

**功能说明**：高等级对低等级有优势。

| 机制 | 说明 |
|------|------|
| 伤害加成 | 等级高造成更多伤害 |
| 伤害减免 | 等级高受到更少伤害 |
| 命中加成 | 等级高命中率更高 |
| 经验惩罚 | 打低级怪经验少 |

# 六、各类型游戏应用

- 等级决定内容
- 组队练级

```

每日经验上限：无限/有限
```

### 等级特点
- 多重等级系统
- 经验有上限
- 快速前期

### 配置
```
角色等级：1-100
账号等级：1-999
通行证等级：1-100
每日经验上限：有
```

---

## 3. Roguelike

### 等级特点
- 局内等级
- 快速升级
- 选择成长

### 配置
```
局内等级：1-20
升级选择：三选一
局外成长：永久加成
```

| 方法 | 说明 | 参数 |
|------|------|------|
| getLevel | 获取当前等级 | 角色 |
| getExp | 获取当前经验 | 角色 |

| 事件 | 数据 | 说明 |
