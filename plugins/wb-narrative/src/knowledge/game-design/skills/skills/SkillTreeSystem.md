# 一、系统概述

## 什么是技能树系统

技能树系统让玩家通过分配技能点来解锁和强化技能/天赋，实现角色的个性化成长。

## 核心职责

| 职责 | 说明 |
|------|------|
| 节点管理 | 管理技能树节点 |
| 点数分配 | 分配技能点 |
| 前置检查 | 检查解锁条件 |
| 效果应用 | 应用节点效果 |
| 重置机制 | 支持洗点重置 |

## 适用游戏类型

| 游戏类型 | 天赋复杂度 | 特点 |
|---------|----------|------|
| **Diablo-like** | 极高 | 大量节点选择 |
| **RPG** | 高 | 职业技能树 |
| **MMORPG** | 高 | 多分支天赋 |
| **ARPG** | 中高 | 动作技能树 |
| **策略** | 中 | 科技树 |

# 二、核心功能模块

## 1. 技能点获取

**功能说明**：玩家获取可分配的技能点。

| 来源 | 说明 |
|------|------|
| 升级获得 | 每次升级获得 |
| 任务奖励 | 特定任务奖励 |
| 成就奖励 | 达成成就获得 |
| 道具使用 | 使用技能点道具 |

**示例**：

> **升级获得**：每升1级获得1技能点，满级100级共100点可分配。
>
> **任务奖励**：完成主线剧情，额外获得5技能点。

---

## 2. 节点解锁

**功能说明**：使用技能点解锁节点。

| 功能 | 说明 |
|------|------|
| 点击节点 | 选择要解锁的节点 |
| 条件检查 | 检查前置条件 |
| 消耗点数 | 扣除技能点 |
| 效果生效 | 应用节点效果 |

**示例**：

> **解锁节点**：点击"火焰强化"节点，消耗1技能点，火焰技能伤害+10%。
>
> **前置要求**：解锁"火焰强化II"需要先解锁"火焰强化I"。

---

## 3. 节点升级

**功能说明**：部分节点可多次升级。

| 功能 | 说明 |
|------|------|
| 最大等级 | 节点可升级的次数 |
| 递增效果 | 每级效果递增 |
| 费用变化 | 高等级可能费用更高 |

**示例**：

> **多级节点**："生命强化"最大5级，每级+100生命值，满级+500生命值。

---

## 4. 技能重置

**功能说明**：重置已分配的技能点。

| 重置方式 | 说明 |
|---------|------|
| 全部重置 | 重置所有节点 |
| 部分重置 | 只重置特定分支 |
| 单点重置 | 重置单个节点 |
| 重置费用 | 可能需要金币/道具 |

**示例**：

> **洗点系统**：消耗洗点道具，重置所有技能点，重新分配。
>
> **免费重置**：每周一次免费重置机会。

# 三、技能树设计

## 1. 树形结构

```
                    [起始节点]
                         │
            ┌────────────┼────────────┐
            ▼            ▼            ▼
        [攻击系]      [防御系]      [辅助系]
            │            │            │
       ┌────┼────┐  ┌────┼────┐  ┌────┼────┐
       ▼    ▼    ▼  ▼    ▼    ▼  ▼    ▼    ▼
      节点 节点 节点 节点 节点 节点 节点 节点 节点
```

---

## 2. 分支类型

| 分支 | 说明 | 示例 |
|------|------|------|
| 攻击分支 | 提升伤害能力 | 暴击、穿透 |
| 防御分支 | 提升生存能力 | 生命、护甲 |
| 辅助分支 | 提升效用能力 | 冷却、移速 |
| 职业分支 | 职业专属能力 | 战士/法师 |

---

## 3. 节点类型

| 类型 | 说明 |
|------|------|
| 属性节点 | 提升属性数值 |
| 技能节点 | 解锁新技能 |
| 强化节点 | 强化已有技能 |
| 被动节点 | 解锁被动效果 |
| 关键节点 | 重大能力改变 |

# 四、节点配置

## 1. 节点结构

```
SkillTreeNode {
  id: "fire_mastery_1"
  name: "火焰精通 I"
  icon: "node_fire_1.png"
  description: "火焰技能伤害+10%"
  
  // 位置
  position: { x: 100, y: 200 }
  branch: "fire"
  
  // 等级
  maxLevel: 1
  pointCost: 1
  
  // 前置
  prerequisites: [
    { nodeId: "fire_novice", level: 1 }
  ]
  
  // 效果
  effects: [
    { type: "stat_modifier", stat: "fire_damage", value: 10, isPercent: true }
  ]
}
```

---

## 2. 前置条件

| 条件类型 | 说明 |
|---------|------|
| 节点前置 | 需要解锁特定节点 |
| 等级前置 | 需要角色达到等级 |
| 投入前置 | 该分支需投入一定点数 |
| 技能前置 | 需要学会特定技能 |

**示例**：

> **节点前置**：解锁"火焰大师"需要"火焰精通III"满级。
>
> **投入前置**：解锁第3层节点需要在该分支已投入10点。

---

## 3. 节点效果

| 效果类型 | 说明 |
|---------|------|
| 属性加成 | 增加属性数值 |
| 技能解锁 | 解锁新技能 |
| 技能强化 | 改变技能效果 |
| 被动效果 | 添加被动能力 |
| 变形效果 | 改变技能形态 |

**示例**：

> **属性加成**：攻击力+50
>
> **技能变形**：火球术变为三连发

# 五、高级功能

## 1. 预设方案

**功能说明**：保存和切换技能树配置。

| 功能 | 说明 |
|------|------|
| 保存方案 | 保存当前配置 |
| 加载方案 | 加载已保存配置 |
| 快速切换 | 一键切换配置 |

---

| 功能 | 说明 |

| 费用预估 | 显示所需技能点 |
| 一键解锁 | 按路径批量解锁 |

---

## 3. 节点模拟

**功能说明**：预览技能树效果。

| 功能 | 说明 |
|------|------|
| 模拟分配 | 模拟点击分配 |
| 效果预览 | 显示总效果 |
| 确认应用 | 确认后真正应用 |
| 取消还原 | 取消返回原状态 |

# 六、各类型游戏应用

## 1. Diablo-like

### 技能树特点
- 大量节点（100+）
- 多条路径
- 关键节点

### 配置
```
节点总数：100+
分支数量：5-10
关键节点：10-20个
重置费用：逐渐增加
```

---

## 2. MMORPG

### 技能树特点
- 职业专属
- 3-4分支
- 可切换

### 配置
```
分支：战斗/防护/辅助
每分支：15-20节点
切换：付费切换
```

---

## 3. 策略/4X

### 技能树特点
- 科技树形式
- 时间解锁
- 互斥选择

### 配置
```
科技数量：50-100
解锁方式：时间+资源
时代划分：4-6个时代
```

# 七、功能速查表

## 技能树系统方法

| 方法 | 说明 | 参数 |
|------|------|------|
| getSkillPoints | 获取可用点数 | 角色 |
| unlockNode | 解锁节点 | 节点ID |
| upgradeNode | 升级节点 | 节点ID |
| canUnlock | 检查能否解锁 | 节点ID |
| resetTree | 重置技能树 | 分支ID? |
| getNodeLevel | 获取节点等级 | 节点ID |
| getUnlockedNodes | 获取已解锁节点 | - |
| savePreset | 保存预设 | 预设名 |
| loadPreset | 加载预设 | 预设名 |

## 节点类型

| 类型 | 说明 |
|------|------|
| STAT | 属性节点 |
| SKILL | 技能节点 |
| PASSIVE | 被动节点 |
| KEYSTONE | 关键节点 |
| UTILITY | 效用节点 |

## 技能树事件

| 事件 | 数据 | 说明 |
|------|------|------|
| skilltree:node_unlocked | {nodeId, level} | 节点解锁 |
| skilltree:node_upgraded | {nodeId, level} | 节点升级 |
| skilltree:reset | {branch} | 技能树重置 |
| skilltree:preset_saved | {presetName} | 预设保存 |
| skilltree:preset_loaded | {presetName} | 预设加载 |
| skilltree:points_gained | {amount, source} | 获得技能点 |
