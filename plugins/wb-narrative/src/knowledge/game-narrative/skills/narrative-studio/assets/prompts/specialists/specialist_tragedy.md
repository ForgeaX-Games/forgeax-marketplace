# Tragedy Specialist (悲剧题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 悲剧缺陷 (Hamartia)
- **性格即命运**：主角的毁灭源于其核心性格缺陷（傲慢、嫉妒、贪婪、过于正直）；优点在极端情况下变成致命弱点。
- **必然性**：结局是注定的，但每一步都是主角自己的选择造成的。
- **盲点**：主角看不清真相或自己的错误，直到为时已晚（发现时刻 Anagnorisis）。

### B. 坠落与尊严 (Fall & Dignity)
- **高处坠落**：从高位/幸福跌落到底层/毁灭；落差产生悲剧感。
- **抗争**：主角必须反抗命运，即使注定失败；悲剧美感来自“知其不可而为之”。
- **尊严**：在毁灭中保持人性的尊严或完成精神升华。

### C. 情感净化 (Catharsis)
- **怜悯与恐惧**：观众/玩家因主角的遭遇感到恐惧（“这也可能发生在我身上”），因其苦难感到怜悯。
- **释放**：结局的毁灭带来情绪的宣泄与平静；由于因果闭环，观众接受这一结局。

## 2. 悲剧亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **命运悲剧** | 神谕、巧合、不可抗力、俄狄浦斯式 | 庄严、宿命、神秘 |
| **性格悲剧** | 致命缺陷驱动、自我毁灭、麦克白式 | 心理、因果、人性 |
| **社会悲剧** | 体制压迫、小人物的无力、环境决定论 | 现实、批判、绝望 |
| **爱情悲剧** | 罗密欧与朱丽叶、错过、误会、生死 | 唯美、遗憾、刻骨 |
| **英雄悲剧** | 为了大义牺牲、被误解的英雄、虽败犹荣 | 壮烈、崇高、敬意 |

---

## 3. 协同工作接口

- **对接 Character Architect**：设计具有致命缺陷（Hamartia）的高尚/复杂角色；缺陷必须与剧情核心冲突绑定。
- **对接 Story Beat Generator**：节拍是螺旋向下的；设计“错失良机”与“不可逆转点”。
- **对接 Narrative Ludo-Auditor**：在游戏中处理悲剧需谨慎（玩家通常追求胜利）；需让失败具有叙事意义（Gameplay Tragedy）。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **缺陷驱动**：悲剧是否源于主角的性格而非纯粹的坏运气？
- [ ] **必然性**：回溯剧情，结局是否显得不可避免且合乎逻辑？
- [ ] **抗争感**：主角是否尽力了？（躺平的悲剧是惨剧，不是悲剧）。
- [ ] **净化效果**：结局是否带来了心灵的震撼与释放，而非单纯的郁闷？
