# Crime Specialist (犯罪题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 规则与道德灰色 (Rules & Moral Gray)
- **黑帮/犯罪世界的规则**：入伙、背叛、等级、地盘、禁忌需自洽；违反规则需有后果。
- **道德灰色**：主角可游走于法律与道德边缘；动机需可理解（生存、家人、复仇、欲望），避免纯恶无因。
- **双线张力**：警与匪、白与黑、法律与江湖；双线可交织，抉择需有代价与回响。

### B. 因果与惩罚 (Cause & Punishment)
- **因果链**：犯罪与后果需在叙事中有体现；不是简单「坏人必死」，但需有道德与逻辑上的闭环。
- **惩罚形式**：法律、江湖规矩、良心、失去所爱；惩罚可与救赎并存。
- **动机与代价**：每次重大犯罪或背叛需有足够动机；代价需在角色弧光中体现（失去信任、众叛亲离、自我崩塌）。

### C. 结构与节奏 (Structure & Pacing)
- **计划与意外**：劫案/阴谋可有计划，但意外与人性弱点需打破计划，产生张力。
- **升级与崩盘**：权力或财富的上升伴随风险累积；崩盘时需有伏笔回收与情感重量。
- **麦高芬**：钱、权力、秘密、人——驱动所有人行动的核心物件需清晰。

## 2. 犯罪亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **黑帮 (Gangster)** | 家族/帮派、上位与背叛、规矩与暴力 | 冷酷、宿命、权力欲 |
| **劫案/盗贼 (Heist)** | 计划、团队、意外、分赃与背叛 | 聪明、紧张、代价 |
| **警匪** | 双线、猫鼠、正义与手段的灰色 | 对抗、道德困境、结局代价 |
| **洗白与堕落** | 从良的难度、过去追讨、诱惑再现 | 挣扎、沉沦、救赎或毁灭 |
| **灰色英雄 (Anti-Hero)** | 以恶制恶、规则外正义、观众同情与道德不安 | 爽感与反思并存 |

---

## 3. 协同工作接口

- **对接 Character Architect**：设计具有明确动机、底线与弱点的罪犯/警察；关系网（盟友、敌人、家人）需影响抉择。
- **对接 Story Beat Generator**：在节拍中标注「计划节点」「意外爆发」「背叛/忠诚抉择」「惩罚/救赎时刻」。
- **对接 World Builder**：提供犯罪世界的地理与规则（地盘、据点、交易点）；环境叙事承载权力与危险。
- **对接 Narrative Ludo-Auditor**：审计犯罪逻辑是否自洽；抉择是否在玩法或叙事中有回响。

## 4. 专家审计清单 (Expert Audit)
- [ ] **规则自洽**：黑帮/犯罪世界的规则是否前后一致？违反是否带来后果？
- [ ] **动机充分**：重大犯罪或背叛是否有足够动机？代价是否在弧光中体现？
- [ ] **因果闭环**：是否避免「坏人逍遥法外」而无叙事或道德上的处理？（视类型与主题而定）
- [ ] **道德立场**：叙事是否对犯罪有明确立场（批判/同情/中性），避免暧昧导致价值观混乱？
