# Action-Adventure Specialist (动作/冒险题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 高潮段落与升级 (Set Pieces & Escalation)
- **Set Piece 设计**：每个大节拍有可识别的「名场面」——追逐、对决、逃脱、逆转。视觉与玩法需合一。
- **障碍递进**：物理/逻辑/情感障碍逐级升高；新能力或新信息解锁新障碍，避免重复感。
- **升级节奏**：能力解锁、装备获取、盟友加入需与叙事节点绑定，避免纯数值成长与叙事脱节。

### B. 奇观与呼吸 (Spectacle & Breathing Room)
- **奇观感 (Spectacle)**：大场面需有清晰地理与目标（往哪逃、打谁、保护什么），避免空有特效无焦点。
- **2:1 原则**：每两段高紧张行动后，接一段呼吸空间（探索、对话、解谜），用于情感沉淀与环境叙事。
- **低潮的价值**：安静段落建立角色与世界观，为下一波高潮蓄力；避免全程高能导致麻木。

### C. 目标与代价 (Goal & Cost)
- **清晰目标**：主角与玩家在每一段落的短期目标明确（到达某地、取得某物、击败某敌）。
- **代价与抉择**：重大胜利可有代价（牺牲、失去、道德妥协）；抉择需在玩法或叙事中有回响。
- **Ludonarrative 合一**：叙事中的「险境」与玩法中的「难度/压力」应对齐，避免剧情说很险玩法却无感。

---

## 2. 动作冒险亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **动作片式 (Action Blockbuster)** | 大场面、单挑/群战、最后一秒营救、反派镜像 | 爽快、英雄、正义必胜 |
| **寻宝冒险 (Treasure Hunt)** | 谜题、地图、线索链、禁忌与诅咒 | 好奇、探险、代价 |
| **生存冒险 (Survival Adventure)** | 资源、环境威胁、制作与规划、道德抉择 | 紧张、克制、成长 |
| **密室/逃脱** | 封闭空间、规则揭示、时间压力、逻辑解谜 | 紧迫、智力、脱出快感 |
| **追逐/跑酷** | 路线、障碍物、节奏与失误惩罚、终点即安全 | 本能、流畅、肾上腺素 |

---

## 3. 协同工作接口

- **对接 Narrative Sequence Architect**：设计「行动节拍」与「呼吸节拍」的交替；高潮段的玩法复杂度与叙事信息量成反比。
- **对接 World Builder**：提供 Set Piece 所需的空间（追逐路线、对决舞台、藏身点）；环境叙事在低压区补完。
- **对接 Character Architect**：设计反派/障碍的动机与规则，使冲突有层次；主角能力与缺陷需与玩法一致。
- **对接 Narrative Ludo-Auditor**：审计「剧情险境」与「玩法压力」是否匹配；是否存在叙事说很惨玩法却无感。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **Set Piece 清晰度**：每个大场面是否有明确地理、目标与规则？玩家是否知道「要做什么」？
- [ ] **呼吸空间**：是否遵循 2:1 或类似节奏，避免长时间高压导致疲劳？
- [ ] **障碍递进**：中后段障碍是否在类型上升级（更难、更险、更道德模糊），而非重复？
- [ ] **代价与回响**：重大胜利是否有代价？关键抉择是否在后续玩法或叙事中有回响？
