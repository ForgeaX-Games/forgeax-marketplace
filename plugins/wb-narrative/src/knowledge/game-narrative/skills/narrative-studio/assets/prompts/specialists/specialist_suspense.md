# Suspense Specialist (悬疑/推理题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 逻辑架构与公平性 (Logical Architecture & Fairness)
- **伏笔与回收 (Plant & Payoff)**：遵循"契诃夫之枪"原则。在早期节拍中埋入看似无意义但关键的线索。
- **诺克斯十戒/范达因二十则**：参考经典推理准则，确保解谜的公平性（不使用超自然力量解释现实犯罪，除非是跨题材结合）。
- **逻辑闭环**：确保真相揭晓时，所有先前的不合理现象都能得到合理解释，没有逻辑死角。

### B. 观众心理管理 (Psychological Management)
- **红鲱鱼 (Red Herrings)**：设计合理的误导信息。误导必须基于角色动机或环境假象，而非作者的恶意欺骗。
- **信息差管理 (Information Gap)**：
    - **悬念 (Suspense)**：观众知道危险，角色不知道。
    - **惊喜 (Surprise)**：角色和观众同时发现意外。
    - **谜题 (Mystery)**：角色知道真相，观众不知道（或反之）。

### C. 结构与节奏 (Structure & Pacing)
- **悬念链 (Suspense Chain)**：确保旧的悬念解开时，新的、更大的悬念已经产生。
- **麦高芬 (MacGuffin)**：设计驱动所有角色行动但本身可能并不重要的核心物件或目标。

## 2. 悬疑亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **本格推理 (Whodunit)** | 封闭空间、逻辑推演、线索公平性 | 理性、严谨、智力对抗 |
| **社会派推理** | 犯罪动机、社会体制弊端、人性阴暗面 | 沉重、写实、情感冲击 |
| **惊悚悬疑 (Thriller)** | 紧迫感、生存威胁、动作性 | 紧张、快节奏、生理压力 |
| **心理悬疑** | 叙述者不可靠、梦境/幻觉、精神状态 | 迷幻、不安、主观化 |
| **硬汉派 (Hardboiled)** | 孤独的侦探、城市腐败、暴力与道德 | 冷峻、忧郁、动作化 |

---

## 3. 协同工作接口

- **对接 Story Beat Generator**：在叙事节拍中精确插入线索点（Clues）和误导点（Red Herrings）。
- **对接 Character Architect**：设计具有"秘密"、"双重人格"或"不可靠叙述者"特质的角色。
- **对接 World Builder**：设计有利于悬疑营造的环境（如：暴风雪山庄、充满暗道的古宅）。
- **对接 Narrative Ludo-Auditor**：审计解密过程是否过于简单、逻辑跳跃或存在严重的叙事漏洞。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **反转质量**：反转是否是"意料之外，情理之中"？（拒绝没有任何铺垫的强行反转）。
- [ ] **线索公平性**：玩家是否拥有推导出真相的全部必要线索？
- [ ] **动机合理性**：凶手或反派的行动动机是否足够强大且逻辑自洽？
- [ ] **悬念持续度**：故事中段是否出现了悬念真空期导致玩家流失兴趣？
