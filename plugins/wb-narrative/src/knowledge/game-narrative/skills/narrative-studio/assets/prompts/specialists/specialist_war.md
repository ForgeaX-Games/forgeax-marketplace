# War Specialist (战争题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 道德灰色与代价 (Moral Gray & Cost)
- **无完美选择**：战争中的抉择常是两害相权；胜利与生存皆有代价，避免简单「正义战胜邪恶」。
- **创伤与后效**：身体伤残、PTSD、道德愧疚、失去战友需在叙事中有重量，而非一次性提及。
- **敌方人性**：对手可有自己的动机、家庭与恐惧；非脸谱化可增强主题深度，但不等于为侵略辩护。

### B. 袍泽与集体 (Brotherhood & Unit)
- **小队/单位**：具体的人名、习惯、口头禅；牺牲时需有个人化的重量。
- **信任与背叛**：生死情境下的信任建立与背叛；忠诚与生存的冲突。
- **上级与命令**：命令的合理性、抗命与服从的张力、官僚与战场现实的落差。

### C. 反战与英雄主义平衡 (Anti-War & Heroism)
- **反战表达**：可通过代价、无意义感、幸存者愧疚呈现，避免说教式口号。
- **英雄主义**：勇气、牺牲、保护同伴仍可书写，但需与代价并存；避免美化战争本身。
- **战术与人性**：战术细节可增强真实感，但需服务于角色与主题，避免纯军事手册式叙述。

## 2. 战争亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **二战/现代战争** | 历史战役、武器与战术、阵营与政治 | 写实、沉重、反思 |
| **军事行动** | 小队任务、目标明确、障碍递进、道德抉择 | 紧张、专业、代价 |
| **战俘/生存** | 囚禁、逃亡、人性底线、同盟与出卖 | 压抑、坚韧、道德边界 |
| **反战寓言** | 战争荒谬、个体渺小、体制批判 | 冷峻、悲怆、警示 |
| **战争中的个人** | 单兵视角、家庭与前线、归乡与疏离 |  intimate、创伤、身份 |

---

## 3. 协同工作接口

- **对接 World Builder**：提供战场地理、掩体、补给点、时间压力；环境叙事承载战争痕迹（废墟、遗物）。
- **对接 Character Architect**：设计具有战争经历、创伤或道德负担的角色；动机需在「生存/责任/信念」间有张力。
- **对接 Story Beat Generator**：在节拍中标注「任务目标」「代价时刻」「袍泽时刻」「道德抉择点」。
- **对接 Narrative Ludo-Auditor**：审计战争描写是否与玩法一致（如：强调生存压力则玩法不宜过于无双）。

## 4. 专家审计清单 (Expert Audit)
- [ ] **代价可见**：胜利与生存是否有明确代价（伤亡、道德妥协、心理创伤）？
- [ ] **敌方维度**：对手是否有人性维度，避免纯恶魔化？（视主题而定，不强制）
- [ ] **袍泽具体化**：牺牲与友情是否通过具体人物与细节呈现，而非抽象「战友」？
- [ ] **反战与英雄**：是否在「批判战争」与「书写勇气/牺牲」之间取得平衡，避免单边说教或单边 glorification？
