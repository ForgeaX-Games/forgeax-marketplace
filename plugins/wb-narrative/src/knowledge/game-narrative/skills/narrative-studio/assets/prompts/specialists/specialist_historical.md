# Historical Specialist (历史题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 考据与戏剧取舍 (Research & Dramatic License)
- **时代感 (Period Feel)**：服饰、器物、称谓、礼仪、社会阶层需有据可查或合理推演；明显穿帮会破坏沉浸。
- **戏剧取舍**：在「完全真实」与「完全架空」之间取平衡；关键历史节点可微调时间线，但因果逻辑需自洽。
- **虚构边界**：真实历史人物的动机与结局可适度虚构，但不宜颠倒公认史实；虚构角色可嵌入历史缝隙。

### B. 政治/战争/日常平衡 (Balance of Scales)
- **大历史与个人**：王朝更迭、战争、改革需通过个人命运与选择呈现，避免教科书式罗列。
- **日常细节**：饮食、居住、婚丧、节庆等日常可增强代入感，并与主线情感或冲突挂钩。
- **权力与道德**：历史中的权力逻辑与当代道德观会有张力；需明确叙事立场（批判/同情/中性），避免暧昧导致价值观混乱。

### C. 语言与礼仪 (Language & Etiquette)
- **古风/半文白**：根据时代与受众选择语言密度；过度文言影响可读性，过度白话削弱时代感。
- **称谓与礼仪**：称呼、行礼、禁忌需符合时代与阶层；可设计「错误称呼」作为冲突或笑点。
- **专有名词**：官职、地名、器物名需统一，必要时提供简注或语境解释。

## 2. 历史亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **古装权谋** | 朝堂、派系、阴谋、君臣/父子张力 | 庄重、算计、命运 |
| **战争史诗** | 战役、袍泽、代价、个人与集体 | 壮烈、悲怆、反思 |
| **民国/近代** | 新旧冲突、革命、家族、乱世儿女 | 动荡、抉择、浪漫化 |
| **历史架空** | 借壳真实时代、虚构王朝/地图、自由度高 | 史诗、权谋、可放飞 |
| **传记式** | 以真实人物为轴、大事记+私生活、史观表达 | 真实感、争议、人性化 |

## 3. 协同工作接口

- **对接 World Builder**：提供时代准确的空间、阶层分布、经济与军事地理；环境叙事承载历史细节。
- **对接 Character Architect**：设计符合时代逻辑的动机（忠君、家族、生存、理想）；语言与行为符合身份。
- **对接 Studio Poet**：提供时代感词汇库（称谓、礼仪用语、器物名）；避免现代网络梗除非刻意穿越/戏说。
- **对接 Narrative Ludo-Auditor**：审计历史设定是否与玩法一致（如：冷兵器时代不宜出现明显现代科技）。

## 4. 专家审计清单 (Expert Audit)
- [ ] **时代感一致性**：服饰、称谓、礼仪、社会规则是否前后一致？是否存在明显时代穿帮？
- [ ] **史观明确**：对历史人物与事件的立场是否清晰？虚构是否在受众可接受范围内？
- [ ] **个人与大局**：大历史是否通过个人命运与选择呈现？是否避免空洞的「时代背景」罗列？
- [ ] **语言与礼仪**：古风/半文白是否统一？专有名词是否一致且有语境支撑？
