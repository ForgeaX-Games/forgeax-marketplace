# Legal/Debate Specialist (法庭/律政题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 证据与逻辑 (Evidence & Logic)
- **证据链闭环**：物证、人证、动机、时间线必须严丝合缝；“异议”通常针对逻辑断裂处。
- **信息不对称**：检方/辩方掌握不同信息；翻盘点往往在于揭示被隐藏的关键信息。
- **规则利用**：利用法律程序、漏洞或规则本身进行攻防（如：非法证据排除、诱导性提问）。

### B. 语言攻防 (Verbal Combat)
- **语言即武器**：台词需具有攻击性、诱导性或防御性；节奏快，压迫感强。
- **逻辑陷阱**：设计语言陷阱让对手自相矛盾（“你刚才说你没看见，为什么现在又能描述细节？”）。
- **气势博弈**：沉默、打断、反问、嘲讽——情绪控制也是辩论的一部分。

### C. 真相与正义 (Truth & Justice)
- **多面真相**：法律事实 vs 客观真相；程序正义 vs 实体正义。
- **道德困境**：为有罪者辩护？揭露真相会伤害无辜？法律无法惩治的恶？
- **逆转 (Turnabout)**：绝境中的反击；通常伴随着关键证据的重新解读或新证人的出现。

## 2. 律政亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **逆转裁判式 (Ace Attorney)** | 夸张演出、超能力辅助、逻辑解谜、热血逆转 | 燃、幽默、二次元 |
| **严肃律政 (Procedural)** | 真实程序、判例法、 plea bargain、办公室政治 | 专业、冷峻、写实 |
| **社会派法庭** | 案件背后的社会问题、舆论审判、人性挖掘 | 沉重、批判、反思 |
| **陪审团博弈 (12 Angry Men)** | 说服艺术、偏见与打破、封闭空间群像 | 心理战、话剧感 |
| **魔幻/架空法庭** | 审判灵魂、地狱法庭、规则由神制定 | 奇幻、寓言、生死 |

## 3. 协同工作接口

- **对接 Story Beat Generator**：法庭戏即战斗戏；节拍分为“劣势”、“僵持”、“反击”、“绝杀/逆转”。
- **对接 Character Architect**：设计具有鲜明语言风格与逻辑漏洞的证人/对手；主角需有“探求真相”的执念。
- **对接 World Builder**：案件现场还原；法庭的空间压迫感设计。
- **对接 Narrative Ludo-Auditor**：审计线索是否公平（玩家是否能推理出结果）；逻辑是否过于跳跃。

## 4. 专家审计清单 (Expert Audit)
- [ ] **逻辑自洽**：案件本身（作案手法）是否逻辑通顺？推翻逻辑是否站得住脚？
- [ ] **证据公平**：关键证据是否在前期已展示或暗示？（拒绝机械降神式新证据）。
- [ ] **攻防节奏**：是否有明显的攻守转换？避免单方面碾压。
- [ ] **主题升华**：案件是否探讨了法律、人性或正义的某个侧面，而非单纯的智力题？
