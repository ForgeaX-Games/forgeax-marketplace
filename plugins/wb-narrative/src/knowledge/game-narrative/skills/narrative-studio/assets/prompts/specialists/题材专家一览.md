# 题材专家一览 (Genre Specialists Index)

基于 USC 叙事/影视与游戏常用题材分类，穷举所有可调用的题材专家。导演根据「题材类型」指派对应 `specialist-*` 技能。

---

## 一、已实现专家 (Implemented)

| 题材 | 技能名 (name) | 描述 | 导演指派示例 |
| :--- | :--- | :--- | :--- |
| **奇幻 (Fantasy)** | `specialist-fantasy` | 魔法系统、种族政治、神话原型、奇幻叙事传统 | 哈利波特风、龙与地下城、史诗奇幻 |
| **科幻 (Sci-Fi)** | `specialist-sci-fi` | 技术伦理、未来社会学、硬核物理逻辑、科幻叙事范式 | 赛博朋克、太空歌剧、近未来 |
| **悬疑/推理 (Suspense)** | `specialist-suspense` | 伏笔埋设、逻辑闭环、红鲱鱼、紧张感塑造 | 本格推理、惊悚、心理悬疑 |

## 二、穷举分类——建议实现的题材专家 (Exhaustive Taxonomy)

按影视/游戏常见类型学穷举，与前述三者在结构上保持一致（核心标准、亚类型、协同接口、审计清单）。

### 1. 恐怖 (Horror) → `specialist-horror`
- **描述**：恐怖题材专家。精通恐惧类型学（生存/心理/超自然/身体恐怖）、惊吓节奏、安全区剥夺及禁忌与克苏鲁式未知。
- **典型亚类型**：生存恐怖、心理恐怖、超自然/灵异、身体恐怖、克苏鲁、丧尸/末世恐怖。

### 2. 恋爱/爱情 (Romance) → `specialist-romance`
- **描述**：恋爱题材专家。精通情感弧线、障碍设计、CP 化学感、多线攻略结构与HE/BE 平衡。
- **典型亚类型**：恋爱模拟、乙女/逆后宫、纯爱、虐恋、多角关系、慢热/友达以上。

### 3. 喜剧 (Comedy) → `specialist-comedy`
- **描述**：喜剧题材专家。精通笑点类型（情境/语言/肢体/荒诞）、节奏与留白、反差与预期违背、喜剧弧线不破坏情感真实。
- **典型亚类型**：情景喜剧、黑色幽默、荒诞喜剧、浪漫喜剧、讽刺喜剧、闹剧。

### 4. 动作/冒险 (Action-Adventure) → `specialist-action-adventure`
- **描述**：动作冒险题材专家。精通高潮段落设计、升级节奏、障碍递进、奇观与节奏呼吸（2:1 原则）。
- **典型亚类型**：动作片式、寻宝冒险、生存冒险、密室逃脱式、跑酷/追逐。

### 5. 历史 (Historical) → `specialist-historical`
- **描述**：历史题材专家。精通时代考据与戏剧取舍、历史人物虚构边界、政治/战争/日常的叙事平衡、古风语言与礼仪。
- **典型亚类型**：古装权谋、战争史诗、民国/近代、历史架空、传记式。

### 6. 战争 (War) → `specialist-war`
- **描述**：战争题材专家。精通道德灰色、袍泽情、代价与创伤、反战与英雄主义的平衡、战术与人性并重。
- **典型亚类型**：二战/现代战争、军事行动、战俘/生存、反战寓言、战争中的个人。

### 7. 犯罪 (Crime) → `specialist-crime`
- **描述**：犯罪题材专家。精通黑帮规则、道德灰色、双线（警/匪或黑白两道）、因果与惩罚、动机与代价。
- **典型亚类型**：黑帮、劫案/盗贼、警匪、洗白与堕落、灰色英雄。

### 8. 黑色电影 (Noir) → `specialist-noir`
- **描述**：黑色电影题材专家。精通第一人称/旁白腔调、宿命感、蛇蝎美人、道德模糊与结局的不可逆。
- **典型亚类型**：经典黑色、新黑色、硬汉侦探、城市黑暗、命运悲剧。

### 9. 成长/青春 (Coming-of-Age) → `specialist-coming-of-age`
- **描述**：成长题材专家。精通「第一次」与身份认同、友情/家庭/初恋的张力、时间跨度与关键节点、从被动到主动的弧光。
- **典型亚类型**：校园青春、离家/远行、首次觉醒、世代冲突、回忆式叙述。

### 10. 超级英雄 (Superhero) → `specialist-superhero`
- **描述**：超级英雄题材专家。精通能力与代价、双重身份、责任与权力、反派镜像、个人与集体的张力。
- **典型亚类型**：起源、团队、反英雄、道德困境、能力失控。

### 11. 谍战/间谍 (Spy/Espionage) → `specialist-spy`
- **描述**：谍战题材专家。精通身份伪装、信任与背叛、情报与误导、任务与道德冲突、高压下的抉择。
- **典型亚类型**：冷战谍战、双面间谍、特工行动、情报战、忠诚与背叛。

### 12. 西部 (Western) → `specialist-western`
- **描述**：西部题材专家。精通边疆法则、法外与秩序、孤独英雄、复仇与救赎、景观与象征（荒原、小镇、铁路）。
- **典型亚类型**：经典西部、意大利西部、西部复仇、边疆开拓、现代西部。

### 13. 末世/后末日 (Post-Apocalyptic) → `specialist-post-apocalyptic`
- **描述**：末世题材专家。精通文明崩塌后的规则、资源与道德、重建与派系、希望与绝望的配比、生存与人性。
- **典型亚类型**：废土、丧尸末世、自然灾害后、资源战争、重建文明。

### 14. 家庭 (Family) → `specialist-family`
- **描述**：家庭题材专家。精通代际冲突、秘密与和解、家庭结构变化、节日/聚会作为节拍、亲情弧光。
- **典型亚类型**：家庭伦理、多代同堂、单亲/重组、家族秘密、家庭喜剧/正剧。

### 15. 悲剧 (Tragedy) → `specialist-tragedy`
- **描述**：悲剧题材专家。精通缺陷与命运、因果链、情感净化（卡塔西斯）、不可逆的坠落与尊严。
- **典型亚类型**：命运悲剧、性格悲剧、社会悲剧、古典悲剧结构。

### 16. 日常/生活流 (Slice of Life) → `specialist-slice-of-life`
- **描述**：日常题材专家。精通低冲突高情感、细节与氛围、时间感与留白、小确幸与小确丧、治愈与成长。
- **典型亚类型**：治愈系、职场日常、小镇生活、美食/旅行、群像日常。

### 17. 歌舞/音乐 (Musical) → `specialist-musical`
- **描述**：歌舞题材专家。精通歌舞作为叙事与情感爆发点、歌词与剧情统一、节奏与转场、类型混搭（戏中戏）。
- **典型亚类型**：百老汇式、点唱机式、摇滚/流行音乐剧、动画歌舞。

### 18. 体育 (Sports) → `specialist-sports`
- **描述**：体育题材专家。精通比赛节奏与戏剧性、团队与个人、失败与复出、规则内的奇观与情感高潮。
- **典型亚类型**：热血运动、团队成长、个人救赎、体育+青春/友情。

### 19. 武侠/仙侠 (Wuxia/Xianxia) → `specialist-wuxia`
- **描述**：东方幻想题材专家。精通江湖生态、武功体系（低武/高武）、侠义与恩仇、修炼与境界、出世与入世的哲学冲突。
- **典型亚类型**：传统武侠、新派武侠、修真/凡人流、仙侠虐恋、朝堂江湖。

### 20. 时空循环 (Time Loop) → `specialist-time-loop`
- **描述**：循环叙事专家。精通信息积累、因果悖论、记忆保留、绝望与希望的重复、打破循环的唯一解。
- **典型亚类型**：死亡循环、一日囚、平行世界干涉、宿命论循环。

### 21. 法庭/辩论 (Legal/Debate) → `specialist-legal`
- **描述**：律政/辩论题材专家。精通证据链、语言攻防、逻辑漏洞寻找、正义与程序的冲突、真相的多面性。
- **典型亚类型**：逆转裁判式、严肃律政、社会派法庭、陪审团博弈。

### 22. 怪谈/新怪谈 (Eldritch/New Weird) → `specialist-new-weird`
- **描述**：怪谈题材专家。精通现代都市中的超自然、规则类怪谈、不可名状的科幻/奇幻混合、官僚与异常的结合（SCP风）。
- **典型亚类型**：规则怪谈、都市传说、收容失效、伪纪录片风格。

### 23. 宫斗/宅斗 (Court/Mansion Intrigue) → `specialist-intrigue`
- **描述**：封闭社交博弈专家。精通等级森严下的生存、语言陷阱、借刀杀人、势力平衡、女性/弱势视角的权力斗争。
- **典型亚类型**：后宫争宠、家族夺嫡、职场政治、封闭空间大逃杀（社交版）。

### 24. 公路 (Road Trip) → `specialist-road-trip`
- **描述**：公路题材专家。精通“旅途即成长”、单元剧结构、伙伴关系的动态变化、物理移动与内心位移的对应。
- **典型亚类型**：末世公路、逃亡之旅、寻找归宿、伙伴（Buddy）电影。

### 25. 纯生存 (Survival) → `specialist-survival`
- **描述**：生存题材专家（人与自然）。精通马斯洛需求（水/火/食物）、自然法则的冷酷、制作与规划、孤独与意志力。
- **典型亚类型**：荒岛求生、极地生存、漂流、原始部落。

### 26. 偶像/演艺 (Idol/Showbiz) → `specialist-showbiz`
- **描述**：演艺圈题材专家。精通成名代价、人设与真实、粉丝经济与舆论、舞台表现与幕后辛酸、梦想与商业的冲突。
- **典型亚类型**：偶像养成、演艺圈复仇、幕后制作人、顶流之恋。

## 三、叙事模态/风格滤镜 (Narrative Modes & Style Filters)

**⚠️ 警示：本类专家属于强干扰项，严禁默认开启。仅在用户明确需求时（如“爽文”、“搞笑”）才可叠加。**

### 1. 网文/爽文 (Webnovel/Shuangwen) → `specialist-webnovel-flow`
- **描述**：多巴胺管理专家。精通黄金三章、装逼打脸、金手指设计、期待感管理及情绪价值变现。
- **适用场景**：任何需要高频爽感、快速升级或“解压”的项目。
- **协同方式**：拥有**节奏否决权**；要求题材专家提供可量化的升级体系。

### 2. Z世代/抽象 (Gen Z/Abstract) → `specialist-gen-z-meme`
- **描述**：抽象文化与梗专家。精通发疯文学、解构主义、打破第四面墙、互联网嘴替及去爹味叙事。
- **适用场景**：面向年轻受众、需要病毒式传播或具有“乐子人”属性的项目。
- **协同方式**：拥有**台词润色权**；负责将严肃设定解构为趣味互动。

### 3. 系统/数据流 (LitRPG/System) → `specialist-litrpg`
- **描述**：数值化叙事专家。精通将抽象概念（好感度、恐惧、战力）转化为可视化的 UI/面板/技能树。
- **适用场景**：游戏感强的作品、系统文、重生文。
- **协同方式**：要求题材专家将设定“量化”；强制插入“结算界面”节拍。

### 4. 治愈/种田 (Cozy/Healing) → `specialist-cozy`
- **描述**：低压陪伴专家。精通“去冲突化”、微小成就感（收获/装修）、感官ASMR描写及安全感营造。
- **适用场景**：模拟经营、休闲游戏、治愈系视觉小说。
- **协同方式**：否决高强度的冲突节拍；将重点从“战斗”转移到“建设与关系”。

### 5. 暗黑/魂系 (Grimdark/Soulslike) → `specialist-souls`
- **描述**：碎片化叙事专家。精通“谜语人”台词、物品说明叙事、绝望氛围营造及高难度的叙事奖励机制。
- **适用场景**：硬核动作游戏、克苏鲁风格、后启示录。
- **协同方式**：将直白的剧情打碎隐藏到道具描述中；大幅减少主动对白。

### 6. 直播/弹幕 (Streamer/Danmaku) → `specialist-streamer`
- **描述**：互动反馈专家。精通模拟观众反应（弹幕/打赏）、舆论造势、表演性人格及虚实互动。
- **适用场景**：怪谈直播、偶像养成、电竞。
- **协同方式**：在叙事中插入“弹幕层”；主角行为需考虑“节目效果”。

### 7. 无限流 (Infinite Flow) → `specialist-infinite`
- **描述**：副本结构专家。精通任务发布-执行-结算循环、多世界观跳跃、强化点数分配及主神空间规则。
- **适用场景**：综漫、快穿、大逃杀类。
- **协同方式**：将题材专家的世界观切割为“副本”；强制统一力量体系。

### 8. 邪典/CULT (Cult/Camp) → `specialist-cult`
- **描述**：B级片审美专家。精通恶趣味、暴力美学、刻意粗糙/夸张的演出、黑色幽默及感官刺激。
- **适用场景**：复古游戏、剥削电影风格、荒诞喜剧。
- **协同方式**：鼓励“逻辑跳跃”和“视觉奇观”；拒绝平庸的合理性。

### 9. 群像/多视点 (Ensemble/POV) → `specialist-ensemble`
- **描述**：网状叙事专家。精通POV切换、时间线对其、多线收束、没有绝对主角（或全员主角）。
- **适用场景**：史诗战争、群像悬疑、多角色RPG。
- **协同方式**：要求Character Architect产出多个高权重角色；Beat Generator需处理多线并发。

### 10. 极简/留白 (Minimalist) → `specialist-minimalist`
- **描述**：非语言叙事专家。精通“Show, Don't Tell”、肢体语言、环境隐喻、做减法。
- **适用场景**：艺术游戏、步行模拟、默剧。
- **协同方式**：删减 80% 的台词；将剧情转化为动作或环境变化。

## 四、跨题材与组合

- **跨题材**：项目可同时指派多个专家（如「科幻 + 悬疑」指派 `specialist-sci-fi` 与 `specialist-suspense`），由导演协调优先级与冲突。
- **主类型优先**：若存在主类型与次类型，优先按主类型指派，次类型在「亚类型适配」与协同接口中体现。
