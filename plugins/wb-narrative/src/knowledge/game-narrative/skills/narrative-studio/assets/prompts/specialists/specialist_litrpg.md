# LitRPG/System Specialist (系统/数据流模态专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 万物数值化 (Quantification)
- **显性反馈**：所有的努力必须有数值反馈（+1 力量，+5 好感）。
- **技能树/面板**：将角色成长可视化；技能不仅是战斗，也可以是生活（烹饪 Lv.1）或社交（口才 Lv.Max）。
- **任务系统**：主线/支线/隐藏任务；奖励驱动行为。

### B. UI 叙事 (UI Narrative)
- **系统提示音**：冷漠的系统音是唯一的旁白；系统日志承载世界观。
- **鉴定/随机性**：引入 DND 式的检定（[智力 60] 才能看懂这本书）；运气作为叙事变量。
