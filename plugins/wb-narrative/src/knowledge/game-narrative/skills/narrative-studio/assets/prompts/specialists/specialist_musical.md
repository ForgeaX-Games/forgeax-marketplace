# Musical Specialist (歌舞/音乐题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 歌舞叙事功能 (Functions of Song & Dance)
- **情感爆发**：当说话不足以表达时，便唱歌；当唱歌不足以表达时，便跳舞。
- **独白 (I Want Song)**：主角唱出核心欲望（Disney 传统）；确立目标。
- **对决/合唱**：通过对唱展现冲突、谈判或坠入爱河；合唱展现群体意志或混乱。
- **蒙太奇**：利用歌曲压缩时间、展示过程（如：训练、建设、成长）。

### B. 歌词与潜台词 (Lyrics & Subtext)
- **歌词即台词**：歌词必须推进剧情或深化角色，不能只是好听的废话。
- **潜台词外化**：角色在歌中唱出平时不敢说的话（内心独白外化）。
- **旋律动机**：利用Leitmotif（主导动机）代表特定角色或情感，在变奏中讲故事。

### C. 奇观与转场 (Spectacle & Transition)
- **现实与幻想的边界**：起舞时世界随之改变（灯光、布景、物理规则）；从现实平滑过渡到幻想。
- **群舞奇观**：大规模的肢体律动带来的视觉冲击；配合音乐的高潮。
- **打破第四面墙**：角色直接对观众演唱；自我指涉。

## 2. 歌舞亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **百老汇式** | 爵士、踢踏、Showstopper、大团圆或大悲剧 | 华丽、戏剧、经典 |
| **点唱机 (Jukebox)** | 利用现有流行金曲重新编排剧情 | 怀旧、共鸣、后现代 |
| **摇滚/流行音乐剧** | 现代配器、叛逆精神、演唱会风格 | 激情、先锋、狂野 |
| **动画歌舞 (Disney)** | 童话感、角色曲、魔法与歌声结合 | 梦幻、纯真、感染力 |
| **印度/宝莱坞** | 华丽群舞、色彩斑斓、超长篇幅、悲喜交加 | 热闹、神话、情感浓烈 |

---

## 3. 协同工作接口

- **对接 Studio Poet**：歌词创作是核心；需兼顾韵律与叙事功能。
- **对接 Narrative Sequence Architect**：设计“歌曲节点”；玩法可是音游、QTE 或与音乐卡点的战斗。
- **对接 World Builder**：场景需具备“舞台感”；环境互动可产生音乐（如：雨声、脚步声汇成节奏）。
- **对接 Character Architect**：角色的声音特质需符合性格。

## 4. 专家审计清单 (Expert Audit)
- [ ] **必要性**：这首歌是否必须存在？删掉是否影响剧情理解？
- [ ] **情绪衔接**：从对话到歌唱的过渡是否自然（或风格化地突兀）？
- [ ] **视觉奇观**：画面是否配合音乐产生了超越现实的表现力？
- [ ] **节奏感**：歌曲的分布是否疏密得当？
