# Showbiz/Idol Specialist (偶像/演艺题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 梦想与代价 (Dream & Cost)
- **成名之路**：练习生→出道→爆红→过气/转型；每个阶段的压力与挑战。
- **等价交换**：为了梦想牺牲什么？（隐私、自由、健康、爱情、自尊）。
- **商业与艺术**：做自己想做的 vs 市场想要的；资本的裹挟。

### B. 人设与真实 (Persona vs Reality)
- **面具**：台前的光鲜完美 vs 幕后的疲惫破碎；双重生活的撕裂感。
- **塌房与危机**：秘密暴露的风险；舆论的反噬；公关战。
- **粉丝关系**：爱与控制；私生饭、黑粉、死忠粉；粉丝是水，能载舟亦能覆舟。

### C. 舞台与表现 (Performance)
- **舞台高潮**：演出不仅是展示，更是叙事高潮（在舞台上宣泄情感、对决、告白）。
- **技能成长**：歌舞、演技、魅力的数值化与可视化。
- **竞争机制**：选秀、排行榜、颁奖礼；零和博弈的残酷性。

## 2. 演艺亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **偶像养成** | 练习生日常、选秀竞争、团魂、C位之争 | 热血、青春、成长 |
| **演艺圈复仇** | 华丽归来、揭露黑幕、用演技/作品打脸 | 爽感、腹黑、逆袭 |
| **幕后制作人** | 运营策略、危机公关、挖掘新人、资本博弈 | 策略、职场、群像 |
| **顶流之恋** | 秘密恋爱、地下情、合约情侣、公费恋爱 | 甜蜜、禁忌、刺激 |
| **过气翻红** | 中年危机、实力沉淀、自我和解 | 现实、励志、温情 |

## 3. 协同工作接口

- **对接 Character Architect**：设计具有独特“星光”与致命“黑料”的角色；反差萌与人设崩塌。
- **对接 Story Beat Generator**：节拍多为通告、舞台、热搜事件；高潮在于大型演出或颁奖礼。
- **对接 Specialist-Musical**：如果是音乐题材，需配合歌曲叙事；歌词即心声。
- **对接 Narrative Ludo-Auditor**：审计养成数值是否与剧情匹配；演出环节的操作反馈是否强烈。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **人设反差**：台前幕后是否有足够的反差与张力？
- [ ] **舆论压迫感**：是否展现了网络舆论对角色的真实影响？
- [ ] **舞台叙事**：关键演出是否承载了剧情转折或情感爆发？
- [ ] **行业逻辑**：是否符合娱乐圈的基本规则（而非纯粹的玛丽苏幻想）？
