# Comedy Specialist (喜剧题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 笑点类型学 (Comedy Taxonomy)
- **情境喜剧 (Situational)**：角色陷入尴尬、误会、连锁反应；逻辑自洽的荒诞。
- **语言喜剧 (Verbal)**：双关、反讽、节奏化对话、重复与变奏、吐槽与接梗。
- **肢体/动作喜剧 (Physical)**：夸张动作、时机差、倒霉蛋连锁、与玩法反馈结合。
- **荒诞/黑色 (Absurd/Black)**：逻辑断裂、道德灰色、笑与不安并存；需有情感落点避免纯虚无。

### B. 节奏与留白 (Pacing & Setup-Payoff)
- **预期建立与违背**：先建立「正常」预期，再在时机点违背，笑点来自落差而非单纯怪癖。
- **留白**：不解释所有笑点；留给观众/玩家自己补全。过度解释会削弱喜感。
- **呼吸空间**：连续笑点后需情感或节奏上的缓冲，否则疲劳；与紧张玩法搭配时遵循 2:1 原则。

### C. 喜剧弧线与情感真实 (Arc & Truth)
- **角色认真**：角色在自己的世界里是认真的；观众笑的是情境与反差，不是角色「在演喜剧」。
- **情感落点**：喜剧中段或结尾需有真实情感（友情、成长、和解），否则易流于空洞。
- **代价与底线**：笑料可以有代价（尴尬、失败），但需有底线（不嘲笑弱势、不纯恶意的羞辱）。

## 2. 喜剧亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **情景喜剧 (Sitcom)** | 固定场景、误会链、角色化学、每集小闭环 | 轻松、可重复、安全 |
| **黑色幽默 (Black Comedy)** | 禁忌话题、道德模糊、笑与不适并存 | 讽刺、冷峻、反思 |
| **荒诞喜剧 (Absurdist)** | 逻辑断裂、规则自洽的疯狂世界、存在主义玩笑 | 迷幻、疏离、释然 |
| **浪漫喜剧 (Rom-Com)** | 相遇→障碍→误会→和解；笑点与心动点交织 | 甜、虐、HE 预期 |
| **讽刺喜剧 (Satire)** | 对制度、阶层、文化的夸张与讽刺；有明确靶子 | 尖锐、冒犯、思辨 |
| **闹剧 (Slapstick)** | 肢体、倒霉、连锁反应、低语言高动作 | 直给、爽快、合家欢 |

## 3. 协同工作接口

- **对接 Character Architect**：设计具有鲜明缺陷、口头禅或行为模式的角色；缺陷需可爱或可成长，避免纯负面。
- **对接 Story Beat Generator**：在节拍中标注「笑点时刻」「误会爆发」「情感落点」；与玩法高潮错峰避免信息过载。
- **对接 Studio Poet**：提供吐槽腔、双关、重复句式等语言风格建议；台词节奏与阅读/语音节奏一致。
- **对接 Narrative Ludo-Auditor**：审计喜剧节奏是否与操作节奏匹配（如：高难度操作时不宜依赖长段台词笑点）。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **预期与违背**：主要笑点是否建立在「建立预期→违背预期」之上，而非单纯怪癖堆砌？
- [ ] **角色认真**：角色是否在自己的逻辑里认真行动？观众笑的是情境还是角色被写成「小丑」？
- [ ] **情感落点**：喜剧是否在适当节点有真实情感（感动、成长、和解），避免全程飘忽？
- [ ] **底线**：是否避免对弱势、创伤、真实苦难的恶意嘲笑？讽刺是否有明确对象与边界？
