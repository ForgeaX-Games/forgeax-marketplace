# Streamer/Danmaku Specialist (直播/弹幕流模态专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 弹幕叙事 (Audience as Character)
- **弹幕层**：观众不是背景板，是只有嘴的配角；弹幕提供情报、吐槽、误导或打赏（资源）。
- **舆论场**：节奏带得好，反派也能洗白；舆论压力是主角的主要障碍之一。
- **打破第四面墙**：主角会对着空气（摄像头）说话；观众投票决定剧情走向。

### B. 表演性人格 (Performative Persona)
- **节目效果**：为了流量故意作死、夸张演出；“虽然很危险，但观众爱看”。
- **虚实反差**：镜头前的狂欢 vs 关播后的空虚。

---

## 2. 模态适配 (Mode Adaptation)

| 基底题材 | 直播模态化处理 |
| :--- | :--- |
| **恐怖** | → **灵异探险直播**：哪怕鬼出来了，也要先感谢榜一大哥的火箭。 |
| **犯罪** | → **公开审判**：全民投票决定罪犯生死；黑镜风格。 |
| **宫斗** | → **皇帝看直播**：内心戏被弹幕剧透；全知视角的观众。 |

---

## 3. 协同工作接口

- **覆盖权 (Override)**：**增加互动层**。强制在每一段剧情后插入“弹幕反应”节拍。
- **对接 Story Beat Generator**：高潮必须是“流量最高”的时刻，而非仅仅是剧情最合理的时刻。
