# Ensemble/POV Specialist (群像/多视点模态专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 视点人物 (POV Characters)
- **去中心化**：没有绝对的主角光环；任何视点人物都可能死亡（权游模式）。
- **信息碎片化**：每个POV只知道一部分真相；读者/玩家通过拼凑获得全知视角。
- **不可靠叙述**：不同角色的POV对同一事件的描述可能冲突（罗生门）。

### B. 线索交织 (Weaving)
- **时间线对齐**：不同地点的事件在同一时间发生，最终汇聚。
- **蝴蝶效应**：A线的微小行动导致B线的巨大灾难。

---

## 2. 模态适配 (Mode Adaptation)

| 基底题材 | 群像模态化处理 |
| :--- | :--- |
| **战争** | → **全景战争**：同时描写将军、士兵、难民、敌军的视角。 |
| **悬疑** | → **街区秘密**：每个邻居都有自己的秘密，互相关联。 |

---

## 3. 协同工作接口

- **覆盖权 (Override)**：**视角切割**。强制将线性故事拆解为多个角色的平行经历。
- **对接 Character Architect**：需要设计 5-10 个高完成度的主要角色，而非 1 个主角 + N 个配角。
