# Family Specialist (家庭题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 结构与动力 (Structure & Dynamics)
- **家庭角色**：不仅是称谓，更是功能（控制者、调停者、替罪羊、隐形人）。
- **权力结构**：经济大权、话语权、情感勒索；权力的转移（如：父母老去，子女掌权）是冲突源。
- **代际冲突**：价值观、生活方式、沟通方式的断裂；爱的表达方式错位。

### B. 秘密与创伤 (Secrets & Trauma)
- **家庭秘密**：私生子、破产、过往罪恶、遗传病；秘密维持着表面的和平，揭开即风暴。
- **原生家庭创伤**：童年阴影对成年的投射；“想逃离却最终变成了父母的样子”。
- **和解或决裂**：和解不一定是原谅，也可以是接纳或划清界限；结局需真实。

### C. 仪式与空间 (Rituals & Space)
- **家庭聚会**：晚餐、葬礼、婚礼、春节；高压社交场，冲突集中爆发点。
- **空间叙事**：老房子、餐桌、上了锁的抽屉；空间承载记忆与关系。
- **羁绊**：血缘是无法切断的纽带；即使恨也纠缠不清。

## 2. 家庭亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **家庭伦理剧** | 婆媳、出轨、育儿、财产争夺 | 现实、狗血、琐碎 |
| **多代同堂 (Saga)** | 家族兴衰、时代变迁对家庭的冲击、传承 | 史诗、厚重、时间感 |
| **单亲/重组** | 新关系的建立、排斥与接纳、继父母/子女 | 敏感、磨合、成长 |
| **寻亲/归乡** | 寻找根源、面对过去、重新定义家 | 治愈、公路、感伤 |
| **功能失调家庭** | 酗酒、暴力、精神控制、逃离 | 压抑、惊悚、救赎 |

---

## 3. 协同工作接口

- **对接 Character Architect**：设计深受家庭影响的角色；性格缺陷往往能追溯到父母。
- **对接 Story Beat Generator**：节拍多为“餐桌对话”、“秘密泄露”、“爆发争吵”、“温情时刻”。
- **对接 World Builder**：设计充满生活痕迹的家；物品（如：缺角的碗）即叙事。
- **对接 Narrative Ludo-Auditor**：审计情感逻辑是否真实；避免为了戏剧性而牺牲人物逻辑。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **冲突真实**：争吵是否基于真实的生活逻辑与性格差异？
- [ ] **非黑白化**：家人之间往往不是绝对的善恶，而是立场与爱的错位。
- [ ] **和解难度**：和解是否太容易？（深刻的矛盾需要深刻的契机）。
- [ ] **聚会高潮**：是否有利用家庭聚会作为剧情高潮点？
