# Spy/Espionage Specialist (谍战/间谍题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 身份伪装与信任 (Identity & Trust)
- **多重身份**：表面身份、真实身份、任务身份；切换时的风险与破绽。
- **信任建立与利用**：为获取情报建立的信任，与道德代价（利用感情、背叛盟友）。
- **暴露与代价**：身份暴露的触发条件与后果；接近暴露时的紧张感设计。

### B. 情报与误导 (Intel & Misdirection)
- **情报链**：获取—传递—验证—使用；每个环节可有失败与代价。
- **误导与反误导**：假情报、双面间谍、信息差；观众/玩家与角色的信息不对称可制造悬念。
- **麦高芬**：名单、密码、武器——驱动所有人行动的核心情报需清晰。

### C. 任务与道德冲突 (Mission & Moral Conflict)
- **任务优先 vs 人性**：为完成任务是否牺牲无辜、背叛同伴、违背原则？
- **高压抉择**：时间压力、多重目标、不可逆选择；选错即死的设计需有公平性与叙事意义。
- **忠诚与背叛**：对组织、对同伴、对家人、对自己的忠诚如何排序？背叛需有动机与代价。

## 2. 谍战亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **冷战谍战** | 东西方对峙、核威胁、双面间谍、意识形态 | 冷峻、猜疑、代价 |
| **双面间谍** | 多重忠诚、信息博弈、身份崩塌风险 | 高压、反转、信任崩塌 |
| **特工行动** | 任务目标、障碍递进、装备与技能、道德抉择 | 紧张、专业、代价 |
| **情报战** | 密码、窃听、假情报、信息差 | 智力、悬念、反转 |
| **忠诚与背叛** | 组织 vs 个人、同伴 vs 任务、归宿与无根 | 沉重、抉择、身份认同 |

---

## 3. 协同工作接口

- **对接 Character Architect**：设计具有多重身份、秘密与道德负担的角色；关系网（上线、怀疑者、家人）需影响抉择。
- **对接 Story Beat Generator**：在节拍中标注「身份危机」「情报获取/传递」「信任/背叛抉择」「暴露风险」。
- **对接 World Builder**：提供有利于谍战的空间（安全屋、接头点、禁区）；环境叙事承载监视与危险。
- **对接 Narrative Ludo-Auditor**：审计「选错即死」或高后果选项是否公平（玩家是否有足够信息）；失败是否叙事化。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **身份逻辑**：多重身份是否在时间、地点、人际关系上可维持？破绽是否有设计？
- [ ] **情报链清晰**：获取—传递—使用是否在叙事中有体现？失败是否有后果？
- [ ] **道德代价**：为任务牺牲的人性是否在角色弧光中有重量？背叛是否有动机与回响？
- [ ] **高压选项公平性**：高后果选项是否给予玩家足够信息与反应时间？失败是否可叙事化而非单纯 Game Over？
