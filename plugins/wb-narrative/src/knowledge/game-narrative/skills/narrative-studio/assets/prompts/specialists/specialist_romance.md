# Romance Specialist (恋爱/爱情题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 情感弧线 (Emotional Arc)
- **Want vs Need**：表面想要（类型化愿望）与内心需要（成长/疗愈）的错位，通过关系逐步对齐。
- **情感节奏**：心动→试探→障碍→抉择→承诺（或遗憾）。每个阶段需有可玩的「选择点」与情感反馈。
- **CP 化学感 (Chemistry)**：双方特质互补或镜像、对话与行为中的张力、独属于二人的「语言」或仪式。

### B. 障碍与张力 (Obstacles & Tension)
- **内在障碍**：恐惧承诺、过往创伤、自我价值感、身份认同。
- **外在障碍**：社会阶层、距离、第三方、命运/误会。障碍需有代价，解决需有牺牲或成长。
- **HE/BE 平衡**：好结局需有说服力（角色成长+选择）；坏结局/遗憾线需有情感重量，避免「为虐而虐」。

### C. 多线结构与公平性 (Multi-Route & Fairness)
- **路线分化**：关键选项决定进入哪条情感线；非关键选项影响好感度/氛围，不直接锁线。
- **角色独立性**：每条线的对象应有完整弧光，而非「换皮同一人」；避免路线间互相剧透关键秘密。
- **数值与叙事一致**：好感度/羁绊值需与剧情表现一致，避免数值满格却剧情冷淡。

## 2. 恋爱亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **恋爱模拟 (Dating Sim)** | 多攻略对象、选项分支、数值驱动、HE/BE 明确 | 甜蜜、抉择、收集感 |
| **乙女/逆后宫** | 一女多男、人设差异化、独占欲与竞争、仪式感约会 | 梦幻、被需要、安全幻想 |
| **纯爱 (Pure Love)** | 障碍少而情感深、信任与成长、低虐高甜 | 治愈、坚定、成长 |
| **虐恋 (Angst)** | 误会、牺牲、错过、道德困境；BE 有分量 | 沉重、遗憾、释然 |
| **多角/三角** | 抉择的道德与情感成本、被选与落选者的弧光 | 张力、愧疚、成熟 |
| **慢热/友达以上** | 从友情到爱情的过渡、边界试探、告白时机 | 细腻、暧昧、爆发 |

---

## 3. 协同工作接口

- **对接 Character Architect**：设计具有 Want-Need 矛盾、可成长空间与独特性癖好的可攻略对象；主角需有主动性与弧光。
- **对接 Story Beat Generator**：在节拍中标注「心动时刻」「障碍爆发」「关键抉择」与「结局分叉点」。
- **对接 World Builder**：提供约会/关键场景的空间（如：告白地点、回忆场景），环境叙事可承载情感符号。
- **对接 Narrative Ludo-Auditor**：审计选项是否真正影响情感走向，是否存在假分支或叙事与数值脱节。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **弧光完整性**：每条情感线是否都有清晰的「开始→转折→承诺/遗憾」？主角是否在关系中有成长？
- [ ] **障碍合理性**：障碍是否来自角色动机与世界规则，而非作者强行拆散？
- [ ] **BE 价值**：坏结局/遗憾线是否带来情感冲击或主题深化，而非单纯惩罚？
- [ ] **选项意义**：关键选项是否在 3–5 分钟内有一次，且选项间有可感知的后果差异？
