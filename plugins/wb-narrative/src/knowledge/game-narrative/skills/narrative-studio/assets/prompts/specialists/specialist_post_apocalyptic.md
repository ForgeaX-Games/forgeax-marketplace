# Post-Apocalyptic Specialist (末世/后末日题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 文明崩塌后的规则 (Rules After Collapse)
- **新规则**：法律失效后的替代规则——帮派、宗教、以物易物、暴力即权力；规则需自洽且可被挑战。
- **资源即权力**：水、食物、燃料、弹药、药品——稀缺资源的分配决定权力结构。
- **安全区与危险区**：哪里可休息、哪里必争、哪里禁忌；空间的政治与生态需清晰。

### B. 资源与道德 (Resources & Morality)
- **道德边界**：为生存可做到哪一步？食人、掠夺、牺牲他人——触碰底线需有叙事与角色代价。
- **抉择与代价**：救谁、抢谁、与谁结盟；抉择需在后续有回响（信任、报复、资源）。
- **希望与绝望配比**：纯绝望易疲劳，纯希望易虚假；小确幸、小共同体、微光可平衡末世感。

### C. 重建与派系 (Rebuilding & Factions)
- **派系逻辑**：不同群体有不同的生存哲学（独狼、公社、军阀、宗教）；冲突来自理念与资源。
- **重建叙事**：从求生到建家、从建家到建秩序；阶段升级需有叙事与玩法支撑。
- **过去与未来**：旧世界的遗物与记忆；新世界的种子与代价。末世常探讨「人何以为人」。

## 2. 末世亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **废土 (Wasteland)** | 辐射/污染、变异、拾荒与制作、道德灰色 | 荒凉、坚韧、微光 |
| **丧尸末世** | 传染、群体、逃亡与据点、人性考验 | 紧张、抉择、共同体 |
| **自然灾害后** | 气候、地震、海平面；幸存者社群与资源战 | 写实、重建、代价 |
| **资源战争** | 能源、水源、稀土；派系与战争 | 政治、争夺、道德边界 |
| **重建文明** | 从求生到建城、法律与秩序、下一代 | 希望、代价、传承 |

---

## 3. 协同工作接口

- **对接 World Builder**：提供末世空间（废墟、据点、危险区、资源点）；环境叙事承载崩塌与幸存痕迹。
- **对接 Character Architect**：设计具有生存创伤、道德底线与派系归属的角色；动机在「生存/归属/信念」间有张力。
- **对接 Story Beat Generator**：在节拍中标注「资源危机」「道德抉择」「派系冲突」「重建节点」。
- **对接 Narrative Ludo-Auditor**：审计「生存压力」与玩法是否一致；资源与道德抉择是否在后续有回响。

## 4. 专家审计清单 (Expert Audit)
- [ ] **规则自洽**：末世后的权力与规则是否前后一致？资源稀缺是否影响抉择？
- [ ] **道德有重量**：触碰底线（食人、出卖、屠杀）是否有叙事与角色弧光中的代价？
- [ ] **希望与绝望平衡**：是否避免纯绝望导致疲劳或纯希望导致虚假？微光是否有具体载体（人、地、记忆）？
- [ ] **派系有逻辑**：不同群体的生存哲学与利益是否清晰？冲突是否来自理念与资源而非脸谱化善恶？
