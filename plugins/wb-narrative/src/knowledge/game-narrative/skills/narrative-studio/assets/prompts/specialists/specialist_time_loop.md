# Time Loop Specialist (时空循环题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 循环规则 (Loop Rules)
- **触发条件**：死亡、特定时间点、特定事件失败；规则需在初期确立且不可随意更改。
- **保留机制**：什么能带入下一轮？（记忆、知识、纹身/伤痕、极少数物品）。
- **重置范围**：世界完全重置 vs 某些改变是永久的（如：量子纠缠、平行世界影响）。

### B. 信息积累与知识即力量 (Knowledge as Power)
- **背板与预判**：主角利用已知信息“预言”未来、操纵他人、规避危险；爽点来自“全知全能感”。
- **解谜拼图**：每一轮揭示真相的一角；不同时间/地点的线索需拼凑才能看到全貌。
- **试错成本**：利用循环尝试疯狂、危险或不道德的选项（“反正会重置”），探索人性边界。

### C. 心理演变 (Psychological Arc)
- **困惑与恐惧**：初期的不解与对重复的恐惧。
- **放纵与虚无**：利用不死之身享乐、发泄、犯罪，随后陷入无意义感（“上帝的孤独”）。
- **决心与打破**：找到目标（救人、真相），为了打破循环而精密的筹划与无数次练习。

## 2. 循环亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **死亡循环 (Die & Retry)** | Roguelike 机制、高难度挑战、死得其所 | 紧张、硬核、试错 |
| **一日囚 (Groundhog Day)** | 日常生活的变奏、自我完善、人际关系攻略 | 荒诞、哲思、温情 |
| **谋杀解谜 (Loop Mystery)** | 阻止谋杀、找出凶手、时间线与不在场证明 | 悬疑、烧脑、紧迫 |
| **平行世界干涉** | 跨越时空的对话、过去影响未来、多线收束 | 宏大、宿命、感动 |
| **逃离规则** | 循环是囚笼/实验、对抗规则制定者 | 惊悚、反叛、智斗 |

## 3. 协同工作接口

- **对接 Narrative Sequence Architect**：设计“时间表”——NPC 在不同时间点的行动轨迹；设计“最短路径”与“完美路线”。
- **对接 Character Architect**：设计在不同轮回中展现不同侧面的 NPC（因主角干涉而变）；主角的心理创伤与麻木感。
- **对接 Story Beat Generator**：节拍不再是线性的，而是螺旋上升的；标注“关键信息获取点”与“解锁新路径”。
- **对接 Narrative Ludo-Auditor**：审计重复体验是否枯燥；是否提供了“快速跳过”或“差异化反馈”机制。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **规则严谨性**：循环逻辑是否有硬伤？（如：为何某次行动没有触发惯例？）
- [ ] **信息继承价值**：新的一轮是否获得了推进剧情的关键信息？避免无效重复。
- [ ] **打破条件唯一性**：打破循环的条件是否足够苛刻且具有戏剧性？（“完美的一天”或“唯一的真相”）。
- [ ] **主角心理变化**：是否展现了主角对时间的态度变化（从恐惧到利用到厌倦到珍惜）？
