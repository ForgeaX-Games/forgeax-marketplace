# Horror Specialist (恐怖题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 恐惧类型学 (Fear Taxonomy)
- **生存恐怖 (Survival Horror)**：资源稀缺、不可战胜的威胁、逃亡与躲藏。恐惧来自无力感与资源管理。
- **心理恐怖 (Psychological Horror)**：不可靠叙述、幻觉与真实边界模糊、内在罪恶与愧疚。恐惧来自认知崩塌。
- **超自然/灵异 (Supernatural)**：禁忌、诅咒、不可解释现象。恐惧来自规则被打破与未知。
- **身体恐怖 (Body Horror)**：异化、寄生、变形。恐惧来自「自我」与「身体」的剥离。

### B. 节奏与安全区 (Pacing & Safe Zones)
- **惊吓节奏**：建立预期后违背（Jump Scare 需有铺垫）、长时间压抑后的释放、呼吸空间与高压的 2:1 配比。
- **安全区剥夺**：原本安全的空间逐渐被侵蚀，玩家/观众失去「安全锚点」。
- **未知与不可见**：看不见的威胁比可见的更具持续性恐惧；信息控制（不解释 vs 过度解释）决定恐怖寿命。

### C. 禁忌与克苏鲁式 (Taboo & Cosmic Horror)
- **禁忌触碰**：死亡、身体、伦理边界。触碰需有叙事代价与道德重量。
- **克苏鲁式未知**：人类渺小、宇宙冷漠、知识带来疯狂。不可名状优于具体怪物设计。

## 2. 恐怖亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **生存恐怖 (Survival Horror)** | 资源、躲藏、有限对抗、逃生路线 | 压抑、无助、喘息即奢侈 |
| **心理恐怖 (Psychological)** | 叙述者不可靠、记忆/现实错位、内在罪疚 | 不安、迷幻、认知崩塌 |
| **超自然/灵异** | 诅咒、鬼怪、规则与禁忌 | 阴冷、宿命、不可违逆 |
| **身体恐怖 (Body Horror)** | 异化、寄生、医学/实验伦理 | 恶心、身份焦虑、失去自我 |
| **克苏鲁/宇宙恐怖** | 不可名状、知识即危险、人类渺小 | 绝望、冷漠、理智边缘 |
| **丧尸/末世恐怖** | 传染、群体、文明崩塌、道德边界 | 逃亡、抉择、人性考验 |

## 3. 协同工作接口

- **对接 World Builder**：设计有利于恐怖氛围的空间（狭窄通道、视野盲区、声源不明、安全区逐渐失效）。
- **对接 Character Architect**：设计具有创伤、秘密或「不可靠叙述者」特质的角色；反派/威胁的动机与规则需自洽。
- **对接 Story Beat Generator**：在节拍中精确放置「惊吓点」「压抑累积点」与「呼吸空间」，避免疲劳。
- **对接 Narrative Ludo-Auditor**：审计恐怖节奏是否与玩法节奏匹配（如：高操作压力时不宜叠加复杂叙事信息）。

## 4. 专家审计清单 (Expert Audit)
- [ ] **廉价惊吓**：Jump Scare 是否建立在铺垫与氛围之上，而非单纯音效/闪现？
- [ ] **过度解释**：威胁一旦被完全解释，恐惧是否随之消散？是否保留「未知」余量？
- [ ] **安全区滥用**：是否存在永远安全的空间导致紧张感崩盘？
- [ ] **伦理边界**：触碰禁忌是否服务于主题，而非为吓而吓？是否考虑受众边界？
