# Sports Specialist (体育题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 比赛叙事 (Match Narrative)
- **比赛即战斗**：将比赛拆解为节拍（劣势→僵持→觉醒→逆转）；每一分都有叙事意义。
- **最后一分钟**：时间的压迫感；关键球的慢动作与心理独白。
- **必杀技/战术**：将技术动作戏剧化、命名、视觉化；战术博弈（剪刀石头布逻辑）。

### B. 团队与个人 (Team & Individual)
- **位置与性格**：前锋（冲动/英雄）、后卫（稳重/牺牲）、队长（责任）、替补（渴望）；位置对应性格。
- **信任与羁绊**：无声的默契；传球即信任；更衣室冲突与和解。
- **天才与凡人**：天赋的碾压 vs 努力的极限；凡人击败天才的爽点。

### C. 胜负与成长 (Victory, Defeat & Growth)
- **输的意义**：失败往往比胜利更能塑造角色；各种输法（惜败、惨败、被黑哨）。
- **复出与救赎**：老将不死、伤病归来、洗刷耻辱。
- **超越胜负**：最终的胜利不仅是比分，更是战胜了自我或赢得了对手尊重。

## 2. 体育亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **热血竞技 (Shonen)** | 必杀技、友情努力胜利、全国大赛 | 燃、夸张、激情 |
| **写实/职业** | 转会、伤病、商业、战术数据、生涯规划 | 专业、残酷、现实 |
| **弱队逆袭 (Underdog)** | 杂牌军、魔鬼教练、从零开始、奇迹 | 励志、幽默、感动 |
| **个人项目** | 拳击/赛车、孤独的王者、与自己的身体对话 | 孤傲、内省、极限 |
| **智力竞技** | 棋牌/电竞、心理战、算力博弈、微表情 | 烧脑、紧张、静态 |

## 3. 协同工作接口

- **对接 Narrative Sequence Architect**：比赛节奏与玩法节奏高度对齐；QTE 或操作挑战对应关键赛点。
- **对接 Character Architect**：设计对手（劲敌）；劲敌必须强大且值得尊重。
- **对接 World Builder**：赛场氛围（观众、解说、灯光）；更衣室作为私密空间。
- **对接 Narrative Ludo-Auditor**：审计“剧情杀”与玩家操作的冲突（玩家赢了但剧情判输是大忌）。

## 4. 专家审计清单 (Expert Audit)
- [ ] **规则清晰**：不懂规则的观众是否能看懂局势优劣？
- [ ] **动作叙事**：关键得分/失误是否有前因后果与心理铺垫？
- [ ] **对手魅力**：对手是否足够强大且有个性？（战胜弱鸡没有成就感）。
- [ ] **情感高潮**：比赛结束时是否伴随情感的爆发（泪水、欢呼、释然）？
