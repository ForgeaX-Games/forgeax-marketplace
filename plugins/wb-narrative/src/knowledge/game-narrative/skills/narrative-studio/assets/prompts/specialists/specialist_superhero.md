# Superhero Specialist (超级英雄题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 能力与代价 (Power & Cost)
- **能力来源与规则**：超能力需有来源（变异、科技、神赐、修炼）与限制（代价、弱点、道德约束）。
- **代价**：能力使用可有身体/心理/社会代价；「无所不能」会削弱戏剧性，需有障碍与抉择。
- **升级与失控**：能力可随叙事成长，但失控或滥用需有后果；避免能力膨胀无边界。

### B. 双重身份与责任 (Dual Identity & Responsibility)
- **双重身份**：日常身份与英雄身份的张力；秘密身份带来的关系困境（隐瞒、暴露、信任）。
- **责任与权力**：能力越大责任越大——需具体化为抉择（救谁、牺牲什么、何时越界）。
- **个人与集体**：拯救世界与保护所爱、公众期待与私人欲望的冲突。

### C. 反派与镜像 (Villain & Mirror)
- **反派镜像**：反派可镜像主角的欲望、创伤或另一种选择（「若我走错一步」）；避免纯恶无因。
- **动机与规则**：反派需有自洽动机与能力边界；对决需有逻辑与代价。
- **道德灰色**：以暴制暴、越权执法、牺牲少数——超级英雄叙事可探讨权力与道德的边界。

## 2. 超级英雄亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **起源 (Origin)** | 获得能力、第一次抉择、身份确立 | 觉醒、责任、代价 |
| **团队 (Team)** | 协作、分歧、领袖与服从、集体责任 | 群像、冲突、团结 |
| **反英雄 (Anti-Hero)** | 道德模糊、以暴制暴、观众同情与不安 | 暗黑、爽感、反思 |
| **道德困境** | 救谁、杀谁、越权、秘密与公开 | 沉重、抉择、后果 |
| **能力失控** | 能力反噬、伤害无辜、赎罪与重建 | 创伤、责任、救赎 |

---

## 3. 协同工作接口

- **对接 World Builder**：提供英雄与反派活动的空间（城市、基地、异空间）；能力对环境的破坏需有规则与后果。
- **对接 Character Architect**：设计具有 Want-Need 矛盾、秘密身份与道德底线的英雄；反派需有镜像或对照。
- **对接 Story Beat Generator**：在节拍中标注「能力觉醒/失控」「身份危机」「对决与代价」。
- **对接 Narrative Ludo-Auditor**：审计「能力表现」与「玩法反馈」是否一致；叙事中的代价是否在玩法中有体现。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **能力有边界**：超能力是否有明确限制、代价或道德约束？是否避免「万能解」？
- [ ] **双重身份有张力**：秘密身份是否带来关系困境或抉择？暴露是否有后果？
- [ ] **反派有维度**：反派是否有动机与镜像价值？对决是否有逻辑与情感重量？
- [ ] **道德立场**：以暴制暴、越权等行为是否有叙事或角色弧光中的反思与代价？
