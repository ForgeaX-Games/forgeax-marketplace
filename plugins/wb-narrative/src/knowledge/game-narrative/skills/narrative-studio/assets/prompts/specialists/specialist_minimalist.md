# Minimalist Specialist (极简/留白模态专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 减法叙事 (Subtractive Narrative)
- **沉默**：能不说话就不说话；用眼神、动作、交互代替台词。
- **去背景化**：不交代前因后果；主角醒来就在旅途上；世界观靠猜。
- **核心符号**：用一个简单的符号（如：山、光、红围巾）贯穿始终。

### B. 环境与交互 (Environment & Interaction)
- **步行模拟**：走路本身就是叙事；风景的变化暗示心理的变化。
- **机械叙事**：通过玩法的机制（牵手、呼唤）来传达情感，而非文字。

---

## 2. 模态适配 (Mode Adaptation)

| 基底题材 | 极简模态化处理 |
| :--- | :--- |
| **冒险** | → **ICO/风之旅人**：没有文字，只有陪伴与到达。 |
| **恐怖** | → **Limbo**：没有对白，只有黑白画面与死亡陷阱。 |

---

## 3. 协同工作接口

- **覆盖权 (Override)**：**静音键**。删掉 90% 的对话与旁白。
- **对接 Studio Poet**：如果有文字，必须极度精炼，甚至使用造语（无法听懂的语言）。
