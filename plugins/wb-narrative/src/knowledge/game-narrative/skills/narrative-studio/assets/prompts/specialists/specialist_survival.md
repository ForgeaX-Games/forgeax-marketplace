# Survival Specialist (纯生存题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 马斯洛底层 (Maslow's Basement)
- **生理极限**：饥饿、干渴、体温、疲劳、疾病；这些不仅是数值，更是叙事驱动力（为了水可以冒险）。
- **资源管理**：每一份资源（火柴、子弹、抗生素）都极其珍贵；浪费带来死亡焦虑。
- **制作与工具**：从无到有；工具的获得代表能力的跃升（获得生火工具 = 获得温暖与熟食 = 安全感）。

### B. 自然法则 (Laws of Nature)
- **冷酷无情**：自然不邪恶，也不仁慈，只是冷漠。暴风雪不会因为你可怜而停止。
- **掠食者与猎物**：生态链的位置；从被狩猎到狩猎者的转变。
- **环境叙事**：天气、地形、动植物不仅是背景，更是对手或盟友。

### C. 孤独与意志 (Solitude & Will)
- **心理熵**：长期的孤独导致幻觉、自言自语、对无生命物体的拟人化（如《荒岛余生》的排球）。
- **活下去的理由**：求生必须有精神锚点（家人的照片、一个承诺、复仇的执念）。
- **文明的剥离与重建**：回归野兽 vs 保持人性；在荒野中重建微型文明仪式。

## 2. 生存亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **荒岛求生 (Castaway)** | 从零开始、鲁滨逊式建设、孤独感 | 探索、建设、孤独 |
| **极地/极端环境** | 严寒/酷热、时间压力（天黑前）、身体机能对抗 | 紧迫、极限、痛苦 |
| **漂流 (Drifting)** | 空间受限、资源极度匮乏、未知的恐惧 | 压抑、无助、听天由命 |
| **原始部落** | 族群生存、图腾与信仰、与其他部落/野兽的战争 | 野性、残酷、繁衍 |
| **都市求生 (Urban Survival)** | 灾后废墟、拾荒、避开人群、隐形人 | 警惕、冷漠、社会达尔文 |

## 3. 协同工作接口

- **对接 World Builder**：提供资源分布图、动植物生态、天气变化规律；环境即玩法。
- **对接 Character Architect**：设计角色的体能、技能（生火、医疗）与心理弱点；身体状态影响心理状态。
- **对接 Narrative Sequence Architect**：设计“资源循环”与“探索半径”；白天探索/黑夜防守的节奏。
- **对接 Narrative Ludo-Auditor**：审计数值压力是否过大导致无法体验叙事，或过小导致失去生存感。

## 4. 专家审计清单 (Expert Audit)
- [ ] **需求驱动**：角色的行动是否主要由生理需求驱动？（渴了找水，冷了找火）。
- [ ] **自然的压迫感**：环境是否展现出足够的冷漠与威力？
- [ ] **成长可视化**：生存状况的改善是否可见（庇护所升级、食物储备增加）？
- [ ] **心理刻画**：是否描写了孤独、恐惧或绝望对角色心理的侵蚀？
