# Archetype A — Epic Narrative Baseline

> 适用品类家族：JRPG、ARPG、3A线性动作、开放世界RPG、SRPG
> 当品类缺少独立 .skill.ts 时，PromptComposer 回退到此 baseline。

---

## 核心叙事哲学

Epic 叙事的本质是「英雄之旅」——一个普通人在不可逆转的命运召唤下，穿越多层递进的考验，最终以牺牲或超越完成蜕变。

关键支柱：
- **深度世界构建**：世界不是布景板，而是推动故事的引擎——每个地区、种族、历史遗产都应与主角弧光产生共振
- **多幕递增结构**：典型三幕或五幕，每幕结尾有不可逆转折点（导师牺牲 / 真相揭露 / 背叛）
- **同伴即情感锚点**：队友不是工具人，每位同伴承载独立的价值观冲突和成长弧光

---

## 世界观构建指引

### 结构原则
- 从「神话-历史-当代」三层递进构建时间纵深
- 地理即叙事：每个区域对应一种文化/冲突主题，主角旅程 = 文化碰撞之旅
- 古代遗产/预言系统为后续真相揭露埋伏笔
- 元素/力量体系必须与世界文化绑定（非纯战斗机制）

### 关键检查项
- [ ] 世界有清晰的「当前危机」作为主线引擎
- [ ] 至少存在 2-3 个派系/势力的多角博弈
- [ ] 主角故乡/出发地已定义（第一幕情感锚点）
- [ ] 世界史中至少埋有 3 处伏笔种子

---

## 角色塑造指引

### 主角
- 必须有「私人动机」驱动（非抽象的拯救世界）
- 初始状态 = 某种「不完整」，终局 = 通过选择完成自我定义
- 核心抉择时刻：当私人诉求与大义冲突时怎么选

### 同伴体系
- 每位同伴的三要素：加入理由 / 个人创伤 / 战斗哲学
- 同伴之间需要存在「关系张力对」（互补或冲突的价值观）
- 至少 1 位同伴在中后期面临信仰崩塌/背叛/牺牲

### 反派
- 反派 = 主角的暗面镜像（「如果我做了另一个选择」）
- 必须拥有可理解的动机和合理的内在逻辑
- 终局对决应包含价值观辩论，不只是武力碾压

---

## 故事框架指引

### 推荐节拍结构
1. **平凡世界** → 建立日常 / 情感基线
2. **召唤与拒绝** → 触发事件 + 内心犹豫
3. **跨越门槛** → 不可回头的第一步（故乡被毁 / 承诺 / 离别）
4. **试炼之路** → 组队 / 地区叙事 / 同伴线插入 / 能力成长
5. **最深洞穴** → 真相揭露 / 导师牺牲 / 团队分裂
6. **最终试炼** → 重聚 / 价值确认 / 决战
7. **回归与蜕变** → 新秩序 / 代价与获得

### 伏笔管理
- 第一幕埋设的伏笔必须在二/三幕明确回收
- 同伴个人线触发点附着在主线分支节点上
- 最终决战前必须有「团队凝聚」节拍

## 剧本写作指引

### 场景节奏
- 紧张-舒缓交替：每场高压场景后安排 1 场日常/Skit 缓冲
- 关键情感场景使用「慢节奏」：减少动作描写，增加内心独白和沉默留白
- 决战场景用「碎片化」节奏：短句、快速切换、交叉剪辑视角

### 对白原则
- 禁止机制裸露（「你被毒了-3HP」必须改写为情绪表达）
- 角色说话方式必须一致（方言/语气词/句式长度 = 声音指纹）
- 每章给至少 1 位同伴 1 句「名场面级」台词
- 转折点由角色弧光驱动，不由外部事件突兀触发

### 情感曲线
- 全流程情感强度呈「W型」：起→跌→回升→更深跌落→最终高峰
- 牺牲场景不要过度渲染——克制 = 更大冲击力
- 结尾留有「余韵」：不把所有事说完，让玩家自己感受
