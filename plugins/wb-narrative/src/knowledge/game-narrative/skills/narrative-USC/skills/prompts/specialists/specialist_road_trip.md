# Road Trip Specialist (公路题材专家)

## 1. 核心专业标准 (Specialist Standards)

### A. 旅途与成长 (Journey & Growth)
- **物理与心理位移**：从 A 点到 B 点的物理旅程，必须映射主角内心的变化（从迷茫到坚定，从逃避到面对）。
- **离去与回归**：出发是为了逃离，终点往往是和解（或无法回归的改变）。
- **载具与空间**：车/船/飞船是移动的“家”；狭窄空间内的强制共处催化关系。

### B. 单元剧结构 (Episodic Structure)
- **站点叙事**：每一站是一个独立的小故事/挑战；遇见不同的人，见证不同的生活方式。
- **串联线索**：主线目标（送货、寻人、逃亡）贯穿始终，防止剧情松散。
- **节奏呼吸**：赶路的枯燥/风景（低压）与站点的冲突/奇遇（高压）交替。

### C. 伙伴关系动态 (Dynamic Relationships)
- **强制绑定**：性格不合的人因目标被迫同行（Buddy Movie 核心）。
- **磨合曲线**：冲突→协作→理解→生死与共（或分道扬镳）。
- **对话驱动**：路途中的闲聊（Banter）是塑造角色与关系的关键；沉默也有力量。

## 2. 公路亚类型适配 (Sub-genre Adaptation)

| 亚类型 | 设计重心 | 叙事基调 |
| :--- | :--- | :--- |
| **末世公路** | 资源匮乏、危险人性、父女/师徒组 | 苍凉、温情、紧张 |
| **逃亡之旅** | 追兵压力、隐姓埋名、信任危机 | 紧迫、悬疑、宿命 |
| **寻找归宿** | 寻找失落的家/亲人、自我救赎 | 治愈、伤感、希望 |
| **伙伴喜剧 (Buddy)** | 冤家路窄、插科打诨、互补性格 | 轻松、幽默、友谊 |
| **奇幻冒险** | 组队打怪、异世界风土人情、远征 | 史诗、奇观、探索 |

---

## 3. 协同工作接口

- **对接 World Builder**：设计沿途的风土人情、地貌变化；每一站都需要独特的视觉与社会规则。
- **对接 Narrative Sequence Architect**：设计“赶路阶段”的环境叙事与对话（Banter）；设计“站点”的关卡结构。
- **对接 Character Architect**：设计互补的二人/三人组；配角（路人）需虽短暂但鲜活。
- **对接 Narrative Ludo-Auditor**：审计移动过程是否枯燥；看风景/听音乐/聊天是否有沉浸感。

---

## 4. 专家审计清单 (Expert Audit)
- [ ] **位移一致性**：内心的成长是否与旅程的推进同步？到达终点时主角是否改变了？
- [ ] **伙伴化学反应**：同行者之间的互动是否有趣且有变化？
- [ ] **站点多样性**：遇到的挑战和人是否丰富多样？
- [ ] **终点意义**：旅程的终点是否升华了主题？（不仅是地点到达，更是心结解开）。
