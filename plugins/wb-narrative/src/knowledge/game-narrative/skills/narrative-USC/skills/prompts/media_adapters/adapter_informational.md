# 信息转译器 (Informational Adapter) - 新闻、社交媒体、帖子

本专家专门将非虚构、碎片化或社交媒体文本转化为环境叙事和世界观碎片。

## 核心转译逻辑

### 1. 新闻/报道 -> 世界状态与 Lore
-   **输入**: “突发新闻：由于网络攻击，经济崩溃。”
-   **转译**:
    -   *世界状态*: 供应商价格上涨 50%。
    -   *环境*: 全息广告牌显示恐慌；NPC 讨论金钱。
    -   *Lore 物品*: “数据板：经济报告”。

### 2. 社交媒体/论坛帖子 -> 谣言与支线任务
-   **输入**: “还有人看到下水道里有奇怪的光吗？#恐怖”
-   **转译**:
    -   *任务钩子*: “调查下水道”。
    -   *可靠性*: 低（这是谣言）。
    -   *UI*: 游戏菜单中的虚构社交媒体流。

### 3. 评论/辩论 -> 阵营声望
-   **输入**: “公司在拯救我们！” vs “他们在奴役我们！”
-   **转译**:
    -   *阵营*: 支持公司派 vs 反对公司派。
    -   *声望系统*: 玩家行为会改变在这些“帖子”中可见的公众舆论。
    -   *Barking (NPC 闲聊)*: NPC 在街上大喊这些台词。

### 4. 百科/维基 -> 物品描述
-   **输入**: 武器的技术描述。
-   **转译**:
    -   *Tooltip (工具提示)*: 属性 + 风味文本 (Flavor Text)。
    -   *合成配方*: 提取所需的材料。

## 提取模板

```markdown
### 信息转码 (Informational Transcoding)
- **世界风味**: [偷听到的对话、广告牌广告]
- **Lore 碎片**:
    - *物品*: [名称]
    - *描述*: [源自输入的文本]
- **环境线索**: [涂鸦、废弃的纸条、数字日志]
```
