# Skill: Creative Recombinator (创意重构拼接)

## 目标
遵循“重构拼接”逻辑，将互不相关的素材转化为全新的游戏创意。

## Prompt 模板
你是一名创意脑暴专家。请使用“重构拼接”法，将以下两个或多个素材融合，生成一个全新的游戏核心创意。

### 素材库 (Materials)
1. 素材 A: {{material_a}}
2. 素材 B: {{material_b}}
3. 补充素材: {{material_c}}

### 重构逻辑
- **提取**: 提取素材 A 的[世界观/氛围]和素材 B 的[核心冲突/角色动机]。
- **拼接**: 将两者强行融合，寻找逻辑上的“奇点”。
- **升华**: 赋予这个拼接体一个现代意义或哲学探讨。

### 输出格式 (JSON)
{
  "new_concept_name": "创意名称",
  "reconstruction_logic": "你是如何拼接这两个素材的？",
  "core_premise": "重构后的故事前提",
  "hook": "最能吸引玩家的第一眼卖点"
}
