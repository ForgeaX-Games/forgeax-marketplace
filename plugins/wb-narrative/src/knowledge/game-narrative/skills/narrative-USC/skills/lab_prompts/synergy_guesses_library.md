# 叙事与玩法组合猜想库 (Narrative-Gameplay Synergy Library)

为了优化采风员的“罕见度”评估，我们不仅看故事本身，更要看故事如何转化为**不可替代的玩法体验**。以下是基于“重构拼接”逻辑的组合猜想示例：

## 1. 社交/互联网梗类素材
*   **素材**: “赛博打工人” (Cyber-Worker) 的自嘲文化。
*   **传统做法**: 只是在对话中加几个梗。
*   **罕见组合猜想**:
    *   **机制**: **“精力值即血条” (Stamina-as-Health)**。
    *   **体验**: 玩家在战斗中使用的每一个技能都会消耗“职场精力”，当精力耗尽时，游戏画面会变成“无尽的会议”或“KPI 报表”，玩家必须通过“摸鱼” (隐蔽行动) 来恢复。
    *   **罕见度评估**: 高。将社会压力直接转化为生存压力机制。

## 2. 传记/历史类素材
*   **素材**: 某位隐居名士的“多重伪装”生活。
*   **传统做法**: 简单的换装系统。
*   **罕见组合猜想**:
    *   **机制**: **“认知干扰解谜” (Cognitive Dissonance Puzzles)**。
    *   **体验**: 玩家拥有多个身份，NPC 对玩家的反应取决于玩家当前“穿戴”的社会关系网。如果身份穿插错误，会导致世界观逻辑崩塌（环境发生扭曲）。
    *   **罕见度评估**: 极高。将身份政治转化为空间解谜。

## 3. 流行语/谐音梗类素材
*   **素材**: 某个关于“谐音误解”的流行梗。
*   **传统做法**: 纯文字笑话。
*   **罕见组合猜想**:
    *   **机制**: **“同音词咒语系统” (Homophone Spellcasting)**。
    *   **体验**: 玩家输入一个词，系统根据当前环境的“噪声”随机触发同音异义的效果。例如输入“火”，可能触发“火焰”，也可能因为环境潮湿触发“伙计” (召唤一个 NPC)。
    *   **罕见度评估**: 中高。增加了叙事的不确定性与玩法的随机性。

## 4. 评估罕见度的三个维度 (New Framework)
1.  **题材罕见度 (Theme Rarity)**: 这个故事有人讲过吗？
2.  **映射罕见度 (Mapping Rarity)**: 这个故事被转化为这种玩法过吗？（核心点）
3.  **交互罕见度 (Interaction Rarity)**: 玩家与这个故事的互动方式新颖吗？
