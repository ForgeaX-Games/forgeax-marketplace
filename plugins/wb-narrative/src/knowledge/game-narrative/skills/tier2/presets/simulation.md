# 模拟经营叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **核心体验** | 日常生活感 + NPC 关系 + 长期经营成就感 |
| **叙事节奏** | 缓慢、温馨、有节奏感（日/季节/年循环） |
| **叙事载体** | NPC 对话 + 节日事件 + 解锁内容 |
| **代表作** | 星露谷物语、动物森友会、牧场物语、模拟市民、珊瑚岛 |

---

## 叙事结构：生活节奏 × 关系线

```
[玩家的日常经营循环]
        │
  NPC 日常互动（每天）
        │
  特殊事件触发（节日/好感/解锁）
        │
  NPC 个人故事线（深度解锁）
        │
  大叙事弧线（若有主线）
```

---

## 核心叙事需求

### 角色设计 (C) ★★★
- 每个 NPC 有**日常生活感**：有工作/爱好/烦恼
- NPC 性格多样：内向/外向/热情/神秘等
- NPC 有**成长弧光**（随好感度提升/时间推移变化）
- 家庭/朋友关系网：NPC 之间也有关系，增加世界真实感

### 对话剧本 (D) ★★★
- 日常对话：轻松有趣，随季节/玩家进度变化
- 赠礼/互动反应：对应各种玩家行为的 NPC 反应
- 好感度触发对话：解锁更深层的角色故事
- 节日特殊对话：节日氛围专属台词

### 支线任务 (Q) ★★
- NPC 请求类任务：帮忙找东西/完成委托
- 解锁类任务：触发 NPC 故事线的条件
- 节日任务：节庆活动参与

### 世界观 (W) ★★
- 世界设定：乡村/小镇/海岛/宇宙等背景
- 镇子历史：有深度但不需要展开，供玩家探索
- 经济生态：为什么这个经营系统存在？

## 好感度叙事阶梯

| 好感等级 | 解锁内容 |
|---------|---------|
| 初识（0-2颗心） | 日常寒暄，基本信息 |
| 熟识（3-5颗心） | 个人爱好/背景，开始聊私事 |
| 朋友（6-8颗心） | 个人烦恼，邀请活动 |
| 好友/浪漫（9-10颗心） | 核心秘密，故事线解锁 |
| 特殊（满好感） | 结婚/BFF 事件，终章台词 |

---

## 主线弱化模式（模拟经营特有）

大多数模拟经营不需要强主线，但可以有：
- **引导主线**：教程式，前期引导玩家上手
- **镇子大事件**：1-2 个大型叙事事件推动世界变化
- **开放型结局**：没有明确"通关"，玩家自定义成就感

## 质量检查

- [ ] 每个 NPC 是否有"生活感"（不只是服务玩家的工具）？
- [ ] NPC 日常对话是否随季节/玩家进度变化？
- [ ] 好感度解锁是否层层深入（非一次全说完）？
- [ ] 节日事件是否有背景故事（非纯功能活动）？
- [ ] NPC 之间是否有关系网（增加世界真实感）？
