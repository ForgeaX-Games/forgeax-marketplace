# 战棋 RPG / SRPG 叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **核心体验** | 战棋战术 + 角色情感羁绊，战场既是挑战也是舞台 |
| **叙事特点** | 战前/战后叙事 + 角色支援对话 + 章节战役叙事 |
| **生死设计** | 角色可能永久死亡，增加叙事张力 |
| **代表作** | 火焰纹章系列、皇家骑士团、三角战略、战场女武神 |

---

## 叙事结构：章节战役制

```
[战前情报] → [战场事件/触发对话] → [战后叙事/角色互动]
     │                                       │
主线推进叙事（战争/政治）          支援对话（角色羁绊）
```

---

## 核心叙事需求

### 角色设计 (C) ★★★
- 角色即战棋单位，设计必须**兼顾叙事和战术意义**
- 每个角色需要：参战原因、战场哲学（如何看待战斗与杀戮）
- **支援系统**：角色两两之间的羁绊对话
- **生死设计**：是否支持永久死亡？角色死亡前的最后台词

### 剧情结构 (S) ★★★
- 战争题材：政治博弈、多方势力
- 每章开头：战前简报（叙事 + 战术情报）
- 每章结尾：战后叙事（胜利/失败的后果）
- 中期转折：盟友反目/背叛/真相揭露

### 对话剧本 (D) ★★★
- 主线对话：推进战争叙事
- 战场触发对话：特定条件触发（相遇/存活/击败某单位）
- 支援对话：C/B/A/S 级别羁绊对话
- 死亡台词：每个角色的最后一句话（若支持永久死亡）

### 世界观 (W) ★★★
- 政治地图：势力版图、边界争议
- 战争背景：为什么打仗？利益/信仰/复仇？
- 各方立场：无绝对善恶的多方战争

## 支援对话系统模板

```markdown
## [角色A] × [角色B] 支援对话

**C 级** — 相互认识，初步了解
> A：[开场白，略带距离感]
> B：[回应，揭示一点角色特质]
> A：[反应，建立连接]

**B 级** — 产生信任
> [一个共同经历/对话，揭露角色背景]

**A 级** — 深度羁绊
> [核心矛盾/互相理解，揭露人物最深层的东西]

**S 级**（若有浪漫路线）— 承诺同行
> [情感表达，两人决定彼此守护]
```

```markdown
**关键台词**：[主角/将领的出战宣言]

### 战场触发事件
- [触发条件]：[触发对话/特殊事件]
- [触发条件]：[...]

### 战后叙事
**胜利走向**：[战胜后的叙事后果]
**失败走向**（若支持）：[失败后的叙事]
**牺牲处理**（若支持永久死亡）：[阵亡角色的告别叙事]
```

## 战争叙事设计原则

1. **战争无赢家**：展示战争的代价，而非纯粹的英雄主义
2. **敌人也是人**：精英敌方单位有背景故事
3. **战术即选择**：战术决策具有叙事意义（放弃某翼保护某人）
4. **牺牲有重量**：永久死亡机制下，每次牺牲都改变叙事

## 质量检查

- [ ] 战争是否呈现了多方的合理动机（非单纯正义vs邪恶）？
- [ ] 每个角色是否有战场哲学（为什么而战）？
- [ ] 支援对话是否层层深入（C/B/A/S 递进）？
- [ ] 战前/战后叙事是否与战术决策有连接？
- [ ] 角色死亡（若有）是否有充分的叙事重量？
