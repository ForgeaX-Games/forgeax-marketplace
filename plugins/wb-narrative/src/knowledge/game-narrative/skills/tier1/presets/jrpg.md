# JRPG / 日式 RPG 叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **核心体验** | 英雄传记 + 同伴羁绊 + 拯救世界 |
| **叙事模式** | 线性主线 + 支线个人线，同伴成为家人 |
| **战斗叙事** | 战斗是情感的延伸，BOSS 是故事的高潮 |
| **代表作** | 最终幻想系列、勇者斗恶龙、女神异闻录、仙剑奇侠传、古剑奇谭 |

---

## 叙事结构：英雄之旅变体

```
第一幕：平凡世界
  → 主角日常 → 触发事件（家园被毁/召唤/使命揭晓）

第二幕：踏上旅程
  → 组队（同伴逐一加入）→ 冒险（地区/副本叙事）
  → 内部危机（同伴矛盾/背叛/牺牲）
  → 最低谷（失去一切，重新站起）

第三幕：最终决战
  → 真相揭露（幕后黑手/世界真相）
  → 团队凝聚 → 决战
  → 结局（拯救/代价/新的起点）
```

## 核心叙事需求

### 世界观 (W) ★★★
- 世界圣经：地图大陆 + 势力阵营 + 魔法/技术体系
- 神话/历史：古代战争、神明、预言等背景层
- 地区叙事：每个游历地区有独立历史和冲突

### 角色设计 (C) ★★★
- **主角**：普通人出发 → 命中注定的成长弧光
- **核心同伴**：4-8 人，每人有独立个人线和弧光
- **反派**：有理由、有背景，不是单纯的"坏人"——最好是主角的镜像
- **导师/父亲形象**：通常中期牺牲，激发主角成长

**同伴三要素**：
1. 与主角的**情感纽带**（为什么跟你走）
2. 个人**创伤/秘密**（个人线的核心）
3. **战斗哲学**（怎么看待力量/战斗/牺牲）

### 剧情结构 (S) ★★★
- 三幕式结构 + 英雄之旅节拍
- 每个章节有本章节的小高潮
- 全局伏笔：第一章埋下，三分之二处揭示，结局落地

### 对话剧本 (D) ★★★
- 过场对话：推进剧情、揭露信息
- 队伍对话（Skit/庭院对话）：角色日常互动，轻松有趣
- 战前/战后台词：Barks，简短有力
- 关键台词：每个角色需要 1-3 句"名场面台词"

### 支线任务 (Q) ★★
- 角色个人线（必须）：每个核心同伴 1 条深度支线
- 地区支线：对应地区冲突的短故事
- 探索支线：隐藏内容、隐藏 BOSS 前置故事

## 反派设计模板

```markdown
## [反派名] — 主要对立角色

**表面动机**：[他宣称的目标，让人觉得有道理]
**真实目标**：[底层驱动，通常源于创伤]
**与主角的关系**：[为什么他们是镜像/对立面]
**转化点**（可选）：[是否有可能被救赎/理解他的那一刻]

**为什么他不是单纯的坏人**：
[用 2-3 句话说明玩家应该理解（不是认同）他的动机]
```

## 同伴个人线结构

```
触发条件：主角亲密度 / 章节进度到达某节点
↓
前置：日常对话暗示个人创伤/秘密
↓
触发事件：外部事件揭开伤疤
↓
内部冲突：同伴面对选择（逃避 vs 直面）
↓
关键场景：主角介入，支持/理解/一起面对
↓
解决：创伤化解，角色成长
↓
后续：与主角关系加深，战斗技能/对话变化
```

## 质量检查

- [ ] 反派有可以理解（不必认同）的动机？
- [ ] 每个同伴有独立的弧光和创伤？
- [ ] 章节有独立的小高潮和叙事完整性？
- [ ] 关键伏笔在第一幕埋下，最终有回收？
- [ ] 团队对话（Skit）与主线剧情形成对比节奏？
- [ ] 最终决战前有充分的情感积累？
