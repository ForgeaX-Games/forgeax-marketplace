# CRPG / 西式 RPG 叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **核心体验** | 玩家 Agency 最大化——"我的角色，我的故事" |
| **叙事模式** | 网状分支 + 多解法 + 道德灰色地带 |
| **内容形式** | 对话树 + 技能检定 + 世界状态追踪 |
| **代表作** | 博德之门3、神界原罪2、辐射系列、异域镇魂曲、Disco Elysium |

---

## 叙事结构：网状 + 收敛

```
[主线主干]
    │
    ├─ 派系A路线 ─────────────────┐
    │  └─ A1/A2 子选择            │
    │                            ├─ 收敛点（幕末）── 主线继续
    ├─ 派系B路线 ─────────────────┤
    │  └─ B1/B2/B3 子选择         │
    │                            │
    └─ 独行路线（无派系）──────────┘
```

**每一幕结束**必须有收敛点——分支在此汇合，但带着不同后果。

## 核心叙事需求

### 世界观 (W) ★★★
- **极深 Lore 体系**：历史层叠（远古/中古/近代/现在）
- 派系政治：每个派系有合理动机，无绝对善恶
- 规则体系：魔法/科技/宗教/种族关系必须内在一致
- 世界圣经应支持**持续扩展**

### 角色设计 (C) ★★★
- **同伴设计**（6-10人）：
  - 每人有独立世界观和价值立场
  - 会对玩家选择做出**反应**（认可/不认可/离队）
  - 有个人任务线（深度支线）
- **NPC 设计**：支持多种互动路径，有记忆系统
- **玩家角色**：可自定义，但部分背景影响剧情

### 剧情结构 (S) ★★★
- **多幕制**（3-5幕），每幕有核心冲突
- **多结局**：5-15 个，由变量累积触发
- **道德光谱**：玩法允许扮演英雄、混乱中立、甚至反派
- **旁白系统**（部分 CRPG 特有）：环境描写 + 技能检定结果

### 对话剧本 (D) ★★★
- **技能检定对话**：成功/失败版本各需编写
- **分支对话**：平均每段对话 3-8 个选项
- **态度变化对话**：友好/中立/敌对版本
- **失败叙事**：CRPG 的特点，失败也要有故事

### 支线任务 (Q) ★★★（CRPG 核心内容）
- 独立支线数量：至少 30-50 条（完整游戏）
- 每条支线应有独立的叙事弧线，非仅"杀X个怪"
- 派系任务线：跟随某派系的专属支线
- 随机/程序化事件模板

### 物品叙事 (I) ★★★
- **装备 Lore**：每件重要物品有来历故事
- **碎片化历史传递**：通过物品拼凑世界历史

### Lore 碎片 (L) ★★★
- 图书馆/读物系统
- 碑文/壁画
- NPC 的"传说/谣言"系统

## 对话树设计模式

### 技能检定节点
```
[对话触发]
  │
  ├─ 感知检定 DC 15
  │   ├─ 成功 → 发现隐藏信息，获得额外选项
  │   └─ 失败 → 正常对话，缺少信息
  │
  ├─ 说服检定 DC 12
  │   ├─ 成功 → NPC 妥协，信息披露
  │   └─ 失败 → NPC 拒绝，可能战斗/离开
  │
  └─ [通用选项] → 无论检定结果均可选
```

### 态度追踪节点
```
对话选择积累 → 变量 npc_stance: hostile/neutral/friendly
  → 影响后续交互、任务、战斗/合作
```

```markdown
**为什么有问题**：[他们的弱点/极端/代价]

**对玩家的要求**：
- 加入条件：
- 忠诚任务：
- 背叛代价：

**与其他派系的冲突**：
- vs [派系B]：因为 [核心矛盾]
```

## 质量检查

- [ ] 派系设计是否无绝对善恶，每方都有合理动机？
- [ ] 技能检定对话的成功/失败版本均已编写？
- [ ] 失败路径是否也有有趣的叙事？
- [ ] 支线任务是否有独立叙事弧线（非仅杀怪收集）？
- [ ] 同伴是否会对玩家选择做出合理反应？
- [ ] 多结局触发条件是否清晰且可达？
- [ ] 物品 Lore 是否形成一致的历史拼图？
