# 开放世界 RPG 叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **核心体验** | 探索即叙事——玩家在世界中发现故事，而非被故事引导 |
| **叙事模式** | 碎片化 + 空间叙事 + 主线弱引导 |
| **内容形式** | 环境叙事为主，对话/文档/场景考古为辅 |
| **代表作** | 塞尔达传说、上古卷轴5、原神、太吾绘卷、古剑奇谭OL |

---

## 叙事结构：星系模型

```
[中心主线——世界危机]
        │
  ┌─────┼─────┐
  │     │     │
地区A  地区B  地区C    ← 每个地区有完整的本地叙事
  │     │     │
支线群  支线群  支线群   ← 探索触发，非线性
  │
碎片叙事（物品/环境/NPC 对话）
```

主线是**指南针**，地区叙事才是**主要内容体量**。

---

## 核心叙事需求

### 世界观 (W) ★★★
- 世界圣经：可扩展的模块化世界观
- 每个地区有**独立历史、文化、冲突**
- 远古历史（供玩家考古发现的"洋葱层"）
- 世界规则：魔法/修炼/科技必须内在一致

### 环境叙事 (E) ★★★（开放世界核心）
- 场景考古：废墟/古战场/遗迹背后的故事
- 视觉叙事：建筑风格/植被/残骸传递历史
- NPC 行为叙事：路人对话/日程传递世界生活感

### 物品叙事 (I) ★★★（开放世界核心）
- 每件命名装备/道具有 Lore 文本
- 收集品（图鉴/典籍/信件）拼凑碎片历史
- 道具叙事优先于长篇文档

### 角色设计 (C) ★★★
- **主线角色**：少而精，支撑世界危机弧线
- **地区 NPC**：为各地区冲突服务，不需要全局登场
- **隐藏角色**：探索奖励，有完整背景故事

### 支线任务 (Q) ★★
- 类型多样：人物故事线 / 地区冲突线 / 寻宝/解谜
- 支线应和主线**主题呼应**（微缩版主题）
- 支线结局有多种可能（开放世界必须支持多解法）

### Lore 碎片 (L) ★★★
- 日记/信件/典籍系统
- 考古/遗迹调查
- 地区传说/民间故事

## 地区叙事设计模板

```markdown
## [地区名] 叙事设计

**地区定位**：[在主线中的地理/政治位置]

**地区历史**（3层洋葱）：
1. 远古层：[玩家通过考古才能发现的历史]
2. 近代层：[本地 NPC 普通人知道的历史]
3. 当下层：[玩家抵达时正在发生的冲突]

**地区核心冲突**：
[用一句话描述当地的矛盾/张力]

**关键地标**：
- [地标A]：场景考古揭示 [历史片段]
- [地标B]：视觉叙事暗示 [世界真相]

**地区主要 NPC**：
- [NPC1]：[动机/秘密]
- [NPC2]：[与地区冲突的关系]

**地区主要支线**（2-4条）：
- [支线1]：[一句话概述]
- [支线2]：[一句话概述]
```

## 空间叙事设计原则

1. **故事在场景中，不在文字框里**：优先用场景传达，对话补充
2. **玩家发现优于 NPC 告知**：让玩家自己找到线索
3. **每个探索点有奖励感**：发现场景背后的故事本身就是奖励
4. **时间层叠**：同一地点展示不同时代的状态（现在/废墟/幻象）

---

## 主线弱化模式

开放世界主线应该是**吸引而非驱动**：

| 主线强度 | 描述 | 适用 |
|---------|------|------|
| 强弱型（塞尔达式） | 主线仅提供方向，探索是主要内容 | 探索优先游戏 |
| 均衡型（原神式） | 主线与探索并重，章节制推进 | 长线运营游戏 |
| 叙事型（上古卷轴式） | 主线有完整弧线，开放世界丰富侧面 | 单机大作 |

## 质量检查

- [ ] 地区是否有独立的历史/冲突（不只是战斗地图）？
- [ ] 环境叙事是否足以让玩家理解场景背后的故事？
- [ ] 物品 Lore 是否形成连贯的世界拼图？
- [ ] 主线是否是"吸引"而非强制玩家？
- [ ] 支线结局是否支持多种解法？
- [ ] 隐藏内容是否有完整背景（值得探索的奖励感）？
