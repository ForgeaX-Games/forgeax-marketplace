# 视觉小说 / AVG 叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **核心体验** | 阅读/观看叙事，情感沉浸，角色羁绊 |
| **Agency 类型** | 以 Expressive 为主，Branching 控制结局走向 |
| **内容形式** | 立绘 + 背景 + 对话框 + BGM |
| **代表作** | 星之梦、寒蝉鸣泣之时、月姬、Clannad、VA-11 Hall-A |

---

## 叙事结构模式

### 模式 A：线性主干 + 结局分支
```
主线章节（固定）
    ↓
关键选择点（每章 1-2 个）
    ↓
角色路线（3-6 条）
    ↓
对应结局（True End / Good End / Bad End）
```

### 模式 B：时间循环 / 惊奇叙事
```
表层故事（误导性）
    ↓
关键信息点（逐渐揭露）
    ↓
真相章节（颠覆认知）
    ↓
True Route 解锁
```

### 模式 C：群像 + 全员攻略
```
共通路线（世界观建立）
    ↓
角色个人线（各自独立）
    ↓
真实路线（需攻略条件）
```

## 核心叙事需求

### 世界观 (W) ★★★
- 背景设定（时代/地域/氛围）
- 核心规则（魔法/科幻/现实等特殊设定）
- 必须在共通路线开头自然嵌入，不过度Info-dump

### 角色设计 (C) ★★★
- 主要可攻略角色：3-6 个（男/女/无性别）
- 每个角色必须有：外貌描述、性格核心、Want（表面欲求）、Need（真实需求）
- 角色弧光设计：开场缺陷 → 转折点 → 成长/改变
- 关系初始状态与变化轨迹

### 剧情结构 (S) ★★★
- 共通路线：占总字数 20-35%
- 个人路线：每条占 15-25%
- 结局设计：至少 Good End + Bad End，True End 需要特殊条件
- Pacing 节奏：日常（舒缓）↔ 高潮（紧张）交替

### 对话剧本 (D) ★★★
- 台词必须体现角色个性（禁止角色同质化）
- 日常对话：轻松、有趣、建立信任
- 关键对话：情感张力，高信息密度
- 内心独白：主角视角，引导玩家共情

### UI 文案 (U) ★★
- 选择肢文案：简洁，体现选择权重（"直接说出来" vs "先听他说"）
- 系统提示：沉浸式（不打破第四堵墙）

## Agency 设计模板

### Expressive Agency（语气/态度选择）
```
情境：[角色]问你是否要一起去...
选项：
  A. "当然！我早就想去了" → 态度：热情
  B. "好吧...如果你坚持的话" → 态度：犹豫
  C. "今天可以，但不是习惯就好" → 态度：理性
结果：不改变剧情走向，影响[角色]对你的印象值
```

### Branching Agency（剧情分叉）
```
情境：[角色]陷入危机，你需要选择...
选项：
  A. 直接站出来帮助 → 走向：好感+信任，但暴露自己
  B. 暗中调查先弄清楚 → 走向：风险降低，但延误
  C. 告知第三方处理 → 走向：安全，但[角色]心寒
结果：直接影响角色路线走向和结局触发条件
```

|---------|------|---------|
| 日常互动场景 | 建立关系、角色塑造 | 占比 40-50% |
| 剧情推进场景 | 信息揭露、矛盾升级 | 占比 25-35% |

| 结局场景 | 情感收束、主题升华 | 占比 5-10% |
