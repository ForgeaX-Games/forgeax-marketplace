# 《星星归乡记》

## 一句话
三颗星星手拉手，就能飞回天上的家！

## 故事
夜空突然变得黯淡，原来是调皮的风把星星们吹散到了人间。它们落在树梢、花丛、屋顶，失去了回家的力量。

你是月亮姐姐派来的星星向导。只要让三颗同色的星星靠在一起，它们就能重新发光，手拉手飞回天上。从后院开始，穿过田野，翻过山顶，一路收集散落的星星。

当最后一颗星星归位，整片夜空将重新闪耀。

## 玩法映射
| 元素 | 叙事包装 |
|-----|---------|
| 你是 | 月亮姐姐的星星向导 |
| 核心行动 | 让三颗同色星星靠在一起 |
| 收集/消除 | 星星手拉手飞回天上 |
| 失败意味着 | 步数用尽，星星继续流浪 |
| 最终目标 | 让夜空重新闪耀 |

## 关卡拓展
- 场景线：后院→田野→森林→山顶→云端
- 难度线：星星更分散/步数更少/特殊障碍（云雾遮挡）
- 最终章：收集最后的北极星，点亮整片银河

点点决定独自出发。它用微弱的光照亮前路，把散落在树洞、花丛、蘑菇下的光点一颗颗找回来。每找到一颗，身后就多一只苏醒的伙伴，队伍越来越长——但森林小路很窄，一不小心就会撞上自己的队伍。

**品类识别**：tower ✅ | **模式**：融合（按用户主题方向）

# 《植物守护花园》

## 一句话
僵尸来袭，用植物大军守住你的花园！

## 故事
午夜的钟声敲响，地底的僵尸们苏醒了。它们闻到了花园里向日葵的香气，排着队从墓地涌来。铁桶僵尸、撑杆僵尸、巨人僵尸……一波比一波凶猛。

你是花园的主人。种下豌豆射手挡住第一波，向日葵提供阳光能量，坚果墙拖延时间。每一株植物都是你的战士，每一寸土地都不能让出。从前院到后院，从白天到黑夜，僵尸永不停歇。
