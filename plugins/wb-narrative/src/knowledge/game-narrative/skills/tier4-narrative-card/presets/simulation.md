# 模拟经营品类预设

## 组合逻辑

- [主角] 开始经营 [店铺/农场/事业]
- 通过 [服务/生产] 满足 [顾客需求]，赚取资源升级
- 失败 = 顾客不满意，生意惨淡
- 成功 = 从小店发展成 [最终形态]

**故事必须解释**：
- 为什么要满足顾客？（赚钱/口碑/梦想）
- 最终想要达成什么样的事业？

## 要素库

### 经营对象类型
| 类型 | 变体示例 |
|-----|---------|
| 餐饮类 | 餐厅/咖啡馆/甜品店/快餐店/酒吧 |
| 服务类 | 美容院/宠物店/医院/酒店/健身房 |
| 生产类 | 农场/工厂/矿场/渔场/牧场 |
| 零售类 | 超市/服装店/花店/书店/玩具店 |
| 特殊类 | 动物园/游乐园/水族馆/博物馆 |

### 核心行动
| 类型 | 变体示例 |
|-----|---------|
| 制作类 | 烹饪/调制/制造/种植 |
| 服务类 | 接待/服务/治疗/照顾 |
| 管理类 | 进货/定价/装修/招聘 |

### 顾客/需求类型
| 类型 | 变体示例 |
|-----|---------|
| 人物顾客 | 普通顾客/VIP/挑剔顾客/急性子 |
| 动物顾客 | 宠物/野生动物/神奇生物 |
| 订单需求 | 简单订单/复杂订单/限时订单 |

### 资源类型
| 类型 | 变体示例 |
|-----|---------|
| 金钱 | 金币/钻石/星星 |
| 材料 | 食材/原料/零件 |
| 声望 | 好评/星级/口碑 |
| 经验 | 等级/技能点 |

### 升级路径
| 类型 | 示例 |
|-----|------|
| 规模升级 | 更大的店面/更多的设施 |
| 品种升级 | 更多的产品/更丰富的菜单 |
| 效率升级 | 更快的速度/更高的品质 |
| 员工升级 | 更多/更熟练的员工 |

### 场景递进
| 发展线 | 示例 |
|-------|------|
| 规模线 | 小摊→小店→大店→连锁→帝国 |
| 地域线 | 村庄→城镇→城市→全国→全球 |
| 品质线 | 普通→精品→高端→顶级→传说 |

## 组合示例

### 示例1：美味餐厅
- 经营对象：餐厅
- 核心行动：烹饪美食、服务顾客
- 顾客：各种食客（普通/VIP/挑剔/急性子）
- 资源：金币、食材、好评
- 升级：菜单→厨具→装修→员工→分店
- 场景：小餐车→街边小店→高档餐厅→连锁品牌
- 结局：成为米其林星级餐厅

### 示例2：开心农场
- 经营对象：农场
- 核心行动：种植、收获、售卖
- 顾客：收购商、市场订单
- 资源：金币、种子、农具
- 升级：田地→作物种类→设施→动物→加工厂
- 场景：小菜园→家庭农场→大型农庄→农业帝国
- 结局：建成最美丽富饶的农场

### 示例3：宠物美容院
- 经营对象：宠物店
- 核心行动：洗澡、美容、照顾宠物
- 顾客：各种宠物和主人
- 资源：金币、美容工具、好评
- 升级：工具→服务项目→店面→员工
- 场景：小工作室→宠物店→宠物乐园→宠物王国
- 结局：成为宠物们最爱的美容天堂

### 示例4：甜品梦工厂
- 经营对象：甜品店/蛋糕店
- 核心行动：制作甜品、装饰蛋糕
- 顾客：甜品爱好者、订制订单
- 资源：金币、食材、配方
- 升级：配方→设备→装修→员工→分店
- 场景：家庭厨房→小甜品店→甜品工坊→甜品帝国
- 结局：成为甜品界的传奇大师

### 示例5：梦想动物园
- 经营对象：动物园
- 核心行动：照顾动物、建设展区、接待游客
- 顾客：游客、动物
- 资源：金币、动物、设施
- 升级：动物种类→展区→设施→员工→新区域
- 场景：小型动物角→动物园→野生动物园→世界级动物王国
- 结局：建成动物们的天堂，游客们的乐园

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 规模 | 店面/设施/员工 |

| 挑战 | 限时/特殊顾客/高峰期 |
| 目标 | 营业额/好评/星级 |
