# 射击品类预设

## 组合逻辑

- [入侵者] 大举进攻 [守护对象]
- [主角] 驾驶 [载具/武器] 迎击，一波波消灭敌人
- 失败 = 被击落 / 守护对象被摧毁
- 成功 = 击败最终Boss，守护成功

**故事必须解释**：
- 入侵者为什么要进攻？想要什么？
- 为什么只有主角能阻止它们？

## 要素库

### 主角类型
| 类型 | 变体示例 |
|-----|---------|
| 战机 | 战斗机/宇宙飞船/直升机/飞艇 |
| 角色 | 小战士/宇航员/魔法师/小精灵 |
| 载具 | 坦克/潜艇/机甲/飞毯 |
| 生物 | 喷火龙/战斗蜜蜂/攻击海豚 |

### 入侵者阵营
| 类型 | 变体示例（从弱到强） |
|-----|---------------------|
| 外星人 | 侦察机→战斗机→母舰→外星领主 |
| 机器人 | 小型机→中型机→重型机→机械霸主 |
| 怪物 | 小怪→中怪→大怪→魔王 |
| 虫族 | 虫兵→虫将→虫后→虫皇 |
| 海盗 | 小船→战船→旗舰→海盗王 |

### 守护对象
| 类型 | 示例 |
|-----|------|
| 星球 | 地球/家园星/和平星球 |
| 王国 | 天空王国/海底王国/精灵王国 |
| 家园 | 村庄/城市/基地/蜂巢 |
| 抽象 | 和平/光明/希望 |

### 武器/攻击方式
| 类型 | 变体示例 |
|-----|---------|
| 子弹类 | 普通弹→散弹→追踪弹→激光 |
| 魔法类 | 火球→冰锥→雷电→元素风暴 |
| 特殊类 | 炸弹/护盾/召唤僚机/必杀技 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 太空线 | 大气层→近地轨道→小行星带→敌方母星 |
| 天空线 | 云层→高空→平流层→太空边缘 |
| 海洋线 | 浅海→深海→海沟→海底基地 |
| 幻想线 | 森林→火山→冰川→魔王城 |

### 道具/升级
| 类型 | 作用 |
|-----|------|
| 武器升级 | 火力增强/弹道变化 |
| 防护道具 | 护盾/无敌/回血 |
| 特殊道具 | 清屏炸弹/时间减速/僚机 |

## 组合示例

### 示例1：雷电战机（经典）
- 主角：地球联盟战斗机
- 入侵者：外星机械军团
- 守护对象：地球
- 武器：激光炮、导弹、护盾
- 场景：城市上空→海洋→沙漠→敌方基地→母舰
- Boss：区域指挥官→舰队司令→外星领主
- 结局：击败外星领主，地球恢复和平

### 示例2：小蜜蜂保卫蜂巢
- 主角：勇敢的小蜜蜂
- 入侵者：害虫军团（蚜虫→甲虫→黄蜂→螳螂）
- 守护对象：蜂巢和蜂后
- 武器：蜂刺、花粉弹、召唤蜂群
- 场景：花园→果园→森林→蜂巢入口
- Boss：各区域虫王
- 结局：击退害虫，蜂巢安全

### 示例3：魔法小精灵
- 主角：光明小精灵
- 入侵者：黑暗生物（暗影→幽灵→恶魔→黑暗领主）
- 守护对象：精灵王国的光明之树
- 武器：光球、星尘、彩虹光线
- 场景：精灵森林→迷雾沼泽→黑暗山谷→魔王城堡
- Boss：各区域黑暗守卫
- 结局：驱散黑暗，光明重现

### 示例4：海底守护者
- 主角：守护海豚/潜艇
- 入侵者：污染怪物（垃圾怪→油污怪→毒素怪→污染之王）
- 守护对象：海底王国和珊瑚礁
- 武器：声波炮、泡泡弹、净化光线
- 场景：珊瑚礁→沉船区→深海→污染源
- Boss：各区域污染首领
- 结局：净化海洋，生物回归

### 示例5：糖果战机
- 主角：糖果飞船
- 入侵者：蛀牙军团（小蛀虫→蛀牙兵→蛀牙将军→蛀牙魔王）
- 守护对象：糖果王国
- 武器：糖果炮、棒棒糖导弹、彩虹激光
- 场景：糖果镇→巧克力山→棉花糖云→蛀牙巢穴
- Boss：各区域蛀牙首领
- 结局：消灭蛀牙军团，糖果王国欢庆

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 敌人 | 种类/密度/弹幕模式 |
| 地形 | 障碍物/可破坏物 |
| Boss | 攻击模式/弱点/变身 |
| 挑战 | 限时/无伤/特定武器 |
