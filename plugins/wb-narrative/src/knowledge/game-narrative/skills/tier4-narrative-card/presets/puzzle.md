# 解谜品类预设

## 组合逻辑

- [目标/出口] 被 [障碍/机关] 挡住
- 通过 [操作] 移动/触发机关，一步步解开
- 失败 = 被困住，无法前进
- 成功 = 解开谜题，到达目标

**故事必须解释**：
- 为什么目标被挡住了？（封印/机关/迷宫）
- 为什么这样操作就能解开？

## 要素库

### 解谜类型
| 类型 | 代表玩法 |
|-----|---------|
| 推箱子 | 把箱子推到指定位置 |
| 华容道 | 移动方块让目标物通过 |
| 滑块谜题 | 滑动方块拼成图案 |
| 路径规划 | 找到从起点到终点的路 |
| 机关解谜 | 按正确顺序触发机关 |

### 主角/操作对象
| 类型 | 变体示例 |
|-----|---------|
| 角色类 | 小人/机器人/精灵/小动物 |
| 物品类 | 箱子/方块/钥匙/光线 |
| 抽象类 | 光点/能量/数据 |

### 障碍/机关类型
| 类型 | 变体示例 |
|-----|---------|
| 阻挡类 | 墙壁/石块/箱子/门 |
| 机关类 | 开关/按钮/压力板/齿轮 |
| 陷阱类 | 坑洞/尖刺/激光/火焰 |
| 移动类 | 移动平台/传送门/电梯 |

### 目标类型
| 类型 | 示例 |
|-----|------|
| 到达目标 | 让角色/物品到达指定位置 |
| 配对目标 | 让所有物品归位 |
| 连接目标 | 连通起点和终点 |
| 解锁目标 | 打开门/解除封印 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 建筑线 | 仓库→工厂→实验室→神秘基地 |
| 遗迹线 | 入口→走廊→大厅→宝藏室 |
| 自然线 | 森林→洞穴→地下→神殿 |
| 抽象线 | 简单→复杂→高级→大师 |

### 解谜机制（为什么这样能解开）
| 类型 | 叙事解释 |
|-----|---------|
| 物理规则 | 推动、滑动、重力、反射 |
| 魔法规则 | 触发魔法阵、能量流动 |
| 机械规则 | 齿轮联动、开关控制 |
| 逻辑规则 | 正确的顺序和组合 |

## 组合示例

### 示例1：仓库管理员（推箱子）
- 主角：仓库小工
- 解谜类型：把箱子推到指定位置
- 障碍：墙壁、其他箱子
- 目标：所有箱子归位
- 场景：小仓库→大仓库→迷宫仓库→超级仓库
- 机制：箱子只能推不能拉
- 结局：成为最优秀的仓库管理员

### 示例2：救出小车（华容道）
- 主角：红色小车
- 解谜类型：移动其他车让红车开出停车场
- 障碍：其他车辆
- 目标：红车从出口离开
- 场景：小停车场→大停车场→地下车库→超级停车场
- 机制：车只能前后移动
- 结局：红色小车成功逃出拥挤的停车场

### 示例3：精灵回家路（路径规划）
- 主角：迷路的小精灵
- 解谜类型：找到从起点到家的路
- 障碍：墙壁、陷阱、怪物
- 目标：安全到达家门口
- 场景：森林→迷宫→洞穴→精灵村
- 机制：避开危险，收集钥匙开门
- 结局：小精灵平安回到温暖的家

### 示例4：古墓探险家（机关解谜）
- 主角：小探险家
- 解谜类型：按正确顺序触发机关
- 障碍：机关门、陷阱、谜题石板
- 目标：打开宝藏室的门
- 场景：古墓入口→走廊→大厅→宝藏室
- 机制：观察线索，按顺序触发
- 结局：打开宝藏室，发现传说中的宝藏

### 示例5：光线大师（反射解谜）
- 主角：光线/镜子
- 解谜类型：调整镜子角度让光线到达目标
- 障碍：墙壁、障碍物
- 目标：光线照亮所有水晶
- 场景：暗室→神殿→水晶洞→光明圣殿
- 机制：光线遇镜反射，遇障碍阻挡
- 结局：光芒照亮整个神殿

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|

| 元素 | 物品数量/种类 |
| 机制 | 新机关/新规则/组合 |
| 目标 | 步数/时间/完美通关 |
