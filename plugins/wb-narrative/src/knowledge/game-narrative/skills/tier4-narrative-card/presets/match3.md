# 三消品类预设

## 组合逻辑

- [世界/被拯救者] 陷入 [困境]
- 三个 [消除元素] 连在一起会 [魔法效果]，从而解救一点点
- 失败 = 步数用尽，被困者继续沉睡
- 成功 = 所有人获救，世界恢复光明

**故事必须解释**：
- 为什么三个一样的连在一起会消失/产生效果？
- 消除和「解救」之间的因果关系是什么？

## 要素库

### 困境类型（为什么要消除）
| 类型 | 叙事解释 |
|-----|---------|
| 黑暗笼罩 | 三颗宝石聚齐 → 爆发光芒驱散黑暗 |
| 封印困住 | 三个相同配对 → 解除封印释放被困者 |
| 订单任务 | 三个同类凑齐 → 完成一组订单/任务 |
| 魔法收集 | 三颗聚齐 → 收集魔力/能量 |
| 污染清理 | 三个配对 → 净化一片区域 |

### 被拯救者（目标对象）
| 类型 | 变体示例 |
|-----|---------|
| 生物类 | 精灵/小动物/村民/公主 |
| 区域类 | 村庄/森林/王国/星球 |
| 物品类 | 星星/宝石/糖果/花朵 |
| 抽象类 | 光明/希望/记忆/梦想 |

### 消除元素（三消的对象）
| 类型 | 变体示例 |
|-----|---------|
| 宝石类 | 红宝石/蓝宝石/绿宝石/钻石 |
| 糖果类 | 棒棒糖/软糖/巧克力/饼干 |
| 自然类 | 花朵/水果/树叶/星星 |
| 生物类 | 小动物头像/精灵/表情 |
| 抽象类 | 符文/魔法球/能量块 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 王国线 | 边境村庄→森林→山谷→城镇→王宫 |
| 自然线 | 花园→森林→雪山→沙漠→火山 |
| 深度线 | 地表→洞穴→地下湖→熔岩层→核心 |
| 甜蜜线 | 糖果镇→巧克力河→棒棒糖森林→蛋糕山→糖果城堡 |

### 反派/障碍（可选）
| 类型 | 作用 |
|-----|------|
| 女巫/魔王 | 制造困境的源头，最终Boss |
| 黑暗魔法 | 笼罩世界的威胁 |
| 时间限制 | 太阳落山前/魔法消散前 |
| 特殊方块 | 冰块/铁链/石头（需要多次消除） |

## 组合示例

### 示例1：宝石王国保卫战
- 困境：黑暗魔法笼罩，居民沉睡
- 被拯救者：王国的村庄和居民
- 消除元素：魔法宝石
- 机制解释：三颗宝石聚齐爆发光芒驱散黑暗
- 场景：边境村庄→森林村→山顶村→城镇→王宫

### 示例2：糖果精灵救援
- 困境：女巫把精灵变成糖果封印
- 被拯救者：糖果森林的小精灵
- 消除元素：彩色糖果
- 机制解释：三个同色糖果配对解除封印
- 场景：森林入口→糖果沼泽→棒棒糖山→巧克力洞穴→女巫城堡

### 示例3：星星送回家
- 困境：星星从天上掉落散落各地
- 被拯救者：月亮姐姐的星星宝宝
- 消除元素：彩色星星
- 机制解释：三颗同色星星手拉手飞回天上
- 场景：后院→田野→森林→山顶→云端

### 示例4：海底珊瑚修复
- 困境：污染让珊瑚失去颜色
- 被拯救者：海底的珊瑚礁和海洋生物
- 消除元素：彩色珊瑚/贝壳
- 机制解释：三个同色聚齐恢复珊瑚生机
- 场景：浅海→珊瑚礁→海底森林→深海→海底宫殿

### 示例5：梦境修复师
- 困境：噩梦入侵，美梦碎片散落
- 被拯救者：孩子们的美梦
- 消除元素：梦境碎片（云朵/星星/彩虹）
- 机制解释：三个碎片聚齐修复一段美梦
- 场景：卧室→云端→彩虹桥→星空→梦境核心
