# 合成/2048品类预设

## 组合逻辑

- 传说中的 [最终目标物] 拥有神奇力量
- 两个相同的 [低级物] 碰撞会 [合成] 成更高级的
- 失败 = 空间被填满，无法继续合成
- 成功 = 终极目标物诞生，愿望成真

**故事必须解释**：
- 为什么两个一样的碰在一起会变成更高级的？
- 最终目标物为什么重要/有什么神奇之处？

## 要素库

### 合成逻辑（为什么A+A=B）
| 类型 | 叙事解释 |
|-----|---------|
| 进化 | 两个相同生物碰撞 → 基因融合进化成更高形态 |
| 合体 | 两个相同怪物合体 → 变成更大更强的怪物 |
| 成长 | 两个相同植物融合 → 长成更大的植物 |
| 升级 | 两个相同数字/等级碰撞 → 能量叠加升级 |
| 锻造 | 两个相同材料合并 → 锻造成更高级材料 |

### 合成对象（从低到高）
| 类型 | 进化链示例 |
|-----|-----------|
| 数字 | 2→4→8→16→32→64→128→256→512→1024→2048 |
| 水果 | 葡萄→樱桃→橙子→柠檬→猕猴桃→苹果→梨→桃子→菠萝→椰子→西瓜 |
| 生物 | 蛋→幼崽→少年→成年→王者→神兽 |
| 建筑 | 帐篷→小屋→房子→别墅→城堡→宫殿 |
| 武器 | 木剑→铁剑→钢剑→银剑→金剑→神剑 |

### 最终目标（合成的终点）
| 类型 | 示例 |
|-----|------|
| 生物类 | 神兽/龙王/凤凰/守护神 |
| 物品类 | 神器/圣杯/世界树果实/宇宙之心 |
| 数字类 | 2048/4096/传说数字 |
| 建筑类 | 天空之城/神殿/世界奇观 |

### 空间限制（失败条件）
| 类型 | 叙事解释 |
|-----|---------|
| 棋盘满 | 世界被占满，没有空间继续合成 |
| 入侵者 | 入侵者太多挤满了空间 |
| 时间 | 魔法消散前必须完成 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 等级线 | 新手村→普通区→精英区→传说区→神话区 |
| 区域线 | 村庄→城镇→城市→王国→大陆 |
| 难度线 | 4×4棋盘→5×5棋盘→6×6棋盘→特殊形状 |

## 组合示例

### 示例1：数字王国（经典2048）
- 合成逻辑：相同数字碰撞能量叠加
- 进化链：2→4→8→...→2048
- 最终目标：传说中的2048
- 空间限制：4×4格子被数字填满
- 叙事：数字精灵合体变强，召唤传说数字守护王国

### 示例2：合成大西瓜
- 合成逻辑：相同水果融合成长
- 进化链：葡萄→樱桃→...→西瓜
- 最终目标：传说中的大西瓜
- 空间限制：容器被水果填满溢出
- 叙事：水果精灵融合成长，培育出传说中的大西瓜

### 示例3：怪物进化岛
- 合成逻辑：相同怪物合体进化
- 进化链：史莱姆→哥布林→兽人→巨魔→龙→神龙
- 最终目标：神龙守护神
- 空间限制：岛屿空间被占满
- 叙事：怪物合体进化，培育出守护岛屿的神龙

### 示例4：建造天空之城
- 合成逻辑：相同建筑合并升级
- 进化链：帐篷→小屋→房子→别墅→城堡→天空之城
- 最终目标：传说中的天空之城
- 空间限制：土地被建筑占满
- 叙事：建筑不断升级合并，最终建成传说中的天空之城

### 示例5：锻造神剑
- 合成逻辑：相同武器熔炼升级
- 进化链：木剑→铁剑→钢剑→银剑→金剑→神剑
- 最终目标：封印魔王的神剑
- 空间限制：熔炉空间有限
- 叙事：武器不断熔炼升级，锻造出封印魔王的神剑

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 目标 | 合成到指定等级/数量/限步达成 |
| 棋盘 | 大小/形状/障碍格子 |
| 机制 | 炸弹/锁定格/传送门/随机障碍 |
