# 时机类品类预设（跳跃/接东西/躲避）

## 组合逻辑

- [主角] 想要 [到达目标/收集东西/躲避危险]
- 在正确的时机 [跳跃/接住/躲开]
- 失败 = 掉落 / 接漏 / 碰到危险
- 成功 = 到达目的地 / 收集足够 / 存活到最后

**故事必须解释**：
- 为什么时机很重要？（平台会消失/东西会掉/危险会来）
- 到达目的地/收集东西之后会怎样？

## 要素库

### 玩法子类型
| 类型 | 代表玩法 | 核心操作 |
|-----|---------|---------|
| 跳跃类 | 跳一跳、Doodle Jump | 按压蓄力跳跃到下一个平台 |
| 接东西 | 接金币、接水果 | 左右移动接住下落的物品 |
| 躲避类 | 别踩白块、躲避球 | 躲开危险物，只碰安全物 |
| 踩点类 | 钢琴块、别踩白块 | 在正确位置/时机点击 |

### 主角类型
| 类型 | 变体示例 |
|-----|---------|
| 可爱生物 | 小青蛙/小兔子/小企鹅/小鸟 |
| 小人物 | 小忍者/小精灵/小棋子/小机器人 |
| 容器类 | 篮子/盘子/帽子/手（接东西用） |
| 抽象物 | 小方块/小球/光点 |

### 目标物/收集物
| 类型 | 变体示例 |
|-----|---------|
| 平台类 | 方块/云朵/荷叶/石柱/蘑菇 |
| 下落物 | 金币/水果/星星/礼物/糖果 |
| 危险物 | 炸弹/毒药/石头/仙人掌 |

### 动机类型
| 类型 | 叙事解释 |
|-----|---------|
| 向上攀登 | 要跳到最高处/云端/月亮 |
| 收集任务 | 收集足够的东西完成任务 |
| 逃离危险 | 地面在崩塌/水在上涨，必须往上跳 |
| 追逐目标 | 追着某个东西一直向前 |

### 失败条件
| 类型 | 叙事解释 |
|-----|---------|
| 掉落 | 跳空了/踩歪了，掉下去 |
| 接漏 | 重要的东西没接住 |
| 碰到危险 | 碰到炸弹/毒药/敌人 |
| 被追上 | 被上涨的水/崩塌的地面追上 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 高度线 | 草地→树梢→云端→星空→月球 |
| 季节线 | 春天花园→夏日海滩→秋天森林→冬日雪地 |
| 难度线 | 平台大且稳→平台小且动→平台会消失 |

## 组合示例

### 示例1：跳一跳小青蛙
- 子类型：跳跃类
- 主角：小青蛙
- 目标：跳到月亮上找妈妈
- 操作：按压蓄力，松开跳跃
- 失败：跳空了掉进水里
- 场景：荷叶→石头→云朵→星星→月亮
- 结局：在月亮上和妈妈团聚

### 示例2：小熊接蜂蜜
- 子类型：接东西
- 主角：饥饿的小熊
- 目标：接住足够的蜂蜜罐过冬
- 操作：左右移动接住下落的蜂蜜
- 失败：接到炸弹/蜂蜜罐摔碎太多
- 场景：蜂蜜树下→蜂蜜瀑布→蜂蜜雨
- 结局：收集满仓蜂蜜，温暖过冬

### 示例3：忍者跳跳跳
- 子类型：跳跃类（带上升追逐）
- 主角：小忍者
- 目标：跳到塔顶救师父
- 操作：点击跳跃，躲避障碍
- 失败：被上涨的岩浆追上/掉落
- 场景：塔底→塔中→塔顶→天守阁
- 结局：救出师父，成为真正的忍者

### 示例4：企鹅躲雪球
- 子类型：躲避类
- 主角：南极小企鹅
- 目标：躲过雪崩，收集小鱼
- 操作：左右移动躲避雪球，接住小鱼
- 失败：被雪球砸中
- 场景：雪山坡→冰川→暴风雪区
- 结局：安全到达温暖的海边

### 示例5：云端跳跳兔
- 子类型：跳跃类（Doodle Jump）
- 主角：长耳朵小兔
- 目标：跳到彩虹尽头找到传说的胡萝卜
- 操作：左右控制方向，自动弹跳
- 失败：掉出屏幕下方
- 场景：草地→云朵→彩虹→星空
- 结局：找到金色胡萝卜，愿望成真

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 平台 | 静止→移动→消失→弹跳 |
| 物品 | 种类/速度/危险物 |
| 机制 | 弹簧/传送门/道具 |
| 挑战 | 限时/目标分数/连续 |
