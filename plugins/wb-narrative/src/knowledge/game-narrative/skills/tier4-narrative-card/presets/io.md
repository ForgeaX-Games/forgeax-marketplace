# IO/大作战品类预设

## 组合逻辑

- 在 [竞技场] 中，所有 [参与者] 都想成为最强
- 吃 [食物/小目标] 会变大，够大就能吃掉其他玩家
- 失败 = 被更大的吞噬
- 成功 = 成为最大最强，称霸竞技场

**故事必须解释**：
- 为什么吃东西就能变大？
- 为什么大的能吃小的？（食物链/物理规则）

## 要素库

### 主角类型（可成长的角色）
| 类型 | 变体示例 |
|-----|---------|
| 生物类 | 细胞/蛇/鱼/虫子/史莱姆 |
| 物体类 | 黑洞/雪球/纸团/气球 |
| 角色类 | 小战士/小法师/小坦克 |
| 载具类 | 小车/飞机/坦克/飞船 |

### 成长方式
| 类型 | 叙事解释 |
|-----|---------|
| 吞噬成长 | 吃掉比自己小的，身体变大 |
| 击败成长 | 击败对手获得经验/能量 |
| 收集成长 | 收集散落的资源变强 |
| 合并成长 | 与同类合并变得更大 |

### 吞噬/攻击目标
| 类型 | 变体示例 |
|-----|---------|
| 食物 | 光点/食物/能量球/星星 |
| 其他玩家 | 比自己小的玩家 |
| NPC | 散落的小生物/小物体 |
| 建筑 | 树木/房子/城市（黑洞类） |

### 危险因素
| 类型 | 叙事解释 |
|-----|---------|
| 被吞噬 | 被比自己大的吃掉 |
| 被击败 | 被其他玩家攻击消灭 |
| 边界 | 碰到边界受伤/死亡 |
| 陷阱 | 地图上的危险区域 |

### 成长阶段
| 阶段 | 示例 |
|-----|------|
| 初生期 | 最小最弱，只能吃食物 |
| 成长期 | 可以吃小型玩家 |
| 强大期 | 可以吃中型玩家 |
| 霸主期 | 几乎无敌，统治竞技场 |

### 场景类型
| 类型 | 示例 |
|-----|------|
| 自然场景 | 草原/海洋/森林/沙漠 |
| 城市场景 | 街道/城市/都市/世界 |
| 抽象场景 | 细胞世界/数据空间/宇宙 |
| 幻想场景 | 糖果世界/玩具世界/梦境 |

## 组合示例

### 示例1：球球大作战
- 主角：彩色小球
- 成长方式：吞噬比自己小的球
- 食物：散落的光点
- 危险：被大球吞噬
- 场景：彩色竞技场
- 成长链：小球→中球→大球→巨球→霸主球
- 结局：成为竞技场最大的球

### 示例2：贪吃蛇大作战
- 主角：小蛇
- 成长方式：吃光点变长，撞死其他蛇吃残骸
- 食物：光点、其他蛇的残骸
- 危险：撞到其他蛇身体
- 场景：无边界蛇场
- 成长链：小蛇→中蛇→长蛇→巨蛇→蛇王
- 结局：成为最长的蛇

### 示例3：黑洞大作战
- 主角：小黑洞
- 成长方式：吞噬物体变大
- 食物：路灯→汽车→房子→大楼→城市
- 危险：被更大的黑洞吞噬
- 场景：城市街道
- 成长链：小洞→中洞→大洞→巨洞→世界吞噬者
- 结局：吞噬整个城市

### 示例4：细胞大作战
- 主角：小细胞
- 成长方式：吞噬小细胞和营养
- 食物：营养物质、小细胞
- 危险：被大细胞吞噬、病毒
- 场景：微观世界/培养皿
- 成长链：细菌→原生动物→多细胞→巨型细胞
- 结局：成为培养皿霸主

### 示例5：纸团大作战
- 主角：小纸团
- 成长方式：滚动收集纸张变大
- 食物：便签→纸张→书本→纸箱
- 危险：被大纸团碾压、掉进垃圾桶
- 场景：办公室/教室/图书馆
- 成长链：小纸团→纸球→大纸球→纸团怪兽
- 结局：成为办公室最大的纸团

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 场景 | 主题/大小/障碍物 |
| 规则 | 经典/团队/限时/生存 |
| 胜利 | 最大/存活/积分/击杀数 |
| 特殊 | 道具/技能/进化分支 |
