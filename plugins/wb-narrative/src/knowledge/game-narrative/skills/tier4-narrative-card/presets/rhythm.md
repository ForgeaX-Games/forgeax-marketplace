# 音游/节奏品类预设

## 组合逻辑

- [音乐世界] 需要 [主角] 用节奏的力量来 [目的]
- 跟随音乐节拍，在正确时机 [操作]
- 失败 = 节奏混乱，Miss太多
- 成功 = 完美演奏，达成目标

**故事必须解释**：
- 为什么要踩准节拍？（演奏/魔法/舞蹈）
- 节奏和目标之间的关系是什么？

## 要素库

### 主角/操作对象
| 类型 | 变体示例 |
|-----|---------|
| 演奏者 | 钢琴家/吉他手/鼓手/DJ |
| 舞者 | 街舞者/芭蕾舞者/偶像 |
| 角色 | 音乐精灵/节奏小人/音符精灵 |
| 物品 | 钢琴键/鼓/音符/光点 |

### 节奏行动（玩家操作）
| 类型 | 变体示例 |
|-----|---------|
| 点击类 | 点击下落的音符/方块 |
| 滑动类 | 滑动连接音符/跟随轨迹 |
| 长按类 | 长按保持音符 |
| 组合类 | 多种操作混合 |

### 目的/动机
| 类型 | 叙事解释 |
|-----|---------|
| 演奏音乐 | 完美演奏一首曲子 |
| 拯救世界 | 用音乐的力量驱散黑暗 |
| 追逐梦想 | 成为顶级音乐家/舞者 |
| 唤醒沉睡 | 用音乐唤醒沉睡的精灵/世界 |
| 传递快乐 | 用音乐让大家开心 |

### 音乐世界设定
| 类型 | 示例 |
|-----|------|
| 乐器世界 | 钢琴王国/吉他岛/鼓之森林 |
| 音符世界 | 音符精灵的家园 |
| 舞台世界 | 从街头到世界舞台 |
| 抽象世界 | 节奏空间/音乐维度 |

### 场景/曲目递进
| 难度线 | 示例 |
|-------|------|
| 技术线 | 简单→普通→困难→专家→大师 |
| 舞台线 | 练习室→小舞台→大舞台→演唱会→世界巡演 |
| 故事线 | 新手村→城镇→王城→世界→传说舞台 |

### 评价系统
| 等级 | 叙事包装 |
|-----|---------|
| Perfect | 完美演奏/神级表演 |
| Great | 精彩演奏/出色表演 |
| Good | 不错的演奏/良好表演 |
| Miss | 失误/节奏偏离 |

## 组合示例

### 示例1：钢琴块（经典）
- 主角：钢琴演奏者
- 操作：点击下落的黑白琴键
- 目的：完美演奏每一首曲子
- 世界：钢琴音乐厅
- 场景：练习室→小音乐厅→大音乐厅→世界舞台
- 结局：成为世界级钢琴家

### 示例2：音乐精灵救世界
- 主角：音乐小精灵
- 操作：跟随节拍收集音符
- 目的：用音乐驱散笼罩世界的黑暗
- 世界：被黑暗笼罩的音乐王国
- 场景：音符森林→旋律山谷→和弦城堡→黑暗核心
- 结局：音乐之光驱散黑暗，世界恢复色彩

### 示例3：偶像养成记
- 主角：追梦少女/少年
- 操作：跟随节拍跳舞
- 目的：从练习生成为顶级偶像
- 世界：偶像养成世界
- 场景：练习室→地下演出→电视节目→演唱会→世界巡演
- 结局：站上梦想的舞台，成为超级偶像

### 示例4：节奏跑酷
- 主角：跑酷小人
- 操作：跟随节拍跳跃/滑行/闪避
- 目的：跟着音乐节奏跑到终点
- 世界：音乐跑道
- 场景：城市街道→霓虹隧道→云端跑道→星空大道
- 结局：完成最难的节奏跑道，成为节奏大师

### 示例5：唤醒沉睡的乐器
- 主角：音乐学徒
- 操作：演奏各种乐器（点击/滑动/长按）
- 目的：唤醒被诅咒沉睡的乐器精灵
- 世界：乐器博物馆
- 场景：钢琴室→弦乐厅→打击乐区→交响大厅
- 结局：所有乐器精灵苏醒，奏响最美的交响乐

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 曲目 | 歌曲/风格/BPM |
| 难度 | 音符密度/速度/复杂度 |
| 机制 | 长按/滑动/多轨道/变速 |
| 目标 | 通关/全连/满分/特殊成就 |
