# 贪食蛇品类预设

## 组合逻辑

- [主角] 想要成长到 [最终形态]
- 吃 [食物] 会变长，但身体越长越难控制
- 失败 = 撞墙 / 咬到自己
- 成功 = 成长到最终形态，完成蜕变

**故事必须解释**：
- 为什么吃了东西身体会变长？
- 为什么撞到自己/墙就会失败？

## 要素库

### 主角类型（会变长的角色）
| 类型 | 变体示例 |
|-----|---------|
| 蛇类 | 小蛇/彩虹蛇/龙蛇/水晶蛇 |
| 虫类 | 毛毛虫/蚯蚓/蜈蚣 |
| 列车 | 小火车/地铁/过山车 |
| 队伍 | 蚂蚁队伍/小鸭子队伍/精灵队伍 |
| 抽象 | 光线/彩带/音符串 |

### 食物类型（吃了变长）
| 类型 | 变体示例 |
|-----|---------|
| 水果 | 苹果/樱桃/草莓/葡萄 |
| 生物 | 小虫子/小鱼/萤火虫 |
| 能量 | 星星/宝石/能量球/光点 |
| 乘客 | 小动物乘客/精灵/小人 |

### 成长机制（为什么吃了会变长）
| 类型 | 叙事解释 |
|-----|---------|
| 进化成长 | 吃东西获得能量，身体自然成长 |
| 队伍扩张 | 每救一个伙伴，队伍多一个成员 |
| 收集连接 | 每收集一个，就连接到队伍后面 |
| 吸收合并 | 吸收能量，身体变得更长更强 |

### 危险因素（失败条件）
| 类型 | 叙事解释 |
|-----|---------|
| 撞墙 | 撞到边界，旅程结束 |
| 咬到自己 | 身体太长绕回来咬到自己 |
| 撞到障碍 | 撞到石头/陷阱/敌人 |
| 被追上 | 被追逐者追上 |

### 最终目标
| 类型 | 示例 |
|-----|------|
| 长度目标 | 成长到指定长度 |
| 进化目标 | 进化成最终形态（龙/蝴蝶） |
| 收集目标 | 收集齐所有伙伴/宝物 |
| 填满目标 | 身体填满整个空间 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 自然线 | 草地→花园→森林→洞穴→神秘之地 |
| 水域线 | 小溪→池塘→河流→湖泊→大海 |
| 空间线 | 小房间→大房间→迷宫→开放世界 |

## 组合示例

### 示例1：小蛇成龙记
- 主角：小青蛇
- 食物：能量珠
- 成长机制：吃能量珠身体变长，积累足够能量进化
- 危险：撞墙、咬到自己
- 最终目标：进化成神龙
- 场景：草地→森林→山洞→云端
- 结局：化身神龙翱翔天际

### 示例2：毛毛虫变蝴蝶
- 主角：小毛毛虫
- 食物：树叶
- 成长机制：吃树叶长大，吃够了就能结茧
- 危险：掉下树枝、被鸟吃掉
- 最终目标：变成美丽的蝴蝶
- 场景：嫩叶→树枝→树干→树冠
- 结局：破茧成蝶飞向花园

### 示例3：小火车接乘客
- 主角：小火车头
- 食物：车站的乘客车厢
- 成长机制：每到一站接一节车厢，队伍变长
- 危险：撞到障碍、脱轨
- 最终目标：接满所有乘客到达终点站
- 场景：小镇站→森林站→山地站→海边站→终点站
- 结局：所有乘客安全到达，欢呼庆祝

### 示例4：萤火虫收集光
- 主角：小萤火虫
- 食物：散落的光点
- 成长机制：收集光点，尾巴的光带越来越长
- 危险：撞到黑暗障碍、光带缠住自己
- 最终目标：收集足够的光照亮黑暗森林
- 场景：树洞→灌木丛→大树下→森林深处
- 结局：光芒照亮整片森林

### 示例5：小鸭子找妈妈
- 主角：鸭妈妈
- 食物：走散的小鸭子
- 成长机制：每找到一只小鸭子，队伍多一个
- 危险：撞到障碍、队伍太长走散
- 最终目标：找回所有小鸭子
- 场景：池塘→草地→农场→小溪→家
- 结局：一家团聚回到温暖的窝

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 空间 | 大小/形状/有无边界 |
| 障碍 | 固定/移动/随机出现 |
| 食物 | 密度/特殊效果 |
| 模式 | 限时/目标长度/生存/对战 |
