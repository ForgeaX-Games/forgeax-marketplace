# 放置/挂机品类预设

## 组合逻辑

- [主角] 怀揣梦想，开始经营 [事业]
- 通过 [核心行动] 积累 [资源]，不断升级扩张
- [自动机制] 让离开时也能持续收益
- 成功 = 从零建成 [最终形态]，实现梦想

**故事必须解释**：
- 为什么离开了还能自动赚钱/成长？
- 最终要建成什么，为什么值得期待？

## 要素库

### 经营对象类型
| 类型 | 变体示例 |
|-----|---------|
| 商业 | 柠檬水摊→小店→连锁店→商业帝国 |
| 农场 | 小菜园→农场→庄园→农业王国 |
| 工厂 | 小作坊→工厂→集团→工业帝国 |
| 王国 | 小村庄→城镇→城市→王国 |
| 冒险 | 新手村→城镇→王城→世界 |

### 核心资源类型
| 类型 | 变体示例 |
|-----|---------|
| 金币类 | 金币/钻石/宝石/星星 |
| 材料类 | 木材/石头/铁矿/魔法水晶 |
| 产品类 | 农作物/商品/装备/魔法物品 |
| 能量类 | 体力/魔力/经验/声望 |

### 自动机制（为什么挂机也能收益）
| 类型 | 叙事解释 |
|-----|---------|
| 员工/助手 | 雇佣的员工会自动工作 |
| 自动化设备 | 建造的机器会自动生产 |
| 魔法/科技 | 魔法/科技让一切自动运转 |
| 自然生长 | 种下的东西会自然生长 |

### 升级路径
| 类型 | 示例 |
|-----|------|
| 规模升级 | 更大的场地/更多的设施 |
| 效率升级 | 更快的生产/更高的收益 |
| 种类升级 | 解锁新产品/新功能 |
| 自动化升级 | 更多自动收益/离线收益 |

### 场景递进
| 发展线 | 示例 |
|-------|------|
| 规模线 | 摊位→小店→大店→连锁→帝国 |
| 地域线 | 村庄→城镇→城市→王国→世界 |
| 时代线 | 原始→农业→工业→科技→未来 |

### 里程碑事件
| 类型 | 示例 |
|-----|------|
| 解锁新区域 | 开放新的经营场所 |
| 获得新能力 | 解锁新的生产方式 |
| 达成成就 | 完成特定目标获得奖励 |
| 剧情推进 | 触发新的故事章节 |

## 组合示例

### 示例1：柠檬水大亨
- 经营对象：柠檬水生意
- 资源：金币
- 核心行动：卖柠檬水赚钱
- 自动机制：雇佣员工自动卖柠檬水
- 升级路径：摊位→小店→连锁店→柠檬水帝国
- 最终愿景：成为柠檬水大亨
- 结局：建立全球柠檬水帝国

### 示例2：开心农场主
- 经营对象：农场
- 资源：金币、农作物
- 核心行动：种植、收获、售卖
- 自动机制：作物自然生长，收割机自动收获
- 升级路径：小菜园→农场→大庄园→农业王国
- 最终愿景：拥有最美丽的农场
- 结局：成为传说中的农场主

### 示例3：矿业大亨
- 经营对象：矿场
- 资源：矿石、金币
- 核心行动：挖矿、冶炼、售卖
- 自动机制：矿工和机器自动挖掘
- 升级路径：小矿洞→矿场→矿业集团→矿业帝国
- 最终愿景：挖到世界最深处的宝藏
- 结局：发现传说中的宝藏矿脉

### 示例4：魔法学院院长
- 经营对象：魔法学院
- 资源：魔力、金币、学生
- 核心行动：招收学生、教授魔法、完成任务
- 自动机制：学生自动学习和练习
- 升级路径：小教室→学院→魔法大学→魔法圣殿
- 最终愿景：培养出最强的魔法师
- 结局：学院成为大陆第一魔法学府

### 示例5：猫咪咖啡馆
- 经营对象：猫咪咖啡馆
- 资源：金币、猫咪、顾客满意度
- 核心行动：收养猫咪、招待顾客、升级设施
- 自动机制：猫咪自动卖萌吸引顾客
- 升级路径：小角落→咖啡馆→猫咪乐园→猫咪王国
- 最终愿景：建造猫咪们的天堂
- 结局：成为最受欢迎的猫咪咖啡馆

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 规模 | 场地/设施/员工数量 |
| 解锁 | 新产品/功能/区域/角色 |
| 目标 | 收益/收集/成就 |
| 事件 | 随机事件/节日活动/限时挑战 |
