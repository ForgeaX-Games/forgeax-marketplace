# 连连看品类预设

## 组合逻辑

- [配对对象] 们散落各处，想要 [配对目的]
- 找到两个相同的，用线连起来，它们就能 [团聚/消失]
- 失败 = 时间耗尽 / 无法连接
- 成功 = 所有配对完成，皆大欢喜

**故事必须解释**：
- 为什么它们会散落各处？
- 为什么连线就能让它们团聚/消失？

## 要素库

### 配对对象类型
| 类型 | 变体示例 |
|-----|---------|
| 动物 | 小猫/小狗/小鸟/小鱼/小兔 |
| 水果 | 苹果/橙子/草莓/葡萄/西瓜 |
| 物品 | 礼物/玩具/糖果/宝石/星星 |
| 表情 | 笑脸/爱心/太阳/月亮/彩虹 |
| 角色 | 精灵/小人/怪物/机器人 |

### 配对目的（为什么要配对）
| 类型 | 叙事解释 |
|-----|---------|
| 团聚 | 走散的伙伴/家人要团聚 |
| 配对 | 找到另一半/伴侣 |
| 收集 | 成对收集才能带走 |
| 解封 | 配对才能解除封印 |
| 整理 | 成对整理才能归位 |

### 连接机制（为什么要连线）
| 类型 | 叙事解释 |
|-----|---------|
| 心灵感应 | 相同的伙伴心灵相通，连线就能感应到 |
| 魔法通道 | 画出魔法通道让它们相遇 |
| 路径引导 | 为它们指引相遇的路 |
| 能量连接 | 相同的能量可以连接共振 |

### 连接规则（路径限制）
| 类型 | 叙事解释 |
|-----|---------|
| 直线/折线 | 通道只能转弯有限次（0-2次） |
| 无障碍 | 通道不能穿过其他物品 |
| 边界可绕 | 可以绕到场景边缘 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 自然线 | 花园→森林→山谷→神秘之地 |
| 建筑线 | 小屋→城镇→城堡→宫殿 |
| 节日线 | 春节→情人节→万圣节→圣诞节 |
| 主题线 | 海洋→天空→太空→梦境 |

### 特殊道具
| 类型 | 作用 |
|-----|------|
| 提示道具 | 显示一对可连接的配对 |
| 重排道具 | 打乱重新排列 |
| 时间道具 | 增加/暂停时间 |
| 炸弹道具 | 消除指定区域 |

## 组合示例

### 示例1：小动物找伙伴（经典）
- 配对对象：各种小动物图案
- 配对目的：走散的小动物伙伴要团聚
- 连接机制：画出路径让它们相遇
- 场景：农场→森林→动物园→神奇动物园
- 结局：所有小动物都找到了伙伴

### 示例2：精灵配对回家
- 配对对象：各种颜色的小精灵
- 配对目的：同色精灵配对才能打开回家的门
- 连接机制：魔法通道连接
- 场景：精灵森林→蘑菇村→彩虹桥→精灵王国
- 结局：所有精灵都回到了精灵王国

### 示例3：礼物配对送达
- 配对对象：各种礼物盒
- 配对目的：相同的礼物配对才能送出
- 连接机制：快递路线连接
- 场景：仓库→街道→社区→全城
- 结局：所有礼物都送到了小朋友手中

### 示例4：星座配对
- 配对对象：星座图案
- 配对目的：相同星座的星星要连成星座
- 连接机制：星光连线
- 场景：夜空一角→半边天空→整片星空→银河
- 结局：所有星座都在夜空中闪耀

### 示例5：糖果配对装盒
- 配对对象：各种糖果图案
- 配对目的：相同糖果配对装进礼盒
- 连接机制：传送带连接
- 场景：糖果车间→包装线→礼盒区→发货区
- 结局：所有糖果礼盒都装好送出
