# 俄罗斯方块品类预设

## 组合逻辑

- [下落物] 不断从天而降，填满 [空间]
- 排列整齐填满一行，就会消除腾出空间
- 失败 = 堆到顶部，空间溢出
- 成功 = 坚持足够久 / 达成目标分数

**故事必须解释**：
- 为什么下落物是不同形状的？（不是普通方块）
- 为什么一行满了会消失/被带走？

## 要素库

### 下落物类型（方块的叙事包装）
| 类型 | 变体示例 |
|-----|---------|
| 建筑材料 | 砖块/积木/石块/木块 |
| 货物 | 箱子/包裹/行李/集装箱 |
| 自然物 | 石头/冰块/云朵/陨石 |
| 食物 | 蛋糕块/巧克力块/果冻块 |
| 生物 | 小怪物/小精灵/小动物 |

### 空间/容器（为什么有边界）
| 类型 | 叙事解释 |
|-----|---------|
| 建筑工地 | 在有限空间里堆砌建筑 |
| 仓库 | 货物不断运来，要整理好 |
| 容器 | 东西不断掉落，要装好 |
| 世界边界 | 东西从天而降，堆到顶就溢出 |

### 消除机制（为什么一行满了会消除）
| 类型 | 叙事解释 |
|-----|---------|
| 完工运走 | 一层建好就被运走/验收 |
| 整理清空 | 一行整理好就清空腾出空间 |
| 魔法消失 | 排列整齐触发魔法，整行消失 |
| 压缩打包 | 一行满了就压缩打包送走 |

### 目标类型
| 类型 | 示例 |
|-----|------|
| 生存目标 | 坚持尽可能长的时间 |
| 分数目标 | 达到指定分数 |
| 消除目标 | 消除指定行数 |
| 建造目标 | 完成指定层数的建筑 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 建筑线 | 地基→低层→中层→高层→摩天楼 |
| 速度线 | 慢速→中速→快速→极速→疯狂 |
| 难度线 | 新手→熟练→专家→大师→传说 |

### 特殊方块/道具
| 类型 | 作用 |
|-----|------|
| 炸弹方块 | 落地爆炸清除周围 |
| 直线方块 | 直接消除一整行 |
| 重力方块 | 压实下方空隙 |
| 时间道具 | 暂停/减速下落 |

## 组合示例

### 示例1：建筑大师（经典）
- 下落物：彩色积木/砖块
- 空间：建筑工地
- 机制：一层建好就验收运走
- 目标：建造尽可能高的建筑
- 场景：小工地→大工地→摩天楼工地
- 结局：成为传说中的建筑大师

### 示例2：仓库整理员
- 下落物：各种形状的包裹箱
- 空间：仓库货架
- 机制：一行整理好就发货清空
- 目标：处理尽可能多的订单
- 场景：小仓库→中型仓库→超级仓库
- 结局：成为最佳仓库管理员，获得晋升

### 示例3：糖果工厂
- 下落物：彩色糖果块/果冻块
- 空间：糖果生产线
- 机制：一行糖果排好就打包出厂
- 目标：生产足够多的糖果礼盒
- 场景：小作坊→工厂→超级糖果帝国
- 结局：糖果王国的孩子们都吃到了糖果

### 示例4：星际防御
- 下落物：太空陨石/碎片
- 空间：防御力场
- 机制：一行碎片聚齐就被力场粉碎
- 目标：保护星球不被陨石淹没
- 场景：小行星带→陨石雨→流星暴
- 结局：成功保护星球，成为星际英雄

### 示例5：小怪物排排站
- 下落物：各种形状的小怪物组合
- 空间：怪物收容所
- 机制：一行怪物排好就被送回家
- 目标：帮助所有迷路的小怪物回家
- 场景：森林收容所→山洞收容所→魔法收容所
- 结局：所有小怪物都回到了自己的家

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 速度 | 下落速度/加速条件 |
| 方块 | 形状种类/特殊方块 |
| 规则 | 预览/暂存/连消奖励 |
| 挑战 | 限时/目标/对战 |
