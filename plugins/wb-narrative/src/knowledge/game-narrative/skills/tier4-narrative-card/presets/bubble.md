# 泡泡龙品类预设

## 组合逻辑

- [被困者] 被 [泡泡/宝石] 困在上方
- 发射 [泡泡]，三个同色连在一起就会消除
- 失败 = 泡泡堆太低 / 步数用尽
- 成功 = 消除所有泡泡，被困者获救

**故事必须解释**：
- 为什么被困者会被泡泡/宝石困住？
- 为什么三个同色连在一起就会消除？

## 要素库

### 主角类型（发射者）
| 类型 | 变体示例 |
|-----|---------|
| 可爱生物 | 小恐龙/小龙/青蛙/小猫 |
| 小人物 | 小精灵/小巫师/小弓箭手 |
| 拟人物品 | 泡泡机/弹弓/魔法棒 |

### 泡泡/球类型
| 类型 | 变体示例 |
|-----|---------|
| 彩色泡泡 | 红/蓝/绿/黄/紫泡泡 |
| 宝石球 | 红宝石/蓝宝石/绿宝石/钻石 |
| 水果球 | 苹果/橙子/葡萄/草莓 |
| 魔法球 | 火球/冰球/雷球/光球 |
| 表情球 | 笑脸/星星/爱心/月亮 |

### 被困者（需要解救的对象）
| 类型 | 变体示例 |
|-----|---------|
| 小动物 | 小鸟/小兔/小熊/小鱼 |
| 精灵 | 森林精灵/水精灵/花精灵 |
| 星星 | 掉落的星星宝宝 |
| 宝物 | 被封印的宝石/钥匙 |

### 困境类型（为什么被困）
| 类型 | 叙事解释 |
|-----|---------|
| 泡泡封印 | 被魔法泡泡包裹困住 |
| 宝石困住 | 被宝石/水晶包围 |
| 蜘蛛网 | 被蜘蛛网缠住 |
| 冰封 | 被冰块冻住 |

### 消除机制（为什么三个能消除）
| 类型 | 叙事解释 |
|-----|---------|
| 魔法共振 | 三个同色魔法共振，产生能量爆发 |
| 颜色融合 | 三个同色融合，释放封印 |
| 能量聚集 | 三个同色聚集，积累足够能量破除 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 森林线 | 森林入口→大树下→树冠→云端 |
| 海洋线 | 海滩→珊瑚礁→海底洞穴→海底宫殿 |
| 天空线 | 草地→山顶→云层→星空 |
| 地下线 | 洞穴入口→水晶洞→熔岩洞→地心 |

### 特殊泡泡/道具
| 类型 | 作用 |
|-----|------|
| 彩虹泡泡 | 可以匹配任何颜色 |
| 炸弹泡泡 | 爆炸消除周围泡泡 |
| 闪电泡泡 | 消除同一行/列 |
| 火焰泡泡 | 穿透消除 |

## 组合示例

### 示例1：泡泡龙救小恐龙（经典）
- 主角：小恐龙泡泡龙
- 泡泡：彩色魔法泡泡
- 被困者：被泡泡困住的小恐龙伙伴
- 机制：三个同色泡泡连接产生魔法共振，释放被困者
- 场景：森林→洞穴→火山→恐龙岛
- 结局：救出所有小恐龙伙伴，大家一起玩耍

### 示例2：星星送回天空
- 主角：月亮精灵
- 泡泡：彩色星星球
- 被困者：从天上掉落的星星宝宝
- 机制：三颗同色星星聚齐，产生升力飞回天空
- 场景：草地→山顶→云层→银河
- 结局：所有星星回到天空，夜空重新闪耀

### 示例3：珊瑚礁救小鱼
- 主角：小章鱼
- 泡泡：彩色珊瑚球
- 被困者：被珊瑚困住的小鱼
- 机制：三个同色珊瑚连接碎裂，小鱼游出
- 场景：浅海→珊瑚礁→海底洞穴→沉船
- 结局：所有小鱼获救，海底恢复热闹

### 示例4：魔法森林解封印
- 主角：小精灵弓箭手
- 泡泡：魔法宝石球
- 被困者：被宝石封印的森林精灵
- 机制：三颗同色宝石共振，封印解除
- 场景：森林边缘→精灵村→古树→精灵女王宫殿
- 结局：所有精灵解封，森林恢复生机

### 示例5：糖果泡泡救甜心
- 主角：棉花糖小人
- 泡泡：彩色糖果球
- 被困者：被糖果困住的甜心精灵
- 机制：三颗同色糖果融化，甜心精灵获救
- 场景：糖果镇→巧克力河→棉花糖山→糖果城堡
- 结局：所有甜心精灵获救，糖果王国欢庆

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 布局 | 初始形状/被困者位置/障碍物 |
| 目标 | 救出数量/清除/限步 |
| 机制 | 下降/移动/特殊障碍 |
| 道具 | 特殊泡泡频率 |
