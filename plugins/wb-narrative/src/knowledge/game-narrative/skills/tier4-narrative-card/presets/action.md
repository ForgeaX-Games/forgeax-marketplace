# 动作/打砖块品类预设

## 组合逻辑

- [被困者/目标] 被 [障碍物] 层层包围
- 用 [弹射物] 击碎障碍，弹来弹去
- 失败 = 球掉落，机会用尽
- 成功 = 击碎所有障碍，解救目标

**故事必须解释**：
- 为什么球/弹射物能一直弹来弹去不停？
- 为什么击碎障碍就能解救目标？

## 要素库

### 主角/发射者类型
| 类型 | 变体示例 |
|-----|---------|
| 板子类 | 弹板/球拍/盾牌/蹦床 |
| 角色类 | 小骑士/小精灵/小动物 |
| 工具类 | 弹弓/大炮/魔法棒 |

### 弹射物类型
| 类型 | 变体示例 |
|-----|---------|
| 球类 | 弹球/能量球/光球/魔法球 |
| 物品类 | 星星/宝石/水果/飞镖 |
| 生物类 | 小精灵/小动物/小怪物 |

### 障碍物/砖块类型
| 类型 | 变体示例 |
|-----|---------|
| 建筑类 | 砖块/石块/冰块/水晶 |
| 敌人类 | 小怪物/入侵者/坏蛋 |
| 封印类 | 魔法屏障/牢笼/锁链 |
| 自然类 | 岩石/荆棘/乌云 |

### 被困者/目标物
| 类型 | 示例 |
|-----|------|
| 生物类 | 小动物/精灵/公主/伙伴 |
| 物品类 | 宝藏/星星/钥匙/宝石 |
| 区域类 | 被封印的土地/被占领的城堡 |

### 弹射机制（为什么能弹来弹去）
| 类型 | 叙事解释 |
|-----|---------|
| 魔法弹球 | 魔法球永不停止，直到任务完成 |
| 能量反弹 | 充满能量的球会不断反弹 |
| 物理反射 | 光滑表面让球反弹 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 王国线 | 边境→村庄→城镇→城堡→王座 |
| 自然线 | 森林→山洞→火山→云端→神殿 |
| 建筑线 | 地牢→塔楼→城墙→主堡→天空城 |

### 特殊砖块/道具
| 类型 | 作用 |
|-----|------|
| 坚硬砖块 | 需要多次击打 |
| 爆炸砖块 | 击中后爆炸影响周围 |
| 道具砖块 | 击碎掉落道具（加长板/多球/激光） |
| 传送砖块 | 球穿过后传送到其他位置 |

## 组合示例

### 示例1：勇者救公主（经典打砖块）
- 主角：魔法弹板
- 弹射物：光之球
- 障碍物：魔王城堡的砖块
- 被困者：被困在城堡顶层的公主
- 机制：光之球击碎砖块，一层层打上去
- 场景：城堡外墙→塔楼→主堡→王座室
- 结局：击碎所有砖块，救出公主

### 示例2：星星回家路
- 主角：月亮弹板
- 弹射物：小星星
- 障碍物：乌云/陨石/黑暗块
- 被困者：被乌云困住的星星伙伴
- 机制：星星撞碎乌云，释放被困的星星
- 场景：夜空→云层→风暴区→银河
- 结局：所有星星回到银河闪耀

### 示例3：糖果城堡保卫战
- 主角：糖果弹板
- 弹射物：彩虹糖球
- 障碍物：蛀牙怪物组成的墙
- 被困者：糖果精灵
- 机制：糖球击败蛀牙怪，解救糖果精灵
- 场景：糖果镇→巧克力墙→棉花糖堡→糖果王座
- 结局：击败所有蛀牙怪，糖果王国恢复甜蜜

### 示例4：海底宝藏猎人
- 主角：潜水艇弹板
- 弹射物：声波球
- 障碍物：珊瑚/岩石/沉船残骸
- 被困者：海底宝藏
- 机制：声波击碎障碍，宝藏掉落收集
- 场景：浅海→珊瑚礁→沉船区→海底洞穴
- 结局：收集所有宝藏，成为海底探险家

### 示例5：精灵解封印
- 主角：魔法盾牌
- 弹射物：元素精灵球
- 障碍物：黑暗封印/魔法屏障
- 被困者：被封印的元素精灵
- 机制：精灵球击碎封印，释放对应元素精灵
- 场景：封印之地→元素神殿→精灵圣所
- 结局：所有精灵解封，元素之力回归世界

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 布局 | 排列图案/层数/形状 |
| 砖块 | 普通/坚硬/爆炸/移动 |
| 道具 | 加长板/多球/激光/护盾 |
| 挑战 | 限时/限球数/全清 |
