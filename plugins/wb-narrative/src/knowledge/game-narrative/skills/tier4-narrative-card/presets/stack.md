# 堆叠/叠高高品类预设

## 组合逻辑

- [主角] 想要建造 [建筑目标]，越高越好
- 一层层往上叠，要对齐，偏差会导致面积变小
- 失败 = 偏差太大，无法继续
- 成功 = 建成通天塔，到达目标高度

**故事必须解释**：
- 为什么要往高处建？（到达某处/挑战/梦想）
- 为什么偏差会导致面积变小？

## 要素库

### 堆叠物类型
| 类型 | 变体示例 |
|-----|---------|
| 建筑类 | 楼层/砖块/积木/石板 |
| 食物类 | 汉堡层/蛋糕层/三明治/冰淇淋球 |
| 自然类 | 石头/木块/冰块/云朵 |
| 生物类 | 小动物叠罗汉/精灵塔 |
| 抽象类 | 方块/几何体/能量块 |

### 建筑目标
| 类型 | 示例 |
|-----|------|
| 建筑类 | 高塔/摩天楼/城堡/天空之城 |
| 食物类 | 超级汉堡/巨型蛋糕/冰淇淋塔 |
| 自然类 | 通天石塔/冰雪城堡 |
| 抽象类 | 无限高塔/传说之塔 |

### 堆叠机制（为什么要对齐）
| 类型 | 叙事解释 |
|-----|---------|
| 物理平衡 | 不对齐会不稳定，容易倒塌 |
| 面积递减 | 偏差部分会被切掉，越来越小 |
| 完美追求 | 只有完美对齐才能继续建造 |

### 失败条件
| 类型 | 叙事解释 |
|-----|---------|
| 面积归零 | 偏差太多，可堆叠面积变成零 |
| 倒塌 | 不平衡导致整个建筑倒塌 |
| 掉落 | 新层没放上去，掉落了 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 高度线 | 地面→云层→平流层→太空→宇宙 |
| 建筑线 | 小屋→楼房→高楼→摩天楼→天空之城 |
| 挑战线 | 简单→普通→困难→极限→传说 |

### 特殊机制
| 类型 | 作用 |
|-----|------|
| 完美奖励 | 完美对齐获得额外分数/恢复面积 |
| 道具 | 减速/自动对齐/面积恢复 |
| 障碍 | 风/移动障碍/不规则形状 |

## 组合示例

### 示例1：盖楼大师（经典）
- 堆叠物：楼层/建筑块
- 建筑目标：摩天大楼
- 机制：楼层左右移动，点击放下，偏差部分切掉
- 失败：可堆叠面积变成零
- 场景：城市地面→云层→平流层→太空
- 结局：建造出世界最高的摩天大楼

### 示例2：超级汉堡
- 堆叠物：汉堡配料（面包/肉饼/生菜/番茄/芝士）
- 建筑目标：传说中的超级汉堡
- 机制：配料落下，对齐堆叠
- 失败：汉堡倒塌
- 场景：小餐车→快餐店→高级餐厅→美食大赛
- 结局：制作出载入吉尼斯的超级汉堡

### 示例3：动物叠叠乐
- 堆叠物：各种小动物
- 建筑目标：动物金字塔
- 机制：动物左右移动，点击让它们叠上去
- 失败：动物塔倒塌
- 场景：农场→动物园→马戏团→动物王国
- 结局：动物们完成最高的叠罗汉表演

### 示例4：冰淇淋塔
- 堆叠物：冰淇淋球
- 建筑目标：通天冰淇淋
- 机制：冰淇淋球摆动，点击叠上
- 失败：冰淇淋倒塌/融化
- 场景：冰淇淋车→甜品店→冰淇淋城堡
- 结局：制作出世界上最高的冰淇淋

### 示例5：通天塔传说
- 堆叠物：魔法石块
- 建筑目标：通往天界的通天塔
- 机制：石块移动，对齐堆叠
- 失败：塔身不稳倒塌
- 场景：地面→云端→天界入口→神殿
- 结局：通天塔建成，连接人间与天界

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 目标 | 层数/高度/分数 |
| 速度 | 移动速度递增 |
| 形状 | 不同/不规则形状 |
| 机制 | 风力/障碍/奖励层 |
