# 找茬/找不同品类预设

## 组合逻辑

- [场景] 中藏着 [隐藏目标/不同之处]
- 仔细观察，找出所有目标
- 失败 = 时间耗尽 / 点错太多
- 成功 = 全部找到，任务完成

**故事必须解释**：
- 为什么要找这些东西？（破案/寻宝/修复/救援）
- 找到之后会发生什么？

## 要素库

### 玩法子类型
| 类型 | 代表玩法 | 核心操作 |
|-----|---------|---------|
| 找不同 | 大家来找茬 | 对比两张图，找出不同之处 |
| 找隐藏 | 找隐藏物品 | 在场景中找出隐藏的物品 |
| 找相同 | 找出相同的 | 在众多物品中找出指定的 |
| 找规律 | 找出不属于的 | 找出不符合规律的那个 |

### 场景类型
| 类型 | 变体示例 |
|-----|---------|
| 室内场景 | 卧室/厨房/客厅/教室/图书馆 |
| 室外场景 | 公园/森林/海滩/城市街道/游乐园 |
| 幻想场景 | 魔法城堡/糖果屋/海底世界/外太空 |
| 历史场景 | 古埃及/中世纪/恐龙时代/未来城市 |

### 隐藏目标
| 类型 | 变体示例 |
|-----|---------|
| 差异点 | 颜色不同/形状不同/位置不同/多了/少了 |
| 隐藏物品 | 钥匙/宝石/小动物/线索/字母 |
| 特定物品 | 找出所有苹果/找出所有星星 |
| 异常物品 | 不属于这个场景的东西 |

### 动机类型
| 类型 | 叙事解释 |
|-----|---------|
| 侦探破案 | 找出线索解开谜团 |
| 寻宝探险 | 找到隐藏的宝藏 |
| 修复错误 | 找出被施了魔法的错误之处 |
| 救援任务 | 找到藏起来的小伙伴 |
| 整理收纳 | 找出不属于这里的东西 |

### 限制条件
| 类型 | 叙事解释 |
|-----|---------|
| 时间限制 | 魔法消失前/天黑前必须找到 |
| 次数限制 | 只有有限次点击机会 |
| 提示限制 | 提示道具数量有限 |

### 场景递进
| 难度线 | 示例 |
|-------|------|
| 复杂度 | 简单场景→复杂场景→超级复杂 |
| 差异度 | 明显差异→细微差异→极难发现 |
| 数量 | 找3处→找5处→找7处→找10处 |

## 组合示例

### 示例1：小侦探找线索
- 子类型：找隐藏
- 场景：案发现场（各种房间）
- 目标：隐藏的线索物品
- 动机：收集线索破解谜案
- 限制：时间限制
- 递进：简单房间→复杂房间→超级乱的房间
- 结局：找齐线索，真相大白

### 示例2：魔法找茬师
- 子类型：找不同
- 场景：被施了魔法的两幅画
- 目标：魔法造成的差异
- 动机：找出差异才能解除魔法
- 限制：魔法消失前（时间限制）
- 递进：3处不同→5处→7处→10处
- 结局：解除魔法，画中世界恢复正常

### 示例3：海底寻宝
- 子类型：找隐藏
- 场景：海底世界各处
- 目标：藏在珊瑚/海草/沉船里的宝藏
- 动机：收集宝藏建造海底城堡
- 限制：氧气有限（时间限制）
- 递进：浅海→深海→海底洞穴→沉船
- 结局：收集齐宝藏，建成海底城堡

### 示例4：躲猫猫大作战
- 子类型：找隐藏
- 场景：各种场景（家里/公园/学校）
- 目标：躲起来的小动物朋友
- 动机：找到所有躲起来的朋友
- 限制：天黑前找到（时间限制）
- 递进：简单躲藏→高级躲藏→伪装躲藏
- 结局：找到所有朋友，一起开派对

### 示例5：博物馆奇妙夜
- 子类型：找不同/找异常
- 场景：博物馆各展厅
- 目标：被调皮精灵改动的地方
- 动机：在天亮前把一切恢复原状
- 限制：天亮前（时间限制）
- 递进：小展厅→大展厅→整个博物馆
- 结局：一切恢复，博物馆平安开门

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 场景 | 主题/复杂度/动态 |
| 目标 | 数量/隐蔽度/细微度 |
| 限制 | 时间/点击次数 |
| 特殊 | 移动/消失/干扰 |
