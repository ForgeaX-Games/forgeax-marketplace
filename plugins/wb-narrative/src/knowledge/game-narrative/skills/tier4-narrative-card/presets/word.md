# 猜词/文字品类预设

## 组合逻辑

- [神秘词语] 隐藏在谜题中，等待被发现
- 通过猜测，[反馈机制] 告诉你离答案有多近
- 失败 = 机会用尽，词语未解开
- 成功 = 解开谜题，获得成就感

**故事必须解释**：
- 为什么要猜这个词？（解谜/破案/寻宝）
- 反馈机制是什么原理？（颜色提示/距离提示）

## 要素库

### 游戏类型
| 类型 | 代表玩法 |
|-----|---------|
| 猜词类 | Wordle/猜单词/成语接龙 |
| 填字类 | 填字游戏/字谜 |
| 拼字类 | 拼单词/找单词 |
| 成语类 | 成语填空/看图猜成语/成语消消乐 |

### 主角/玩家身份
| 类型 | 变体示例 |
|-----|---------|
| 学者 | 语言学家/文字考古学家/词汇大师 |
| 侦探 | 文字侦探/密码破译员 |
| 学生 | 好学的学生/词汇挑战者 |
| 守护者 | 文字守护者/成语传承人 |

### 谜题形式
| 类型 | 变体示例 |
|-----|---------|
| 字母格 | 5字母格/6字母格/自定义格 |
| 填空 | 成语填空/句子填空 |
| 图片 | 看图猜词/表情猜词 |
| 线索 | 根据提示猜词 |

### 反馈机制（为什么能逐步接近答案）
| 类型 | 叙事解释 |
|-----|---------|
| 颜色提示 | 绿色=正确位置，黄色=有但位置错，灰色=没有 |
| 距离提示 | 告诉你离正确答案有多近 |
| 线索解锁 | 每次猜测解锁一条新线索 |
| 部分显示 | 正确的部分会显示出来 |

### 词语来源/主题
| 类型 | 示例 |
|-----|------|
| 日常词汇 | 常用单词/生活用语 |
| 成语典故 | 四字成语/历史典故 |
| 专业领域 | 科学术语/行业用语 |
| 流行文化 | 网络热词/影视名称 |

### 场景递进
| 难度线 | 示例 |
|-------|------|
| 词汇线 | 简单词→常见词→生僻词→专业词→传说词 |
| 长度线 | 3字→4字→5字→6字→更长 |
| 主题线 | 日常→自然→科技→历史→哲学 |

## 组合示例

### 示例1：每日一词（Wordle类）
- 游戏类型：猜单词
- 主角：词汇爱好者
- 目的：每天猜出神秘单词，保持连胜
- 反馈：绿色=正确，黄色=有但位置错，灰色=没有
- 场景：每日挑战→周挑战→月挑战→年度挑战
- 结局：成为猜词大师，连胜记录无人能破

### 示例2：成语大冒险
- 游戏类型：成语填空/看图猜成语
- 主角：成语小学者
- 目的：收集所有成语，成为成语大师
- 反馈：正确的字会亮起来
- 场景：成语启蒙→成语故事→成语典故→成语殿堂
- 结局：集齐所有成语，成为成语传承人

### 示例3：文字侦探
- 游戏类型：根据线索猜词
- 主角：文字侦探
- 目的：通过猜词破解密码，解开谜案
- 反馈：每次猜测解锁一条新线索
- 场景：简单案件→复杂案件→连环案件→终极谜案
- 结局：破解所有谜案，成为传奇侦探

### 示例4：汉字拼拼乐
- 游戏类型：拼字/组词
- 主角：汉字小精灵
- 目的：把散落的汉字组合成词语
- 反馈：正确组合会发光
- 场景：汉字花园→词语森林→成语山→文字圣殿
- 结局：收集所有汉字，建成文字王国

### 示例5：表情猜猜猜
- 游戏类型：看表情/图片猜词
- 主角：表情翻译官
- 目的：翻译表情密码，理解隐藏的信息
- 反馈：猜对会显示完整答案
- 场景：简单表情→组合表情→故事表情→谜题表情
- 结局：成为表情语言大师
