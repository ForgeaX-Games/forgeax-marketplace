# 跑酷品类预设

## 组合逻辑

- [主角] 因为 [动机] 必须不停地跑
- 一路上要躲避 [障碍]，收集 [物品]
- 失败 = 被 [追逐者] 抓住 / 撞到障碍
- 成功 = 到达 [目的地]，实现目标

**故事必须解释**：
- 为什么不能停下来？（被追/地面崩塌/时间紧迫）
- 收集的东西有什么用？

## 要素库

### 主角类型
| 类型 | 变体示例 |
|-----|---------|
| 可爱动物 | 小鸡/小猫/小狗/兔子/仓鼠 |
| 小人物 | 小男孩/小女孩/精灵/小矮人 |
| 拟人物品 | 跑步的寿司/逃跑的姜饼人/飞翔的纸飞机 |
| 英雄角色 | 小骑士/小忍者/小海盗/小牛仔 |

### 动机类型（为什么要跑）
| 类型 | 叙事解释 |
|-----|---------|
| 逃离追捕 | 被坏人/怪物/猎人追赶，跑慢就被抓 |
| 寻找亲人 | 和妈妈/家人走散，要跑回家 |
| 拯救伙伴 | 伙伴被抓走，要追上去救回来 |
| 追寻梦想 | 要跑到传说中的地方实现梦想 |
| 送达任务 | 必须在时间内把东西送到目的地 |

### 障碍类型
| 类型 | 变体示例 |
|-----|---------|
| 地形障碍 | 坑洞/悬崖/河流/岩石/荆棘 |
| 移动障碍 | 滚石/飞镖/摆锤/火焰 |
| 敌人障碍 | 巡逻的怪物/飞行的乌鸦/追赶的猎人 |
| 机关障碍 | 陷阱/弹簧/传送门/消失的平台 |

### 收集物品
| 类型 | 作用叙事 |
|-----|---------|
| 金币/星星 | 积攒能量/收集希望 |
| 食物 | 补充体力/变得更强 |
| 钥匙/碎片 | 打开通往下一区域的门 |
| 道具 | 获得临时能力（磁铁/护盾/加速） |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 自然线 | 森林→河流→山地→雪原→火山 |
| 城市线 | 街道→屋顶→地铁→高架→摩天楼 |
| 幻想线 | 糖果路→云端→彩虹桥→星空→月球 |
| 地下线 | 洞穴→矿井→地下河→熔岩→地心 |

### 追逐者（可选）
| 类型 | 示例 |
|-----|------|
| 怪物 | 大灰狼/怪兽/僵尸/幽灵 |
| 人物 | 猎人/警察/厨师/巫婆 |
| 自然 | 洪水/雪崩/岩浆/龙卷风 |

## 组合示例

### 示例1：小鸡回家路
- 主角：走散的小鸡
- 动机：和妈妈走散，要跑回家
- 障碍：坑洞、河流、老鹰、狐狸
- 收集：小虫子（补充体力）
- 追逐者：大灰狼
- 场景：农场→森林→河边→山坡→鸡窝
- 结局：和妈妈团聚

### 示例2：姜饼人大逃亡
- 主角：刚出炉的姜饼人
- 动机：不想被吃掉，要逃出厨房
- 障碍：刀叉、热锅、面粉堆
- 收集：糖霜（变得更强）
- 追逐者：饥饿的厨师
- 场景：烤箱→厨房→餐厅→花园→森林
- 结局：逃到森林获得自由

### 示例3：小骑士救公主
- 主角：勇敢的小骑士
- 动机：公主被抓，要追上恶龙
- 障碍：陷阱、岩浆、怪物
- 收集：金币（升级装备）
- 追逐者：无（主动追击）
- 场景：城堡→森林→火山→龙穴
- 结局：打败恶龙救出公主

### 示例4：地铁跑酷少年
- 主角：涂鸦少年
- 动机：被巡逻员发现，要逃跑
- 障碍：火车、栏杆、路障
- 收集：金币、滑板
- 追逐者：巡逻员和狗
- 场景：站台→铁轨→隧道→屋顶→终点站
- 结局：成功逃脱成为传说

### 示例5：纸飞机环游世界
- 主角：小纸飞机
- 动机：想飞到世界的尽头看看
- 障碍：气流、乌云、飞鸟、高楼
- 收集：星星（保持飞行）
- 追逐者：无
- 场景：窗台→城市→海洋→沙漠→极光
- 结局：看到世界尽头的美景

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 场景 | 环境主题、视觉风格 |
| 障碍 | 类型、密度、移动速度 |
| 速度 | 跑步速度、反应时间 |
| 特殊 | Boss追逐、限时收集、无尽模式 |
