# 弹弓/投掷品类预设

## 组合逻辑

- [目标/敌人] 在远处，需要被 [击中/击倒/投进]
- 拉动瞄准，调整角度力度，发射 [弹射物]
- 失败 = 弹药用尽，目标未完成
- 成功 = 击倒所有目标，大获全胜

**故事必须解释**：
- 为什么要用弹弓/投掷而不是直接过去？
- 击倒目标之后能得到什么？

## 要素库

### 玩法子类型
| 类型 | 代表玩法 | 核心操作 |
|-----|---------|---------|
| 弹弓类 | 愤怒的小鸟 | 拉弹弓发射，击倒建筑/敌人 |
| 投篮类 | 投篮机、纸团投篮 | 调整角度力度，投进目标 |
| 高尔夫类 | 迷你高尔夫 | 瞄准击球，进洞 |
| 飞镖类 | 飞镖、射箭 | 瞄准投掷，击中靶心 |
| 抛物类 | 抛硬币、套圈 | 抛出物品，落到目标位置 |

### 弹射物/投掷物
| 类型 | 变体示例 |
|-----|---------|
| 生物类 | 小鸟/小猪/小青蛙/小企鹅 |
| 球类 | 篮球/高尔夫球/足球/弹珠 |
| 物品类 | 纸团/飞镖/石头/水果/炸弹 |
| 特殊类 | 魔法球/冰球/火球/分裂球 |

### 目标类型
| 类型 | 变体示例 |
|-----|---------|
| 建筑类 | 木头/石头/玻璃搭建的结构 |
| 容器类 | 篮筐/球洞/杯子/圈 |
| 敌人类 | 绿猪/怪物/坏蛋 |
| 靶子类 | 靶心/气球/瓶子 |

### 动机类型
| 类型 | 叙事解释 |
|-----|---------|
| 复仇/夺回 | 坏蛋偷了东西，要打败它们夺回来 |
| 挑战/竞技 | 挑战高分/击败对手 |
| 救援 | 击倒障碍救出被困的伙伴 |
| 破坏 | 拆除危险建筑/清理障碍 |
| 收集 | 击中目标收集奖励 |

### 物理机制
| 类型 | 叙事解释 |
|-----|---------|
| 抛物线 | 重力影响，形成弧线轨迹 |
| 反弹 | 碰到墙壁/物体会反弹 |
| 连锁 | 击中一个引发连锁反应 |
| 特殊能力 | 分裂/加速/爆炸/穿透 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 难度线 | 近距离→中距离→远距离→超远+障碍 |
| 结构线 | 简单结构→复杂结构→机关结构 |
| 环境线 | 无风→有风→移动目标→多重障碍 |

## 组合示例

### 示例1：愤怒的小鸟（经典）
- 子类型：弹弓类
- 弹射物：各种能力的小鸟
- 目标：绿猪和它们的建筑
- 动机：绿猪偷了鸟蛋，要夺回来
- 物理：抛物线+建筑倒塌连锁
- 场景：森林→沙漠→雪地→城堡
- 结局：打败所有绿猪，夺回鸟蛋

### 示例2：纸团投篮王
- 子类型：投篮类
- 弹射物：废纸团
- 目标：垃圾桶
- 动机：无聊的办公室，挑战投篮纪录
- 物理：抛物线+反弹
- 场景：办公桌→会议室→老板办公室→天台
- 结局：成为办公室投篮传奇

### 示例3：小青蛙救伙伴
- 子类型：弹弓类
- 弹射物：小青蛙（弹舌头）
- 目标：困住伙伴的泡泡/笼子
- 动机：伙伴被坏蛇困住，要救出来
- 物理：舌头弹射+击破障碍
- 场景：池塘→森林→沼泽→蛇窝
- 结局：救出所有伙伴，赶走坏蛇

### 示例4：迷你高尔夫冒险
- 子类型：高尔夫类
- 弹射物：高尔夫球
- 目标：球洞
- 动机：完成传说中的高尔夫球场
- 物理：滚动+反弹+坡度
- 场景：花园→城堡→火山→太空
- 结局：一杆进洞成为高尔夫大师

### 示例5：水果忍者弹弓版
- 子类型：弹弓类
- 弹射物：小忍者飞镖
- 目标：飞起来的水果（躲开炸弹）
- 动机：收集水果完成忍者修行
- 物理：抛物线+连击加分
- 场景：道场→果园→市场→忍者考核
- 结局：通过考核成为水果忍者

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 布局 | 目标位置/结构/障碍物 |
| 数量 | 弹射物数量限制 |
| 机制 | 风力/移动目标/特殊弹射物 |
| 目标 | 星级/全击倒/限弹数 |
