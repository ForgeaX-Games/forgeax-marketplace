# 棋牌品类预设

## 组合逻辑

- [主角] 进入 [牌/棋] 的世界，想要 [目的]
- 运用智慧和策略，与 [对手] 对决
- 失败 = 输掉比赛
- 成功 = 一路过关斩将，成为 [最高称号]

**故事必须解释**：
- 为什么要通过牌/棋来解决问题？
- 赢了能得到什么？

## 要素库

### 游戏类型
| 类型 | 代表玩法 |
|-----|---------|
| 纸牌类 | 扑克/斗地主/21点/接龙/蜘蛛纸牌 |
| 麻将类 | 麻将/碰碰胡 |
| 棋类 | 象棋/围棋/五子棋/跳棋 |
| 休闲牌 | UNO/卡牌配对/记忆翻牌 |

### 主角/玩家身份
| 类型 | 变体示例 |
|-----|---------|
| 牌手 | 扑克高手/麻将大师/棋圣 |
| 角色 | 小侦探/小魔法师/冒险家 |
| 身份 | 挑战者/守护者/传承者 |

### 目的/动机
| 类型 | 叙事解释 |
|-----|---------|
| 成为最强 | 击败所有对手成为冠军 |
| 解谜破案 | 通过牌局获取线索 |
| 赢得奖励 | 赢取金币/宝物/荣誉 |
| 拯救/守护 | 通过胜利来保护重要的东西 |
| 传承技艺 | 学习并传承古老的牌/棋技艺 |

### 对手类型
| 类型 | 变体示例（从弱到强） |
|-----|---------------------|
| 新手 | 初学者/业余玩家 |
| 普通 | 普通玩家/街头高手 |
| 高手 | 地区冠军/职业选手 |
| 大师 | 全国冠军/传说人物 |
| 传说 | 世界冠军/不败神话 |

### 场景递进
| 环境线 | 示例 |
|-------|------|
| 竞技线 | 新手桌→普通桌→高手桌→大师桌→传说桌 |
| 地域线 | 街头→茶馆→俱乐部→锦标赛→世界赛 |
| 故事线 | 学徒→游历→挑战→决战→登顶 |

### 奖励/成就
| 类型 | 示例 |
|-----|------|
| 金币 | 赢取筹码/金币 |
| 道具 | 特殊牌背/棋盘/头像框 |
| 称号 | 段位/头衔/荣誉 |
| 剧情 | 解锁新故事/新对手 |

## 组合示例

### 示例1：扑克大师之路
- 游戏类型：扑克/德州扑克
- 主角：年轻的扑克爱好者
- 目的：成为世界扑克冠军
- 对手：业余→职业→冠军→传说
- 场景：地下牌局→赌场→锦标赛→世界大赛
- 结局：赢得世界扑克大赛冠军

### 示例2：麻将江湖
- 游戏类型：麻将
- 主角：麻将世家传人
- 目的：找回失传的麻将秘籍，成为麻将之王
- 对手：街坊→茶馆高手→地区霸主→麻将宗师
- 场景：家乡茶馆→各地挑战→全国大赛→宗师对决
- 结局：集齐秘籍，成为新一代麻将之王

### 示例3：纸牌接龙冒险
- 游戏类型：纸牌接龙/蜘蛛纸牌
- 主角：小魔法师学徒
- 目的：通过纸牌魔法解除古老的封印
- 对手：无（解谜向）
- 场景：魔法塔一层→二层→...→塔顶
- 结局：解除封印，成为真正的魔法师

### 示例4：五子棋小天才
- 游戏类型：五子棋
- 主角：喜欢下棋的小学生
- 目的：代表学校参加比赛，为学校争光
- 对手：同学→校队→区赛对手→市赛对手→省赛对手
- 场景：教室→校队→区赛→市赛→省赛
- 结局：为学校赢得省赛冠军

### 示例5：记忆翻牌大挑战
- 游戏类型：记忆翻牌/配对
- 主角：小侦探
- 目的：通过记忆配对找到隐藏的线索
- 对手：无（解谜向）
- 场景：案发现场→嫌疑人家→秘密基地→真相大白
- 结局：找到所有线索，破解谜案
