# 钓鱼品类预设

## 组合逻辑

- [水域] 里藏着各种 [鱼类/宝物]
- 抛竿等待，在正确时机收杆
- 失败 = 鱼跑了 / 线断了
- 成功 = 钓到目标鱼 / 收集齐图鉴

**故事必须解释**：
- 为什么要钓鱼？（收集/养家/寻找传说/比赛）
- 钓到传说之鱼会怎样？

## 要素库

### 玩法子类型
| 类型 | 代表玩法 | 核心操作 |
|-----|---------|---------|
| 时机钓鱼 | 休闲钓鱼 | 等待咬钩，时机点击收杆 |
| 拉扯钓鱼 | 钓鱼大亨 | 鱼咬钩后拉扯对抗，控制张力 |
| 瞄准钓鱼 | 捕鱼达人 | 瞄准射击/撒网捕鱼 |
| 收集钓鱼 | 钓鱼日记 | 收集各种鱼类图鉴 |

### 钓鱼者/主角
| 类型 | 变体示例 |
|-----|---------|
| 人物类 | 小渔夫/老爷爷/小女孩/探险家 |
| 动物类 | 钓鱼猫/企鹅/熊/鹈鹕 |
| 幻想类 | 小精灵/人鱼/海盗 |

### 猎物类型
| 类型 | 变体示例（从普通到稀有） |
|-----|------------------------|
| 普通鱼 | 小鱼→中鱼→大鱼 |
| 稀有鱼 | 金鱼→彩虹鱼→传说之鱼 |
| 特殊物 | 宝箱/靴子/海星/珍珠 |
| Boss鱼 | 鲨鱼/鲸鱼/海怪/龙鱼 |

### 水域/场景
| 类型 | 变体示例 |
|-----|---------|
| 淡水 | 小池塘→河流→湖泊→瀑布 |
| 海水 | 海滩→近海→深海→海底 |
| 幻想 | 云海→星空湖→熔岩河→时空裂缝 |

### 装备/升级
| 类型 | 变体示例 |
|-----|---------|
| 鱼竿 | 竹竿→钢竿→碳纤维→传说鱼竿 |
| 鱼饵 | 蚯蚓→虫子→特殊饵→魔法饵 |
| 配件 | 浮漂/鱼线/鱼钩/探鱼器 |

### 动机类型
| 类型 | 叙事解释 |
|-----|---------|
| 收集图鉴 | 收集所有种类的鱼 |
| 养家糊口 | 钓鱼卖钱/喂饱家人 |
| 寻找传说 | 钓到传说中的神鱼 |
| 比赛竞技 | 钓鱼大赛夺冠 |
| 探索秘密 | 钓上来的东西揭示秘密 |

### 挑战机制
| 类型 | 叙事解释 |
|-----|---------|
| 时机判断 | 太早太晚都会跑鱼 |
| 张力控制 | 拉太紧线断，太松鱼跑 |
| 耐心等待 | 稀有鱼需要等很久 |
| 资源管理 | 鱼饵有限，要精打细算 |

## 组合示例

### 示例1：小猫钓鱼记
- 主角：爱吃鱼的小猫
- 水域：村边池塘→小河→大湖→神秘瀑布
- 动机：钓鱼给猫妈妈做生日大餐
- 猎物：小鱼→鲤鱼→金鱼→传说锦鲤
- 挑战：时机点击收杆
- 结局：钓到最大的鱼，妈妈过了最棒的生日

### 示例2：深海钓鱼王
- 主角：年轻渔夫
- 水域：海滩→近海→深海→海沟→传说海域
- 动机：寻找爷爷说的传说巨鱼
- 猎物：普通海鱼→稀有鱼→Boss鱼→传说海王
- 挑战：拉扯对抗，控制张力
- 结局：钓到传说海王，完成爷爷的遗愿

### 示例3：钓鱼图鉴大师
- 主角：鱼类研究员
- 水域：世界各地的水域
- 动机：收集全世界所有鱼类的图鉴
- 猎物：各种真实/幻想鱼类
- 挑战：不同鱼需要不同鱼饵和技巧
- 结局：完成图鉴，成为世界级鱼类学家

### 示例4：企鹅捕鱼大作战
- 主角：南极小企鹅
- 水域：冰洞→冰川→南极海→深海
- 动机：为企鹅村储备过冬的鱼
- 猎物：小鱼→中鱼→大鱼→巨型章鱼
- 挑战：冰面会裂，要快速钓鱼
- 结局：储备满仓，企鹅村安全过冬

### 示例5：星空垂钓者
- 主角：小精灵
- 水域：云海→星河→银河→宇宙尽头
- 动机：钓起掉落的星星送回天空
- 猎物：小星星→流星→彗星→传说星座
- 挑战：星星会挣扎逃跑
- 结局：所有星星回归，夜空重新闪耀

## 关卡拓展维度

| 要素 | 变化方向 |
|-----|---------|
| 水域 | 环境/鱼种分布/天气 |
| 目标 | 特定鱼/数量/重量 |
| 限制 | 时间/鱼饵/装备 |
| 特殊 | Boss鱼/比赛/稀有鱼 |
