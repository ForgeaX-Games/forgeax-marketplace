# Roguelike / Roguelite 叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **叙事特点** | 循环叙事——每次失败都是故事的一部分 |
| **叙事载体** | 循环解锁叙事 + 随机事件 + NPC 碎片对话 |
| **核心挑战** | 如何让"反复死亡/重来"有叙事意义 |
| **代表作** | 哈迪斯、以撒的结合、死亡细胞、杀戮尖塔、挺进地牢 |

---

## Roguelike 叙事核心：循环叙事

Roguelike 最独特的叙事能力是**将游戏机制本身变成叙事**：

- **哈迪斯式**：角色知道自己在死亡循环中，每次死亡是故事进展
- **以撒式**：叙事是隐喻层，玩家需要解读，无明确叙事推进
- **死亡细胞式**：叙事碎片化，通过环境/物品暗示

**选择哪种模式**由世界观设定决定。

## 世界观框架（一页纸）

```markdown
**世界背景**：[为什么循环/重来是合理的？]
示例设定：
- "角色死后会复活，但必须一次次突破诅咒"（哈迪斯式）
- "这是梦/幻象/模拟，失败只是重置"（Meta式）
- "每次冒险都是不同的探索者"（无连续性设定）

**核心目标**：[最终要完成什么？]

**世界规则**（2-3条）：[与循环机制相关的世界规则]

**叙事解锁机制**：[通过什么解锁更多叙事背景？]
```

## 循环叙事设计

### 死亡处理（叙事层面）

| 循环类型 | 死亡叙事 | 推荐度 |
|---------|---------|-------|
| 角色知道循环 | 死亡台词有记忆感（"又一次..."） | ★★★ 推荐 |
| 角色不知道 | 每次是"新的"故事，但玩家积累叙事 | ★★ 可用 |
| 叙事隐喻 | 死亡是隐喻（心理/象征），不解释 | ★★ 高阶 |

### 渐进解锁设计

```
第1-5次运行：基础世界背景（NPC 初识，基本世界介绍）
第6-15次：深化叙事（NPC 关系，世界秘密揭露）
第16-30次：核心真相（为什么循环，最终目标揭露）
通关后：终章叙事（解开所有谜团，完整故事）
```

## NPC 循环对话系统

Roguelike 的 NPC 需要**随循环次数变化**的对话：

```markdown
## [NPC 名字] 对话轨迹

**初见**（1-3次循环）：
"第一次见到你...或者不是？反正，欢迎来到XX。"

**熟识**（4-10次）：
"哦，又是你。你这次打算走多远？"

**了解**（11-20次）：
"每次你回来，我都觉得你离那个答案更近了。"

**信任**（20次+）：
"我不太明白你为什么一直这样做...但我支持你。"

**特殊触发**（特定条件）：
[揭露 NPC 背景或世界秘密的对话]
```

## 随机事件设计

Roguelike 随机事件要有**叙事背景但结果可预测**（避免信息不对等打击体验）：

```markdown
**事件名**：[简短标题]
**场景**（1-2句）：[快速建立情境]
**选项**：
  A：[描述] → 效果：[明确的奖励/代价]
  B：[描述] → 效果：[明确的奖励/代价]
  （可选 C：特殊条件解锁选项）
```

**设计原则**：
- 选项效果要**透明**（Roguelike 玩家讨厌信息不对等）
- 叙事是包装，不影响功能性决策
- 事件名和描述体现世界观氛围

## 死亡台词与复活台词

```markdown
**死亡台词**（每角色 5-10 句，随机）：
1. "这次...走不远。"（冷静）
2. "不行了吗——下次再说吧。"（淡定）
3. "...明明快到了。"（不甘）
4. "该死的陷阱。"（愤怒，简短）
5. "好痛...但活过来过。"（循环感）

**复活台词**（每角色 3-5 句，随机）：
1. "又回来了。"（淡然）
2. "知道哪里出错了，这次不一样。"（成长感）
3. "...我记得上次。"（循环感）
```

## 能力/道具描述规范

Roguelike 的 buff/道具描述同样需要叙事感：

```markdown
**道具名**：[有意象感]
**效果**：[清晰功能说明]
**背景**（可选）：[1句，暗示道具来历或世界背景]

示例：
道具名：《碎裂的王冠》
效果：每层最多携带一件物品，但该物品效果+100%
背景："一个王国的象征——虽然王国早已不在。"
```
