# 卡牌构筑叙事预设

## 品类特征

| 维度 | 详情 |
|------|------|
| **叙事载体** | 卡牌文案 + 角色台词 + 随机事件 |
| **叙事密度** | 低密度，玩家选择性阅读 |
| **核心需求** | 卡牌描述风格统一，有叙事感但不影响功能理解 |
| **代表作** | Slay the Spire、怪物火车、夜之城、幻兽帕鲁卡牌 |

---

## 叙事核心：文案即叙事

卡牌游戏的叙事主要通过**文案**传递，而非剧情场景。

**卡牌文案 = 功能说明 + 叙事风味（可选）**

---

## 世界观框架（一页纸）

```markdown
**世界背景**（2-3句）：
[交代是什么世界，有什么危机，玩家为何战斗]

**核心力量体系**（2-3条）：
[解释卡牌/能力的叙事逻辑，为什么这些技能存在]

**主要对立势力**：
[玩家对抗的是什么，为什么是敌人]

**主角定位**：
[玩家扮演谁，有什么特别之处]
```

---

## 卡牌文案规范

### 功能优先原则
卡牌描述必须首先保证功能清晰：
1. **功能行**：清晰说明效果（XX 伤害 / 获得 X 格挡 / 抽 X 张）
2. **风味行**（可选）：1句有叙事感的描述

### 风味文案写作模板

```markdown
**卡牌名**：[有意象感的命名，非纯功能描述]
**功能**：[清晰的效果说明]
**风味文本**：
"[1句话，从角色/物品视角，暗示世界背景或角色性格]"
```

**好例子**：
- 卡牌名：「最后的防线」
  风味文：「这面盾牌挡下过灭绝之火，但主人没能活着看到胜利。」
- 卡牌名：「狂战士之怒」
  风味文：「战场上，每一滴鲜血都是献给混沌之神的礼物。」

**坏例子**：
- ❌ 「攻击1」——无叙事意象
- ❌ 「使用一张使你增加5点攻击力的卡牌，同时抽一张牌」——功能文当名字

---

## 角色台词（极简）

卡牌游戏的角色台词要**短而有力**：

| 场景 | 台词要求 | 示例 |
|-----|---------|------|
| 战斗开始 | 1-2 句，确立氛围 | "又一波敌人——好，正好热身。" |
| 胜利 | 1 句，有反差或余韵 | "他们没想到，连一次都没有。" |
| 死亡/失败 | 1-2 句，有重量感 | "这次...走不远了。下次吧。" |
| 特殊事件 | 1-3 句，推进叙事 | "这个地方有点不对劲——我以前来过这里？" |
| Boss 战前 | 1-2 句，建立对立 | "就是你毁了这里的一切？" |

## 随机事件文案设计

Roguelike 卡牌的随机事件应有叙事背景：

```markdown
**事件名**：[有叙事感的标题]
**场景描述**：[2-3句，建立情境]
**选项**：
  A：[选项描述] → 结果：[奖励/代价]
  B：[选项描述] → 结果：[奖励/代价]
  C：（可选）[特殊选项] → 结果：
```

**示例**：
```
事件名：《陌生商人》
场景：一个衣衫褴褛的老人守在十字路口，身旁摆着几件奇怪的物件。
他用浑浊的眼睛打量你："命运送我一位顾客，我送你一个选择。"

A：购买神秘物品（消耗 50 金币）→ 获得随机强力卡牌
B：拒绝继续前进 → 无事发生，但老人低声说了什么...（触发隐藏lore）
C：【力量>=15】抢劫他 → 获得物品，但后续遭遇更多敌人
```

```markdown
**击败后叙事**（可选，2-3句）：
```

## 质量检查

- [ ] 卡牌功能描述是否清晰易懂（优先于风味）？
- [ ] 风味文本是否有叙事感而非解释性？
- [ ] 随机事件是否有世界观背景（非纯功能选项）？
- [ ] 角色台词是否短而有力，不打断节奏？
- [ ] 不同类型卡牌的风味文案是否风格一致？
