# 注释 (Annotation)

拖入画布后创建一段可编辑、可拖动的自由文本注释，用于说明管线结构、标记区域或记录调试信息。

## 功能特点

1. **自由文本**：拖入后立即进入编辑状态，可直接输入说明文字
2. **画布定位**：注释独立浮在画布上，可像节点一样拖动位置
3. **双击编辑**：已有注释可双击重新编辑，失焦后自动保存
4. **不参与执行**：注释只用于说明，不产生端口，也不会触发算法计算

## 适用情况

- 为复杂管线分区添加说明
- 标记某组电池的用途或注意事项
- 在调试过程中临时记录观察结论

## 基本使用方法

从 `SPECIAL / Preview` 中将「注释」拖入画布，松手后直接输入文字。输入完成后点击画布其他位置保存；如果内容为空，注释会自动删除。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| - | - | - | 无输入端口 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| - | - | 无输出端口 |

## 注意事项

1. **仅用于编辑器说明**：注释不会参与管线执行，也不会向下游传递数据
2. **空内容自动删除**：新建后如果不输入文字，失焦时会自动移除
3. **编辑方式**：单击选中并拖动，双击进入文本编辑
