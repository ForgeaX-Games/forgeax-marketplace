# 文本面板 (text_panel)

文本面板：可自由输入或接收上游连线，输出字符串。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| input | any | - | 可选。有连接时用上游值转成字符串输出，覆盖面板内容 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| output | string | 当前面板文本；有上游连接时为上游值转字符串 |

## 注意事项

- 无上游连接时，输出为用户在面板中输入的文本
- 有上游连接时，输出为上游值的字符串形式
