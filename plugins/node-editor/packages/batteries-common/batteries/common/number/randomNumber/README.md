# 随机整数 (random_number)

在指定的最小值和最大值范围内生成一个随机整数，每次执行结果不同。

## 功能特点

1. **闭区间随机**：最小值和最大值均包含在可能的结果中
2. **每次不同**：使用 `Math.random()` 保证每次执行产生不同的结果
3. **自动纠错**：最小值和最大值填反时自动交换，不会报错

## 适用情况

- 需要一个随机的单个整数（如随机骰子点数、随机关卡编号）
- 配合其他电池做随机决策或随机采样
- 不需要固定种子、每次都要不同结果的场景

## 基本使用方法

设置 `min`（最小值）和 `max`（最大值），执行后 `number` 端口输出该范围内的一个随机整数。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| min | number | 0 | 随机整数的最小值（含） |
| max | number | 100 | 随机整数的最大值（含） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| number | number | 生成的随机整数 |

## 注意事项

1. **闭区间**：输出值可能等于 `min` 或 `max`
2. **min > max**：两者会自动交换，正常生成
3. **每次不同**：如需可复现的随机结果，请使用「随机数列表」电池并指定种子
