# 进阶运算 (Advanced Math Op)

对输入数字执行进阶数学运算，输出计算结果。

## 功能特点

1. **8 种运算模式**：支持平方根、立方根、向下取整、向上取整、四舍五入、最小值、最大值、对数
2. **一元运算支持**：`sqrt`、`cbrt`、`floor`、`ceil`、`round` 仅使用数字 A，数字 B 被忽略
3. **灵活的对数底数**：`log` 以 B 为底数，B ≤ 0 或 B = 1 时自动退化为自然对数 ln(A)
4. **下拉选择运算符**：`op` 字段提供下拉菜单，无需手动输入字符串

## 适用情况

- 需要对数值开方或求立方根
- 在流程中对浮点数进行取整处理
- 需要从两个数值中取最大/最小值
- 需要进行对数运算（任意底数）

## 基本使用方法

1. 连接上游数值到「数字 A」和「数字 B」端口（或直接在参数面板输入）
2. 在「运算」下拉框中选择所需运算符
3. 「结果」端口输出计算值，可接入下游电池

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| a | number | 0 | 第一个操作数；单目运算时仅使用此值 |
| b | number | 1 | 第二个操作数；单目运算时忽略；log 时作为底数 |
| op | string | `sqrt` | 运算符，从下拉菜单选择 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| result | number | 运算结果 |

## 支持的运算说明

| 运算符 | 含义 | 公式 | 是否使用 B |
|--------|------|------|-----------|
| `sqrt` | 平方根 | √A | ❌ |
| `cbrt` | 立方根 | ∛A | ❌ |
| `floor` | 向下取整 | ⌊A⌋ | ❌ |
| `ceil` | 向上取整 | ⌈A⌉ | ❌ |
| `round` | 四舍五入 | round(A) | ❌ |
| `min` | 最小值 | min(A, B) | ✅ |
| `max` | 最大值 | max(A, B) | ✅ |
| `log` | 对数 | log_B(A)；B≤0 或 B=1 时退化为 ln(A) | ✅ |

## 使用示例

### 输入示例（平方根，只用 A）

```json
{
  "a": 16,
  "b": 0,
  "op": "sqrt"
}
```

### 输出示例

```json
{
  "result": 4
}
```

### 输入示例（向下取整）

```json
{
  "a": 3.7,
  "b": 0,
  "op": "floor"
}
```

### 输出示例

```json
{
  "result": 3
}
```

### 输入示例（以 2 为底的对数）

```json
{
  "a": 8,
  "b": 2,
  "op": "log"
}
```

### 输出示例

```json
{
  "result": 3
}
```

## 注意事项

1. **一元运算忽略 B**：`sqrt`、`cbrt`、`floor`、`ceil`、`round` 只读取 A，B 的连接不影响结果
2. **对数底数**：`log` 运算以 B 为底数；若 B ≤ 0 或 B = 1（无意义的底数），自动退化为自然对数 ln(A)
3. **负数开方**：`sqrt` 对负数返回 NaN，请确保输入 A ≥ 0
4. **浮点精度**：运算结果为 JavaScript 原生浮点数，大数或极小数可能存在精度误差
