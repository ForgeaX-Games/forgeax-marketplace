# 基础运算 (Basic Math Op)

对两个输入数字执行基础数学运算，输出计算结果。

## 功能特点

1. **7 种运算模式**：支持加、减、乘、除、幂、取余、绝对值
2. **一元运算支持**：`abs` 仅使用数字 A，数字 B 被忽略
3. **安全除法/取余**：除数或模数为 0 时自动返回 0，避免运行时错误
4. **下拉选择运算符**：`op` 字段提供下拉菜单，无需手动输入字符串

## 适用情况

- 需要在节点图中对数值进行基本四则运算
- 在流程中动态计算坐标、比例、偏移等数值
- 需要快速进行幂运算或取余操作

## 基本使用方法

1. 连接上游数值到「数字 A」和「数字 B」端口（或直接在参数面板输入）
2. 在「运算」下拉框中选择所需运算符
3. 「结果」端口输出计算值，可接入下游电池

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| a | number | 0 | 第一个操作数；abs 运算时仅使用此值 |
| b | number | 1 | 第二个操作数；abs 运算时忽略 |
| op | string | `+` | 运算符，从下拉菜单选择 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| result | number | 运算结果 |

## 支持的运算说明

| 运算符 | 含义 | 公式 | 是否使用 B |
|--------|------|------|-----------|
| `+` | 加法 | A + B | ✅ |
| `-` | 减法 | A - B | ✅ |
| `*` | 乘法 | A × B | ✅ |
| `/` | 除法 | A ÷ B（B=0 返回 0） | ✅ |
| `^` | 幂运算 | A^B | ✅ |
| `%` | 取余 | A mod B（B=0 返回 0） | ✅ |
| `abs` | 绝对值 | \|A\| | ❌ |

## 使用示例

### 输入示例（加法）

```json
{
  "a": 10,
  "b": 3,
  "op": "+"
}
```

### 输出示例

```json
{
  "result": 13
}
```

### 输入示例（幂运算）

```json
{
  "a": 2,
  "b": 8,
  "op": "^"
}
```

### 输出示例

```json
{
  "result": 256
}
```

### 输入示例（绝对值，只用 A）

```json
{
  "a": -5,
  "b": 0,
  "op": "abs"
}
```

### 输出示例

```json
{
  "result": 5
}
```

## 注意事项

1. **除法/取余安全**：B 为 0 时结果为 0，不会抛出错误，使用时注意逻辑合理性
2. **一元运算忽略 B**：`abs` 只读取 A，B 的连接不影响结果
3. **浮点精度**：运算结果为 JavaScript 原生浮点数，大数或极小数可能存在精度误差
