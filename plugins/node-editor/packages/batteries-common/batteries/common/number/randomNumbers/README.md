# 随机数列表 (random_numbers)

生成一组随机数列表，支持指定范围、数量和随机种子，可在浮点数和整数模式之间切换。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| count | number | 1 | 生成的随机数数量（自动取整且 >=0） |
| min | number | 0 | 随机数最小值 |
| max | number | 1 | 随机数最大值 |
| seed | number | 12345 | 随机种子（整数），相同种子产生相同结果 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| numbers | array | 生成的随机数列表 |

## 注意事项

1. **可复现性**：使用 mulberry32 有种子伪随机算法，相同种子产生相同结果
2. **整数/浮点切换**：右键节点可切换整数模式（`ceil(min)` ~ `floor(max)` 均匀整数）或浮点模式
3. **自动交换 min/max**：若 `min > max`，自动交换两者
4. **count=0**：输出空列表
