# 长度 (list_get_length)

读取一级外层 list 的元素数量。List 不是基础类型，这里表示最外层 shape。

> List 在当前系统中不是基础类型；本节点处理的是基础类型值外侧的一级 list / rank-1 shape。

## 输入

| 参数名 | 类型/Rank | 标签 | 说明 |
|--------|-----------|------|------|
| list | any rank 1 | 列表 | 一级外层 list |

## 输出

| 参数名 | 类型/Rank | 标签 | 说明 |
|--------|-----------|------|------|
| length | number rank 0 | 长度 | 元素数量 |
