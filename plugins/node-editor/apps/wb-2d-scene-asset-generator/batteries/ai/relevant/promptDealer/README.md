# 提示词模板填充 (prompt_dealer)

将模板中的占位符 {0}, {1}, {2}… 用对应输入端口的字符串替换，输出完整提示词。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| template | string | "" | 包含 {0}, {1}, {2}… 的模板字符串 |
| value_0, value_1, … | string | "" | 动态端口，依次替换 {0}, {1}, {2}… |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| prompt | string | 替换所有占位符后的完整字符串 |

## 注意事项

- 占位符与端口一一对应：value_0 → {0}，value_1 → {1}，依此类推
- 未连接的 value 端口按空字符串替换
- 端口数量可随连接增加
