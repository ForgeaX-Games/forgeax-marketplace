# house_template — 房屋模板

**只画房顶掩码，程序自动生成立面、窗，门由掩码控制，渲染成房屋灰度图。**

输入一个**只表达房顶**的二维数组（1=房顶，0=空，**2=在该列开门**）；立面由程序按「高度」沿屋顶形状向上投影生成，窗自动添加。**门不再随机生成，而是被掩码控制**：把某列底部的 `1` 改成 `2`，程序就在该列的**立面墙脚**开一扇**固定 1 格高**的门；相邻列的 `2` 自然连成更宽的门。为让灰度图更像房屋、更易被大模型识别，渲染在「扁平纯色」基础上做了若干修饰：**① 窗户**——立面上按网格自动排布窗洞；**② 屋顶类型**（`roofType`）——`pitched` 坡屋顶：从屋顶轮廓做多源 BFS（草火法）算出真实屋顶结构，按朝向（朝上/下/左/右）把屋面拆成纯色坡面，坡面交界经描边自动成线，得到**真实屋脊拓扑**（矩形=直脊、L=L 脊、T=T 脊、十字=十字脊），四角斜脊以 45° 连到中轴端点、异形凹角自然出天沟；`flat` 平屋顶：屋面板 + 外缘**女儿墙/檐口收边圈**强化平顶感；**③ 外轮廓描边**——沿建筑外缘与各区域交界画深色细线，强化建筑硬直线特征；**④ 门半虚掩**——门靠一侧留一道近黑门缝（纯示意）。

| 步 | 作用 |
|---|---|
| 1 expand | 房顶顶部插入 `height` 行 0 |
| 2 offset | 房顶(1)按 height 向上投影，路径标记为立面(2) |
| 3 differentiate | 多段下沿：第 2 段起把其下方连续立面改为 3、4、5…（用于分段着色区分） |
| 4 placeDoors | **按掩码门标记开门**：门标记 `(r,c)`（值 2）映射到立面 `(r+height,c)`，在该墙脚立面格置门（标 9），固定 **1 格高**；相邻列标记连成更宽的门 |
| 4b resize | 等比缩放居中到 `imageSize²`（门随网格一并缩放，故门尺寸/位置只由掩码 + height 决定，与 imageSize 无关） |
| 4c insetDoors | 已缩放像素网格上把门横向两侧各内缩几像素（还原成立面），让门与墙体/墙角间留一圈立面边框，描边后门框不与墙轮廓糊在一起 |
| 5 placeWindows | 在各立面连通块上按网格自动排布窗户（标 8），窗数/行列由立面尺寸推导，**窗较窄呈竖向窗洞**，避开门与边缘留白 |
| 5b makeDoorsAjar | 门靠一侧留一道竖向门缝（标 10，近黑室内），做半虚掩示意 |
| 6 render | 顶面按 `roofType` 着色：pitched=`ComputeRoof` 多源 BFS 算坡面（屋脊/斜脊/天沟由描边成线，真实屋脊拓扑）/ flat=屋面板(120)+女儿墙**顶帽(158)→内侧投影带(96)**两层（在顶帽下方自动成一条「看线」并以阴影体现女儿墙厚度/立体感）；立面=浅灰(192) 平涂（多段微压暗）；窗=较深灰(132)；门扇=深灰(100)、门缝=近黑(58)；背景=白；最后描深色细线轮廓(64) |

## 输入

| 端口 | 类型 | 默认 | 说明 |
|------|------|------|------|
| spec | string | — | 只表达房顶的二维数组字符串，如 `[[1,1,0],[1,1,1],[1,1,1]]`（1=房顶，0=空，**2=在该列底部开门**）。也兼容三维 `[[[...]],[[...]]]` 批量产出多栋 |
| height | number | 1 | 房屋高度（立面向上投影的行数），越大立面越高 |
| roofType | string | pitched | 屋顶类型：`pitched`=坡屋顶（屋脊渐变两面坡）/ `flat`=平屋顶（顶面纯色平涂） |

## 参数

| 参数 | 类型 | 默认 | 说明 |
|------|------|------|------|
| imageSize | number | 300 | 输出灰度图边长（正方形像素） |

## 输出

| 端口 | 类型 | 说明 |
|------|------|------|
| image | image[] | 房屋灰度图引用（资产库 alias），每栋一张 |
| error | string | 失败信息，成功为空串 |

## 落点（双写）

每张灰度图同时：
1. 归档到仓库顶层 **`.forgeax/grayscale/`**（专属栏目，可用 `FORGEAX_GRAYSCALE_ROOT` 覆盖根目录）；
2. 导入资产库，作为 `image` 端口输出 —— 画布可预览、下游可消费、进 All Images 面板。

> 门**完全确定**：同一份掩码恒得同一结果（无随机）。要开门，把目标列底部的 `1` 改成 `2` 即可；多列连续的 `2` 形成一扇更宽的门。
