# furniture_template — 室内家具模板

**按「家具类型 + 朝向」程序化绘制可识别的家具 2.5D 剪影，渲染成灰度图。**

与 `house_template` 同角色：输出的灰度图作为**形状底图**喂给 `image_gen`，配合「严格遵守灰度图形状」的提示词约束生图。两个下拉：`kind`（家具）+ `orientation`（朝向）。采用**扁平灰度 + 外轮廓描边**：各部件以不同明度填充，再沿外缘与部件接缝画深色细线，使家具轮廓与接缝清晰可辨，便于大模型识别。

**视角为 2.5D 斜投影（cavalier oblique，深度朝右上方后退）**：每个正面部件在其后上方补画退缩的顶面（更亮）与侧面（更暗）平行四边形，使家具能看到一部分顶面，而非纯水平立面。`right` 朝向为 `left` 的水平镜像（深度方向一并镜像）。

## 家具与朝向

朝向 `down`=正面 / `up`=背面 / `left` / `right`（`right` 为 `left` 的水平镜像）。**并非每种家具都区分所有朝向**：

| 家具 | down（正面） | up（背面） | left / right（侧） |
|---|---|---|---|
| table 桌子 | 桌面 + 桌腿 | 同形 | 同形（桌子基本无朝向） |
| chair 椅子 | 座面 + 靠背面板 + 前腿 | 靠背面板 + 腿 | 侧影：靠背 + 座面 + 前后腿 |
| wardrobe 衣柜 | 双门 + 把手 + 底座 | 素柜面 | 较窄侧面 |
| bed 床 | 床尾（矮框 + 床垫 + 床头露头） | 床头（宽床头板 + 床垫） | 侧影：床头板 + 床垫 + 枕头 + 腿 |

## 输入

| 端口 | 类型 | 默认 | 说明 |
|------|------|------|------|
| kind | string | chair | 家具类型：`table` / `chair` / `wardrobe` / `bed` |
| orientation | string | down | 朝向：`down`=正面 / `up`=背面 / `left` / `right` |

## 参数

| 参数 | 类型 | 默认 | 说明 |
|------|------|------|------|
| imageSize | number | 300 | 输出灰度图边长（正方形像素） |
| seed | number | 0 | 绘制随机变体（如桌腿数）的种子；0 = 非确定性随机 |

## 输出

| 端口 | 类型 | 说明 |
|------|------|------|
| image | image[] | 家具灰度图引用（资产库 alias），一项 |
| error | string | 失败信息，成功为空串 |

## 落点（双写）

与 `house_template` 一致：每张灰度图同时归档到仓库顶层 **`.forgeax/grayscale/`**（可用 `FORGEAX_GRAYSCALE_ROOT` 覆盖根目录），并导入资产库作为 `image` 端口输出（画布可预览、下游可消费、进 All Images 面板）。
