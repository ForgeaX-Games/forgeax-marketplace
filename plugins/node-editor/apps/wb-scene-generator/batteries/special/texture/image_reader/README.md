# image_reader — 图像读取

## 功能

从本地文件系统选择一张图片，上传到资产库后输出指向该图片的 ImageRef（alias + blobId），供下游节点直接使用。

节点在画布上以紧凑双格布局呈现：左侧显示名称，右侧为图片预览区域，**双击**预览区打开系统文件浏览器选择图片。

## 输入

无输入端口。

## 输出

| 名称 | 类型 | 说明 |
|------|------|------|
| `image` | `image` | 编码后的 ImageRef（JSON 字符串：`{"alias":"...","blobId":"sha256..."}`），可连接到任何接收 `image` 的下游节点 |

## 参数（内部存储）

| 名称 | 类型 | 说明 |
|------|------|------|
| `imageRef` | `string` | 编码后的 ImageRef（JSON 字符串），由前端节点在用户选择并上传成功后写入 |

## 使用方法

1. 从电池库拖入画布
2. 双击右侧预览区，选择本地图片（支持 PNG、JPG、WebP 等常见格式）
3. 图片预览出现后，节点会把图片上传到资产库并保存 alias
4. 将输出连接到消费 `image` alias 的下游节点

## 典型用法

```
image_reader → image_preview
```
