# 菱形方块地形 (diamond_square)

使用 Diamond-Square（菱形-方块 / 中点位移）算法生成随机分形地形高度图。

## 功能特点

1. **经典分形地形**：Diamond-Square 是最常用的地形生成算法之一，产生自然的山脉、丘陵效果
2. **粗糙度控制**：通过 roughness 参数调节地形从平滑到崎岖的过渡
3. **确定性随机**：相同种子产生相同地形，便于调试和复现
4. **归一化输出**：自动将高度值归一化到 0~1，便于后续处理

## 算法原理

Diamond-Square 算法在正方形网格上交替执行两个步骤：

1. **Diamond 步骤**：对每个方形区域，取四角平均值加随机偏移，设为中心点
2. **Square 步骤**：对每个菱形区域，取四个正交邻居的平均值加随机偏移，设为中心点

每一层迭代后，随机偏移幅度按 `2^(-roughness)` 衰减，网格分辨率加倍，直到填满所有格子。

## 基本使用方法

1. 设置 `power` 确定网格大小（边长 = 2^power + 1）
2. 调整 `roughness` 控制地形起伏程度
3. `grid` 输出为 0~1 归一化高度图，可直接用于渲染或与其他电池叠加

## 输入参数

| 参数名 | 类型 | 默认值 | 范围 | 说明 |
|--------|------|--------|------|------|
| power | number | 7 | 2–10 | 网格幂次，边长 = 2^power + 1 |
| roughness | number | 0.5 | 0–1 | 粗糙度，值越大越崎岖 |
| initHeight | number | 0.5 | 0–1 | 四角初始高度 |
| spread | number | 1.0 | - | 初始扰动幅度 |
| seed | number | 0 | - | 随机种子，0 = 默认种子 |

### power 与网格大小对照

| power | 边长 | 总格数 |
|-------|------|--------|
| 2 | 5 | 25 |
| 5 | 33 | 1,089 |
| 7 | 129 | 16,641 |
| 8 | 257 | 66,049 |
| 9 | 513 | 263,169 |
| 10 | 1025 | 1,050,625 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 地形高度图，值域 0~1（连续浮点值，渲染器以单色透明度渐变显示） |

## 注意事项

1. **渲染显示**：输出为连续浮点值网格（0~1），渲染器自动识别为连续值网格，使用单色 + 透明度渐变显示（值越大越不透明），而非多色分类渲染
2. **网格尺寸**：Diamond-Square 要求边长为 2^n + 1，由 power 参数控制，不支持任意尺寸
3. **粗糙度效果**：roughness=0 接近完全平滑，roughness=1 极度崎岖
4. **性能**：power=9（513×513）约 26 万格子仍可即时完成；power=10（1025×1025）约 100 万格子可能稍有延迟
5. **与 Perlin 噪声区别**：Diamond-Square 产生更大尺度的自相似结构，适合宏观地形；Perlin 噪声更适合细节纹理。两者可通过 blend 电池叠加使用
