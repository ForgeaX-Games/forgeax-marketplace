# OpenSimplex2S 噪声 (opensimplex2s_noise)

基于 FastNoiseLite 的 OpenSimplex2S 噪声生成器，是 OpenSimplex2 的平滑变体（Smooth），质量更高但计算略慢，输出 0~1 值域的二维噪声网格。

## 功能特点

1. **OpenSimplex2S 算法**：OpenSimplex2 的平滑变体，使用更多贡献点计算，产生更圆润的等值线和更丝滑的过渡
2. **4 种分形模式**：单层 / FBm / 脊状 / 乒乓，满足不同地形和纹理需求
3. **完整参数控制**：频率、八度数、间隙度、增益、偏移、种子全部可调
4. **确定性输出**：相同参数和种子始终产生相同结果

## 与 OpenSimplex2 的区别

| 特性 | OpenSimplex2 | OpenSimplex2S |
|------|-------------|---------------|
| 等值线形状 | 略带棱角 | 更圆润平滑 |
| 贡献点数量 | 较少 | 较多 |
| 计算速度 | 较快 | 略慢 |
| 视觉质量 | 良好 | 更高 |

## 适用情况

- 需要高质量平滑噪声的地形高度图
- 生成云彩、水面等需要柔和过渡的自然纹理
- 对噪声质量有较高要求、可接受少量性能开销的场景
- PCG 管线中配合区域划分、纹理混合使用

## 基本使用方法

1. 设置 `width` 和 `height` 确定输出网格尺寸
2. 调整 `frequency` 控制噪声缩放（值越大细节越密）
3. 选择 `fractalType` 和 `octaves` 控制细节层次
4. `grid` 输出值域为 0~1，可直接用于渲染或后续处理

## 输入参数

| 参数名 | 类型 | 默认值 | 范围 | 说明 |
|--------|------|--------|------|------|
| width | number | 128 | 1–1024 | 输出网格宽度 |
| height | number | 128 | 1–1024 | 输出网格高度 |
| frequency | number | 0.02 | - | 噪声采样频率 |
| fractalType | string | "FBm" | 下拉选择 | 分形叠加方式（见下方说明） |
| octaves | number | 4 | 1–8 | 分形叠加层数 |
| lacunarity | number | 2.0 | - | 每层频率倍增系数 |
| gain | number | 0.5 | - | 每层振幅衰减系数 |
| offsetX | number | 0 | - | X 方向采样偏移 |
| offsetY | number | 0 | - | Y 方向采样偏移 |
| seed | number | 1337 | - | 随机种子 |

### fractalType 分形类型

| 值 | 名称 | 效果 |
|----|------|------|
| `None` | 无分形 | 单层 OpenSimplex2S 噪声，最简单平滑 |
| `FBm` | 分形布朗运动 | 多层叠加，自然地形效果，最常用 |
| `Ridged` | 脊状 | 取噪声绝对值的反转，产生山脊/裂缝效果 |
| `PingPong` | 乒乓 | 噪声在 0~1 间来回折叠，产生条纹/波浪纹理 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 噪声网格，值域 0~1 |

## 注意事项

1. **值域映射**：原始 OpenSimplex2S 噪声输出 -1~1，电池内部自动线性映射到 0~1
2. **坐标变换**：算法内部使用 simplex 偏斜变换（F2 skew），将正方形网格坐标变换到三角形网格空间
3. **渲染显示**：输出为连续浮点值网格，渲染器自动识别为连续值网格，使用单色 + 透明度渐变显示
4. **频率选择**：对 128×128 网格，frequency=0.02 约产生 2~3 个噪声周期；frequency=0.05 约 6~7 个周期
5. **分形性能**：octaves 越多计算量线性增长，128×128 网格 8 层仍可即时完成
6. **偏移平移**：修改 offsetX / offsetY 可以在不改变种子的情况下浏览同一噪声场的不同区域
7. **算法来源**：核心算法移植自 FastNoiseLite v1.1.1（MIT License, Jordan Peck 2023）
