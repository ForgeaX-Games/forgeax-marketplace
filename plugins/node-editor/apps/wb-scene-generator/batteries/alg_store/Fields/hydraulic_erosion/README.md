# 水力侵蚀 (hydraulic_erosion)

雨滴模拟水力侵蚀（Sebastian Lague 算法）。在已有的高度图上滚动数万滴雨水，按坡度携带泥沙、平地沉积，反复后山脉自然出现沟壑、河谷与冲积平原。

## 功能特点

1. **基于雨滴的物理模拟**：每滴雨独立运动，记录速度/水量/沉积量，符合真实水力规律。
2. **双线性插值**：高度采样使用双线性插值，输出梯度更平滑，不出现像素化沟壑。
3. **侵蚀刷子**：每次侵蚀按半径以加权方式作用于周围像素，避免单像素抖动。
4. **参数丰富**：惯性、携沙、侵蚀/沉积速率、蒸发、重力、刷子半径全部可调。

## 适用情况

- 给 perlin / simplex / diamond-square 输出的高度图加自然山脉感
- 制作冲积河谷、沉积扇
- 真实感地形预处理（接 grid_classify 区分高地/河谷/平原）

## 基本使用方法

1. 上游接入连续高度图（推荐 0~1）。
2. 默认 50000 滴雨足以产生明显效果；想更精细到 200000 以上。
3. 渲染输出 heightGrid，对比侵蚀前后效果。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| heightGrid | grid | — | 输入高度图（必填） |
| iterations | number | 50000 | 雨滴总数 |
| inertia | number | 0.05 | 方向惯性 0~1 |
| sedimentCapacity | number | 4 | 携沙系数 |
| erodeSpeed | number | 0.3 | 侵蚀速率 |
| depositSpeed | number | 0.3 | 沉积速率 |
| evaporateSpeed | number | 0.01 | 水蒸发速率 |
| gravity | number | 4 | 重力 |
| maxLifetime | number | 30 | 单滴雨寿命 |
| radius | number | 3 | 侵蚀刷子半径 |
| seed | number | 0 | 随机种子（0=时间戳） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| heightGrid | grid | 侵蚀后高度图 |

## 注意事项

1. **耗时**：iterations 与 maxLifetime 决定总成本，128×128 地图、50k 雨滴大约 1~2 秒；超大地图请相应减少 iterations。
2. **刷子半径**：增大 radius 会使沟壑更宽，但牺牲细节。
3. **参数互锁**：默认值是经过验证的良好预设，调参时建议一次只改一个参数观察影响。
4. **输入未归一化**：若高度图值很大（如 0~100），可能需相应调小 erodeSpeed / depositSpeed，否则一步沉积过多导致地形塌方。
