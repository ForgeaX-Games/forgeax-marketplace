# 热侵蚀 (thermal_erosion)

经典热风化（talus angle）模拟：邻居高差超过临界坡度的格子向较低邻居输送材料，反复迭代后陡崖崩塌为碎石坡。

## 功能特点

1. **物理直观**：基于"超过临界坡度即崩塌"的简单规则，参数少而有效。
2. **8 邻居支持**：默认对角参与，崩塌方向更自然。
3. **守恒**：双缓冲实现，确保每次迭代总材料量不变。
4. **配合水力侵蚀**：通常先热侵蚀（崩平陡崖）再水力侵蚀（开沟壑），效果最好。

## 适用情况

- 山脉边缘的碎石坡 / 山脚扇形堆积
- 软化噪声生成的过陡崖壁
- 配合高度图二次处理，模拟长期风化

## 基本使用方法

1. 输入高度图（可来自 perlin_noise / diamond-square / midpoint-displacement）。
2. 默认参数 50 次迭代、talusAngle=0.05、rate=0.5 适用于 0~1 归一化的高度图。
3. 视觉过软可调小 talusAngle（更多坡度被认为陡）；过硬则调大。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| heightGrid | grid | — | 输入高度图 |
| iterations | number | 50 | 整图扫描次数 |
| talusAngle | number | 0.05 | 临界坡度阈值 |
| transportRate | number | 0.5 | 单次输送比例 0~1 |
| diagonal | boolean | true | 是否考虑对角邻居 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| heightGrid | grid | 风化后高度图 |

## 注意事项

1. **talusAngle 单位与高度图一致**：若高度图值范围 0~100，请把 talusAngle 调到 5 左右；0~1 归一化时用默认 0.05。
2. **迭代过多会让山脉整体变平**：通常 30~100 次足够；超过 200 次往往过度。
3. **建议串联顺序**：原始噪声 → 热侵蚀 → 水力侵蚀，得到自然山脉效果。
