# 掩码应用 (grid_mask_apply)

将二值掩码应用到网格上，保留掩码为 1 的位置的原始值，掩码为 0 的位置全部置为 0。适合用于裁剪、区域提取等场景。

## 功能特点

- 简单掩码过滤：`mask = 1 → 保留原值，mask = 0 → 置零`
- 两个输入网格尺寸必须相同
- 输出网格与输入网格尺寸一致

## 输入参数

| 参数 | 类型 | 说明 |
|------|------|------|
| grid | grid | 待过滤的输入网格 |
| mask | grid | 二值掩码网格（0/1），尺寸需与输入网格相同 |

## 输出参数

| 参数 | 类型 | 说明 |
|------|------|------|
| grid | grid | 过滤后的网格 |

## 使用示例

- Perlin 噪声 → 网格二值化 → **掩码应用**（原始噪声 + 二值掩码） → 提取噪声中的特定区域
- 地形高度图 + 洞穴掩码 → **掩码应用** → 仅保留洞穴区域的地形数据
- 任意网格 + 细胞自动机输出 → **掩码应用** → 用 CA 生成的形状裁剪网格

## 注意事项

- 掩码网格应为二值网格（值仅为 0 或 1），非 1 的值均视为 0（不保留）
- 两个网格尺寸不一致时返回空网格
- 若未提供掩码，则直接返回原始网格
