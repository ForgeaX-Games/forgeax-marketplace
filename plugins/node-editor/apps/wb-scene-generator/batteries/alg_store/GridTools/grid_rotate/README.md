# 网格旋转 (grid_rotate)

将二维网格按 90°、180°、270° 顺时针旋转，输出旋转后的新网格。

## 功能特点

1. **三种角度**：支持 90°、180°、270° 顺时针旋转
2. **尺寸自适应**：90°/270° 旋转后行列数自动互换，180° 保持原尺寸

## 适用情况

- 地图/关卡旋转变体生成
- 对称图案构建时旋转部件
- 瓷砖/贴图预处理

## 基本使用方法

1. 将网格连接到 `输入网格` 端口
2. 从下拉框选择 `旋转角度`（90 / 180 / 270）
3. 从 `旋转网格` 端口获取输出

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 待旋转的二维网格 |
| angle | string | 90 | 顺时针旋转角度：90、180、270 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 旋转后的二维网格 |

## 注意事项

1. **尺寸变化**：90° 和 270° 旋转后网格的行数和列数会互换（如 3×5 变为 5×3）
2. **空网格**：输入空网格时返回错误提示
