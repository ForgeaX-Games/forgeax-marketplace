# 网格二值化 (grid_binarize)

将输入网格按阈值二值化，大于阈值的格子变为 1，否则变为 0。适合将 Perlin 噪声、地形高度图等连续值网格转换为二值掩码。

## 功能特点

- 简单阈值二值化：`value > threshold → 1, otherwise → 0`
- 可配置阈值，默认 0.5
- 输出纯二值网格（0/1），可直接作为其它电池的掩码输入

## 输入参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| grid | grid | — | 待二值化的输入网格 |
| threshold | number | 0.5 | 二值化阈值，值 > 阈值 → 1，否则 → 0 |

## 输出参数

| 参数 | 类型 | 说明 |
|------|------|------|
| grid | grid | 二值化后的网格，值仅为 0 或 1 |

## 使用示例

- Perlin 噪声 → 网格二值化（threshold=0.5）→ 得到随机岛屿掩码
- 地形高度图 → 网格二值化（threshold=0.3）→ 提取低海拔区域
- 任意连续值网格 → 网格二值化 → 作为细胞自动机、随机行走等电池的输入掩码

## 注意事项

- 判定条件为严格大于（`>`），等于阈值的格子归为 0
- 对于全零网格，输出仍为全零
