# 网格混合 (grid_blend)

将两个相同尺寸的网格按混合系数线性插值，输出混合后的网格。

## 功能特点

1. **线性插值**：`output = A * (1 - factor) + B * factor`，简单高效
2. **系数钳位**：factor 自动限制在 0~1 范围内
3. **容错处理**：若仅一个网格有效，直接输出该网格；尺寸不匹配时输出空网格

## 适用情况

- 混合两张噪声图（如 Perlin + 随机行走）产生复合地形
- 在两个掩码之间做平滑过渡
- 调节 factor 实现"淡入淡出"动画效果
- PCG 管线中对多个生成结果做加权叠加

## 基本使用方法

1. 连接两个网格到 `gridA` 和 `gridB` 输入端口
2. 设置 `factor`（0 = 完全 A，1 = 完全 B，0.5 = 各半）
3. `grid` 输出即为混合结果

## 输入参数

| 参数名 | 类型 | 默认值 | 范围 | 说明 |
|--------|------|--------|------|------|
| gridA | grid | - | - | 第一个输入网格 |
| gridB | grid | - | - | 第二个输入网格（尺寸需与 A 相同） |
| factor | number | 0.5 | 0–1 | 混合系数 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 混合后的网格 |

## 注意事项

1. **尺寸要求**：两个输入网格的行数和列数必须完全相同，否则输出空网格
2. **值域**：输出值域取决于输入，若 A 和 B 都在 0~1 范围内，输出也在 0~1 内
3. **单输入容错**：若只有一个网格有效（另一个为空或未连接），直接透传该网格
