# 网格裁剪 (grid_crop)

从二维网格中裁剪指定矩形区域，通过起始行列和宽高定义裁剪范围。

## 功能特点

1. **精确定位**：通过起始行、起始列指定裁剪起点
2. **灵活尺寸**：宽度/高度为 0 时自动裁到边界，方便截取尾部区域
3. **边界安全**：自动将超出范围的参数裁剪到有效区间

## 适用情况

- 从大地图中提取局部区域
- 去除网格边缘的无用区域
- 从生成结果中截取感兴趣的子区域

## 基本使用方法

1. 将网格连接到 `输入网格` 端口
2. 设置 `起始行` 和 `起始列` 指定裁剪起点
3. 设置 `裁剪宽度` 和 `裁剪高度`（0 = 裁到边界）
4. 从 `裁剪网格` 端口获取输出

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 待裁剪的二维网格 |
| startRow | number | 0 | 裁剪区域左上角行索引（从 0 开始） |
| startCol | number | 0 | 裁剪区域左上角列索引（从 0 开始） |
| cropWidth | number | 0 | 裁剪列数，0 表示裁到最右列 |
| cropHeight | number | 0 | 裁剪行数，0 表示裁到最底行 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 裁剪后的二维网格 |

## 注意事项

1. **越界处理**：起始位置超出网格范围时会被自动限制到有效区间
2. **空结果**：裁剪区域完全在网格外时返回错误提示
3. **索引从 0 开始**：第一行/列的索引为 0
