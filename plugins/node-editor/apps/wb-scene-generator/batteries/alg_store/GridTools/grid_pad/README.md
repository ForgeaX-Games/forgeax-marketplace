# 网格填充边界 (grid_pad)

在二维网格的上、下、左、右四个方向分别填充指定宽度的边界行列，支持自定义填充值。

## 功能特点

1. **四方向独立控制**：上下左右边距可分别设置不同值
2. **自定义填充值**：边界区域可填充任意数值（默认 0）
3. **原数据保留**：原网格内容完整保留在新网格中心区域

## 适用情况

- 为地图/关卡添加边界围墙（填充 1）
- 算法预处理时扩展网格防止越界
- 为噪声图添加留白区域

## 基本使用方法

1. 将网格连接到 `输入网格` 端口
2. 分别设置 `上边距`、`下边距`、`左边距`、`右边距`
3. 设置 `填充值`（如 0 表示空地，1 表示墙壁）
4. 从 `填充网格` 端口获取输出

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 待填充边界的二维网格 |
| top | number | 1 | 顶部填充行数 |
| bottom | number | 1 | 底部填充行数 |
| left | number | 1 | 左侧填充列数 |
| right | number | 1 | 右侧填充列数 |
| fillValue | number | 0 | 边界区域填充的数值 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 填充边界后的二维网格 |

## 注意事项

1. **负数处理**：边距值小于 0 会被自动置为 0
2. **尺寸变化**：输出网格尺寸 = 原尺寸 + 上下边距之和（行） + 左右边距之和（列）
3. **填充值类型**：填充值为数值类型，非数值输入默认为 0
