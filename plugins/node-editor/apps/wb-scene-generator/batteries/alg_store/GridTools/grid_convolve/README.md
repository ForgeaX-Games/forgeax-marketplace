# 网格卷积 (grid_convolve)

使用预设或自定义卷积核对二维网格进行卷积运算，支持多种边界填充模式和归一化选项。

## 功能特点

1. **10 种预设核**：均值模糊（3x3/5x5）、高斯模糊（3x3/5x5）、锐化、拉普拉斯边缘检测、Sobel 边缘检测（X/Y）、浮雕
2. **自定义核**：支持任意奇数尺寸的自定义卷积核
3. **4 种边界填充**：补零（zero）、复制边缘（clamp）、环绕（wrap）、镜像反射（reflect）
4. **智能归一化**：自动/手动/关闭三种归一化模式

## 适用情况

- 地形网格模糊/平滑处理
- 边缘检测（找到地形变化的边界）
- 锐化增强地形细节
- 自定义滤波器实现特殊效果

## 基本使用方法

1. 将网格连接到 `输入网格` 端口
2. 从 `预设核` 下拉框选择卷积核类型（或选 custom 使用自定义核）
3. 如选 custom，将二维数组连接到 `自定义核` 端口
4. 从 `边界填充` 下拉框选择边界处理方式
5. 从 `卷积结果` 端口获取输出

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 待卷积的二维网格 |
| preset | string | blur3x3 | 预设核类型（见下方参数说明） |
| kernel | grid | [] | 自定义卷积核（二维数组，行列须为奇数），仅 preset=custom 时生效 |
| padding | string | clamp | 边界填充模式（见下方参数说明） |
| normalize | string | auto | 归一化模式（见下方参数说明） |

### 参数说明

**预设核 (preset)**

| 值 | 尺寸 | 效果 |
|----|------|------|
| blur3x3 | 3×3 | 均值模糊 |
| blur5x5 | 5×5 | 均值模糊（更强） |
| gaussian3x3 | 3×3 | 高斯模糊 |
| gaussian5x5 | 5×5 | 高斯模糊（更强） |
| sharpen | 3×3 | 锐化增强 |
| edge_laplacian | 3×3 | 拉普拉斯边缘检测 |
| edge_sobel_x | 3×3 | Sobel 水平边缘检测 |
| edge_sobel_y | 3×3 | Sobel 垂直边缘检测 |
| emboss | 3×3 | 浮雕效果 |
| custom | — | 使用自定义核 |

**边界填充 (padding)**

| 值 | 说明 |
|----|------|
| zero | 边界外视为 0 |
| clamp | 复制最近边缘值 |
| wrap | 环绕（对端取值） |
| reflect | 镜像反射 |

**归一化 (normalize)**

| 值 | 说明 |
|----|------|
| auto | 模糊类预设自动归一化，边缘检测/锐化类不归一化 |
| yes | 强制归一化（核权重和为 1） |
| no | 不归一化 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 卷积运算后的二维网格 |

## 注意事项

1. **自定义核尺寸**：行数和列数必须为奇数（如 3×3、5×5、7×7），偶数尺寸会报错
2. **边缘检测输出**：边缘检测核（laplacian/sobel）的输出可能包含负值和零，适合配合二值化电池做后处理
3. **性能**：卷积是 O(rows × cols × kernelSize²) 的运算，大网格配合大核时计算量较大
4. **归一化**：模糊类核需归一化以保持值范围不变；边缘检测核不应归一化，否则效果会被稀释
