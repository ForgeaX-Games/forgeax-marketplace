# 网格缩放 (grid_resize)

将输入网格缩放到指定尺寸，支持三种插值方法。

## 功能特点

1. **三种插值方法**：最近邻（像素风）、双线性（平滑）、双三次 Catmull-Rom（最平滑）
2. **任意缩放**：支持放大和缩小，输入输出尺寸无需是 2 的幂
3. **边界处理**：采样超出边界时自动钳位到最近边缘像素

## 适用情况

- 将小尺寸地形高度图（如 Diamond-Square 的 129×129）放大到渲染所需的分辨率
- 缩小大网格以降低后续处理计算量
- 在不同分辨率的网格之间做转换以便混合（blend）

## 基本使用方法

1. 连接输入 `grid`
2. 设置 `width` 和 `height` 为目标尺寸
3. 选择 `method` 插值方法
4. `grid` 输出即为缩放后的网格

## 输入参数

| 参数名 | 类型 | 默认值 | 范围 | 说明 |
|--------|------|--------|------|------|
| grid | grid | - | - | 待缩放的输入网格 |
| width | number | 256 | 1–2048 | 目标宽度 |
| height | number | 256 | 1–2048 | 目标高度 |
| method | string | "bilinear" | 下拉选择 | 插值方法（见下方说明） |

### method 插值方法

| 值 | 名称 | 效果 |
|----|------|------|
| `nearest` | 最近邻 | 直接取最近像素值，无平滑，锐利的像素风格 |
| `bilinear` | 双线性 | 对 2×2 邻域线性插值，平滑自然，通用选择 |
| `bicubic` | 双三次 | 对 4×4 邻域 Catmull-Rom 插值，最平滑，保留更多细节 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 缩放后的网格 |

## 注意事项

1. **尺寸相同**：若输入输出尺寸完全一致，直接返回输入的副本，无额外计算
2. **值域**：缩放不改变值域范围；双三次插值在极端梯度处可能产生微小的过冲/欠冲
3. **性能**：双三次对每个像素采样 4×4 = 16 个邻居，大网格（>512×512）时比双线性慢约 4 倍
4. **典型管线**：Diamond-Square (129×129) → 网格缩放 (512×512, bicubic) → blend with Perlin noise
