# 最大矩形查找 (max_rectangle)

在网格的非零区域中查找面积最大的轴对齐矩形，输出矩形掩码网格及宽度、高度、面积。

## 功能特点

1. **柱状图 + 单调栈算法**：将二维问题转化为逐行的"柱状图中最大矩形"问题，时间复杂度 O(rows × cols)
2. **轴对齐矩形**：查找的矩形边与网格行列平行，适用于建筑占地、功能区划分等场景
3. **掩码输出**：结果为与输入同尺寸的 0/1 网格，可直接叠加或传给下游电池

## 适用情况

- 在不规则地形中寻找最大可建造区域
- 从噪声/采样生成的非零区域中提取最大矩形平台
- 配合其他算法（如细胞自动机、Voronoi 地形）做后处理，定位最大连通平面

## 基本使用方法

1. 将包含非零区域的网格连接到 `grid` 输入端口
2. 执行后从 `outputGrid` 获取矩形掩码（1 = 矩形内，0 = 矩形外）
3. 从 `width`、`height`、`area` 端口获取矩形尺寸信息

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 输入网格，非零值为可用区域，零值为障碍 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 与输入同尺寸的掩码网格，最大矩形区域为 1，其余为 0 |
| width | number | 最大矩形的宽度（列数） |
| height | number | 最大矩形的高度（行数） |
| area | number | 最大矩形的面积（宽度 × 高度） |

## 注意事项

1. **非零即可用**：输入网格中任何非零值（正数、负数、浮点数）均视为可用区域，仅值为 0 的格子视为障碍
2. **多个等面积矩形**：若存在多个面积相同的最大矩形，算法返回最先扫描到的（行优先、左优先）
3. **全零网格**：若输入全为 0，输出矩形尺寸均为 0，掩码网格全 0
