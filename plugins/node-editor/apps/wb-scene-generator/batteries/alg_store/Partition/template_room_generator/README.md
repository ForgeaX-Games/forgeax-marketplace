# 模板房间生成 (template_room_generator)

在网格上放置预定义房间模板并通过走廊连接。算法通过约束随机选择模板、随机旋转、走廊挖掘和连通性修复，生成多样化的房间布局。

## 功能特点

1. **7 种内置房间模板**：空房间、中心柱块、柱阵、F 形隔墙、环形房间、条纹走廊、双通道房间
2. **模板约束放置**：支持出现次数上限和相邻不重复约束，避免单调
3. **随机旋转**：每个模板随机旋转 0°/90°/180°/270°，增加多样性
4. **自动走廊连接**：相邻房间通过可调宽度的走廊连接
5. **连通性保证**：通过 Flood Fill 检测孤立区域，多源 BFS 搜索最近点对，L 形走廊修复连通
6. **确定性随机**：相同种子产生相同地图，方便调试和复现
7. **三值输出网格**：地板(0) / 墙壁(1) / 柱子(2)，语义清晰，便于差异化渲染

## 适用场景

- Roguelike / 地牢爬行游戏的关卡地图生成
- 后室（Backrooms）风格迷宫场景
- 室内建筑平面图自动布局
- 作为后续细化（道具分布、NPC 放置、光照设计）的基础布局

## 基本使用方法

1. 将节点拖入画布，设置网格宽高和各项参数
2. 执行后从 `grid` 输出获取二维网格（0=地板，1=墙壁，2=柱子）
3. 可接入渲染节点或下游处理节点（如 A* 寻路、细胞自动机等）

## 算法流程

1. **初始化**：创建指定尺寸的全墙壁网格，根据 cellSize 计算可容纳的房间行列数
2. **模板放置**：遍历房间网格，每个槽位随机选择模板（约束：出现上限 + 相邻不重复 + 留空概率），随机旋转后印入网格。模板中 `'.'` → 地板(0)，`'#'` → 柱子(2)
3. **走廊连接**：对水平/垂直相邻的已放置房间，在间隙处挖掘指定宽度的走廊
4. **Flood Fill**：对所有地板格进行泛洪填充，标记连通区域
5. **连通修复**：对每个孤立区域，多源 BFS 搜索到主区域的最近点对，挖掘 L 形走廊连接
6. **封边**：强制四周边界为实体墙

## 网格值语义

| 值 | 含义 | 来源 | 可通行 |
|----|------|------|--------|
| 0 | 地板 | 模板中的 `'.'` 字符、走廊挖掘、连通修复 | 是 |
| 1 | 墙壁 | 初始填充的结构边界、房间之间的间隔 | 否 |
| 2 | 柱子 | 模板中的 `'#'` 字符（房间内部的柱体、隔墙等障碍物） | 否 |

墙壁(1)和柱子(2)的区别：**墙壁**是房间之间的结构性边界（从未被房间模板覆盖的区域），**柱子**是房间内部由模板定义的障碍物（柱体、隔墙、围栏等）。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 100 | 网格宽度（30~512） |
| height | number | 100 | 网格高度（30~512） |
| cellSize | number | 12 | 房间槽位的瓦片边长（10~40），决定房间间距 |
| emptyChance | number | 0.1 | 空槽概率（0~0.5），越大空旷区域越多 |
| corridorWidth | number | 3 | 走廊宽度（1~7 瓦片） |
| minRegionSize | number | 10 | 孤立区域最小面积，低于此值不连接 |
| seed | number | 0 | 随机种子，0 使用默认种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 地图网格：0=地板，1=墙壁，2=柱子 |
| gridWidth | number | 输出网格的实际宽度 |
| gridHeight | number | 输出网格的实际高度 |

## 内置模板说明

| 编号 | 名称 | 最大使用次数 | 说明 |
|------|------|-------------|------|
| 0 | 空房间 | 3 | 全空的开放空间 |
| 1 | 中心柱块 | 不限 | 中央有大型方形柱体 |
| 2 | 柱阵 | 不限 | 均匀排列的柱子网格 |
| 3 | F 形隔墙 | 2 | F 型隔墙分割空间 |
| 4 | 环形房间 | 不限 | 矩形环状围墙 |
| 5 | 条纹走廊 | 不限 | 水平交替的墙壁和走廊 |
| 6 | 双通道房间 | 不限 | 上下两个封闭区域，中间通道连接 |

## 注意事项

1. **cellSize 最小为 10**：内置模板均为 10×10，cellSize 设为 10 时模板恰好填满槽位
2. **实际房间数**：房间列数 = floor((width − 1) / cellSize)，行数同理；若网格较小或 cellSize 较大，可能只容纳少量房间
3. **大地图注意性能**：width 和 height 设为最大值(512)且有多个孤立区域时，BFS 连通修复可能较慢
4. **走廊长度**：相邻房间之间的走廊实际长度取决于 cellSize 与模板尺寸(10)的差值
