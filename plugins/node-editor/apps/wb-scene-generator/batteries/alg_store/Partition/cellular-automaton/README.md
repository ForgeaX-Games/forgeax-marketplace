# 细胞自动机 (cellular_automaton)

细胞自动机算法电池，在网格内按诞生/存活阈值对二值化网格进行迭代演化。常用于程序化生成洞穴、岛屿、有机形态等地形结构。

## 功能特点

- Moore 邻域（8 方向）细胞自动机
- 可配置诞生阈值和存活阈值
- **三种初始化模式**：random、threshold、binary
- 可选边界处理策略（边界视为存活或死亡）
- 确定性随机种子，相同参数产生相同结果

## 初始化模式说明

### random 模式（默认）

仅使用输入网格的**尺寸**（行列数），忽略所有格子的值。所有格子按 `initProb` 概率随机初始化为存活(1)或死亡(0)。适合不依赖上游网格内容、纯随机生成的场景。

### threshold 模式

使用输入网格的实际值进行二值化。二值化阈值 = `1 - initProb`，即网格值 >= 阈值的格子为存活(1)，否则为死亡(0)。对于值域在 [0, 1] 均匀分布的输入（如 Perlin 噪声），约有 `initProb` 比例的格子变为存活。

例如：`initProb = 0.45` → 阈值 = 0.55 → 值 >= 0.55 的格子存活 → 约 45% 存活。

### binary 模式

检测输入网格是否为二值网格（所有值均为 0 或 1）。如果是，直接使用网格值作为初始状态。如果不是（存在 0、1 以外的值），自动回退到 threshold 模式进行二值化。

适合接收来自其它二值化电池（如网格二值化）或细胞自动机的输出，进行二次迭代。

## 算法流程

1. **初始化**：根据 `initMode` 选择初始化方式
   - random：仅用网格尺寸，所有格子按 `initProb` 随机设定
   - threshold：按 `1 - initProb` 阈值对网格值二值化
   - binary：检测是否为二值网格，是则直接使用，否则回退到 threshold
2. **迭代演化**：每次迭代对所有格子执行以下规则：
   - **诞生**：死亡格子的 8 个邻居中，存活数 >= `birthThreshold` → 变为存活
   - **存活**：存活格子的 8 个邻居中，存活数 >= `survivalThreshold` → 保持存活
   - **死亡**：存活格子的 8 个邻居中，存活数 < `survivalThreshold` → 变为死亡
3. 重复步骤 2 共 `iterations` 次
4. 输出最终网格

## 输入参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| grid | grid | — | 输入网格；random 模式仅用尺寸，threshold 按值二值化，binary 直接使用 |
| initMode | string | "random" | 初始化模式：random / threshold / binary |
| initProb | number | 0.45 | random：存活概率；threshold/binary 回退：阈值 = 1 - initProb |
| birthThreshold | number | 5 | 诞生阈值（0~8），邻居存活数 >= 此值时死亡格子诞生 |
| survivalThreshold | number | 4 | 存活阈值（0~8），邻居存活数 >= 此值时存活格子保持 |
| iterations | number | 5 | 迭代次数（1~100），越多结构越平滑 |
| edgeAlive | boolean | true | 边界视为存活，开启可生成封闭洞穴，关闭可生成开放岛屿 |
| seed | number | 0 | 随机种子（仅 random 模式使用），0 使用默认种子 |

## 输出参数

| 参数 | 类型 | 说明 |
|------|------|------|
| grid | grid | 演化后的网格，存活=1，死亡=0 |

## 常用参数组合

| 场景 | initMode | initProb | birth | survival | iterations | edgeAlive |
|------|----------|----------|-------|----------|------------|-----------|
| 随机洞穴 | random | 0.45 | 5 | 4 | 5 | 1 |
| 噪声驱动洞穴 | threshold | 0.45 | 5 | 4 | 5 | 1 |
| 二值网格平滑 | binary | — | 5 | 4 | 3 | 1 |
| 开放岛屿 | random | 0.40 | 5 | 4 | 5 | 0 |
| 密集迷宫 | random | 0.50 | 5 | 3 | 3 | 1 |

## 注意事项

- random 模式下所有格子均参与模拟，输入网格值被完全忽略
- threshold 和 binary 模式下，值为 0 的格子初始为死亡但仍参与演化（可被诞生）
- binary 模式自动检测，非二值网格会静默回退到 threshold 模式
- `edgeAlive=1` 时，网格边界外被计为存活邻居，有助于生成封闭洞穴
- `edgeAlive=0` 时，边界外视为死亡，适合生成开放式岛屿结构
- 迭代次数过多会导致结构趋于稳定（不再变化）
