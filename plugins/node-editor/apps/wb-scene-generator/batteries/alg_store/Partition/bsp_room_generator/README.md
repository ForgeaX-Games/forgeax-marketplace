# BSP房间生成 (bsp_room_generator)

使用二叉空间分割（Binary Space Partition）算法递归划分二维空间，在叶节点中随机放置矩形房间。

## 功能特点

1. **BSP 递归分割**：根据分割比例范围随机切分空间，偏长轴优先，保证分区均匀
2. **随机房间放置**：在每个叶分区内按最小/最大尺寸约束随机生成房间，带墙壁边距控制
3. **确定性随机**：相同种子产生相同布局，方便调试和复现
4. **多输出**：同时输出网格、房间信息列表和房间数量

## 适用情况

- Roguelike 地牢关卡的房间布局
- 室内建筑平面图划分
- 任何需要非重叠矩形区域的 PCG 场景
- 作为后续细化（走廊连接、家具摆放、怪物分布等）的基础布局

## 基本使用方法

1. 将节点拖入画布，设置网格宽高和房间尺寸参数
2. 执行后从 `grid` 输出获取二维网格，`rooms` 获取房间列表
3. 可接入渲染节点或下游处理节点（如 A* 寻路、细胞自动机等）

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 50 | 网格宽度（16~512） |
| height | number | 50 | 网格高度（16~512） |
| minRoomSize | number | 5 | 房间最小边长（3~50），分区小于此值两倍时停止分割 |
| maxRoomSize | number | 12 | 房间最大边长（3~100），房间尺寸在最小最大间随机 |
| minSplitRatio | number | 0.4 | 分割位置下限（0.2~0.5） |
| maxSplitRatio | number | 0.6 | 分割位置上限（minSplitRatio~0.8） |
| wallPadding | number | 1 | 房间与分区边界的间距（0~5），增大可让房间更稀疏 |
| seed | number | 0 | 随机种子，0 使用默认种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 输出网格：0=墙壁，正整数=房间 ID |
| rooms | array | 房间信息数组 [{id, x, y, w, h, centerX, centerY}] |
| numRooms | number | 生成的房间总数 |

## 注意事项

1. **房间 ID 从 1 开始**：网格中 0 表示墙壁，正整数表示对应房间 ID
2. **尺寸约束**：如果 minRoomSize 设置过大而网格过小，可能只生成极少房间甚至无房间
3. **maxRoomSize 自动钳位**：如果 maxRoomSize 小于 minRoomSize，会被自动提升到 minRoomSize
