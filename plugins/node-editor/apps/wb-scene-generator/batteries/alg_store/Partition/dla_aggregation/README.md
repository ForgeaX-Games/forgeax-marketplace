# 扩散限制聚集 (dla_aggregation)

模拟扩散限制聚集（Diffusion-Limited Aggregation, DLA）过程：大量粒子在网格掩码内进行随机游走，当粒子与已有聚集结构接触时以一定概率粘附，最终形成分形分支图案。

DLA 是一种经典的分形生长模型，由 Witten 和 Sander 于 1981 年提出，广泛应用于模拟自然界中的枝状生长现象。

## 功能特点

1. **经典 DLA 引擎**：忠实实现扩散限制聚集算法，粒子随机游走 + 邻接粘附
2. **可调粘附概率**：`stickiness` 参数从根本上改变输出形态——高粘附产生稀疏树枝，低粘附产生密实团块
3. **5 种种子模式**：中心、随机、底边、边缘、均匀散布，适配不同生长起点需求
4. **方向偏向**：可选的行走偏向角度和强度，实现各向异性生长（如偏向上方生长的树形）
5. **双连通模式**：4 连通产生锐利棱角的分支，8 连通产生圆润粗壮的分支
6. **性能优化**：包围盒自适应发射区 + 击杀区，避免粒子在远离结构的空旷区域浪费步数
7. **确定性复现**：固定 seed 产生完全相同的结果

## 适用场景

- **珊瑚/海绵结构**：水下场景的装饰性地形
- **闪电/裂纹**：从一个点向外扩散的裂纹、闪电纹路
- **矿脉/矿石分布**：地下岩层中的矿脉网络（配合低粘附率产生密实矿体）
- **霜花/冰晶**：冬季场景的窗户霜花或地面冰晶
- **洞穴侵蚀纹理**：岩壁上的侵蚀裂纹、苔藓生长痕迹
- **程序化废墟**：建筑墙面上的裂纹扩展
- **与其他电池组合**：作为 mask 传递给噪声或细胞自动机电池，产生混合纹理

## 算法原理

### 1. 种子放置

在掩码区域内放置初始种子格子，作为聚集结构的起点：

| 模式 | 行为 |
|------|------|
| center | 在掩码中心放置（按到中心距离排序取最近的 N 个） |
| random | 在掩码内随机放置 |
| bottom | 在掩码底边放置 |
| edges | 在掩码边界（与空区域相邻的格子）上放置 |
| scatter | 最远点采样，均匀散布在整个掩码内 |

### 2. 粒子发射

每个粒子从聚集结构包围盒附近的随机位置发射：

- **发射区**：包围盒向外扩展 `margin` 格（margin = max(5, 对角线长度 × 50%)）
- **击杀区**：包围盒向外扩展 `2 × margin` 格
- 粒子超出击杀区时被放弃，避免无效游走
- 发射区和击杀区随结构生长自动扩大

### 3. 随机游走与粘附

每步执行：

1. **邻接检测**：检查当前位置的邻居是否有聚集结构格子
2. **粘附判定**：如果相邻，以概率 `stickiness` 粘附
   - 粘附成功 → 加入结构，更新包围盒，完成该粒子
   - 粘附失败（弹跳）→ 继续行走
3. **随机步进**：选择一个方向移动一格（可带偏向）
   - 目标格必须在网格内、在掩码内、不在现有结构内
   - 不满足则原地不动
4. **击杀检测**：超出击杀区则放弃该粒子

### 4. 方向偏向

当 `biasStrength > 0` 时，每步有 `biasStrength` 的概率向 `biasAngle` 方向移动：

$$P(\text{向偏向方向}) = \text{biasStrength}$$
$$P(\text{随机方向}) = 1 - \text{biasStrength}$$

偏向角度使用与其他电池相同的约定：0° = 上，90° = 右，180° = 下，270° = 左。

### 5. 粘附概率对形态的影响

这是 DLA 最关键的参数：

| stickiness | 形态 | 物理类比 |
|-----------|------|----------|
| 1.0 | 极度稀疏的树枝状分形 | 理想 DLA（极端非平衡生长） |
| 0.3~0.5 | 中等密度的分支结构 | 电化学沉积 |
| 0.05~0.2 | 较密实的团状结构 | 缓慢结晶 |
| 0.01~0.03 | 近似圆形的密实团块 | 接近平衡态的结晶 |

原理：低粘附率使粒子在结构表面多次弹跳，有更多机会进入凹陷处粘附，从而填充间隙，产生更密实的结构。

## 基本使用方法

1. 连接一个网格（如矩形全 1 掩码）到 `grid` 输入端口
2. 选择 `seedMode`（推荐从 center 开始）
3. 调整 `particleCount` 控制结构大小（建议 200~800）
4. 用 `stickiness` 控制分支形态（1.0=稀疏树枝，0.1=密实团块）
5. 设置 `seed=0` 获得随机结果，或指定固定值重复相同布局

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 输入掩码，非零区域为仿真域 |
| particleCount | number | 300 | 聚集粒子数量（10~5000） |
| seedMode | string | center | 种子放置模式（center/random/bottom/edges/scatter） |
| seedCount | number | 1 | 种子数量（1~20） |
| stickiness | number | 1.0 | 粘附概率（0.01~1.0），核心形态参数 |
| neighborMode | string | 8 | 邻接模式（4 或 8） |
| biasAngle | number | 0 | 偏向角度（度，0~360），biasStrength>0 时生效 |
| biasStrength | number | 0 | 偏向强度（0~0.5），0=无偏向 |
| maxStepsPerParticle | number | 5000 | 每粒子最大步数（100~50000） |
| seed | number | 0 | 随机种子，0=自动随机 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 输出网格，聚集结构=1，其余=0 |

## 参数调节指南

### 分支形态
- 更稀疏/分叉：提高 `stickiness`（如 0.8~1.0），减少 `particleCount`
- 更密实/紧凑：降低 `stickiness`（如 0.05~0.2），增加 `particleCount`
- 更粗壮/圆润：使用 `neighborMode=8`
- 更锐利/棱角：使用 `neighborMode=4`

### 方向性生长
- 各向同性（径向扩展）：`biasStrength=0`
- 偏向上方生长（树形）：`biasAngle=0, biasStrength=0.15`
- 偏向右方扩展：`biasAngle=90, biasStrength=0.15`

### 多中心生长
- 单棵树：`seedMode=center, seedCount=1`
- 多棵树合并：`seedMode=scatter, seedCount=5`
- 从底部向上生长：`seedMode=bottom, seedCount=3, biasAngle=0, biasStrength=0.1`
- 从边缘向内侵蚀：`seedMode=edges, seedCount=8`

## 使用示例

### 经典 DLA（中心生长）

```json
{
  "grid": [[1,1,1,...],[1,1,1,...],...]，
  "particleCount": 500,
  "seedMode": "center",
  "seedCount": 1,
  "stickiness": 1.0,
  "seed": 42
}
```

### 密实矿脉

```json
{
  "grid": [[1,1,1,...],[1,1,1,...],...]，
  "particleCount": 800,
  "seedMode": "scatter",
  "seedCount": 3,
  "stickiness": 0.1,
  "neighborMode": "8",
  "seed": 123
}
```

### 输出

```json
{
  "grid": [[0,0,0,...],[0,1,0,...],[0,1,1,...],...]
}
```

## 注意事项

1. **性能**：`particleCount × maxStepsPerParticle` 决定最大计算量。默认 300 × 5000 = 150 万步，在 50×50 网格上通常瞬间完成。超大参数（5000 × 50000）可能需要数秒
2. **粘附概率是核心参数**：`stickiness` 对输出形态的影响远大于其他参数，建议优先调节
3. **包围盒优化**：粒子从结构附近发射、远离结构时被放弃，因此实际行走步数远低于 maxStepsPerParticle
4. **掩码形状**：DLA 输出自然受限于掩码区域。不规则掩码会产生受限方向的分支生长
5. **随机性**：DLA 本质上是随机过程，相同参数不同 seed 会产生完全不同的分支形态。`seed=0` 每次运行随机
6. **与其他电池组合**：输出可通过 blend 电池与地形叠加，或作为 mask 传递给噪声电池增加细节
