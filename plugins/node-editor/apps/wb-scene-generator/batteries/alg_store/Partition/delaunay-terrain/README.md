# Delaunay 三角剖分 (delaunay_terrain)

基于 Bowyer-Watson 算法对种子点进行 Delaunay 三角剖分，将网格按三角形区域划分。每个三角形赋予唯一正整数 ID，凸包外区域为 0。支持自定义种子点或随机生成，并可通过 Lloyd 松弛使三角形更规则。

## 功能特点

1. **Bowyer-Watson 算法**：经典增量式 Delaunay 三角剖分，自动满足空圆性质（无点落在任何三角形外接圆内）
2. **灵活种子点**：支持传入自定义种子点坐标（归一化 [0,1]），或按指定数量随机生成
3. **Lloyd 松弛**：可选迭代松弛使种子点分布更均匀，三角形更接近正三角形
4. **三角形光栅化**：每个三角形在网格中以唯一 ID 填充，方便后续按 ID 分区处理

## 适用情况

- 地形三角网格划分（TIN 地形）
- 基于三角剖分的区域分割和生物群系生成
- Voronoi 图的对偶构建（Delaunay 边对应 Voronoi 邻接）
- 游戏地图中的三角区域划分

## 基本使用方法

1. 设置 `网格宽度` 和 `网格高度`
2. 传入 `种子点列表`（[[x,y],...]，归一化坐标），或留空并设置 `随机种子点数`
3. 可选：设置 `Lloyd 松弛次数` 使三角形更规则
4. 从 `三角区域网格` 获取 ID 网格，从 `三角形列表` 获取顶点坐标

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 128 | 网格列数（8~1024） |
| height | number | 128 | 网格行数（8~1024） |
| seeds | array | [] | 种子点数组 [[x,y],...]，x/y ∈ [0,1]；留空则随机生成 |
| numSeeds | number | 20 | 种子点列表为空时随机生成的点数（3~500） |
| relaxIterations | number | 0 | Lloyd 松弛迭代次数（0~50），0=不松弛 |
| seed | number | 0 | 随机种子，0 使用默认种子 |

### 参数说明

**Lloyd 松弛 (relaxIterations)**：每次迭代将种子点移动到其 Voronoi 区域的质心。松弛后种子点分布更均匀，生成的 Delaunay 三角形更接近正三角形。推荐 2~5 次松弛即可获得明显效果。

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 每个三角形区域为唯一正整数 ID，凸包外区域为 0 |
| triangles | array | 三角形信息数组 [{id, v0:[x,y], v1:[x,y], v2:[x,y]}] |
| numTriangles | number | 生成的三角形总数 |

## 注意事项

1. **凸包外区域**：种子点凸包之外的像素值为 0，不属于任何三角形
2. **最少 3 个点**：Delaunay 三角剖分至少需要 3 个不共线的种子点
3. **归一化坐标**：自定义种子点的 x/y 为 [0,1] 归一化坐标，会自动映射到实际网格尺寸
4. **与 Voronoi 的关系**：Delaunay 三角剖分与 Voronoi 图互为对偶——Delaunay 的每条边连接两个相邻 Voronoi 区域的种子点，两者可配合使用
