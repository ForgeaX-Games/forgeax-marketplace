# Poisson房间放置 (poisson_room_placement)

使用 Poisson 盘采样（Bridson 算法）在二维网格上均匀分布候选点，然后在每个候选点处尝试放置随机尺寸的矩形房间。房间格子的值为房间 ID（从 1 开始），空地为 0。

## 功能特点

1. **均匀分布**：基于 Poisson 盘采样保证候选点之间的最小距离，房间分布均匀自然
2. **无重叠保证**：逐个放置时严格检查碰撞，确保所有房间互不重叠且保持最小间距
3. **灵活控制**：房间宽高、间距、采样半径均可独立配置；采样半径为 0 时自动根据房间尺寸计算
4. **房间 ID 输出**：每个房间在网格中以唯一正整数标识，方便后续按 ID 分别处理

## 适用情况

- 地牢/关卡生成中的房间布局（配合走廊连接算法使用）
- 建筑平面图的房间自动放置
- 任意需要在有限空间内均匀放置矩形区域的场景

## 基本使用方法

1. 设置 `网格宽度` 和 `网格高度` 确定画布大小
2. 设置房间尺寸范围（`最小/最大房间宽度`、`最小/最大房间高度`）
3. 设置 `房间间距` 控制房间之间的空隙
4. 可选：设置 `采样半径` 手动控制候选点密度（0 = 自动计算）
5. 可选：设置 `随机种子` 获得可复现的布局
6. 从 `房间网格` 端口获取带 ID 的网格，从 `房间列表` 端口获取房间详细信息

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 50 | 网格列数（8~512） |
| height | number | 50 | 网格行数（8~512） |
| minRoomW | number | 3 | 房间最小宽度（2~50） |
| maxRoomW | number | 8 | 房间最大宽度（≥minRoomW，上限100） |
| minRoomH | number | 3 | 房间最小高度（2~50） |
| maxRoomH | number | 8 | 房间最大高度（≥minRoomH，上限100） |
| gap | number | 1 | 房间之间最小间距（0~20） |
| radius | number | 0 | Poisson 盘采样半径；0 = 自动计算（基于最大房间尺寸+间距） |
| seed | number | 0 | 随机种子，0 使用默认种子 |

### 参数说明

**采样半径 (radius)**：控制 Poisson 盘采样的候选点密度。值越小候选点越密集（可能产生更多房间），值越大候选点越稀疏。设为 0 时自动计算为 `max(maxRoomW, maxRoomH) / 2 + gap + 1`，适用于大多数场景。

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 输出网格：0=空地，正整数=房间ID |
| rooms | array | 房间信息数组 [{id, x, y, w, h, centerX, centerY}] |
| numRooms | number | 成功放置的房间总数 |

## 注意事项

1. **房间数量取决于参数组合**：网格面积、房间尺寸范围、间距和采样半径共同决定最终放置的房间数量。如果房间过大或间距过大，可能只能放下少量房间
2. **采样半径过小**：会产生大量候选点但大部分被碰撞检测拒绝，浪费计算量但不影响正确性
3. **ID 从 1 开始**：房间 ID 为 1, 2, 3...，0 始终表示空地
4. **房间列表**：rooms 数组中每个元素包含 x/y（左上角坐标）、w/h（宽高）、centerX/centerY（中心坐标），可直接用于后续走廊连接等操作
