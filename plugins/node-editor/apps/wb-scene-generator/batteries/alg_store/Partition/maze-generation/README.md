# 迷宫生成 (maze_generation)

使用递归回溯（随机深度优先搜索）算法生成完美迷宫。完美迷宫是指任意两点之间恰好存在唯一路径、没有环路、没有不可达区域的迷宫。

## 功能特点

- 随机 DFS（递归回溯）迷宫生成算法
- 可分别控制墙壁厚度和通道厚度
- 可选开放入口（左上）和出口（右下）
- 确定性随机种子，相同参数产生相同迷宫
- 使用迭代式 DFS 避免大迷宫的栈溢出
- 输出二值网格：通道 = 1，墙壁 = 0

## 网格尺寸计算

输出网格的实际像素尺寸由迷宫参数决定：

- **网格宽度** = cols × (passageSize + wallSize) + wallSize
- **网格高度** = rows × (passageSize + wallSize) + wallSize

例如：24×24 迷宫、墙壁 1px、通道 1px → 输出 49×49 网格

## 算法流程

1. 创建全 0（墙壁）网格
2. 从 (0, 0) 开始，标记为已访问，挖开该通道格
3. 随机选择一个未访问的相邻格，挖开之间的墙壁，移动过去
4. 若当前格四周均已访问，回溯到上一格
5. 重复直到所有格子都被访问
6. 可选：在左上和右下打开入口/出口

## 输入参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| cols | number | 24 | 迷宫列数（2~200），通道格数 |
| rows | number | 24 | 迷宫行数（2~200），通道格数 |
| wallSize | number | 1 | 墙壁厚度（1~10 像素） |
| passageSize | number | 1 | 通道厚度（1~20 像素） |
| entrance | boolean | true | 是否开放左上入口和右下出口 |
| seed | number | 0 | 随机种子，0 使用默认种子 |

## 输出参数

| 参数 | 类型 | 说明 |
|------|------|------|
| grid | grid | 迷宫网格，通道 = 1，墙壁 = 0 |

## 注意事项

- 生成的是**完美迷宫**（无环路、无孤立区域），任意两格之间恰好有唯一路径
- 迷宫尺寸过大（如 200×200、通道 20px）会产生很大的输出网格，注意内存消耗
- 墙壁和通道厚度影响输出网格的总像素尺寸，但不影响迷宫的拓扑结构
- 输出中 1 = 通道、0 = 墙壁，可直接接入 A* 寻路等下游电池
