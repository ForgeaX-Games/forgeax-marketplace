# 房间走廊漫步 (room_corridor_walk)

从网格中心出发，交替放置矩形房间和走廊，走廊随机向四个方向（上下左右）延伸，可产生分支形成树状结构。灵感来自泰拉瑞亚（Terraria）地牢生成风格。

## 功能特点

1. **交替放置**：维护一个当前坐标，每步先放走廊再放房间，形成自然的房间-走廊-房间链
2. **四方向随机走廊**：走廊在上下左右四个方向中随机选择，长度可控
3. **分支生成**：以可调概率在当前位置产生分支，保存坐标后生成分支，回到原点继续主干
4. **碰撞检测**：房间和走廊均做碰撞检测，保证不重叠
5. **双网格输出**：房间和走廊分别输出到独立网格，便于下游分别处理

## 适用情况

- 泰拉瑞亚风格地牢生成
- 需要房间-走廊交替的线性/树状关卡布局
- Roguelike 地牢生成的基础结构
- 需要走廊和房间分层处理的场景

## 算法流程

1. 在网格中心放置第一个房间
2. 从当前房间出发，随机选方向，放置走廊
3. 在走廊末端放置新房间
4. 以 branchProb 概率将当前位置压入分支栈
5. 更新当前位置到新房间
6. 如果四个方向都无法放置，从分支栈弹出上一个分支点继续
7. 重复直到达到最大房间数或无法继续

## 基本使用方法

1. 将节点拖入画布，设置网格尺寸和房间参数
2. 调节分支概率控制布局的分叉程度（0=纯线性，1=每步都分叉）
3. 执行后 `grid` 输出房间网格，`corridorGrid` 输出走廊网格
4. 可分别接入渲染节点，为房间和走廊使用不同材质

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 50 | 网格宽度（32~512） |
| height | number | 50 | 网格高度（32~512） |
| maxRooms | number | 6 | 最大房间数（1~100） |
| minRoomSize | number | 4 | 房间最小边长（3~30） |
| maxRoomSize | number | 8 | 房间最大边长（3~50） |
| minCorridorLen | number | 2 | 走廊最小长度（1~20） |
| maxCorridorLen | number | 5 | 走廊最大长度（1~40） |
| corridorWidth | number | 2 | 走廊宽度（1~5） |
| branchProb | number | 0.3 | 分支概率（0~1），0=纯线性，1=每步分叉 |
| seed | number | 0 | 随机种子，0 使用默认种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 房间网格：0=空白，正整数=房间 ID（1, 2, 3...） |
| corridorGrid | grid | 走廊网格：0=空白，1=走廊 |
| rooms | array | 房间信息数组 [{id, x, y, w, h, centerX, centerY}] |
| numRooms | number | 实际生成的房间总数 |

## 注意事项

1. **实际房间数可能小于 maxRooms**：当网格空间不足或被已有结构包围时，算法会提前停止
2. **分支概率影响布局形态**：branchProb=0 产生纯线性链状布局，branchProb=1 产生高度分叉的树状布局
3. **走廊与房间分离**：两个输出网格互不重叠，走廊仅存在于 corridorGrid，房间仅存在于 grid
4. **房间 ID 从 1 开始**：第一个房间（中心房间）ID 为 1，后续依次递增
