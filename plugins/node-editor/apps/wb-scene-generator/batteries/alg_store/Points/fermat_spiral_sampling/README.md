# 费马螺旋采样 (fermat_spiral_sampling)

基于黄金角（≈137.508°）的费马螺旋采样，在圆形区域内生成均匀、无聚集的点分布。点按照向日葵种子排列方式分布，是自然界中最均匀的圆盘点分布方案之一。

## 功能特点

1. **黄金角分布**：每个点相对前一个点旋转黄金角（π(3−√5) ≈ 2.3999 弧度），保证无重叠无聚集
2. **确定性生成**：相同参数始终产生相同的点分布（jitter=0 时完全确定）
3. **可选抖动**：通过 jitter 参数为完美螺旋添加随机扰动，使分布更自然
4. **圆盘填充**：点自然填满圆形区域，密度从中心到边缘均匀

## 适用情况

- 需要在圆形区域内均匀分布物件（树木、矿石、建筑等）
- 粒子系统的初始分布
- 作为 Voronoi 图的种子点输入
- 需要确定性且美观的点分布的 PCG 场景

## 算法原理

对于第 i 个点（i = 0, 1, 2, ...）：
- 角度 θ = i × 黄金角
- 半径 r = c × √i（其中 c = R / √(n−1)，R 为圆盘半径）
- 坐标 x = cx + r·cos(θ)，y = cy + r·sin(θ)

黄金角是使得连续点之间角度间隔最不可公约的角度，因此点不会形成明显的径向线条。

## 基本使用方法

1. 设置 width 和 height 指定网格大小
2. 设置 numPoints 控制采样点数量
3. 可选：增加 jitter 使分布更自然（默认 0，完美螺旋）
4. 执行后从 `grid` 获取二值网格，从 `points` 获取坐标列表

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 64 | 网格宽度（4~512） |
| height | number | 64 | 网格高度（4~512） |
| numPoints | number | 200 | 采样点数量（1~10000） |
| jitter | number | 0 | 抖动幅度（0~10 像素），0=完美螺旋 |
| seed | number | 0 | 抖动随机种子，jitter=0 时无效 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 输出网格，采样点=1，其余=0 |
| points | number rank 2 | 采样点坐标数组 `[[x,y],...]`，x 为列号，y 为行号 |
| count | number | 落在网格范围内的实际采样点数 |

## 注意事项

1. **圆形分布**：点分布在以网格中心为圆心、min(width, height)/2 − 1 为半径的圆内，矩形网格的四角区域不会有采样点
2. **实际点数可能小于 numPoints**：取整后坐标重合的点会被去重，超出网格范围的点会被丢弃
3. **点数与密度**：numPoints 越大圆内越密集；若 numPoints 过大超过像素数，多余的点会因坐标重合被丢弃
