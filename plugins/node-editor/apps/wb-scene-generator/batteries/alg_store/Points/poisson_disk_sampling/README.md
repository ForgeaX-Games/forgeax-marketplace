# 泊松盘采样 (poisson_disk_sampling)

使用 Bridson 快速泊松盘采样算法在指定大小的区域内生成均匀分布的采样点。采样点之间保持最小距离，既不会太密集也不会太稀疏，是程序化生成中分布树木、石头、建筑等物件的经典方法。

## 功能特点

1. **Bridson 快速算法**：O(n) 时间复杂度，使用背景网格加速邻域查询
2. **最小距离保证**：任意两个采样点之间的距离不小于指定半径
3. **均匀分布**：相比纯随机采样，泊松盘采样产生视觉上更均匀自然的分布
4. **确定性随机**：相同种子产生相同分布，方便调试和复现

## 适用情况

- 树木、石头、草丛等自然物件的分布
- 城镇、建筑物的布局生成
- 任何需要均匀随机分布点的 PCG 场景
- 可作为 Voronoi 图、Delaunay 三角化等下游算法的输入

## 算法流程

1. 将采样区域划分为 cellSize = radius/√2 的背景网格
2. 随机放置第一个采样点，加入活跃列表
3. 从活跃列表中随机选一个点，在其周围 [radius, 2×radius] 环形区域随机生成候选点
4. 检查候选点与已有点的距离，若均 >= radius 则接受
5. 若连续 maxAttempts 次都被拒绝，将该点从活跃列表移除
6. 重复直到活跃列表为空

## 基本使用方法

1. 设置 width 和 height 指定采样区域大小
2. 设置 radius 控制点的密度
3. 执行后从 `grid` 输出获取二值网格（1=采样点，0=空白），从 `points` 获取坐标列表
4. 可接入渲染节点或下游处理节点

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 64 | 采样区域宽度（4~512） |
| height | number | 64 | 采样区域高度（4~512） |
| radius | number | 5 | 采样点最小距离（1~50），值越大点越稀疏 |
| maxAttempts | number | 30 | 每个活跃点的最大尝试次数（1~100） |
| seed | number | 0 | 随机种子，0 使用默认种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 输出网格，采样点=1，其余=0 |
| points | number rank 2 | 所有采样点的坐标数组 `[[x,y],...]`，x 为列号，y 为行号 |
| count | number | 采样点总数 |

## 注意事项

1. **radius 与区域尺寸的关系**：radius 过大（接近区域尺寸）会导致只有极少采样点；radius=1 时接近随机采样
2. **maxAttempts 影响密度**：值越大采样越接近理论最大密度，但计算时间也越长；30 是常用平衡值
3. **坐标为整数像素坐标**：points 中的 x/y 为取整后的网格坐标，与 grid 中值为 1 的位置一一对应
