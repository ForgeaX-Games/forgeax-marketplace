# 二值网格取反 (grid_invert)

将二值网格中的 0 变为 1、1 变为 0，实现逻辑取反操作。

## 功能特点

1. **简单高效**：逐格取反，0→1，1→0
2. **严格二值**：非零值统一视为 1 并取反为 0

## 适用情况

- 地图正负空间互换（如将通道变成墙壁、墙壁变成通道）
- 遮罩反选操作
- 二值化结果的补集计算

## 基本使用方法

1. 将二值网格连接到 `输入网格` 端口
2. 从 `取反网格` 端口获取输出

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 待取反的二值网格（值为 0 或 1） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 取反后的二值网格 |

## 注意事项

1. **非二值网格**：对于包含非 0/1 值的网格，所有非零值都会被视为 1 并取反为 0
2. **空网格**：输入空网格时返回错误提示
