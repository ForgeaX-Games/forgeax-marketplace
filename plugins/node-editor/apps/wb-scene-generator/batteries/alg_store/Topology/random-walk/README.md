# 随机行走 (random_walk)

在网格掩码内生成随机行走路径，支持可变路径宽度、多种宽度衰减模式、方向偏好控制和可调行走速度。

## 功能特点

1. **8 方向加权行走**：支持上下左右及四个对角方向，可通过方向偏好权重控制各方向的概率
2. **5 种宽度衰减**：不衰减、线性衰减、指数衰减、正弦波动（±50%）、随机浮动（±50%）
3. **掩码约束**：行走仅在输入网格非零区域内进行
4. **触边停止**：可选触碰边界时立即终止行走，或遇到边界自动换向继续行走
5. **确定性随机**：相同种子产生相同路径，便于调试和复现

## 适用情况

- 在区域内生成蜿蜒的河流、道路、隧道
- 生成不规则的自然路径或裂缝纹理
- 利用方向偏好生成有倾向性的路径（如偏水平、偏垂直、偏对角线等）
- PCG 管线中配合区域划分、掩码操作使用

## 基本使用方法

1. 连接 `grid` 输入（提供行走区域的掩码，非零像素为可行走区域）
2. 调整 `steps` 控制路径长度，`pathWidth` 控制路径粗细
3. 选择 `widthDecay` 衰减模式实现由粗到细或波动效果
4. 设置 `dirBias` 控制行走方向偏好（可选）
5. `grid` 输出为二值网格（1 = 路径，0 = 空白）

## 输入参数

| 参数名 | 类型 | 默认值 | 范围 | 说明 |
|--------|------|--------|------|------|
| grid | grid | - | - | 输入网格掩码，行走在非零区域内进行 |
| steps | number | 500 | 1–20000 | 行走总步数，值越大路径覆盖越广 |
| startX | number | -1 | - | 起始点 X 坐标，-1 = 掩码内随机选择 |
| startY | number | -1 | - | 起始点 Y 坐标，-1 = 掩码内随机选择 |
| pathWidth | number | 2 | 1–30 | 路径直径（像素） |
| widthDecay | string | "none" | 下拉选择 | 宽度衰减模式（见下方说明） |
| speed | number | 2 | 1–20 | 每步移动格数，值越大路径越舒展 |
| stopAtBoundary | boolean | true | - | 触碰边界时是否终止行走；false 则换向继续 |
| dirBias | string | "1,1,1,1,1,1,1,1" | 每个值 0–1 | 8 个方向的概率权重（见下方说明） |
| seed | number | 0 | - | 随机种子，0 = 使用默认种子 |

### widthDecay 衰减模式

| 值 | 名称 | 效果 |
|----|------|------|
| `none` | 不衰减 | 路径宽度始终为 pathWidth |
| `linear` | 线性衰减 | 宽度从 pathWidth 线性递减至 1 |
| `exponential` | 指数衰减 | 宽度指数递减，前段快后段慢 |
| `sine` | 正弦波动 | 宽度以 pathWidth 为中心周期性波动 ±50%（范围 0.5× ~ 1.5×） |
| `random` | 随机浮动 | 每步宽度在 pathWidth 的 ±50% 范围内随机取值（范围 0.5× ~ 1.5×） |

### dirBias 方向偏好

8 个数字分别对应以下方向（逗号分隔，每个值 0~1，内部自动归一化）：

| 序号 | 方向 | DX | DY | 说明 |
|------|------|----|----|------|
| 1 | E | +1 | 0 | 右 |
| 2 | NE | +1 | -1 | 右上 |
| 3 | N | 0 | -1 | 上 |
| 4 | NW | -1 | -1 | 左上 |
| 5 | W | -1 | 0 | 左 |
| 6 | SW | -1 | +1 | 左下 |
| 7 | S | 0 | +1 | 下 |
| 8 | SE | +1 | +1 | 右下 |

常用配置示例：

- `1,1,1,1,1,1,1,1` — 均匀（默认）
- `1,0,0,0,1,0,0,0` — 仅水平方向（E + W）
- `0,0,1,0,0,0,1,0` — 仅垂直方向（N + S）
- `0,1,0,0,0,1,0,0` — 对角线偏好（NE + SW）
- `1,0.2,0,0.2,1,0.2,0,0.2` — 以水平为主，少量对角

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 路径网格，路径像素值为 1，其余为 0 |

## 注意事项

1. **掩码约束**：行走者只能在输入网格的非零区域内移动，无法穿越零值区域
2. **方向选择**：每步根据 dirBias 权重加权采样一个方向；不可走方向的权重自动置零后重新归一化
3. **触边停止**：`stopAtBoundary=true`（默认）时，行走者碰到掩码边界或网格边缘立即终止整条路径；`false` 时遇边界换向继续行走直到步数用完
4. **速度效果**：speed > 1 时每步沿同一方向连续移动多格，遇到不可走区域会提前停止
5. **起始点**：若指定的起始点超出网格或位于零值区域，将自动在掩码内随机选择
6. **覆盖率**：默认 steps=500 + speed=2 在 50×50 网格上可产生明显路径；网格更大时可相应增加步数
