# 室内布局清理 (Complex Indoor Cleanup)

对已完成的室内布局进行后处理，修复视觉瑕疵。

## 功能特点

1. **修复过宽门洞**：确保门洞两端保留墙壁，防止门宽超过房间边界
2. **移除孤立墙壁**：清除不与任何房间相邻的墙壁碎片
3. **清除墙壁突起**：移除伸入空地的细墙
4. **清除浮空墙链**：整组移除与所有房间断开的墙壁链

## 输入参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| inputGrid | grid | 含门洞的最终网格 |
| nameList | array | 网格值到名称的映射 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 清理后的网格 |
| nameList | array | 更新后的名称清单 |
