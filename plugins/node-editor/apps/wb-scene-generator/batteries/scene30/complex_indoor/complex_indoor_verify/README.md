# complex_indoor_verify

验证复杂室内布局中所有房间的连通性。从初始房间出发进行 BFS 遍历连接图，对不可达的房间通过 L 形走廊进行修复。

## 功能特点

1. **图级BFS连通性检查**：基于连接列表检查所有房间是否可达
2. **自动修复**：对不可达房间创建L形走廊连接到最近的可达房间
3. **迭代修复**：重复检查直到所有房间可达

## 输入参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| inputGrid | grid | 包含所有房间的网格 |
| roomList | object | 房间数据 |
| connectionList | object | 房间连接关系 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 修复后的网格 |
| roomList | object | 可能含新增走廊 |
| connectionList | object | 含修复连接 |
