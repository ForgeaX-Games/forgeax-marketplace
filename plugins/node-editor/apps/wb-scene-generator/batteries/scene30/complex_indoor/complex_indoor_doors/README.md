# complex_indoor_doors

在已完成的室内布局中，在相连房间的共享墙壁上开门。这是布局管线的最后一步，确保所有房间通过门洞物理连通。

## 功能特点

1. **共享墙壁开门**：在连接列表中记录的共享墙壁上随机选择一段开门
2. **自动发现共享墙壁**：若连接数据缺失共享墙壁信息，自动扫描网格查找
3. **连通性修复**：开门后进行网格级BFS验证，修复仍未连通的区域
4. **自动名称清单**：根据网格中实际出现的值生成完整名称映射

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | - | 完成布局的网格 |
| roomList | object | - | 房间数据 |
| connectionList | object | - | 房间连接关系 |
| doorWidthMin | number | 2 | 最小门宽 |
| seed | number | 0 | 随机种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 含门洞的最终网格 |
| nameList | array | 网格值与名称映射 |
