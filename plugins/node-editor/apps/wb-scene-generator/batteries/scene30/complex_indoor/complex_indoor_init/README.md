# complex_indoor_init

创建网格并在中心放置带墙壁轮廓的初始方形房间。作为复杂室内布局管线的第一步。

## 功能特点

1. **墙壁轮廓模型**：房间由1格宽的墙壁边框包围内部地板区域
2. **居中放置**：初始房间自动放置在网格中心
3. **随机尺寸**：房间内部尺寸在给定范围内随机生成

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 300 | 网格宽度 |
| height | number | 220 | 网格高度 |
| roomMinSize | number | 10 | 初始房间内部最小边长 |
| roomMaxSize | number | 20 | 初始房间内部最大边长 |
| seed | number | 0 | 随机种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 0=空地, 1=墙壁, 2=房间内部 |
| roomList | object | 房间数据数组 |
| nextRoomId | number | 下一个可用房间ID |

## 网格编码

- 0: 空地(建筑外)
- 1: 墙壁(房间轮廓)
- 2: 初始房间内部
