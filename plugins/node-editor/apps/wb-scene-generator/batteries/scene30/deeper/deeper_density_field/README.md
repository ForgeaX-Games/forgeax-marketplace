# 深空密度场 (deeper_density_field)

生成深层空间场景的密度权重图，使用分形叠加噪声配合幂函数极化，产生极度不均匀的空间分布——某些区域高度密集、某些区域几乎空旷。

## 功能特点

1. **分形噪声叠加**：多层值噪声叠加（fBm），兼顾大尺度区域特征和小尺度细节变化
2. **幂函数极化**：对原始噪声施加 `value^polarize`，将分布推向两极，高密度区更密、低密度区更空
3. **平坦地形**：输出全1地面底图，无任何高度差，符合深层空间的平坦地面特征
4. **可重现随机**：seed 参数确保同一参数下每次生成结果一致

## 适用情况

- 为 `deeper_poi_placer` 提供密度引导
- 需要生成空间分布极不均匀的2D场景（太空站、地下城市、废墟区）
- 单独作为热力图/影响场使用（如建筑密度场、怪物刷新权重场）

## 基本使用方法

直接连接到 `deeper_poi_placer` 的对应输入端口：
- `densityGrid` → `deeper_poi_placer.densityGrid`
- `groundGrid` → `deeper_poi_placer.groundGrid`

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 64 | 输出网格列数 |
| height | number | 64 | 输出网格行数 |
| scale | number | 8 | 基础噪声频率（4–16），越小区域块越大 |
| octaves | number | 4 | 分形叠加层数（3–5），越多细节越丰富 |
| persistence | number | 0.5 | 每层振幅衰减系数（0.1–1.0） |
| polarize | number | 3.0 | 极化指数（2–6），越大两极分化越剧烈 |
| seed | number | 0 | 随机种子，0 自动随机 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| densityGrid | grid | 密度权重图，值域 0–100 |
| groundGrid | grid | 全1地面底图 |

## 参数调优建议

| 想要的效果 | 调整方式 |
|---|---|
| 大片空旷区域 + 局部极密集 | 提高 `polarize`（5–8） |
| 整体比较均匀 | 降低 `polarize`（1.5–2） |
| 噪声区域块更大 | 降低 `scale`（3–5） |
| 噪声纹理更细 | 提高 `scale`（12–20） |
| 更多层次感 | 提高 `octaves`（5–6） |

## 注意事项

1. **densityGrid 值域**：输出为整数 0–100，渲染时可直接用颜色映射查看密度分布
2. **groundGrid 用途**：作为 deeper_poi_placer 的地面底图，告知放置器哪些格子是可用地面
3. **极化指数不宜过大**：`polarize > 8` 会导致绝大多数区域密度趋近于0，POI 将集中在极少数热点
