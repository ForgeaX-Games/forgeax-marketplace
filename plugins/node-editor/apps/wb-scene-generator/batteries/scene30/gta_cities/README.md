# GTA Cities 电池组

该小分类面向参考图这类城市形态：有机主路、局部街巷、道路围合地块和方盒建筑。

推荐 MVP 流程：

1. `gta_city_districts`：生成城市功能片区。
2. `gta_city_arterials`：生成主干路、环路和放射路。
3. `gta_city_streets`：在主路服务范围内生成街巷。
4. `gta_city_parcels`：从道路切割后的空间中提取地块。
5. `gta_city_buildings`：在地块内放置建筑足迹。

值约定：

- `300`：主干路
- `301`：街巷
- `410-414`：城市功能区
- `420-423`：道路地块
- `500-503`：建筑足迹

电池保持自包含，不依赖兄弟电池目录的共享工具。
