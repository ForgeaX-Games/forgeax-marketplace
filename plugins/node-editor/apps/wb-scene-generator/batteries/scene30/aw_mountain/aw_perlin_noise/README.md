# 山地柏林噪声 (aw_perlin_noise)

山地场景专用极简 Perlin 噪声生成器，接口只有网格尺寸和随机种子两个参数，其余全部内置为山地地形的最优默认值。

## 功能特点

1. **极简接口**：只需接入一个尺寸数值和种子，无需配置频率、八度、增益等参数
2. **山地调优**：内置 FBm 分形（octaves=4, frequency=0.02, lacunarity=2, gain=0.5），适合生成中等起伏的山地高度场
3. **正方形输出**：宽高相等，与地图尺寸常量直接对接，无需分别连接 width/height

## 固定内部参数

| 参数 | 值 | 说明 |
|------|-----|------|
| fractalType | FBm | 分形布朗运动，地形感最自然 |
| frequency | 0.02 | 低频，产生宏观山体轮廓 |
| octaves | 4 | 4 层叠加，细节适中 |
| lacunarity | 2.0 | 标准倍频系数 |
| gain | 0.5 | 标准振幅衰减 |

## 适用情况

- 野外山地场景的主高度场生成
- 需要快速接入 `height_adjuster` → `biome_classifier` 的标准山地管线
- 也可用于温度场/湿度场等辅助噪声层（替换 opensimplex2_noise）

## 基本使用方法

1. 将 `number_const`（地图尺寸）连接至 `size` 输入
2. 将 `number_const`（随机种子）连接至 `seed` 输入
3. 将本节点的 `grid` 输出连接至 `height_adjuster` 的 `heightGrid` 输入

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| size | number | 200 | 输出网格的边长（宽高相等） |
| seed | number | 42 | 随机种子，相同种子输出相同图案 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 噪声网格，值域 0–1，可直接接入高度调整块 |

## 注意事项

1. **仅正方形**：本节点输出宽高相等的网格；如需非正方形请使用通用 `perlin_noise` 节点
2. **输出范围**：输出值 0–1，`height_adjuster` 会自动检测并缩放至 0–100
