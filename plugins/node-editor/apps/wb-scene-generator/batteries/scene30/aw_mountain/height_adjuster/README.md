# 高度调整块 (height_adjuster)

将穹顶中心点采样与径向高度增益合并为一步：内部自动生成散布均匀的穹顶中心点，并对输入高度场叠加 smoothstep 径向增益，输出可直接送入群系分类器。

## 功能特点

1. **一步到位**：替代原来的「穹顶点采样 → 径向高度增益」两节点组合，减少连线复杂度
2. **参数极简**：仅需输入穹顶数量，其余参数均已内置默认值
3. **均匀分布**：使用拒绝采样保证各穹顶中心之间满足最小间距，避免重叠

## 固定内部参数

| 参数 | 值 | 说明 |
|------|-----|------|
| seed | 0（时间随机）| 每次运行自动随机，保证分布自然 |
| margin | 0.1 | 穹顶中心距边缘至少 10% |
| minSpacing | 0.3 | 任意两穹顶中心归一化间距 ≥ 0.3 |
| radius | 0.3 | 穹顶影响半径（相对于地图最短边的比例） |
| intensity | 45 | 穹顶中心最大增益量（在 0–100 高度 scale 下） |

## 适用情况

- 野外山地地形，需要生成 1–4 个自然分布的山丘/台地穹顶
- 与 `perlin_noise` → `height_adjuster` → `biome_classifier` 的标准三段管线配合使用

## 基本使用方法

1. 将 `perlin_noise` 的 `grid` 接口连接至本节点的 `heightGrid` 输入
2. 用数值常量节点设置 `count`（穹顶数量，建议 1–3）
3. 将本节点的 `heightGrid` 输出连接至 `biome_classifier` 的 `heightGrid` 输入

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| heightGrid | grid | 必填 | 原始高度场；0–1 噪声或 0–100 整数网格均可 |
| count | number | 1 | 穹顶中心点数量（最少 1） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| heightGrid | grid | 叠加所有穹顶增益后的高度场（0–100） |

## 注意事项

1. **数量上限**：当 `count` 较大（> 4）且 `minSpacing=0.3` 时，可用空间不足，部分穹顶会退化为最优放置（仍合法，但间距可能小于设定值）
2. **随机性**：`seed=0` 表示使用当前时间戳作为随机种子，每次执行位置略有不同，这是预期行为
