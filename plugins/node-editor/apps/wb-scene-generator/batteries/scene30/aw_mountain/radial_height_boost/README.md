# 径向高度增益 (radial_height_boost)

在高度图的指定位置叠加一个穹顶形状的高度增益，制造台地、山体、丘陵的同心圆层叠效果。

## 功能特点

1. **穹顶增益**：以指定中心点为圆心向外辐射，中心处增益最大，边缘归零，形成自然的穹顶轮廓
2. **三种衰减曲线**：`smooth`（smoothstep 平滑过渡）、`gaussian`（高斯衰减）、`linear`（线性衰减），覆盖不同地形风格
3. **归一化坐标系**：中心点和半径均使用归一化值（0–1），与网格尺寸无关，便于复用

## 适用情况

- 为已有高度场叠加一个圆形台地或山峰（如图片中央的多层次红棕台地）
- 在噪声高度图基础上人工强调某个区域的海拔高度
- 与 `biome_classifier` 配合，让中央高海拔区域映射为台地/高地群系

## 基本使用方法

1. 将 `perlin-noise` 或其他噪声节点的输出连接到 `heightGrid` 端口
2. 设置 `centerX`/`centerY` 指定穹顶位置（默认地图中央 0.5, 0.5）
3. 调整 `radius`（影响范围）和 `intensity`（增益强度）到合适值
4. 选择 `falloff` 衰减类型，推荐 `smooth` 用于自然地形
5. 输出的 `heightGrid` 可直接送入 `terrain_smoother` 进行平滑

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| heightGrid | grid | - | 原始高度场，值域 0–100 |
| centerX | number | 0.5 | 穹顶中心X，归一化（0=左，1=右）|
| centerY | number | 0.5 | 穹顶中心Y，归一化（0=上，1=下）|
| radius | number | 0.35 | 影响半径，归一化相对于网格最短边 |
| intensity | number | 45 | 中心处最大增益值（叠加到原高度，裁切到100）|
| falloff | string | smooth | 衰减曲线：gaussian / linear / smooth |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| heightGrid | grid | 叠加穹顶增益后的高度场，值域 0–100 |

## 注意事项

1. **增益叠加后裁切**：若原高度 + 增益 > 100，结果自动裁切为 100，不会溢出
2. **半径归一化基准**：`radius` 相对于网格宽度和高度中较短的一边，确保正圆形穹顶
3. **多次叠加**：可串联多个该节点，在不同位置叠加多个穹顶，形成复杂山体
