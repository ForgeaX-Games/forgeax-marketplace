# 网格蒙版应用 (grid_mask_ab)

用网格 B 作为蒙版遮罩网格 A：B 为 0 的位置输出 0，其余位置保留 A 的原值。

## 功能特点

1. **精准遮罩**：完全按 B 的零/非零区域裁剪 A，不改变 A 在非零区域的任何数值
2. **值域保留**：A 的原始值（无论正负、大小）在 B 非零区域原样输出
3. **尺寸容错**：B 超出 A 范围的位置视为 0（遮挡）

## 适用情况

- 将地形高度图（A）限制在某个区域蒙版（B）内
- 用洞穴蒙版裁剪地块分类网格，仅保留洞穴范围内的分类值
- 任何「用一张网格控制另一张网格的有效区域」场景

## 基本使用方法

1. 将数据网格连接到 `gridA`（保留原值的那张）
2. 将区域蒙版连接到 `gridB`（控制哪里有效的那张，非零=有效）
3. `maskedGrid` 输出即为裁剪结果

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| gridA | grid | 必填 | 数据网格，提供实际数值 |
| gridB | grid | 必填 | 蒙版网格，0=遮挡，非零=透过 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| maskedGrid | grid | 遮罩后的网格；B 为 0 处输出 0，其余处为 A 原值 |

## 注意事项

1. **两张网格均必填**：任一缺失时返回 error
2. **B 的值不影响输出大小**：只要 B 非零，输出就是 A 的原值，B 本身的具体数值不会写入结果
3. **尺寸不一致时**：B 超出 A 范围的格子按 0 处理（遮挡）
