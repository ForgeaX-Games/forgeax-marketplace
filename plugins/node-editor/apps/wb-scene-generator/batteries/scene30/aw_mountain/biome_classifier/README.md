# 地块分类器 (biome_classifier)

接受三个独立数值场，按可配置阈值将每个格子分类为 7 种地块类型，输出整数地块网格。

## 地块类型

| ID | 名称 | 渲染颜色 | 触发条件 |
|----|------|----------|---------|
| 1 | 森林山顶 | 🔴 红色 `#e05555` | field1 极高 + field3 高（植被茂盛的山顶） |
| 2 | 森林山脚 | 🟠 深橙色 `#e07730` | field1 低 + field3 高（低地密林） |
| 3 | 草地 | 🟡 橙色 `#e0a830` | field1 低 + field3 低（开阔低地） |
| 4 | 沙滩 | 🟡 黄绿色 `#c8d430` | field1 极低（海拔最低过渡带） |
| 5 | 山脚 | 🟢 草绿色 `#6aba30` | field1 中等 |
| 6 | 山坡 | 💚 翠绿色 `#30ba7a` | field1 中高 |
| 7 | 山顶 | 🩵 青蓝色 `#30bab8` | field1 极高 + field3 低（裸露山顶） |

## 功能特点

1. **语义中立**：输入为三个通用"场"，不绑定具体物理含义（高度/温度/湿度）
2. **森林区分**：同一海拔下，field3（植被场）决定是否生成森林类型
3. **可调阈值**：支持通过 JSON 字符串覆盖所有分类阈值

## 默认阈值

| 阈值 | 默认值 | 说明 |
|------|--------|------|
| beachMax | 42 | field1 < 42 → 沙滩 |
| lowlandMax | 58 | field1 < 58 → 低地区 |
| footMax | 67 | field1 < 67 → 山脚 |
| slopeMax | 76 | field1 < 76 → 山坡；≥76 → 山顶区 |
| forestMin | 50 | field3 > 50 → 触发森林类型 |

## 适用情况

- 野外山地地形的地块分类（搭配 `aw_perlin_noise` + `height_adjuster`）
- 任意需要将三个连续场映射为离散地块类型的场景

## 基本使用方法

1. 将主高度/地形场连接至 `field1`
2. 将植被/湿度场连接至 `field3`
3. `field2` 可选（当前版本保留未用）
4. 将 `terrainGrid` 输出连接至 `terrain_smoother`

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| field1 | grid | 必填 | 主驱动场，决定海拔分层 |
| field2 | grid | 可选 | 次场，当前保留未用 |
| field3 | grid | 可选 | 植被场，决定森林 vs 非森林 |
| thresholds | string | "" | JSON 阈值覆盖（可选） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| terrainGrid | grid | 地块 ID 网格（整数 1–7） |

## 注意事项

1. **field3 可省略**：未连接时默认 field3=50（forestMin 临界值），低地格子将全部分类为草地，山顶区全部分类为山顶
2. **field2 暂无效果**：连接后不报错，但当前版本不影响输出结果
3. **输出端口名**：本版本输出为 `terrainGrid`（旧版为 `biomeGrid`），pipeline 需同步更新连线
