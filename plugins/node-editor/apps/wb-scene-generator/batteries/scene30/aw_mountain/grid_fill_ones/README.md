# 网格填充为1 (grid_fill_ones)

将输入网格中所有非零格子统一置为 1，零格子保持 0，输出纯二值掩码网格。

## 功能特点

1. **全非零值归一**：无论原始网格中有多少种不同的非零数值（1、2、3…），输出一律置为 1
2. **兼容网格列表**：若上游输出的是网格列表，自动取第一张网格处理
3. **错误诊断**：输入格式异常时通过 `错误信息` 端口说明原因

## 适用情况

- 将多值地块网格（如地块分类结果）转换为可用的遮罩
- 将洞穴/道路/河流等多值输出压缩为纯二值掩码
- 任何需要"只要非零就算有"的场景

## 基本使用方法

将任意网格连接到 `输入网格` 端口，执行后 `二值网格` 输出即为二值掩码。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 任意整数二维网格 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 非零→1、零→0 的二值掩码网格 |
| errorMessage | string | 运行成功时为空，失败时说明原因 |

## 注意事项

1. **零值保持为零**：原始网格中值为 0 的格子不会被修改
2. **输入格式**：接受单张网格或网格列表（列表时取第一张）
