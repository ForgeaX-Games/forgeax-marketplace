# 名称清单生成 (name_entry_gen)

用一个数字 ID、名称和图层类型，生成单条格式为 `[{id, name, type}]` 的名称清单。

## 功能特点

1. **快速构造单条名称清单**：直接输入三个字段即可输出标准名称清单格式
2. **type 字段可选**：图层类型留空时，输出条目不含 `type` 字段
3. **与现有管线兼容**：输出格式与地块分类器、填充排序等节点的名称清单完全一致

## 适用情况

- 为河流掩码、洞穴掩码等单一值的网格手动指定名称和图层类型
- 配合"合并"节点将多条名称清单拼接成完整清单列表

## 基本使用方法

填入 ID、名称、图层类型三个参数，执行后将 `名称清单` 输出接入下游节点（如填充排序、合并等）。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| id | number | 1 | 名称清单条目的数字 ID，需与网格中的值对应 |
| name | string | 未命名 | 条目的显示名称 |
| layerType | string | （空） | 图层类型；留空则输出条目不含 type 字段 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| nameList | array | 单条名称清单 `[{id, name, type}]` |

## 注意事项

1. **ID 需与网格值对应**：此节点只生成清单，不修改网格，请确保 ID 与上游网格中的实际值一致
2. **多条清单拼接**：需要多条清单时，将多个此节点的输出接入"合并"节点，再接入下游
