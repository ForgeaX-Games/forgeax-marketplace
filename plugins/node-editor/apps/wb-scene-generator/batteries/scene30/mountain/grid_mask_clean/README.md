# 掩码批量清理 (grid_mask_clean)

对等高线网格列表做批量形态学清理：填充被单色完全包围的内部小孔洞，删除面积过小的孤立碎块。

## 功能特点

1. **批量处理**：输入整个等高线网格列表，对每一层独立执行清理，输出等长的清理后列表，可直接与 `mountain_contour_generate` 的 `等高线列表` 输出对接。
2. **填充内部孔洞**：检测值为 0 的连通域，若不接触地图边界、仅被单一非零值包围、且面积 ≤ `minHoleSize`，则填充为该包围色。
3. **删除孤立小块**：检测每个非零值的连通域，若面积 < `minIslandSize`，则全部置 0。
4. **顺序执行**：先填充孔洞，再删除小块，两步串联保证结果干净。

## 适用情况

- 等高线分层后某层出现大量内部空洞或孤立像素
- 任何基于 BFS/噪声生成的 mask 列表出现碎片化
- 地形、区域划分后处理清理

## 基本使用方法

1. 将 `mountain_contour_generate` 的 `等高线列表` 直接连入 `网格列表`。
2. 调整 `minHoleSize`（越大填充越激进）和 `minIslandSize`（越大删除越多碎块）。
3. `清理后列表` 输出与输入等长，可直接接后续处理节点。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrids | array | — | 待清理的网格列表，0 为背景 |
| minHoleSize | number | 20 | 面积 ≤ 此值的内部 0 连通域被填充 |
| minIslandSize | number | 10 | 面积 < 此值的非零连通域被删除 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrids | array | 清理后的网格列表，长度与输入一致 |

## 注意事项

1. **孔洞填充仅作用于被单色完全包围的区域**：若一个 0 区域同时与多种非零值相邻，不会被填充（避免歧义）。
2. **接触边界的 0 区域不会被填充**：背景区域（外围的 0）不受影响。
3. **先填充后删除**：填孔后若小块恰好超过阈值则不删除，可适当降低 `minIslandSize`。
