# 矩形点位 (rect_poi_place)

对网格列表中每个有值区域，在其边缘随机采样生成 POI 点位，输出为单张点位网格。

## 功能特点

1. **边缘采样**：只在有值区域的边缘格（本格非零且至少有一个4邻格为0）上生成点位，确保点位贴近区域轮廓。
2. **每层独立控制**：每个输入网格独立采样，每层生成 `count` 个点位，所有层结果叠加到同一张输出网格。
3. **统一点位值**：点位写入值为所有输入网格中的全局最大非零值，便于下游道路连接电池统一识别。
4. **可复现**：支持随机种子，相同种子产生相同点位分布。

## 适用情况

- 为各区域色块（如等高线层、地形区域）生成入口/出口点位
- 配合 `道路连接` 电池：本电池生成点位网格 → 接入道路连接的 `POI 网格`
- 田园/城镇场景的区域边界采样

## 基本使用方法

1. 将区域网格列表（如等高线列表、地形分区列表）接入 `网格列表`。
2. 设置 `每层点位数量`（每个区域生成几个点）。
3. 输出的 `点位网格` 直接连入 `道路连接` 的 `POI 网格`，`点位填充值` 设为输入网格最大值即可。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrids | array | — | 输入网格列表，从每层有值区域的边缘采样点位 |
| count | number | 3 | 每个网格生成的点位数量 |
| seed | number | 0 | 随机种子，0 使用当前时间 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 点位网格，点位值=所有输入网格最大非零值，其余=0 |

## 注意事项

1. **点位值自动取最大值**：所有层共用同一个 poiValue（全局最大非零值），下游道路连接电池只需填写这个值即可。
2. **边缘格不足时自动降级**：若某层的边缘格数量 < count，则取该层全部边缘格，不报错。
3. **点位叠加**：多层输出叠加到同一张 grid，若不同层点位落在同一格，后写入的覆盖先写入的（值相同，不影响结果）。
