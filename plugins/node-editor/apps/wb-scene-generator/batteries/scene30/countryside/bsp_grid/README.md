# BSP网格 (bsp_grid)

用二叉空间分割（BSP）算法递归划分输入网格的区域，生成多房间布局，每个房间填充唯一ID的多值网格。

## 功能特点

1. **以网格驱动尺寸**：从输入网格的行列数自动推断生成区域大小，无需手动填写宽高
2. **纯BSP分割**：沿长边递归二叉切割，叶节点即为最终房间区域，生成过程干净简洁
3. **多值网格输出**：每个房间填充不同的正整数ID，墙体区域保持 0，方便后续叠加和筛选
4. **可控随机性**：`splitVariance` 参数调节分割点偏移幅度，既能生成规整均匀的房间，也能生成大小有变化的布局
5. **名称清单随行**：输出 `outputNameList`，每个房间 ID 对应 `房间 N` 名称，可直接传给其他电池使用

## 适用情况

- 地牢生成、Roguelike 场景的初始房间布局
- 需要多个独立房间区域的任何网格生成场景
- 作为更复杂室内流程的第一步（后接走廊连接、家具摆放等）
- 不适用于需要有机曲线房间的场景（BSP生成的均为矩形房间）

## 基本使用方法

连接一个矩形网格到 `grid` 端口，执行后即可在 `outputGrid` 中得到各房间唯一ID填充的多值网格。`outputNameList` 可连接到合并或保存电池使用。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | - | 输入网格，从其行列尺寸推断生成区域大小 |
| minRoomSize | number | 8 | 最小房间尺寸（含墙），小于此值的区域不再分割 |
| maxDepth | number | 5 | BSP最大递归深度，控制房间数量上限（约 2^maxDepth 个） |
| splitVariance | number | 0.2 | 分割点偏离中心的最大比例（0~0.4） |
| wallThickness | number | 1 | 房间边缘保留的墙体格子数（最小1） |
| seed | number | 0 | 随机种子，0 使用当前时间戳 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 多值网格，每个房间区域填充唯一正整数ID，墙体为0 |
| outputNameList | array | 房间ID与名称清单，格式 `[{id, name}]` |
| roomCount | number | 本次生成的房间总数 |

## 参数说明

### splitVariance（分割随机幅度）

| 值 | 效果 |
|----|------|
| 0 | 每次从正中切割，所有房间大小高度均匀 |
| 0.2 | 默认值，轻微随机偏移，略有大小变化 |
| 0.4 | 最大偏移，房间尺寸差异明显 |

### maxDepth 与房间数量关系

BSP 每层最多将空间翻倍，`maxDepth=5` 时最多约 32 个叶节点（实际因 `minRoomSize` 限制会更少）。

## 注意事项

1. **网格尺寸需大于 minRoomSize × 2**：否则根节点无法完成第一次分割，输出将只有1个或0个房间
2. **wallThickness 过大会吞掉小房间**：若收缩后内部尺寸 ≤ 0，该房间会被跳过（视为纯墙），输出 `roomCount` 会小于叶节点数
3. **输出网格中 0 代表墙体**：房间与房间之间的墙体区域均为 0，可叠加到其他网格上使用
4. **输入网格内容不影响结果**：只读取尺寸，不读取格子值，可连接任意已有网格来复用其尺寸
