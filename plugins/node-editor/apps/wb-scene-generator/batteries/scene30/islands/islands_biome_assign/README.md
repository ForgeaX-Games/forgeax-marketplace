# islands_biome_assign (islands_biome_assign)

把高度图和湿度图转换为深水、浅水、沙滩、草地、森林、山地、雪地等岛屿地形。

## 功能特点

1. **直接映射**：沿用原生成器的地形阈值，把连续值转成离散地块。
2. **双图透传**：输出时保留高度图和湿度图，方便后续节点继续使用。
3. **可渲染结果**：输出网格已经是可直接渲染的主地形。

## 适用情况

- 已经有高度图和湿度图，需要进入正式地形阶段时。
- 需要快速观察种子变化对群系分布影响时。
- 不适合只对现成地形做局部修补的场景。

## 基本使用方法

将 `islands_base_noise.heightMap` 和 `moistureMap` 分别连接到本节点。输出的 `outputGrid` 再接 `islands_terrain_refine`。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| heightMap | grid | - | 输入高度图 |
| moistureMap | grid | - | 输入湿度图 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 岛屿地形网格 |
| outputNameList | array | 岛屿地形名称清单 |
| heightMap | grid | 透传高度图 |
| moistureMap | grid | 透传湿度图 |

## 处理规则说明

### 处理规则格式

```json
[]
```

### 处理规则参数

本电池不使用 `processRules`，分类规则固定写在节点内部。

## 使用示例

### 输入示例

```json
{
  "heightMap": [[0.11, 0.31, 0.47, 0.82]],
  "moistureMap": [[0.20, 0.35, 0.74, 0.44]]
}
```

### 输入文件格式

```json
{
  "grid": [[0.11, 0.31, 0.47, 0.82]],
  "districts": []
}
```

### 输出示例

```json
{
  "outputGrid": [[0,1,5,6]],
  "outputNameList": [{"id":5,"name":"森林"}],
  "heightMap": [[0.11,0.31,0.47,0.82]],
  "moistureMap": [[0.20,0.35,0.74,0.44]]
}
```

## 注意事项

1. **尺寸必须一致**：`heightMap` 和 `moistureMap` 的尺寸必须完全相同。
2. **阈值固定**：如果想自定义海平面或雪线，需要再扩展节点逻辑。
3. **本节点不平滑**：如果海岸或林线过碎，请交给后续 `islands_terrain_refine` 处理。
