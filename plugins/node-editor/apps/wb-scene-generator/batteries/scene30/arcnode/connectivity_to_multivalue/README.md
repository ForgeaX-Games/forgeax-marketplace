# 按连通性转多值网格 (connectivity_to_multivalue)

对网格列表中每个网格，将所有 4 连通区域各自赋予唯一整数值，把单值网格转为多值网格。

## 功能特点

1. **4连通标记**：使用 BFS 遍历，上下左右四个方向视为连通，对角不连通
2. **自动起始 ID**：每个网格独立计算，起始值 = 该网格非过滤值的最大值 + 1，避免 ID 与原始值冲突
3. **过滤值保持**：边界值（默认 0）位置保持原值，不参与连通性计算
4. **批处理**：输入网格列表，输出等长的多值网格列表

## 适用情况

- 将建筑占地掩码列表（每个格子值为 1）按连通性区分为多栋建筑，每栋赋予不同 ID
- 将任意单值掩码拆分为可独立识别的多个区域
- 配合 arcnode 系列电池流水线，为每个独立区域后续标注名称清单

## 基本使用方法

1. 将单值掩码网格列表连接到 `gridList`
2. 根据网格中的边界值设置 `filterValue`（默认 0）
3. 执行后 `outputGridList` 输出多值网格列表，每个 4 连通区域具有唯一整数值

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| gridList | any | - | 输入网格列表，每个元素为一个网格 |
| filterValue | number | 0 | 边界值，该值的格子视为隔断，不参与连通性计算 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGridList | array | 与输入等长的多值网格列表，每个 4 连通区域赋予唯一整数值 |

## 注意事项

1. **ID 不跨网格共享**：每个网格独立计算起始 ID，不同网格的区域 ID 可能重叠，若需全局唯一 ID 需后续处理
2. **起始值规则**：起始值 = 当前网格所有非过滤值中的最大值 + 1；若网格全为过滤值则输出空网格
3. **遍历顺序**：区域 ID 按从左到右、从上到下的 BFS 遍历顺序依次递增分配
