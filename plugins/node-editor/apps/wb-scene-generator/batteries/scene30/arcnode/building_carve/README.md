# 建筑雕刻 (building_carve)

对输入网格非零区域的包围盒执行两层分段退线，再将退线结果以中心点为基准等比放大，使最终形状的包围盒与原始包围盒完全一致。

## 功能特点

1. **贴边输出**：最终形状的包围盒与输入包围盒完全一致，建筑紧贴地块边界
2. **两层退线**：第一层对四条边各自独立随机内缩，第二层对每条边分段做精细凹凸处理
3. **中心缩放**：退线完成后以形状中心为基准等比放大，保持凹凸风格不变
4. **可复现**：固定 seed 得到固定结果，seed=0 使用时间戳实现随机

## 与【建筑退线】的区别

| | 建筑退线 (building_profile) | 建筑雕刻 (building_carve) |
|---|---|---|
| 输出包围盒 | 比输入包围盒小 1~4 格 | 与输入包围盒完全一致 |
| 形状风格 | 凹凸分段轮廓 | 凹凸分段轮廓（相同） |
| 额外处理 | 无 | 中心点等比放大 |

## 适用情况

- 需要建筑形状贴紧地块边界的场景（如城市建造、俯视地图）
- 星露谷物语、符文工厂等像素农场/村庄游戏的建筑生成
- 已有精确矩形地块，需要在不改变地块范围的前提下添加轮廓细节

## 基本使用方法

1. 准备一个 grid，用非零值标记目标建筑区域
2. 连接至 `gridList`，可选设置 `seed`
3. 输出 `outputGridList` 中值为 1 的区域即为最终建筑占地，其包围盒与输入完全对齐

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| gridList | array | — | 源网格列表，非零像素的包围盒作为建筑矩形 |
| seed | number | 0 | 随机种子，0 使用当前时间戳 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGridList | array | 与输入等尺寸，建筑区域=1，其余=0；包围盒与输入一致 |

## 算法说明

### 第一步：两层分段退线（同 building_profile）

取输入 grid 非零像素的包围盒，执行两层退线：
- **第一层**：四条边各自从 {1,2,3,4} 按权重 90:70:25:15 独立采样内缩量
- **第二层**：每条边按边长分段（每 7 格一段），每段由第一层退线量调制概率决定内退/外突

### 第二步：中心点等比放大

退线完成后，计算退线结果的实际包围盒（carvedBBox），以其中心为基准，按比例：

```
scaleR = carvedBBox.height / targetBBox.height
scaleC = carvedBBox.width  / targetBBox.width
```

对目标 bbox 内每个像素做反向映射（最近邻采样），将退线形状拉伸到覆盖原始包围盒。

## 注意事项

1. **输入区域大小**：建议包围盒宽高各不小于 8 格，太小时退线可能导致内框过窄
2. **缩放会略微模糊细节**：放大过程使用最近邻采样，形状风格保持但像素边缘可能有轻微变化
3. **seed 复现**：相同 seed + 相同输入 → 完全相同的输出
