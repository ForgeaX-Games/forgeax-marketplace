# 建筑退线 (building_profile)

对输入网格非零区域的包围盒执行两层退线，生成具有凹凸轮廓的建筑占地 mask。

## 功能特点

1. **两层退线**：第一层对四条边各自独立随机内缩，第二层对每条边分段做精细凹凸处理
2. **第一层概率加权**：退 1~4 格，权重 90:70:50:40，退线越多概率越小
3. **第二层调制**：第一层退得少的边，第二层更倾向于继续向内退；退得多的边则几乎不再退
4. **双向退线**：第二层 80% 向内、20% 向外突出，外突不超过原始包围盒边界
5. **可复现**：固定 seed 得到固定结果，seed=0 使用时间戳实现随机

## 适用情况

- 需要生成具有不规则边缘的 2D 建筑占地形状（俯视图）
- 星露谷物语、符文工厂等像素农场/村庄游戏的建筑生成
- 为矩形占地区域添加视觉上的凹凸细节，避免千篇一律的方块建筑
- 不适用于需要精确控制建筑形状的场景

## 基本使用方法

1. 准备一个 grid，用非零值标记目标建筑区域（包围盒即建筑矩形）
2. 连接至 `inputGrid`，可选设置 `seed`
3. 输出 `outputGrid` 中值为 1 的区域即为最终建筑占地

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | — | 源网格，非零像素的包围盒作为建筑矩形 |
| seed | number | 0 | 随机种子，0 使用当前时间戳 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 与输入等尺寸，建筑区域=1，其余=0 |

## 算法说明

### 第一层退线

取输入 grid 非零像素的**包围盒**，对上、下、左、右四条边**各自独立**从 {1,2,3,4} 中按权重随机采样内缩量：

| 退线量 | 权重 | 相对概率 |
|--------|------|---------|
| 1 格 | 90 | 最高 |
| 2 格 | 70 | 较高 |
| 3 格 | 50 | 较低 |
| 4 格 | 40 | 最低 |

四边相互独立，结果为一个略小的不规则矩形内框。

### 第二层退线

对第一层内框的每条边：
- 按边长分成 3~6 段（边越长段越多）
- 每段独立概率判断是否退线，概率由该边**第一层退线量**调制：

| 第一层退线量 | 第二层内退概率 | 第二层外突概率 |
|------------|-------------|-------------|
| 1 格（退得少）| 80% | 10% |
| 2 格 | 60% | 7% |
| 3 格 | 40% | 5% |
| 4 格（退得多）| 20% | 2% |

外突方向受原始包围盒边界限制，不会超出原始矩形范围。

## 注意事项

1. **输入区域大小**：建议包围盒宽高各不小于 8 格，太小时第一层退线可能导致内框过窄而报错
2. **外突限制**：第二层向外突出不超过原始包围盒边界，不会影响到包围盒外的其他区域
3. **seed 复现**：相同 seed + 相同输入 → 完全相同的输出，便于调试和版本管理
