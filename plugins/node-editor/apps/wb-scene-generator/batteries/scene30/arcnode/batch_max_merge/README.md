# 批量归一合并 (batch_max_merge)

对两个网格列表按位置对应，将同一索引处的所有网格归一合并（逐格取最大值），输出等长结果列表。

## 功能特点

1. **按位对应**：两个列表的第 0 项合并、第 1 项合并……依此类推
2. **任意深度支持**：每个 item 可以是单网格，也可以是任意深度的网格嵌套列表，内部递归展平后统一参与合并
3. **归一合并**：逐格取所有参与网格中的最大值，与 `grid_max_merge` 逻辑一致
4. **批处理友好**：配合 arcnode 系列批处理电池，实现多建筑批量图层叠加

## 适用情况

- 将建筑占地列表与内墙列表逐个合并为完整建筑网格列表
- 将门洞列表与窗洞列表合并为开口层列表，再与墙体列表合并
- 任何需要"两组数据按位置叠加"的网格批处理场景

## 基本使用方法

1. 将第一组网格列表连接到 `gridListA`
2. 将第二组网格列表连接到 `gridListB`
3. 执行后 `outputGridList` 输出逐位置合并的结果

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| gridListA | any | - | 第一个网格列表，每个元素可以是单网格或任意深度嵌套列表 |
| gridListB | any | - | 第二个网格列表，每个元素可以是单网格或任意深度嵌套列表 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGridList | array | 按位置对应合并的结果列表 |

## 注意事项

1. **长度对齐**：输出长度等于两个输入列表中较短的那个，多余部分被忽略
2. **尺寸基准**：每对合并时以第一个有效网格的尺寸为基准，其他网格超出范围部分视为 0
3. **归一合并取最大值**：适合掩码类网格叠加；若某位置所有网格值均为 0，输出也为 0
