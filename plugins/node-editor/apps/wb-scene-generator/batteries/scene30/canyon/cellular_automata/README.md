# 元胞自动机 (cellular_automata)

批量在蒙版列表内用元胞自动机规则迭代演化随机噪点，生成有机洞穴地形网格列表。

## 功能特点

1. **批量处理**：接受多个蒙版网格，一次返回对应数量的洞穴网格列表
2. **自然有机形态**：通过生死规则将随机噪点收敛为边缘圆润的洞穴结构
3. **蒙版约束**：仅在每个 mask 非零区域内演化，轮廓由上游蒙版决定
4. **独立随机**：每个蒙版基于 `seed + i×1000` 独立随机，批量结果各不相同但可复现
5. **双输出网格**：同时输出洞穴结构网格（墙/地板）和空洞内部网格（可通行区域蒙版）
6. **名称清单**：输出与网格列表下标对应的 `{id, name}[]` 清单，可直接对接下游标签节点

## 适用情况

- 批量生成多个洞穴区域（如地牢多房间、矿洞多分支）
- 配合地块分类器或 BSP 蒙版生成列表，对每个区域独立演化
- 配合 `name_list_pass` 传入已有名称清单，为每个洞穴命名

## 基本使用方法

1. 将上游蒙版列表（如地块区域拆分后的网格列表）连接到 `maskList` 端口
2. 可选：连接 `nameList` 端口传入名称（`{id,name}[]` 或 `string[]`）；未连接时自动生成「洞穴1」「洞穴2」…
3. 调整 `fillProbability` 控制墙壁密度（建议 0.40–0.55）
4. 将 `caveGrids` / `caveInteriors` 输出连接至渲染或后续处理节点

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| maskList | array | 必填 | 蒙版网格列表（number[][][]），也可传单个网格（number[][]） |
| fillProbability | number | 0.45 | 初始墙壁填充概率（0–1） |
| birthLimit | number | 4 | 地板→墙壁的邻居墙数阈值（含） |
| deathLimit | number | 3 | 墙壁→地板的邻居墙数阈值（不含） |
| iterations | number | 5 | 迭代轮数 |
| borderWall | number | 1 | 1=边界强制为墙，0=边界可通 |
| seed | number | 0 | 基础随机种子，0 使用内置默认值 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| caveGrids | array | 洞穴网格列表（number[][][]），每张：0=蒙版外，1=洞穴墙，2=洞穴空间 |
| nameList | array | 固定名称清单：`[{id:1, name:"洞穴墙"}, {id:2, name:"洞穴空间"}]` |

## 参数说明

### 规则逻辑

每次迭代，对蒙版内每个格子按如下规则判断：

- **当前为墙**：8邻居中墙数 `< deathLimit` → 变地板（孤立墙消亡）
- **当前为地板**：8邻居中墙数 `>= birthLimit` → 变墙（被围则填实）

### 批量种子策略

| 基础 seed | 第 i 个蒙版实际 seed |
|-----------|---------------------|
| 0（默认） | `i × 1000 + 12345` |
| 非零值 N  | `N + i × 1000` |

### 推荐参数组合

| 效果 | fillProbability | birthLimit | deathLimit | iterations |
|------|----------------|------------|------------|------------|
| 标准洞穴 | 0.45 | 4 | 3 | 5 |
| 宽阔大厅 | 0.40 | 5 | 2 | 4 |
| 密集碎石 | 0.55 | 3 | 4 | 3 |
| 圆润平滑 | 0.45 | 4 | 3 | 8 |

## 注意事项

1. **maskList 必填**：未连接或格式错误时节点返回 error，不会输出列表
2. **单网格兼容**：传入单个 `number[][]` 时自动包装，输出列表长度为 1
3. **nameList 与列表对应**：`nameList[i].id === i+1`，与 `caveGrids[i]` / `caveInteriors[i]` 下标对齐
4. **迭代过多风险**：iterations 超过 10 时可能导致结果趋于全墙或全空，建议范围 3–8
