# 宇宙地形细节变化 (cosmos_terrain_variation)

在基础地形上叠加细节噪声，添加苔藓、泥地、砾石、灰烬、黑曜石等子地形变体。

## 功能特点

1. **细节FBM噪声**：使用0.1频率的2层FBM采样，产生连续分布的地形变体斑块
2. **基础地形驱动**：不同基础地形触发不同变体（草→苔藓/泥，火山岩→灰烬/黑曜石等）
3. **强度可调**：variationStrength=0关闭变化，>1增加变化密度

## 适用情况

- 接在cosmos_biome_mapper或cosmos_zone_marker后增加地形细节层次
- 需要丰富地表材质的场景

## 基本使用方法

1. 将上游地形grid接入
2. 设置variationStrength（建议0.5-2.0）
3. 输出variedGrid接入cosmos_object_placer或直接用于渲染

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| terrainGrid | grid | - | 基础地形网格 |
| seed | number | 0 | 随机种子 |
| variationStrength | number | 1.0 | 变化强度倍率 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| variedGrid | grid | 细化后的地形网格 |
| nameList | array | 新增变体类型名称列表 |

## 变体对照表

| 基础地形 | 可能的变体 | 条件 |
|----------|-----------|------|
| 草地(3) | 苔藓(16) | 低噪声区域 |
| 草地(3) | 泥地(18) | 高噪声区域 |
| 泥土(4) | 砾石(17) | 低噪声区域 |
| 泥土(4) | 泥地(18) | 高噪声区域 |
| 沙地(2) | 砾石(17) | 低噪声区域 |
| 火山岩(11) | 灰烬(15) | 低噪声区域 |
| 火山岩(11) | 黑曜石(19) | 高噪声区域 |
| 毒地(10) | 沼泽(12) | 低噪声区域 |
| 浅水(1) | 珊瑚礁(13) | 高噪声区域 |

## 注意事项

1. **随机性**：内部使用Math.random()实现概率判断，seed仅影响噪声采样，多次执行结果可能略有不同
2. **不可通行地形**：水(0,1)、山脉(6)、岩浆(8)不做变化处理
