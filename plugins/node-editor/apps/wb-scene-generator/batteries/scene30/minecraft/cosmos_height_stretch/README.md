# 高度图拉伸 (cosmos_height_stretch)

将高度图每个格子的值乘以拉伸系数，等比例放大或压缩高度信息，完整保留地形细节。

## 功能特点

1. **无损拉伸**：对每个像素做简单乘法，不做任何量化或分级，完整保留高度分布
2. **双向缩放**：系数 >1 放大高度差（地形起伏更明显），系数 <1 压缩（地形更平坦）
3. **整数输出**：结果四舍五入为整数，直接用于渲染器的多值网格渲染
4. **透传兼容**：系数设为 1 时原样输出，可用作高度图的中间透传节点

## 适用情况

- 调整宇宙地形高度图的垂直比例，用于 3D 渲染时控制山峰高度
- 将 0-1000 范围的高度图拉伸到更大范围（如 ×10 → 0-10000）供渲染器精细采样
- 压缩高度差使地形整体更平坦（如系数 0.3）

## 基本使用方法

1. 将 `cosmos_terrain_gen.outputGridList` 或 `cosmos_terrain_variation.variedGridList` 连接到 `inputGridList`
2. 设置 `stretchFactor`（如 `3` 表示高度值扩大为原来的 3 倍）
3. `outputGrid` 即为拉伸后的整数高度网格，可直接送入渲染器

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGridList | array | [] | 高度图列表，取 [0] 作为输入高度图 |
| stretchFactor | number | 1 | 拉伸系数，正数；>1 放大，<1 压缩 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 拉伸后的整数高度网格 |

## 注意事项

1. **系数必须为正数**：传入 0 或负数时自动回退为 1（原样输出）
2. **值域不限制**：拉伸后的值可能超过原始范围（如 1000），渲染器需支持相应值域
3. **精度**：结果做 `Math.round` 取整，拉伸系数极小时（如 0.001）大量格子会归零
