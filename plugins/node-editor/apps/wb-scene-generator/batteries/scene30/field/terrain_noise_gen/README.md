# 噪声地形生成器 (terrain_noise_gen)

基于多层叠加分形噪声（FBM）生成野外地形网格，输出水系、沙滩、草地三层地形分布。

## 功能特点

1. **可复现**：相同 `seed` 始终生成相同地图
2. **自然过渡**：FBM 4层叠加使水系→沙滩→草地边界连续自然
3. **参数灵活**：可独立调节地图尺寸、噪声频率、各地形占比阈值

## 适用情况

- 作为野外场景流水线的第一个节点，后接 `terrain_smoother` 平滑处理
- 宝可梦风、星露谷风、沙盒生存等需要自然户外地形的 2D 游戏

## 基本使用方法

1. 将本节点拖入画布，配置 `seed`、`width`、`height` 等参数
2. 将输出 `terrainGrid` 连接到 `terrain_smoother` 节点的 `terrainGrid` 输入

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| seed | number | 0 | 随机种子，0 使用当前时间戳（每次不同） |
| width | number | 64 | 地图宽度（网格列数） |
| height | number | 64 | 地图高度（网格行数） |
| scale | number | 0.05 | 噪声频率缩放比；越小地形越宏观，越大越破碎 |
| waterThresh | number | 0.35 | 低于此噪声值的格子为水系；调大水域面积增加 |
| sandThresh | number | 0.45 | 噪声值在 waterThresh 与此值之间为沙滩；调大沙滩带变宽 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| terrainGrid | grid | 二维地形网格，0=水系，1=沙滩，2=草地 |

## 注意事项

1. **阈值约束**：`waterThresh` 必须小于 `sandThresh`，否则节点返回错误
2. **scale 调优**：推荐范围 0.02~0.1；小于 0.02 地形过于均匀，大于 0.1 地形过于破碎
3. **种子复现**：固定 `seed`（非 0 值）可锁定特定地图用于关卡设计
