# 野外装饰物采样器 (field_decoration_sampler)

对地形网格进行分层装饰物采样，在草地上用泊松盘采样放置树木和灌木，随机散布岩石，并在水边放置荷叶，输出统一的装饰物列表。

## 功能特点

1. **泊松盘采样树木**：树木之间保持自然间距，避免聚簇或重叠
2. **分层采样灌木**：在树木的空隙中再次泊松采样灌木，形成疏密有致的林相
3. **岩石随机散布**：按密度参数在草地和沙滩随机散点，增加地面细节
4. **水边荷叶**：自动检测水系与陆地交界处，概率性放置荷叶

## 适用情况

- 野外场景流水线的第三环节，接在 `terrain_smoother` 之后
- 需要丰富地面细节、避免大片空旷区域的户外 2D 游戏场景

## 基本使用方法

1. 将 `terrain_smoother` 的 `smoothedGrid` 连接至本节点 `terrainGrid` 输入
2. 同步传入与 `terrain_noise_gen` 相同的 `seed`，保证地形与装饰物可复现
3. 将输出 `decorations` 连接至 `walkable_grid_builder` 和 `field_scene_merger`

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| terrainGrid | grid | — | 平滑后的地形网格（0=水/1=沙/2=草） |
| seed | number | 0 | 随机种子，0 使用时间戳 |
| treeRadius | number | 2.5 | 树木泊松盘最小间距（格）；调大使树木更稀疏 |
| bushRadius | number | 1.5 | 灌木泊松盘最小间距（格）；调大使灌木更稀疏 |
| rockDensity | number | 0.03 | 岩石散布概率（0~1）；调大岩石更多 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| decorations | array | 装饰物对象数组，每项格式：`{ x, y, type, variant }` |

## 参数说明

### decorations 数组结构

每个装饰物对象包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| x | number | 装饰物所在列（格坐标） |
| y | number | 装饰物所在行（格坐标） |
| type | string | 类型：`"tree"` / `"bush"` / `"rock"` / `"lilypad"` |
| variant | number | 外观变体编号（从 0 开始），供渲染层选择贴图帧 |

### type 取值说明

| type | 放置区域 | 采样方式 | 对碰撞的影响 |
|------|---------|---------|------------|
| tree | 草地 | 泊松盘采样（半径 treeRadius） | 阻挡 |
| bush | 草地（树木空隙） | 泊松盘采样（半径 bushRadius） | 可通行 |
| rock | 草地、沙滩 | 随机散布（概率 rockDensity） | 阻挡 |
| lilypad | 水格（靠近陆地） | 随机（概率 8%） | 可通行 |

## 注意事项

1. **泊松盘采样性能**：地图尺寸超过 128×128 且 radius 较小时，采样数量可能较多，执行时间略长（通常在 1 秒内）
2. **荷叶仅在边缘水格**：远离陆地的深水区不会放置荷叶
3. **装饰物不重叠**：树木、灌木、岩石之间会互相避让，同一格最多一种装饰物
