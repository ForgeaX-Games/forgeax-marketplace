# 地形平滑器 (terrain_smoother)

对地形网格执行细胞自动机平滑，消除噪声生成后的孤立飞地，使水系、沙滩、草地边界更加自然连通。

## 功能特点

1. **消除飞地**：单格或小块孤立地形被周围主体地形吸收
2. **保留大型区域**：仅平滑边界，不破坏大面积连通区域的主体结构
3. **可调强度**：通过迭代次数控制平滑程度

## 适用情况

- 接在 `terrain_noise_gen` 之后，作为野外场景生成流水线的第二环节
- 需要去除噪声直出的"锯齿感"和孤立格子

## 基本使用方法

1. 将 `terrain_noise_gen` 的 `terrainGrid` 输出连接至本节点的 `terrainGrid` 输入
2. 调整 `iterations` 控制平滑强度（推荐 1~3）
3. 将本节点的 `smoothedGrid` 连接至下游 `field_decoration_sampler` 和 `walkable_grid_builder`

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| terrainGrid | grid | — | 待平滑的地形网格（0=水/1=沙/2=草） |
| iterations | number | 2 | 平滑迭代次数；0 表示原样输出，不平滑 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| smoothedGrid | grid | 平滑后的地形网格，值编码与输入相同 |

## 注意事项

1. **迭代次数**：1~3 次适合大多数场景；超过 5 次后地形趋于"宏块化"，细节丢失
2. **边界策略**：边界外视为与边界格同地形（Clamp 扩展），不会产生异常边缘
3. **平票规则**：当邻居中多种地形出现次数相同时，保留该格原有地形值
