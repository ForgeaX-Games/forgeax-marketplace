# 悠闲海岛村落 (island_cozy_village)

生成一片悠闲海岛场景，海洋环绕有机形状的小岛，岛上分布沙滩、草地、树林、花丛、民居和码头。

## 功能特点

1. **有机岛屿形状**：使用多控制点扰动生成自然弧线轮廓，避免圆形或方形的机械感
2. **地形分层**：从岛心到海岸按距离分配草地、树林、沙滩，边缘设浅海过渡带
3. **悠闲设施**：自动在草地上分散放置民居小屋和石板路，在沙滩边缘生成码头栈桥
4. **装饰细节**：随机在草地上散布花丛，提升场景活力
5. **多岛支持**：可生成2-4座相互隔开的岛屿，形成群岛格局

## 适用情况

- 休闲/农场风格游戏地图（星露谷、动物森友会类）
- 探险类游戏的热带岛屿关卡
- 需要海岛主题背景的2D场景

## 基本使用方法

直接拖入本电池，设置尺寸和种子，即可输出完整海岛地图和名称清单，可直接接入贴图模式渲染。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 80 | 网格列数，最小40 |
| height | number | 80 | 网格行数，最小40 |
| seed | number | 0 | 随机种子，0使用当前时间戳 |
| islandCount | number | 2 | 岛屿数量，1-4 |
| houseCount | number | 4 | 民居数量，1-12 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 海岛地图网格 |
| outputNameList | array | 掩码名称清单，格式为 [{id, name, type}] |

## 掩码值说明

| ID | 名称 | 类型 |
|----|------|------|
| 1 | 海洋 | tile |
| 2 | 浅海 | tile |
| 3 | 沙滩 | tile |
| 4 | 草地 | tile |
| 5 | 树林 | asset |
| 6 | 民居 | asset |
| 7 | 码头 | tile |
| 8 | 石板路 | tile |
| 9 | 花丛 | asset |

## 注意事项

1. **尺寸建议**：80×80 以上效果最佳，过小可能导致房屋找不到合适位置
2. **多岛间距**：系统会自动保持岛屿间距，若尺寸太小可能只生成1-2个岛屿
3. **贴图模式**：本电池直接输出 outputGrid + outputNameList，可直接切换到贴图模式
