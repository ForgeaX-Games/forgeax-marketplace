# 出血森林生成 (gen_bleeding_forest)

使用元胞自动机（Cellular Automata）算法生成有机森林地图，包含生物发光空地、暗池水坑和随机步行路径。

## 功能特点

1. **元胞自动机**：多轮CA迭代（生规则≥5/存规则≥4）生成有机边界的森林与空地
2. **最大连通区域**：洪水填充找到最大空地簇，保证传送门和出生点可达
3. **随机步行路径**：在随机空地对之间carve路径，增加林间通道
4. **生物发光点**：空地中稀疏散布发光格，营造诡异的生物发光效果
5. **暗池散布**：深色水池随机分布，增加地形层次感

## 适用情况

- 古老森林、精神幻境、生物发光洞穴场景
- 需要有机边界而非方形房间的自然地图
- 恐怖、梦境、奇幻类游戏的场景生成

## 基本使用方法

连接 `outputGrid` 和 `outputNameList` 到渲染电池。通过 `fillRatio` 调整森林密度，`iterations` 控制空地边界的平滑程度。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 72 | 网格宽度（列数），最小30 |
| height | number | 72 | 网格高度（行数），最小30 |
| seed | number | 0 | 随机种子，0使用当前时间戳 |
| iterations | number | 4 | CA迭代次数，1-10 |
| fillRatio | number | 0.55 | 初始森林填充比，0.3-0.8 |
| pathCount | number | 8 | 连通路径数量 |
| poolCount | number | 6 | 暗池数量 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 森林地图网格 |
| outputNameList | array | 掩码ID与名称映射 |
| spawnX | number | 出生点X瓦片坐标 |
| spawnY | number | 出生点Y瓦片坐标 |
| portalX | number | 传送门X瓦片坐标 |
| portalY | number | 传送门Y瓦片坐标 |
| decors | array | 装饰物列表，每项含 {tx, ty, type} |

## 注意事项

1. **fillRatio过高**：若大于0.75，可能导致空地极少，传送门和出生点被迫放在边缘
2. **迭代次数**：iterations=1-2产生粗糙边界，4-6较平滑，超过8边界可能过于圆滑
3. **路径碰撞**：随机步行路径可能部分重叠，这是正常现象
