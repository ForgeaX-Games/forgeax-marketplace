# 恐惧迷宫生成 (gen_fear_labyrinth)

使用迭代DFS（深度优先搜索）算法生成完美迷宫，叠加恐惧房间、血脉纹理和血晶装饰，营造黑暗恐怖地牢氛围。

## 功能特点

1. **DFS完美迷宫**：深度优先搜索生成无环完美迷宫，每两点之间有且仅有一条路径
2. **恐惧房间**：随机将迷宫路口扩展为3×3的宽阔恐惧房间，打破单调通道
3. **血脉墙体**：紧邻通道的墙格有概率变成血脉纹理，增加恐怖氛围
4. **血晶装饰**：恐惧房间内随机生成血晶格，增加游戏交互点
5. **传送门定位**：传送门放置在右下角区域的第一个可用通道格

## 适用情况

- 恐怖地牢、精神恐惧、梦魇迷宫场景
- Roguelike类完美迷宫地图需求
- 需要DFS迷宫基础形态叠加装饰的游戏

## 基本使用方法

直接连接 `outputGrid` 和 `outputNameList` 到渲染电池。`chamberChance` 控制恐惧房间密度，`veinChance` 调整血脉墙比例。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 65 | 网格宽度，最小11，自动转奇数 |
| height | number | 65 | 网格高度，最小11，自动转奇数 |
| seed | number | 0 | 随机种子，0使用当前时间戳 |
| chamberChance | number | 0.2 | 恐惧房间生成概率，0-1 |
| veinChance | number | 0.08 | 血脉墙生成概率，0-1 |
| crystalChance | number | 0.15 | 血晶生成概率，0-1 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 恐惧迷宫网格 |
| outputNameList | array | 掩码ID与名称映射 |
| spawnX | number | 出生点X（左上角，固定为1） |
| spawnY | number | 出生点Y（左上角，固定为1） |
| portalX | number | 传送门X（右下区域） |
| portalY | number | 传送门Y（右下区域） |
| decors | array | 装饰物列表，每项含 {tx, ty, type} |

## 注意事项

1. **奇数尺寸要求**：DFS迷宫算法要求网格宽高为奇数，输入偶数时自动+1
2. **chamberChance过高**：超过0.5时恐惧房间相互连通，可能破坏迷宫结构感
3. **装饰物去重**：恐惧房间装饰会自动去重，同一格只保留一个装饰物
