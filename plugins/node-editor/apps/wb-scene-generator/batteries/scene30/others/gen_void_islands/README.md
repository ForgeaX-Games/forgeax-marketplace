# 虚空岛屿生成 (gen_void_islands)

使用Prim最小生成树算法将圆形浮空岛屿全部连通，生成漂浮于黑暗虚空中的平台群地图。

## 功能特点

1. **区域网格放置**：岛屿均匀分布在网格区域中，保证间距合理且桥梁不会过长
2. **Prim MST连通**：最小生成树保证所有岛屿全部连通，不产生孤立岛屿
3. **2格宽桥梁**：MST每条边生成2格宽的桥，玩家可轻松通过
4. **边缘光晕环**：每个岛屿外围有1格宽光晕（EDGE），营造漂浮发光感
5. **记忆碎片**：岛屿平台上随机散布记忆碎片格，增加收集性

## 适用情况

- 虚空、梦境、次元空间等悬空平台场景
- 需要保证全连通的岛屿平台跑酷地图
- 维度穿越、失忆找回主题的游戏场景

## 基本使用方法

直接连接 `outputGrid` 和 `outputNameList` 到渲染电池。`islandCount` 控制岛屿数量，`islandMinR/MaxR` 控制岛屿大小。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 68 | 网格宽度（列数），最小40 |
| height | number | 68 | 网格高度（行数），最小40 |
| seed | number | 0 | 随机种子，0使用当前时间戳 |
| islandCount | number | 10 | 岛屿数量，3-20 |
| islandMinR | number | 4 | 岛屿最小半径（格数） |
| islandMaxR | number | 8 | 岛屿最大半径（格数） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 虚空岛屿地图网格 |
| outputNameList | array | 掩码ID与名称映射 |
| spawnX | number | 出生点X（第一岛屿中心） |
| spawnY | number | 出生点Y（第一岛屿中心） |
| portalX | number | 传送门X（最后岛屿中心） |
| portalY | number | 传送门Y（最后岛屿中心） |
| decors | array | 装饰物列表，每项含 {tx, ty, type} |

## 注意事项

1. **岛屿半径与网格大小**：islandMaxR过大或islandCount过多会导致岛屿重叠
2. **执行前需连线**：出生点/传送门坐标在运行后才会输出真实值
3. **桥梁方向**：桥梁沿MST边的方向延伸，部分角度的桥梁可能显示为对角线段
