# 文本模板填充 (text_template_fill)

将三个字符串值填入模板中的 `{0}` `{1}` `{2}` 占位符，输出替换后的完整文本。

## 适用情况

- 需要将房间名称、尺寸、描述等信息拼入固定格式的提示词或文本
- 静态固定 3 个插槽，不需要动态扩展

## 基本使用方法

1. 在 `模板` 端口输入含占位符的文本，例如：`房间：{0}，尺寸：{1}x{2}`
2. 分别将三个值连接到 `{0}` ~ `{2}` 端口
3. 输出 `text` 即为替换后的完整文本

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| template | string | "" | 含 {0} {1} {2} 的文本模板 |
| value_0 | string | "" | 替换 {0} 的字符串 |
| value_1 | string | "" | 替换 {1} 的字符串 |
| value_2 | string | "" | 替换 {2} 的字符串 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| text | string | 填充占位符后的完整文本 |

## 注意事项

1. 未填写或未连接的 `value_*` 端口默认为空字符串，对应占位符**保留原样**不被替换
2. 同一占位符在模板中出现多次时，所有出现位置均会被替换
