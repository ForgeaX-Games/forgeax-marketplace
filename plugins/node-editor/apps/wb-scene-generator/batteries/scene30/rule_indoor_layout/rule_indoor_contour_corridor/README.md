# 轮廓偏移走廊 (rule_indoor_contour_corridor)

根据建筑轮廓边缘向内偏移，生成1-2层环形走廊，支持 L/U/T 等异形建筑输入。

## 功能特点

1. **距离场算法**：通过 BFS 计算每个内部像素到边界的最短距离，精确支持任意轮廓
2. **多层走廊**：可配置1-2层环形走廊，偏移距离和宽度均可独立控制
3. **异形支持**：天然支持 L 型、U 型、T 型等不规则建筑平面

## 适用情况

- 需要生成规则室内布局的第一步：从建筑轮廓生成走廊骨架
- 大型建筑需要多层走廊环
- 异形平面的走廊生成

## 基本使用方法

将建筑轮廓网格（非零为内部）连接到 inputGrid，调整偏移距离和走廊宽度参数即可生成走廊环。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | - | 输入网格，非零区域为建筑内部 |
| offsetLayer1 | number | 12 | 第一层走廊距边缘的偏移距离 |
| offsetLayer2 | number | 25 | 第二层走廊偏移距离，0表示不生成 |
| corridorWidth | number | 5 | 走廊宽度（格） |
| wallThickness | number | 1 | 外墙厚度 |
| seed | number | 0 | 随机种子，0使用当前时间戳 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 0=外部/墙, 1=走廊, 2=房间区域 |
| ringCount | number | 实际生成的走廊环层数 |

## 注意事项

1. **偏移距离**：offsetLayer1 建议 10-15，offsetLayer2 建议 20-30
2. **空间不足**：若建筑内部空间不足以容纳第二层走廊，将自动退化为单层
3. **走廊宽度**：建议 4-6 格，过窄会导致后续房间切割困难
