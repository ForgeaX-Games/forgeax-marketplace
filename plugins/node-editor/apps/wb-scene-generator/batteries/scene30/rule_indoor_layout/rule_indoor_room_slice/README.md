# 房间切割 (rule_indoor_room_slice)

将走廊间的房间区域沿长边垂直方向递归切割为独立房间，每个房间分配唯一 ID。

## 功能特点

1. **长边垂直切割**：自动识别每个区域的长轴方向，沿垂直方向进行切割
2. **递归细分**：大区域递归切割直到满足尺寸约束
3. **碎片清理**：面积过小的碎片自动转为墙壁

## 适用情况

- 走廊生成完毕后，需要将走廊之间的空间分割为房间
- 适用于矩形和 L 型等异形区域

## 基本使用方法

将走廊连通后的网格连接到 inputGrid，调整最小/最大房间尺寸参数。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | - | 0=外部/墙, 1=走廊, 2=房间区 |
| minRoomSize | number | 8 | 房间最小边长 |
| maxRoomSize | number | 25 | 超过此值继续切割 |
| wallThickness | number | 1 | 内墙厚度（1-2格） |
| seed | number | 0 | 随机种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 0=墙, 1=走廊, 10+=各房间ID |

## 注意事项

1. **房间ID**：从 10 开始递增，避免与走廊(1)和标记值冲突
2. **碎片处理**：面积小于 4 格的碎片区域自动转为墙壁
3. **内墙厚度**：建议 1 格，2 格会占用更多空间
