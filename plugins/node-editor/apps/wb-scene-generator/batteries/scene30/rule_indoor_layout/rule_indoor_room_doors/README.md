# 房间门洞 (rule_indoor_room_doors)

在每个房间与走廊接壤的墙壁上开门洞，保证所有房间通过走廊系统连通。

## 功能特点

1. **智能选墙**：优先在最长的共享墙段上开门，保证门洞位置合理
2. **多门支持**：每个房间可配置开多个门洞
3. **宽度可控**：门洞宽度 2-4 格可调

## 适用情况

- 房间切割完成后，需要给每个房间添加出入口
- 保证所有房间与走廊系统的连通性

## 基本使用方法

将房间切割后的网格连接到 inputGrid，调整门洞宽度和数量参数。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | - | 0=墙, 1=走廊, 10+=房间ID |
| doorWidth | number | 3 | 门洞宽度（2-4格） |
| maxDoorsPerRoom | number | 2 | 每房间最多门数 |
| seed | number | 0 | 随机种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 门洞标记为走廊值(1)的网格 |

## 注意事项

1. **门洞宽度**：建议 2-4 格，过宽可能破坏墙体完整性
2. **优先级**：门洞优先开在最长的共享墙段上
3. **无可用墙**：若某房间完全不与走廊接壤，则无法开门（需检查走廊布局）
