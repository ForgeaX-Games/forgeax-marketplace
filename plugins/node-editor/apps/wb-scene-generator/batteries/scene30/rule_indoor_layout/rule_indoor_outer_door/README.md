# 外墙开门 (Outer Wall Door)

在建筑外墙上开门，门洞位于走廊与外墙交汇处，连通室内走廊与外部空间。

## 功能特点

1. **精准定位**：仅在走廊接触外墙处开门，保证门通向走廊
2. **间距控制**：自动保持门之间的最小间距，避免过于密集
3. **宽度可控**：门洞宽度通过 doorWidth 参数控制
4. **支持异形**：通过 footprintGrid 支持 L/U/T 等异形建筑的外墙开门

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | - | 处理后的室内网格 |
| footprintGrid | grid | - | 原始建筑轮廓（非零=内部） |
| doorWidth | number | 3 | 外墙门洞宽度 |
| maxDoors | number | 8 | 最多开几个外墙门 |
| maxWallDepth | number | 6 | 最大墙厚搜索深度 |
| seed | number | 0 | 随机种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 外墙开门后的网格 |
