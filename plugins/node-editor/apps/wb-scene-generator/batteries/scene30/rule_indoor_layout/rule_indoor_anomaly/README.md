# 异形/裂隙生成 (rule_indoor_anomaly)

在室内布局中生成异常/裂隙区域，支持自然侵蚀、多边形裂隙和裂缝分支三种形态模式。

## 功能特点

1. **自然侵蚀 (natural)**：从中心点随机扩散，经细胞自动机平滑，生成有机不规则形态
2. **多边形 (polygon)**：生成随机星形/凸多边形并光栅化，适合大型集中异常区
3. **裂缝 (crack)**：从中心向多方向发射分支裂缝线，模拟结构破坏

## 适用情况

- 在完成的室内布局中添加异常/腐蚀/裂隙区域
- 制造游戏中的特殊地形或事件区域
- 参考图中的红色异常侵入效果

## 基本使用方法

将完成开门的室内布局网格连接到 inputGrid，选择形态模式和异常区大小比例。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | - | 完成布局的室内网格 |
| anomalyValue | number | 5 | 异常区标识值 |
| sizeRatio | number | 0.15 | 异常区占内部面积的比例 |
| shape | string | "natural" | 形态模式：natural/polygon/crack |
| seed | number | 0 | 随机种子 |

### 形态模式说明

| 模式 | 说明 |
|------|------|
| natural | 随机扩散+细胞自动机平滑，有机不规则形态 |
| polygon | 随机星形多边形光栅化，集中的异常区 |
| crack | 多方向分支裂缝线，模拟结构破损 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 嵌入异常区后的网格 |

## 注意事项

1. **异常值**：默认为 5，确保不与走廊(1)和房间ID(10+)冲突
2. **比例控制**：sizeRatio 建议 0.05-0.3，过大会破坏大量房间
3. **可选步骤**：异常生成为可选步骤，不需要时可跳过此电池
