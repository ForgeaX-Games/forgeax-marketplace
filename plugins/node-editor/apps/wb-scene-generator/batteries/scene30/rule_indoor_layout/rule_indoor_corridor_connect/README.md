# 走廊环连通 (rule_indoor_corridor_connect)

检测多层走廊环并用垂直/水平连接走廊贯通，确保走廊系统完整连通。

## 功能特点

1. **自动检测**：通过连通分量分析识别独立的走廊环
2. **均匀分布**：连接走廊在建筑内部均匀分布，避免集中
3. **兜底连通**：主连接后仍有断开的走廊分量，用最短路径桥接修复

## 适用情况

- 轮廓偏移走廊电池输出了多层走廊环，需要连通
- 即使只有单层走廊也可安全使用（直接透传）

## 基本使用方法

将 rule_indoor_contour_corridor 的输出网格连接到 inputGrid，调整连接走廊数量和宽度即可。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | - | 0=外部, 1=走廊, 2=房间区 |
| connectorWidth | number | 4 | 连接走廊宽度（2-6格） |
| connectorCount | number | 4 | 连接走廊数量（2-8条） |
| seed | number | 0 | 随机种子 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 所有走廊环连通后的网格 |

## 注意事项

1. **单层走廊**：若只有一个走廊连通分量，直接透传不做修改
2. **连接宽度**：建议 2-6 格，与走廊主宽度匹配
3. **连接数量**：建议 4 条左右，过多会压缩房间空间
