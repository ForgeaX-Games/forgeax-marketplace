# 网格左上角四分之一 (grid_quarter_tl)

提取输入网格左上角四分之一区域，输出子网格。

## 功能特点

1. **尺寸自动计算**：输出尺寸为 `⌈width/2⌉ × ⌈height/2⌉`，奇数尺寸向上取整
2. **零参数**：只需接入网格，无需配置任何参数

## 适用情况

- RTS 地图流水线：将全图网格裁剪为左上角象限，供 `rts_base_shape_gen` 作为掩码输入
- 任何需要取网格左上角四分之一的场景

## 基本使用方法

直接将源网格连接到 `grid` 端口，`quarterGrid` 输出即为左上角四分之一子网格。

```
[全图网格] ──► [grid_quarter_tl] ──► quarterGrid ──► [rts_base_shape_gen]
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 要裁剪的源网格 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| quarterGrid | grid | 左上角 ⌈w/2⌉ × ⌈h/2⌉ 子网格 |

## 注意事项

1. **奇数尺寸**：宽或高为奇数时，四分之一尺寸向上取整（如 201×201 → 101×101）
