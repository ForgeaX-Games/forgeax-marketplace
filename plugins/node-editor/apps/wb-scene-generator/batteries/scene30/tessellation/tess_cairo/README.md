# 开罗五边形镶嵌 (tess_cairo)

生成开罗五边形镶嵌格，所有格子是完全相同的不规则五边形。

## 功能特点

1. **五边形全等**：所有格子是同一个不规则五边形，通过旋转和反射拼合，满足"每块相同"的镶嵌要求
2. **四重对称**：整个图案具有 4 折旋转对称性（每 90° 重复一次）
3. **交替对角线构造**：基于正方格网格 + 交替方向对角线（"/" 和 "\"），每个五边形由两个相邻格子的三角形合并而成

## 算法原理

```
奇偶格交替对角线：
  偶格（cx+cy 偶数）：用 "/" 对角线分成上三角和下三角
  奇格（cx+cy 奇数）：用 "\" 对角线分成左三角和右三角

合并规则（五边形 = 2 个三角形）：
  A 型五边形 = 偶格的上三角（fx+fy<1） + 右侧奇格的左三角（fx>fy）
  B 型五边形 = 偶格的下三角（fx+fy≥1） + 左侧奇格的右三角（fx≤fy）

每个五边形有 5 条边，是一个不规则凸五边形。
```

## 适用情况

- 独特的地砖/路面铺装表示（仿开罗古城效果）
- 五边形分区地图（策略游戏、迷宫设计）
- 与 `tess_edge_deform` 组合产生有机五边形镶嵌

## 基本使用方法

连接 `width`、`height`、`cellSize` 参数即可。每个五边形大约跨越 2 个底层正方格，建议 `cellSize=8~15`。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 80 | 输出网格宽度 |
| height | number | 80 | 输出网格高度 |
| cellSize | number | 10 | 底层正方格边长（格） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| regionGrid | grid | 每格存储五边形单元的 1-based ID |

## 注意事项

1. **五边形大小**：每个五边形实际跨越约 2×1 个底层方格的面积，视觉大小约为 `cellSize × cellSize` 像素
2. **与变形组合**：tess_cairo + tess_edge_deform 可产生有机变形的五边形镶嵌，形似伊斯兰几何纹样
3. **与 tess_hex_grid 对比**：六边形有 6 条边，开罗五边形有 5 条边，图案更加不规则和有趣
