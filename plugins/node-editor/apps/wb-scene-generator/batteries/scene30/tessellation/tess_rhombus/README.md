# 菱形镶嵌格 (tess_rhombus)

使用斜坐标系将平面分割为完全相同的菱形（平行四边形）单元格。

## 功能特点

1. **严格全等**：所有格子是完全相同的平行四边形（形状、大小、角度均一致）
2. **斜坐标系**：通过将直角坐标转换为斜坐标实现，计算简单高效
3. **角度可调**：60°=等边菱形（3D 立方体叠加效果），45°=正方形菱形，其他角度=各种平行四边形

## 适用情况

- 等轴/等距（isometric）视角游戏的地图底层格
- 3D 立方体叠加视觉效果的装饰纹样
- 斜纹格式的地形分区

## 基本使用方法

连接 `width`、`height`、`sideLen`、`angle` 参数，运行即可。推荐 `angle=60` 获得最佳视觉效果。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 80 | 输出网格宽度 |
| height | number | 80 | 输出网格高度 |
| sideLen | number | 12 | 菱形边长（格） |
| angle | number | 60 | 菱形锐角（度），范围 20~160 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| regionGrid | grid | 每格存储菱形单元的 1-based ID |

## 注意事项

1. **angle=60**：产生等边菱形，三组平行线对称，视觉上像三维立方体叠加
2. **angle=90**：退化为方形格（与 rect_grid 等价）
3. **极端角度**：angle 接近 0° 或 180° 时格子会极度扁平，不建议使用
