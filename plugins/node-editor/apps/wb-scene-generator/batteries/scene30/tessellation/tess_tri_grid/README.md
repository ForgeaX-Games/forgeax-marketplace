# 三角形镶嵌格 (tess_tri_grid)

将平面光栅化为等边三角形镶嵌网格，每个像素标注所属三角形单元的 ID。

## 功能特点

1. **上下三角交替**：等边三角形天然形成"上△ + 下▽"交替排列，无缝铺满平面
2. **对角线判断**：使用局部归一化坐标的对角线方向精确区分三角形归属
3. **连续 ID 输出**：每个三角形分配 1-based 连续整数 ID，兼容下游变形电池

## 适用情况

- 生成三角形分区地图或策略游戏底层格
- 作为 `tess_edge_deform` 的输入，进行三角形边界变形
- 制作几何感强的装饰纹样底层结构

## 基本使用方法

连接 `width`、`height`、`cellSize` 三个 `number_const` 节点，运行后得到三角形 ID 网格。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 80 | 输出网格宽度（格） |
| height | number | 80 | 输出网格高度（格） |
| cellSize | number | 12 | 等边三角形边长（格） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| regionGrid | grid | 每格存储所属三角形单元的 1-based ID |

## 注意事项

1. **视觉效果**：三角形格子的视觉密度约是六边形的 2 倍，相同 cellSize 下三角形格更细密
2. **与 tess_edge_deform 配合**：变形后的三角形镶嵌格会产生有趣的锯齿有机纹样
3. **cellSize 建议 ≥ 6**，太小时三角形过细密，计算量大效果不明显
