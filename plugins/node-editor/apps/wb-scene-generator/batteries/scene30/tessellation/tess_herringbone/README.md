# 人字形砖块 (tess_herringbone)

将平面分割为人字形（herringbone）排列的 2:1 矩形砖块，所有砖块形状完全相同。

## 功能特点

1. **严格全等**：所有砖块均为 2:1 矩形，水平砖和垂直砖通过 90° 旋转相互全等
2. **经典人字纹**：水平砖和垂直砖在 4u×4u 超格内以对角象限交替排布，产生传统人字（chevron）图案
3. **建筑质感**：人字形铺装广泛用于地板、路面、砖墙装饰

## 适用情况

- 地板纹样（仿古砖、木地板）
- 路面铺装的地图表示
- 建筑风格装饰纹样底层结构

## 基本使用方法

连接 `width`、`height`、`brickSize` 参数即可。`brickSize=6` 时每块砖 6×12 格，适合中等分辨率地图。

## 超格结构说明

```
┌─────────┬────┬────┐
│ H  H  H │ V  │ V  │  ← 左上 2u×2u：2 块水平砖（2u×u）
│ H  H  H │ V  │ V  │     右上 2u×2u：2 块垂直砖（u×2u）
├────┬────┼─────────┤
│ V  │ V  │ H  H  H │  ← 左下 2u×2u：2 块垂直砖
│ V  │ V  │ H  H  H │     右下 2u×2u：2 块水平砖
└────┴────┴─────────┘
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 80 | 输出网格宽度 |
| height | number | 80 | 输出网格高度 |
| brickSize | number | 6 | 砖块短边长（格），长边 = 2×短边 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| regionGrid | grid | 每格存储砖块的 1-based ID |

## 注意事项

1. **brickSize 建议 ≥ 4**：太小时砖块视觉上过于细密
2. **与 tess_edge_deform 配合**：可对人字形砖块进行边界扭曲，产生不规则石板路效果
