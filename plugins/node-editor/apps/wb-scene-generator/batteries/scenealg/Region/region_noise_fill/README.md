# 噪声填充 (alg_region_noise_fill)

**归类：Region**（grid → 单张 0/1 点掩码）

基于坐标哈希的确定性噪声场，按阈值二值化为 0/1 点掩码。与逐格独立的概率填充不同，噪声场空间相关、呈连片分布。纯算法工具，无领域语义。

## 输入

| 参数 | 类型 | 默认 | 说明 |
|------|------|------|------|
| region | grid | — | 约束区，只在非零有效格内取噪声 |
| density | number | 0.3 | 保留比例（阈值 = 1 - density），0..1 |
| seed | number | 0 | 噪声种子，0=用当前时间 |

## 输出

| 参数 | 类型 | 说明 |
|------|------|------|
| region | grid | 与输入同形状的 0/1 点掩码 |
| count | number | 实际保留的格数 |

## 算法

对每个有效格 `(r,c)` 计算 `hashNoise(r,c,seed) ∈ [0,1)`，当其 `> (1 - density)` 时保留为 1。

- 同坐标 + 同 seed → 结果固定（确定性）。
- 噪声场相邻格相关，故结果呈连片纹理，而非纯随机散点。

来源是老 `natural_decoration` 的 `fillNoise` / `hashNoise`，去除装饰语义后通用化。
