# 矩形网格 (rect_grid)

生成指定宽高的矩形网格，所有格子填充为指定值。

## 功能特点

1. **简单直接**：输入宽高，立即输出对应尺寸的全填充矩形网格
2. **自定义填充值**：可指定任意整数作为填充值，默认为 1
3. **常用起点**：作为 `building_profile`（建筑轮廓生成）等电池的输入源

## 适用情况

- 需要一个矩形占地 mask 作为后续处理的输入
- 与 `building_profile` 配合，快速生成具有退线轮廓的建筑形状
- 需要生成指定填充值（非 1）的初始网格

## 基本使用方法

连接 `width`、`height`、`fillValue` 输入端口，`grid` 输出端口即为对应尺寸的矩形网格。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| width | number | 50 | 网格列数 |
| height | number | 50 | 网格行数 |
| fillValue | number | 1 | 所有格子填充的整数值 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid | grid | 所有格子均为指定填充值的矩形网格 |

## 注意事项

1. **最小尺寸**：宽高均至少为 1，传入小于 1 的值会被自动修正为 1
2. **整数输入**：宽高和填充值均会自动向下取整
