# 点生矩形 (point_to_rect)

将网格中的非零点扩展为指定宽高的矩形掩码区域，矩形掩码值与点的掩码值相同，支持多点同时处理。

## 功能特点

1. **点扩展为矩形**：输入网格中每个非零格（点）都会以自身为中心，向外扩展为指定宽高的矩形区域。
2. **掩码值继承**：矩形区域的掩码值与原点的掩码值完全一致，不做额外变换。
3. **多点支持**：网格中可以存在任意数量的点，每个点独立扩展，最终叠加到同一输出网格。
4. **边界裁剪**：矩形超出网格边界的部分自动裁剪，不会越界。

## 适用情况

- 需要根据离散采样点生成覆盖区域（如建筑占地范围、势力圈、碰撞盒）。
- 结合随机选点电池，快速生成多个分散的矩形区块。
- 将稀疏点集转换为密集掩码图，用于后续区域处理。

## 基本使用方法

1. 将含有点的网格连接到「输入网格」端口。
2. 设置「矩形宽度」和「矩形高度」（单位：格），控制每个点扩展的矩形大小。
3. 执行后，「输出网格」中每个点位置都会被以该点为中心的矩形掩码覆盖。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 含点的源网格，非零格视为点，零格为空白 |
| width | number | 3 | 每个点扩展出的矩形列数（宽度） |
| height | number | 3 | 每个点扩展出的矩形行数（高度） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 每个点扩展为矩形后的网格，矩形掩码值与原点相同 |

## 注意事项

1. **矩形以点为中心**：矩形相对于点位置做居中对齐，偏移量为 `floor(width/2)` 列、`floor(height/2)` 行。
2. **多点覆盖顺序**：若多个点的矩形区域存在重叠，后处理的点会覆盖先处理的点（处理顺序为从上到下、从左到右）。
3. **零值不生成矩形**：输入网格中值为 0 的格子不视为点，不会产生矩形。
4. **输出不保留原点位**：输出网格以全零初始化，仅由各点扩展出的矩形填充，原点本身的单格也会被矩形覆盖。
