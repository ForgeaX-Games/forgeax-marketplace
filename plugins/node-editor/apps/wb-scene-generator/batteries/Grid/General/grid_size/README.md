# 网格尺寸 (grid_size)

计算网格中所有非零值的最小包围盒，输出其宽度和高度。

## 功能特点

1. **包围盒计算**：找到所有非零格子的最小外接矩形，返回其列数（宽）和行数（高）
2. **忽略背景**：值为 0 的格子视为背景，不参与尺寸计算
3. **全零处理**：网格全为 0 时，宽度和高度均返回 0

## 适用情况

- 需要获取地图中某个区域的实际占用尺寸
- 作为其他电池的参数（如按比例缩放、偏移计算等）
- 验证生成结果是否符合预期大小

## 基本使用方法

将任意网格连接到「输入网格」端口，即可从「宽度」和「高度」端口获取其非零区域的包围盒尺寸。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | - | 输入网格，0 为背景 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| width | number | 非零值包围盒的列数 |
| height | number | 非零值包围盒的行数 |

## 注意事项

1. **坐标系**：宽度对应列（水平方向），高度对应行（垂直方向）
2. **全零网格**：所有格子均为 0 时，输出 `width=0, height=0`
3. **包围盒非网格整体尺寸**：输出的是非零内容的紧凑范围，不是网格数组的行列数
