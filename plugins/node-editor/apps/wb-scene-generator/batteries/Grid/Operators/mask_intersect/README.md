# 掩码交集 (mask_intersect)

对两个等尺寸网格求交集：同一点位两个网格均为非零值时输出 1，否则输出 0。

## 功能特点

1. **布尔交集**：仅判断非零/零，不依赖具体像素值，任意掩码均可使用
2. **输出统一为1**：交集点位统一输出 1，便于后续管道继续处理
3. **尺寸校验**：两个输入网格尺寸不一致时返回错误信息，不产生越界

## 适用情况

- 需要找出两个区域的重叠部分时使用
- 用于过滤：只保留同时满足两个条件的格子
- 配合 `mask_edge`、`mask_outline` 等电池做多步掩码运算

## 基本使用方法

将两个同尺寸网格分别连接到 `inputGrid1` 和 `inputGrid2`，`outputGrid` 输出交集结果，交集格为 1，其余为 0。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid1 | grid | — | 第一个输入网格，非零值视为有效区域 |
| inputGrid2 | grid | — | 第二个输入网格，非零值视为有效区域，尺寸须与网格1相同 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 与输入等尺寸，两个网格同一点位均为非零时输出1，否则输出0 |

## 注意事项

1. **尺寸必须一致**：两个输入网格的行数和列数必须完全相同，否则返回错误。
2. **输出固定为1**：交集点位输出值固定为 1，不保留原始像素值；如需保留某一网格的原值，请在交集结果上做进一步处理。
3. **零值判断**：只要任一网格该点为 0，输出即为 0，与具体的非零数值无关。
