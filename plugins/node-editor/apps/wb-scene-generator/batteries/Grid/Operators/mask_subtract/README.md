# 掩码差集 (mask_subtract)

对两个等尺寸网格求差集：网格1中为非零值、且网格2中为零值的点位输出 1，其余输出 0。

## 功能特点

1. **布尔差集**：网格1 减去 网格2，结果为网格1独有的区域
2. **输出统一为1**：差集点位统一输出 1，便于后续管道继续处理
3. **尺寸校验**：两个输入网格尺寸不一致时返回错误信息，不产生越界

## 适用情况

- 需要从一个区域中排除另一个区域时使用
- 用于过滤：保留网格1中、但不在网格2中出现的格子
- 配合 `mask_intersect`、`mask_edge` 等电池做多步掩码运算

## 基本使用方法

将被减网格连接到 `inputGrid1`，减数网格连接到 `inputGrid2`，`outputGrid` 输出差集结果。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid1 | grid | — | 被减网格，非零值视为有效区域 |
| inputGrid2 | grid | — | 减数网格，非零值视为需要排除的区域，尺寸须与网格1相同 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 与输入等尺寸，网格1非零且网格2为零的点位输出1，其余输出0 |

## 注意事项

1. **顺序敏感**：差集不满足交换律，`grid1 - grid2` 与 `grid2 - grid1` 结果不同，注意连接顺序。
2. **尺寸必须一致**：两个输入网格的行数和列数必须完全相同，否则返回错误。
3. **输出固定为1**：差集点位输出值固定为 1，不保留网格1的原始像素值；如需保留原值，请在差集结果上做进一步处理。
