# 掩码边缘提取 (mask_edge)

遍历输入网格的每行每列，提取各行最左/最右非零值、各列最上/最下非零值，输出仅保留这些边缘格的网格。

## 功能特点

1. **行方向边缘**：对每一行扫描，保留该行最左侧和最右侧的非零格
2. **列方向边缘**：对每一列扫描，保留该列最上方和最下方的非零格
3. **取并集输出**：四类边缘取并集，重叠格只记录一次，原始像素值保持不变
4. **零值穿透**：输入中的 0 值不参与边缘判断，输出中非边缘格统一置为 0

## 适用情况

- 需要提取任意形状掩码的轮廓骨架（最外层一圈格子）
- 后续流程需要知道每行/每列的起止位置时使用
- 配合 `building_profile`、`mask_outline` 等电池做进一步处理

## 基本使用方法

将任意网格连接到 `inputGrid` 端口，`outputGrid` 输出一个与输入等尺寸的网格，其中只有边缘格保留原值，其余格清零。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | — | 输入二维整数网格，非零值视为有效区域 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 与输入等尺寸，仅保留每行最左/最右、每列最上/最下的非零值，其余置0 |

## 注意事项

1. **空行/空列**：某行或某列全为 0 时，该行/列不贡献任何边缘格，不影响其他行列的处理。
2. **单格行/列**：若某行只有一个非零格，该格同时作为最左和最右，只保留一次。
3. **输出值不变**：边缘格的输出值与输入值相同，不统一替换为 1，便于保留原始掩码 ID。
