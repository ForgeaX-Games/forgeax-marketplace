# 多值合并 (grid_max_merge)

检测多个相同大小网格中的非零值，逐格取所有输入网格的最大值，合并为一个网格输出。

## 功能特点

1. **动态端口**：初始提供2个网格输入槽，连接最后一个后自动追加新槽位，支持任意数量网格输入
2. **逐格取最大值**：每个点位保留所有输入网格中该位置的最大值，原始掩码ID信息得以保留
3. **尺寸容错**：以第一个有效网格尺寸为基准，其他网格越界部分视为0处理

## 适用情况

- 将多个带有不同掩码ID的网格叠加，高ID覆盖低ID
- 合并多个图层时需要保留原始值（而非归一化为0/1）
- 需要"大值优先"的网格叠合场景
- 若只需要判断某位置是否有内容（结果为0/1），请使用「归一合并」

## 基本使用方法

将需要合并的网格依次连接到 网格0、网格1 等端口，连接最后一个槽位后自动出现新槽位。输出的 outputGrid 中每个位置为所有输入网格在该位置的最大值。

## 输入参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid_0 | grid | 第0个输入网格 |
| grid_1 | grid | 第1个输入网格（动态，连接后自动追加更多槽位） |
| grid_N | grid | 更多动态端口，数量由连接情况决定 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 多值合并结果：逐格取所有输入网格的最大值 |

## 注意事项

1. **所有输入网格应大小相同**：若尺寸不一致，以第一个有效网格为基准，其他网格越界部分视为0
2. **输出保留原始值**：结果不做归一化，输出值为各输入网格在该位置的最大值
3. **至少连接一个网格**：所有端口均未连接时返回错误
