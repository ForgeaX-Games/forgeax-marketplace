# 归一合并 (grid_union_merge)

检测多个相同大小网格中的非零值，执行并集归一化：任意输入网格在某位置非零，输出该位置即为1；所有输入在该位置均为零，输出为0。

## 功能特点

1. **动态端口**：初始提供2个网格输入槽，连接最后一个后自动追加新槽位，支持任意数量网格输入
2. **网格列表输入**：额外提供 gridList 端口，可将网格数组批量接入，与动态端口中的网格共同参与合并
3. **并集归一化**：多个网格逻辑"OR"合并，结果只含0和1，原始掩码值大小不影响结果
4. **尺寸容错**：以第一个有效网格尺寸为基准，其他网格越界部分视为0处理

## 适用情况

- 将多个不同来源的掩码层（道路、建筑、水体等）合并为统一的占用图
- 将若干0/1网格叠加后生成单一的二值结果
- 不适用于需要保留原始掩码值（如取最大值覆盖）的场景，那种情况请使用「网格合并」

## 基本使用方法

将需要合并的网格依次连接到 网格0、网格1 等端口，连接最后一个槽位后自动出现新槽位。也可以将网格数组直接连接到 gridList 端口批量输入。输出的 outputGrid 中任意输入非零的位置均标记为1。

## 输入参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grid_0 | grid | 第0个输入网格 |
| grid_1 | grid | 第1个输入网格（动态，连接后自动追加更多槽位） |
| grid_N | grid | 更多动态端口，数量由连接情况决定 |
| gridList | array | 网格数组，列表中每个元素均参与归一合并 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 归一合并结果：任意输入非零处为1，全部为零处为0 |

## 注意事项

1. **所有输入网格应大小相同**：若尺寸不一致，以第一个有效网格为基准，其他网格越界部分视为0
2. **输出只有0和1**：原始掩码值不论大小，只要非零就输出1
3. **至少连接一个网格**：所有端口均未连接时返回错误
