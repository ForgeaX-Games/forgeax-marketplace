# 按值拆分 (grid_split_by_value)

将多值网格按不同的非零值拆分为多个独立网格组成的列表。

## 功能特点

1. **按值分层**：自动检测网格中所有不同的非零值，每种值生成一个独立子网格
2. **保留原值**：子网格中匹配位置填入原有值，其余位置填0
3. **升序排列**：输出列表中的子网格按值从小到大排列
4. **零值跳过**：值为0的单元格视为背景，不会生成对应子网格

## 适用情况

- 需要将多类型区域网格拆分为单类型图层分别处理
- 对各类型区域单独应用算法后再合并
- 提取特定类型区域的掩码以便后续操作

## 基本使用方法

1. 将含有多种非零值的网格连接到输入端口
2. 执行节点，输出端口返回一个网格数组
3. 数组中每个元素对应原网格中的一个独特非零值

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 待拆分的多值网格 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| grids | array | 拆分后的网格数组，每个元素为只含对应值的二维网格 |

## 使用示例

### 输入网格

```
[[0, 1, 1],
 [2, 2, 0],
 [0, 3, 3]]
```

### 输出 grids（数组，共3个子网格）

```
grids[0] → 值为1的子网格：
[[0, 1, 1],
 [0, 0, 0],
 [0, 0, 0]]

grids[1] → 值为2的子网格：
[[0, 0, 0],
 [2, 2, 0],
 [0, 0, 0]]

grids[2] → 值为3的子网格：
[[0, 0, 0],
 [0, 0, 0],
 [0, 3, 3]]
```

## 注意事项

1. **零值不拆分**：值为0的单元格视为背景，不会单独生成子网格
2. **输出顺序**：子网格按值从小到大排列，与出现顺序无关
3. **全零网格**：若输入网格全部为0，输出为空数组
