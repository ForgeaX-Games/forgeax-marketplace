# 随机退线 (grid_setback)

对每个输入网格随机选取若干条边进行退线处理，输出退线后的网格列表。

## 功能特点

1. **概率化边数选取**：每个网格随机决定退线 0~4 条边，边数越多概率越低
2. **概率化退线量**：每条选中的边随机退线 1~4 格，小退线量概率更高
3. **非破坏性退线**：退线区域的格子值清零，原本为 0 的格子不受影响
4. **批量处理**：支持动态多端口和列表端口同时输入，一次处理所有网格
5. **可复现结果**：通过固定随机种子得到一致输出

## 适用情况

- 建筑或地块生成中，对房间/地块添加随机退线/退距效果
- 为网格增加不规则边缘，避免所有区域整齐对齐
- 与 BSP 分割等电池配合，生成更自然的空间布局

## 基本使用方法

将网格连接到动态端口（`网格0`、`网格1`…）或 `网格列表` 端口，执行后从 `退线网格列表` 获取处理结果。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid_0, grid_1, ... | grid | - | 动态端口，支持任意数量网格输入 |
| gridList | array | - | 网格列表，列表中每个元素均参与处理 |
| seed | number | 0 | 随机种子；0=使用当前时间戳（每次不同），非零=固定结果 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGridList | array | 退线处理后的网格列表，顺序与输入一致 |

## 退线规则说明

### 边数选取概率

| 退线边数 | 概率 |
|----------|------|
| 0 条（不退线） | 50% |
| 1 条 | 20% |
| 2 条 | 10% |
| 3 条 | 10% |
| 4 条（全部） | 10% |

### 退线量概率（每条边独立）

| 退线格数 | 概率 |
|----------|------|
| 1 格 | 30% |
| 2 格 | 30% |
| 3 格 | 20% |
| 4 格 | 20% |

## 注意事项

1. **退线是清零操作**：被选中边缘的格子值设为 0，其余区域保持原值不变
2. **输出顺序**：动态端口网格在前，`gridList` 中的网格追加在后，整体顺序与输入一致
3. **空网格跳过**：行数或列数为 0 的网格直接原样输出，不做处理
4. **种子与批量**：所有网格共用一个 RNG 实例，按输入顺序依次消耗随机数，相同种子产生相同结果
