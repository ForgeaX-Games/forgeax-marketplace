# 过滤网格 (grid_filter)

过滤网格列表，移除非零格数量小于阈值的网格。

## 功能特点

1. **按非零格数过滤**：统计每个网格中值不为 0 的格子数量，低于阈值的网格被丢弃
2. **顺序保留**：输出列表保持输入顺序，仅删除不满足条件的项
3. **防御处理**：空网格、非数组元素自动跳过

## 适用情况

- BSP 分割后过滤掉过小的房间/地块
- 随机退线后移除被清空或过小的区域
- 任何需要按尺寸筛选网格的场景

## 基本使用方法

将网格列表连接到 `gridList`，设置 `minSize`（最小非零格数），执行后从 `outputGridList` 获取保留的网格。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| gridList | array | - | 输入的网格列表 |
| minSize | number | 1 | 非零格数量的最小阈值（含），小于此值的网格被移除 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGridList | array | 过滤后的网格列表 |

## 注意事项

1. **阈值为含边界**：非零格数 `>= minSize` 才保留，等于阈值的网格会被保留
2. **全零网格**：非零格数为 0，始终被过滤掉（除非 `minSize <= 0`）
3. **空输入**：`gridList` 不是数组时输出空列表并返回错误信息
