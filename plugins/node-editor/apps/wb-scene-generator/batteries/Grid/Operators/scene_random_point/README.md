# 随机选点 (scene_random_point)

在网格中随机填充指定数量的点，支持自定义填充值和随机种子。

## 功能特点

1. **随机散点**：在输入网格的任意位置随机放置指定数量的点
2. **自定义填充值**：可指定填充到格子的数值（默认为 1）
3. **可复现结果**：通过设置随机种子，确保每次运行结果一致
4. **非覆盖逻辑**：已有填充值的格子不会被重复计数，保证实际点数准确

## 适用情况

- 需要在场景网格中随机放置资源、宝箱、怪物等点位
- 随机初始化噪声或种子点，用于后续生长算法的起点
- 场景测试时快速生成随机分布的标记点

## 基本使用方法

1. 将输入网格连接到「输入网格」端口
2. 设置「点位数量」（默认 10）
3. 设置「填充值」（默认 1）
4. 可选：设置「随机种子」以固定随机结果（0 表示每次随机）
5. 执行后从「输出网格」获取结果

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 输入的二维整数网格（必填） |
| count | number | 10 | 要随机填充的点的数量 |
| fillValue | number | 1 | 填充到选中格子的值 |
| seed | number | 0 | 随机种子，0 表示每次运行结果不同 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 随机填充点后的输出网格 |

## 注意事项

1. **点数上限**：若网格中非填充值格子数少于 count，实际填充点数以可用格子数为上限，不会无限重试
2. **种子固定**：需要可复现结果时，设置任意非零整数作为种子
3. **输入网格不会被修改**：输出是输入的副本，原始网格数据不变
