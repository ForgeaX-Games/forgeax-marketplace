# 掩码轮廓提取 (mask_outline)

提取非零掩码区域的轮廓，支持向内（正数）和向外（负数）两种厚度模式。

## 功能特点

1. **双向轮廓**：正厚度提取内轮廓（在掩码内部），负厚度提取外轮廓（在掩码外部）
2. **可控厚度**：厚度绝对值越大，轮廓带越宽
3. **边界安全**：外轮廓不超出网格边界，内轮廓不超出原始掩码范围
4. **全掩码支持**：输入网格中所有非零值均视为掩码，无论具体数值

## 适用情况

- 为建筑轮廓、地形区域等添加边框/描边效果
- 提取区域内圈用于绘制墙体、边界装饰
- 生成区域外扩阴影或缓冲区

## 基本使用方法

连接 `inputGrid`，设置 `thickness`，输出 `outputGrid` 即为轮廓区域（值=1）。

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | — | 源网格，非零值视为掩码区域 |
| thickness | number | 1 | 轮廓厚度，正=向内，负=向外，0=返回空网格 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 与输入等尺寸，轮廓区域=1，其余=0 |

## 参数说明

### thickness 详解

| 值 | 效果 |
|----|------|
| `1` | 掩码最外一圈像素（内轮廓，宽1格） |
| `2` | 掩码最外两圈像素（内轮廓，宽2格） |
| `-1` | 掩码外紧贴边缘一圈（外轮廓，宽1格） |
| `-2` | 掩码外扩两圈范围的轮廓带（外轮廓，宽2格） |
| `0` | 返回全零网格 |

## 注意事项

1. **内轮廓限制**：当掩码区域较小，厚度值大于掩码最窄处宽度时，腐蚀后内部为空，轮廓等于整个掩码
2. **厚度取整**：thickness 会自动取整，不支持小数
3. **外轮廓边界**：外轮廓受网格边界限制，不会溢出
