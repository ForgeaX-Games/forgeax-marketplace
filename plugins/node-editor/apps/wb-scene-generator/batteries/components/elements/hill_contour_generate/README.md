# 小山包等高线生成 (hill_contour_generate)

在输入掩码区域内生成圆润的小山包等高线，多层等高线严格一层套一层呈同心状分布，并内置形态学后处理。

## 功能特点

1. **严格嵌套**：等高线层间严格一层套一层，不会出现外层包含间隙或内层溢出的情况，适合表现小山包或丘陵轮廓。
2. **圆度可控**：`roundness` 参数在 0~1 之间调节等高线形状，接近 1 时为圆形，接近 0 时轮廓趋向方形，便于匹配不同地形风格。
3. **有机边缘**：`noiseAmount` 参数为等高线叠加轻微 Value Noise 扰动，避免机械感，呈现自然地形感。
4. **多峰支持**：`hillCount` 可放置多个山包，各自产生独立同心圈，相邻山包之间自然过渡，形成鞍部。
5. **内置后处理**：自动执行孔洞填充、孤立碎块删除、空白归并，无需手动接 `grid_mask_clean` 电池。

## 适用情况

- 游戏地图中需要放置小山丘、土堆等凸起地貌
- 以等高线风格表现小型地形元素（山包、小丘、岛屿）
- 在已有区域掩码内快速叠加一层地形层次感
- 不适用：需要大规模山脉连绵延伸（建议用 `mountain_contour_generate`）

## 基本使用方法

1. 将上游区域掩码（如某块可用地块）接入 `grid` 端口
2. 调整 `contourLevels`（层数，建议 4~8）和 `hillCount`（山包数量，建议 1~3）
3. 调整 `roundness`（0.85 左右圆润感强）和 `noiseAmount`（0.1~0.15 自然边缘）
4. 输出的 `contourLayers` 可直接接渲染器或下游处理节点
5. `heightGrid` 可接 biome 分类或地形纹理节点

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 输入掩码网格，仅对非 0 格子生成等高线 |
| contourLevels | number | 6 | 等高线层数，决定同心圈数量 |
| hillCount | number | 1 | 山头数量，多峰时自动分散放置 |
| roundness | number | 0.85 | 等高线圆度，0~1，越大越圆润 |
| peakRadius | number | 0.35 | 山包影响半径，归一化坐标 0~1 |
| noiseAmount | number | 0.12 | 边缘扰动量，0=完美圆形，0.2=自然边缘 |
| peakPosition | number | 0 | 山头位置（九宫格，见下方说明），0=随机 |
| minHoleSize | number | 20 | 后处理：最大孔洞面积（格子数），超过则保留 |
| minIslandSize | number | 8 | 后处理：最小岛屿面积（格子数），小于则删除 |
| seed | number | 0 | 随机种子，0 使用当前时间 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| contourLayers | array | 网格列表，共 contourLevels 个，外→内层序号递增（1, 2, 3…） |
| heightGrid | grid | 0~100 连续高度场，外低内高，0 格子（边界外）保持 0 |
| outputNameList | array | 各层名称清单，格式 [{id, name}]，name 为"山包层1"等 |

## 山头位置参数说明

`peakPosition` 使用九宫格数字键盘布局（1-9），对应区域内的相对位置：

```
7(左上)  8(上中)  9(右上)
4(左中)  5(正中)  6(右中)
1(左下)  2(下中)  3(右下)
```

- 设为 `0` 时每次随机产生位置（受 `seed` 控制）
- 设为 `1~9` 时，第一个山头固定在该区域，多山头时其余山头随机分散

## 算法说明

### 高度场构建

核心算法使用高斯距离场代替 FBM 噪声，确保等高线形态严格同心：

- 以 Lp 范数控制圆度，`p=lerp(1.2, 2.0, roundness)`
- 多峰使用 soft-max 融合，使最近山峰获得更高权重，峰间自然产生鞍部
- 叠加低频 Value Noise 轻微扰动等高线边缘

### 后处理流程

1. **合并各层** → 构建完整地图
2. **填充内部孔洞** → 面积 ≤ minHoleSize 且被单一颜色包围的空洞填充
3. **删除孤立碎块** → 面积 < minIslandSize 的非零连通域删除
4. **空白归并** → 残余空白格子并入相邻最多的层
5. **拆回各层** → 按层值分配回独立网格

## 注意事项

1. **peakRadius 过小时**：山包范围覆盖不了整个输入区域，边缘格子可能全被归入最外层，适当调大至 0.4~0.5。
2. **hillCount 过多时**：多个山包争抢有限空间，外层等高线会相互干扰，建议每 100×100 格子区域不超过 3 个山包。
3. **noiseAmount > 0.3 时**：扰动较强，某些高度较低的峰可能产生碎块，建议同步调大 `minIslandSize`。
4. **等高线顺序**：`contourLayers[0]` 为最外层（最低），`contourLayers[contourLevels-1]` 为最内层（山顶），渲染时由外向内叠加即可获得正确视觉层次。
