# 农田生成 (farmland_grid)

根据可用区域掩码，用三种布局算法生成农田分区，输出含小径与多种地块类型的多值网格。

## 功能特点

1. **三种布局形式**：规则网格（grid）、水平条带（strip）、BSP 自由划分（bsp），适应不同农场风格
2. **多值输出**：单一网格涵盖小径（1）与多个地块类型（2-6），可直接用于渲染或后续处理
3. **掩码约束**：只在输入掩码非零区域内生成，外部保持 0，天然支持任意形状的农场边界
4. **可复现**：BSP 算法通过随机种子保证结果可复现；grid/strip 为纯确定性算法

## 适用情况

- 星露谷物语/牧场物语风格的农场地图生成
- 需要将一块空地划分为多个可种植地块的场景
- 与室内/室外地形掩码结合，快速填充可用耕地

## 基本使用方法

1. 将可种植区域的掩码网格（非零 = 可用区域）连接到 `grid` 端口
2. 在 `layout` 下拉框选择布局形式
3. 调整 `plotWidth`、`plotHeight`、`pathWidth` 控制地块大小和小径宽度
4. 执行后，`outputGrid` 中地块值为 2-6（循环），小径值为 1，空白为 0

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | - | 可种植区域掩码，非零为有效格子 |
| layout | string | grid | 布局算法（见下方说明） |
| plotWidth | number | 4 | 地块列数，最小 2（grid/bsp 有效） |
| plotHeight | number | 4 | 地块行数，最小 2（所有算法有效） |
| pathWidth | number | 1 | 地块间小径格数，最小 1 |
| seed | number | 0 | 随机种子（仅 bsp），0=时间戳 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 多值网格（0=空，1=小径，2-6=地块类型） |

## 布局算法说明

| 算法值 | 名称 | 特点 |
|--------|------|------|
| `grid` | 规则网格 | 按 plotWidth × plotHeight 均匀划分，相邻地块类型自动区分，排列整齐 |
| `strip` | 水平条带 | 按 plotHeight 行高切分为水平条带，每条带为一种地块类型，路径感强 |
| `bsp` | 自由划分 | 递归 BSP 切分，产生大小不一的地块，更自然有机，受 seed 控制 |

## 输出值含义

| 值 | 含义 |
|----|------|
| 0 | 空（原始掩码为 0 的区域） |
| 1 | 小径（地块间隔离带） |
| 2 | 地块类型 A |
| 3 | 地块类型 B |
| 4 | 地块类型 C |
| 5 | 地块类型 D |
| 6 | 地块类型 E |

地块类型按 2→3→4→5→6→2 循环分配，可通过下游的掩码过滤电池按类型分离单个地块。

## 使用示例

### 输入示例

```json
{
  "grid": "...(20x20 楼层掩码)...",
  "layout": "grid",
  "plotWidth": 5,
  "plotHeight": 4,
  "pathWidth": 1,
  "seed": 0
}
```


## 注意事项

1. **掩码约束**：输出网格尺寸与输入完全一致，输入为 0 的格子输出保持 0
2. **参数适配**：plotWidth/plotHeight 过大时，部分地块会被掩码边界截断，属正常现象
3. **bsp 最小尺寸**：BSP 的最小地块尺寸取 min(plotWidth, plotHeight)，建议设 ≥ 3 以避免过小地块
4. **小径宽度**：pathWidth=1 已足够分隔，设 2 可获得更宽的田间小路效果
