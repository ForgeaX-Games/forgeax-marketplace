# 区域分区生成器 (Region Zone Generator)

在**单张**掩码网格（`grid[y][x]`，非零像素为可用区域）上，根据区域列表（面积权重、九宫格方位）把可用区域切成若干不重叠的不规则分区，输出**一张多值网格**（每个分区一个递增 ID：1, 2, 3, ...）。

> v2.0 起端口契约与 `zone_nesting` / `edge_green_cluster` 对齐：**单网格进、单网格出**，网格列表由引擎按 DataTree 自动逐张 fanout；不再输出 `nameList`，可直接喂 `grid2node`。

## 功能特点

1. **方位驱动布局**：每个区域可指定 1-9 的九宫格方位，种子点被定位到对应位置的最近可用像素
2. **配额感知 Voronoi**：面积权重大的区域自动获得更多像素，实现比例控制
3. **Lloyd 松弛**：迭代质心更新使区域分布更均匀，减少极端形状
4. **四种边界风格**：`organic`（自然有机）、`smooth`（更圆润）、`rectilinear`（矩形化）、`voronoi`（原始 Voronoi 直边）
5. **多值网格输出**：单张网格，分区 k 的像素写入 ID `(k+1)`，未分配像素为 0

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| inputGrid | grid | — | 单张掩码网格，非零像素为可用区域（网格列表由引擎逐张 fanout） |
| regions | array | — | 区域描述数组，见下方格式 |
| boundaryStyle | string | "rectilinear" | 边界风格：organic / smooth / rectilinear / voronoi |
| relaxIterations | number | 5 | Lloyd 松弛迭代次数（推荐 3-8） |
| smoothIterations | number | 5 | 边界平滑 CA 迭代次数（仅 organic/smooth 风格） |
| seed | number | 0 | 随机种子，0=每次不同 |

### regions 格式

支持 `[面积占比, 方位]` 或 `[名称, 面积占比, 方位]`（名称仅作可读注释，不影响输出）；也兼容 JSON 字符串。

```json
[[3, 1], [2, 5], [2, 9], [4, 3]]
```

- 面积占比：相对权重，按比例归一化
- 方位（1-9）：
  ```
  1 | 2 | 3
  4 | 5 | 6
  7 | 8 | 9
  ```

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 单张多值网格，分区 k → ID `(k+1)`，未分配像素为 0 |

## 边界风格说明

| 风格 | 特点 |
|------|------|
| organic | 细胞自动机平滑（4/8 阈值），自然有机感 |
| smooth | 更强平滑（3/8 阈值 + 双倍迭代），边界更圆润 |
| rectilinear | 曼哈顿距离 Voronoi，边界呈直线，类矩形风格（默认） |
| voronoi | 不做后处理，保留原始 Voronoi 直线边界 |

## 注意事项

1. **可用区域 = 非零像素**：只有 `inputGrid` 值 ≠ 0 的像素才参与分区
2. **面积权重仅影响比例**：不保证精确面积，实际分配受掩码形状影响
3. **同方位冲突**：多个区域指定相同方位时，通过随机扰动区分种子点
4. **可复现**：相同 seed + 相同参数可复现结果；seed=0 时使用当前时间戳
