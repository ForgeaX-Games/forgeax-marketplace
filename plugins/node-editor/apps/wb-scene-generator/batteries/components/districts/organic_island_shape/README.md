# 有机海岸轮廓 (organic_island_shape)

基于**柏林噪声 + 椭圆距离场**算法，将输入的矩形网格重塑为自然平滑的有机岛屿轮廓，模拟真实海岸线的不规则感。

## 功能特点

1. **椭圆距离场基础**：以输入网格非零区域的边界框为基础，生成中心向外衰减的椭圆形岛屿基础形状
2. **多层柏林噪声扰动（fBm）**：叠加多个频率的柏林噪声，扰动海岸线边界，产生凹湾、半岛等自然地貌
3. **参数化控制**：可分别调整噪声频率、扰动强度、岛屿大小和噪声细节层数
4. **兼容任意输入网格**：自动识别非零区域作为生成范围，输出与输入同尺寸

## 适用情况

- 星露谷物语风格的孤岛农场地图
- 需要有机自然轮廓的任意陆地区块
- 后接地形细化电池（等高线、装饰等）时的基础形状层

## 基本使用方法

1. 将任意矩形网格（如空白填充的矩形）接入 `grid` 端口
2. 调整 `islandRatio`（建议 0.4~0.6）控制岛屿大小
3. 调整 `noiseStrength`（建议 0.2~0.4）控制海岸线的凹凸程度
4. 修改 `seed` 获得不同形状

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 输入矩形网格，非零区域作为生成范围 |
| noiseScale | number | 3 | 噪声频率：值越大海岸线弯曲细节越丰富，建议 1~6 |
| noiseStrength | number | 0.35 | 噪声扰动强度：0=完美椭圆，0.5=大幅扭曲，建议 0.1~0.5 |
| islandRatio | number | 0.5 | 陆地覆盖率：0.3=小岛，0.6=大岛，范围 0.2~0.75 |
| octaves | number | 4 | 噪声层数：越多细节越丰富，建议 2~5 |
| seed | number | 0 | 随机种子，0=每次随机 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 岛屿轮廓网格，陆地格子=输入最大值，海洋=0 |

## 算法说明

### 核心公式

每个格子的"岛屿值"计算如下：

```
islandValue = (1 - ellipticDist) + noiseStrength × (noise - 0.5) × 2
```

- `ellipticDist`：该格子到边界框中心的归一化椭圆距离（0=中心，1=边界）
- `noise`：多层柏林噪声值（0~1，归一化后 0.5 为均值）
- 当 `islandValue > (1 - islandRatio)` 时判定为陆地

### 多层噪声（fBm）

使用分形布朗运动叠加多个频率的柏林噪声：
- 每层频率翻倍、振幅减半
- `octaves=4` 时：1× + 2× + 4× + 8× 频率叠加

## 使用示例

### 输入

```json
{
  "grid": "[[接入一个100×100的矩形网格]]",
  "noiseScale": 3,
  "noiseStrength": 0.35,
  "islandRatio": 0.5,
  "octaves": 4,
  "seed": 0
}
```

### 输出

```json
{
  "outputGrid": "[[100×100网格，岛屿区域=输入最大值，海洋=0]]"
}
```

## 注意事项

1. **输入网格的值不影响形状**：电池仅用输入网格的非零区域确定生成范围，实际形状完全由噪声参数和 seed 决定
2. **noiseStrength 过大会产生破碎形状**：建议不超过 0.5，超过后可能出现孤立陆地块
3. **islandRatio 与 noiseStrength 相互影响**：较大的 islandRatio（接近 0.7）配合较大 noiseStrength 可能导致岛屿充满整个矩形；建议先固定一个再调另一个
4. **seed=0 每次执行结果不同**：需要固定形状时请设置非零 seed
