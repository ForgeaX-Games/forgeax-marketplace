# 随机矩形区域生成 (random_rect_zone_gen)

在输入网格的指定区域内随机散布指定数量、指定大小范围的轴对齐矩形，矩形严格不超出目标区域边界，支持不规则区域和多值掩码中的异形区域。

## 功能特点

1. **兼容二值与多值网格**：`targetValue=0` 时以 `0` 为禁区，所有非零格均可放矩形；填非零值时精确匹配该掩码ID，支持不规则区域和异形区域（如仅在区域3内生成矩形）
2. **严格边界约束**：矩形的每一格都必须满足目标值条件且不超出网格边界，确保不"溢出"初始区域
3. **随机锚点 + 偏移**：从非零格随机选锚点，再随机偏移矩形位置，分布更均匀自然
4. **软失败机制**：某个矩形在 `maxAttempts` 次内找不到合法位置时直接跳过，不报错，其余矩形继续放置
5. **输出实际数量**：`placedCount` 端口告知最终成功放置了多少个矩形

## 适用情况

- 在地形/地块掩码内快速散布建筑底座、农田、广场等矩形功能区
- 在房间网格中填充桌子、床铺、柜子等矩形家具轮廓
- 任何需要在有界非零区域内随机摆放矩形的 PCG 场景
- 不适用于需要矩形不互相重叠的情况（本电池允许矩形覆盖彼此）

## 基本使用方法

1. 将上游网格连接到 `grid` 输入端口
2. 设置 `fillValue` 为矩形写入的数值（需与输入网格中已有值区分）
3. 调整 `count`、`minW`/`maxW`、`minH`/`maxH` 控制数量和大小
4. 执行后从 `outputGrid` 获取结果，`placedCount` 查看实际放置数

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| grid | grid | — | 输入网格，支持二值或多值掩码 |
| targetValue | number | 0 | 目标区域掩码值，0=任意非零，非零=精确匹配该ID |
| fillValue | number | 2 | 矩形区域写入的数值 |
| count | number | 5 | 目标矩形数量 |
| minW | number | 2 | 矩形最小宽度（列数） |
| maxW | number | 8 | 矩形最大宽度（列数） |
| minH | number | 2 | 矩形最小高度（行数） |
| maxH | number | 8 | 矩形最大高度（行数） |
| maxAttempts | number | 100 | 每个矩形最大随机尝试次数 |
| seed | number | 0 | 随机种子，0 = 自动随机 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| outputGrid | grid | 叠加矩形后的网格 |
| placedCount | number | 实际成功放置的矩形数量 |

## 注意事项

1. **targetValue 用法**：默认 `0` 等同于旧的"任意非零"模式，向下兼容；需要只在某个特定区域（如掩码值3的区域）放矩形时，将 `targetValue` 设为 `3` 即可
2. **fillValue 不要选 0**：0 是禁区标识，用 0 填充会导致后续执行中矩形区域被当作边界
3. **矩形可以重叠**：本电池不做重叠检测，同一区域可能被多个矩形覆写，最终值取最后一次写入
4. **实际放置数可能少于 count**：当非零区域狭窄或矩形较大时，多次尝试仍找不到合法位置，该矩形会被跳过；可提高 `maxAttempts` 增加成功率，或缩小矩形尺寸
5. **seed=0 每次随机**：如需可复现结果，请填写非零种子
