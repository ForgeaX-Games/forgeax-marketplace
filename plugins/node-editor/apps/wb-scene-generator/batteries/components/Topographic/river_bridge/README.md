# 河流架桥 (river_bridge)

分析网格中河流区域的流向，在垂直于流向的指定位置生成桥掩码，支持直线桥与折线桥两种形态。

## 功能特点

1. **自动识别流向**：bounding box 宽 ≥ 高 → 水平流动（桥垂直穿越）；高 > 宽 → 垂直流动（桥水平穿越）
2. **位置可控**：`position` 参数 0~1 控制桥沿主轴的位置（0=上游/左侧，1=下游/右侧）
3. **两种桥形**：直线（`straight`）或水平竖直折线（`zigzag`，可控偏移量）
4. **多值网格支持**：每个非零值视为独立河流，各自生成桥
5. **支持网格列表**：单个网格或网格列表均可，输出同格式

## 流向判断原理（PCA）

对河流所有格子坐标做主成分分析（PCA），取最大方差方向作为流向。  
桥方向 = 垂直于流向，适用于水平、垂直、对角线及任意不规则河流。

- `position` 控制沿**流向轴**的位置：0 = 上游/起点端，1 = 下游/终点端
- 桥 strip 宽度 = `width`，在流向轴方向上量取

## zigzag 折线示意（水平河流）

```
position处           position+offset处
  │  │                   │  │
  │  │  (上半段)          │  │
  └──┴──────────────────┘  │  ← 中段横向连接
                        │  │
                        │  │  (下半段)
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| input | any | — | 单个二维网格或网格列表 |
| width | number | 2 | 桥宽（格数） |
| position | number | 0.5 | 沿主轴的位置，0.0~1.0 |
| algorithm | string | "straight" | 桥形：straight / zigzag |
| zigzagOffset | number | 3 | zigzag模式下中段横向偏移格数（可为负） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| output | any | 桥掩码，桥格子值=河流值，其余为0 |

## 注意事项

1. 只有位于河流内部（原网格非零）的格子才会被标记到桥掩码中，桥不会延伸到河岸
2. zigzagOffset 为 0 时折线退化为直线
3. 同一网格多个非零值 → 每个值独立生成一座桥，共享同一组参数
