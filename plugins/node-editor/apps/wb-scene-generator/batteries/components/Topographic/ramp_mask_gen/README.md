# 坡道掩码生成 (ramp_mask_gen)

读取网格列表中每个区域，在其底边生成一个 2×2 坡道掩码块：上两格在区域内部，下两格在区域外部。支持指定坡道横向位置（0~1），并输出坡道名称清单。

## 功能特点

1. **列表输入/输出**：输入和输出均为网格列表格式（array），与上下游电池对齐
2. **指定或随机坡道位置**：通过 `rampPosition` 端口（0~1）控制坡道在底边的横向位置；不填（-1）则随机
3. **保留原值**：坡道格子值等于所属区域值，便于下游识别归属
4. **输出名称清单**：输出每个坡道区域对应的 `{id, name, type:'tile'}` 条目，支持 ID 顺延

## 坡道示意

```
...  区域内容  ...
[ v ][ v ]         ← 上两格（区域内，底边格子）
[ v ][ v ]         ← 下两格（区域外，底行下一行）
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| input | array | — | 网格列表，每项为一个二维网格 |
| rampPosition | number | -1 | 坡道横向位置 0~1（0=最左，1=最右）；-1 表示随机 |
| seed | number | 0 | 随机种子（仅随机模式生效），0 使用当前时间 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| output | array | 坡道掩码列表，坡道格子为原区域值，其余为 0 |
| outputNameList | array | 坡道名称清单，条目数量与输入网格数量严格对应，每项含 `{id, name, type:'tile'}` |

## rampPosition 说明

- 取值范围 `0~1`，线性映射到区域底边所有合法候选位置（按列从左到右排序）
- `0` = 最左侧合法坡道位置，`1` = 最右侧合法坡道位置，`0.5` ≈ 居中
- `-1`（默认）或超出 `[0, 1]` 范围时退回随机模式，使用 `seed` 控制

## 输出名称清单格式

```json
[
  { "id": 1, "name": "坡道_1", "type": "tile" },
  { "id": 2, "name": "坡道_2", "type": "tile" }
]
```

条目数量与输入网格数量严格对应：输入 N 个网格，输出 N 条清单条目，ID 从 1 开始递增。

## 注意事项

1. 区域底行已在网格最后一行时，下方两格超出边界，该区域坡道跳过
2. 区域底边宽度为 1（单列）时，无法构成 2 格宽坡道，该区域跳过
3. 同一格被多个区域坡道覆盖时，后处理的区域值会覆盖前一个
4. `rampPosition` 指定的是候选位置索引比例，而非绝对列坐标
