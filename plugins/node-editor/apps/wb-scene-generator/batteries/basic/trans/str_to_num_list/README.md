# 字符串转数字列表 (str_to_num_list)

将形如 `[1,2,3]` 或 `["12","1","3"]` 的 JSON 字符串解析为数字列表。

## 功能特点

1. **标准 JSON 数组解析**：支持符合 JSON 规范的数字数组，如 `[10, 20, 30]`
2. **字符串数字兼容**：数组元素可为数字（`1`）或数字字符串（`"1"`），均能正确转为数值
3. **多层嵌套兼容**：支持被多次 `JSON.stringify` 的字符串（最多 3 层解包）
4. **健壮的错误处理**：遇到无法转为数字的元素时返回空列表和错误说明，不会抛出异常

## 适用情况

- 将 AI 模型输出的 JSON 数字数组字符串转为可用数字列表
- 从文本参数中提取一组数值（如权重、坐标、序列等）
- 在节点流中进行字符串与数字列表类型之间的格式转换

## 基本使用方法

1. 将包含 JSON 数字数组的字符串连接到 **输入字符串** 端口
2. 执行节点，**数字列表** 端口输出解析后的列表

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| str | string | - | 形如 `[1,2,3]` 或 `["12","1","3"]` 的 JSON 字符串，元素可为数字或数字字符串 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| list | array | 解析后的数字列表 |

## 使用示例

### 输入示例（数字元素）

```json
{
  "str": "[10, 20, 30, 40]"
}
```

### 输入示例（字符串数字元素）

```json
{
  "str": "[\"12\",\"1\",\"1\",\"3\"]"
}
```

### 输出示例

```json
{
  "list": [12, 1, 1, 3]
}
```

## 注意事项

1. **输入必须为合法 JSON 数组**：字符串内容必须符合 JSON 语法，如 `[1,2,3]`，不支持单引号
2. **元素必须可转为数字**：若数组中含有无法转换的元素（如 `"abc"`），输出空列表并附带 error 字段
3. **浮点数完整支持**：`[1.5, 2.7, 3.14]` 等小数数组可正常解析
4. **非数组输入报错**：若 JSON 解析结果不是数组（如对象 `{}`），输出空列表并附带 error 字段
