# 字符串转列表 (str_to_list)

将形如 `["a","b","c"]` 的 JSON 字符串解析为字符串列表。

## 功能特点

1. **标准 JSON 数组解析**：支持符合 JSON 规范的字符串数组，如 `["apple","banana","cherry"]`
2. **自动类型转换**：数组中的非字符串元素（数字、布尔值等）会被自动转为字符串
3. **多层嵌套兼容**：支持被多次 `JSON.stringify` 的字符串（最多 3 层解包）
4. **健壮的错误处理**：输入非法时返回空列表和错误说明，不会抛出异常

## 适用情况

- 将 AI 模型输出的 JSON 字符串数组转为可用列表
- 从文本数据中提取预定义的字符串集合
- 在节点流中进行字符串与列表类型之间的格式转换

## 基本使用方法

1. 将包含 JSON 字符串数组的字符串连接到 **输入字符串** 端口
2. 执行节点，**字符串列表** 端口输出解析后的列表

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| str | string | - | 形如 `["item1","item2",...]` 的 JSON 字符串 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| list | array | 解析后的字符串列表 |

## 使用示例

### 输入示例

```json
{
  "str": "[\"苹果\",\"香蕉\",\"樱桃\"]"
}
```

### 输出示例

```json
{
  "list": ["苹果", "香蕉", "樱桃"]
}
```

## 注意事项

1. **输入必须为合法 JSON 数组**：字符串内容必须符合 JSON 语法，如 `["a","b"]`，不支持单引号 `['a','b']`
2. **非字符串元素自动转换**：数组中若含有数字、布尔值，会通过 `String()` 转换为字符串后输出
3. **非数组输入报错**：若 JSON 解析结果不是数组（如对象 `{}`），输出空列表并附带 error 字段
